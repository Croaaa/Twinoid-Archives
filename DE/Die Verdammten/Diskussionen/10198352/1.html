Wie Grolnar schon richtig angemerkt hat, werden bei diesem Twinoid-Kasten Rahmengrafiken benutzt. Die müsste man dann erstmal mit einem Bildbearbeitungsprogramm umpinseln (Farbton ändern).

Alternativ kann man den Kasten auch einfach transparent machen, dahinter ist ja sowieso eine dunkle Hintergrundgrafik vorgesehen (die man hier bei DV aktuell noch nicht sieht, weil anscheinend die Grafik noch nicht kopiert wurde).

Also nutzen wir solange einfach den Grafiklink von hordes:
.ghost .guser .night {
background-image: url("../../../../gfx/design/soul...") !important;
}(Die Grafik-URL wird durch die Forensoftware abgekürzt - der vollständige Link muss via Rechtsklick -> Link kopieren da eingefügt werden!)

Und dann setzen wir den Twinoid-Kasten auf transparent:
.tid_userSheet .tid_userSheetBg,
.tid_userSheet .tid_userSheetBg .tid_bgRight,
.tid_userSheet .tid_header,
.tid_userSheet .tid_header .tid_headerRight,
.tid_userSheet .tid_footer,
.tid_userSheet .tid_footer .tid_footerRight {
background-color: transparent !important;
background-image: none !important;
}

Et voilà: http://www10.pic-upload.de/04.05.12/k...

Wem die orangenen Buttons nicht gefallen, kann diese ebenfalls transparent setzen:
.tid_userSheet table.tid_actions .tid_button {
background-color: transparent !important;
background-image: none !important;
border: none !important;
}

Und der Name + Titel ganz oben links ist dann eigentlich auch überflüssig:
.guser .side {
display: none !important;
}
