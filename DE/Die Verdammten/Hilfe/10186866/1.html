Ich hab hier ein Samsung Galaxy S Mini, so ziemlich das billigste was man im Android-Bereich finden kann.

Das Teil kann enorm kosteneffizient sein, aber dafür muss man schon etwas fummeln.

Sprich: Firmware von Samsung runter, vernünftige Firmware ohne schrottigen Launcher und 1000 Werbe/Spionage-Apps drauf (Cyanogen 7), ARM6-Nightly von Fennec (Firefox) und Flash drauf, CPU übertakten und schon kann man damit auf DV praktisch alles machen (ist allerdings aufgrund des kleinen Bildschirms etwas fummelig).

JavaScript kann so ziemlich jedes jedes moderne Handy, dass über einen Internet Browser verfügt (und garantiert jedes Smartphone).
