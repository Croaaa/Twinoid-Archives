Dass das System auf dem Englischen Server anders ist liegt nur daran, dass dieser in verschiedenen Zeitzonen genutzt wird und damit nicht alle zur selben Zeit Mitternacht haben, damit man trotzdem seine Dinoz alle 24h bewegen kann wurde das System eingeführt, hier auf dem Deutschen Server spielen im großen und ganzen nur Leute aus der selben Zeitzone weshalb das einfachere System verwendet werden kann, welches auch auf dem französischen Server zu finden ist.


Grund dafür das man nur alle 24 h eine Zug mit seinen Dinoz machen kann ist der, dass die Entwickler auf diese weise etwas mehr Zeit haben sich weitere Missionen usw. zu überlegen und auch sonst das Spiel besser ausbauen können. Würde jeder Dinoz jeden Tag 3 Aktionen machen können, wäre man viel schneller am ende des Spiels angelangt ohne dass die Entwickler genügend Zeit hätten um es auszubauen.

Wem es trotzdem zu langsam geht kann auf Irmas zurück greifen, würde ich aber auf dauer nicht empfehlen, da sie 1. Gold Kosten und 2. kommt ihr viel zu schnell an den Punkt wo ihr mit euren stärksten Dinoz nichts mehr machen könnt.
