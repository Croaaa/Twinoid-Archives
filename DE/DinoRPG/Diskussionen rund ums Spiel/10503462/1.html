Jain.

Der Bug ist/war das sobald man auf den Button "Transferieren" geklickt hat wurde eine Übersicht geöffnet und es wurde ein Code generiert. Dieser Code wurde nicht aktualisiert, wenn man danach den Dinoz weiter trainiert hat.

Beispiel:

- Ich habe eine Dinoz auf DinoFight der Level 7 ist.
- Ich drück erstmals auf den Transferieren Button weil ich wissen will was da kommt.
- Es öffnet sich eine Seite auf der ich meinen Dinoz und eine Code sehe "XJc8eF2d"
- Ich breche den Vorgang ab und trainieren meinen Dinoz auf DinoFight weiter.
- Auf Level 18 möchte ich ihn jetzt wirklich transferieren und folge den weiteren Anweisungen auf der Seite.
DER BUG:
- Es wird kein neuer Code für meinen Dinoz generiert und ich erhalte meinen Dinoz in DinoRPG auf Level 7!

Wie gesagt, ich weiß nicht ob dieser Bug noch besteht oder nicht.

Fest steht aber:
Wenn man Geduld hat und sich vom "Transferieren" fern hält bis man das wirklich machen möchte, erhält man seinen Dinoz auf genau dem Level!
Ich bin so auch zu einem Dinoz Level 20 gekommen.
