Bonjour,

Pour s'y retrouver plus facilement, et puisque les bugs sont beaucoup moins fréquents que les questions, la règle qui s'applique sur ce forum c'est : un bug = un sujet. C'est plus facile pour un admin de s'y retrouver si jamais il vient à visiter le forum.

Petit rappel : lorsque vous êtes victimes d'un bug, la première étape est d'envoyer un message sur le support puis ensuite venir créer un sujet sur le forum pour savoir si vous êtes plusieurs à en avoir fait l'expérience. Cependant, si vous avez un doute, vous pouvez toujours créer le sujet sur le forum avant l'envoi du message sur le support. Mais, sans message sur le support, le bug prendra certainement plus de temps à être corrigé.

Je ferme, bon jeu.
