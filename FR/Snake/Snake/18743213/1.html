Le 25 novembre 2012
Keikyaku

Le signalement des bugs se faisant désormais uniquement sur les forums, voici le sujet qui vous permettra de remonter les bugs liés à Snake.


Avant toute chose, assurez-vous que le problème ne vient pas de votre machine.

  • videz le cache de votre navigateur.
  • vérifiez que vos plugins sont à jour.
  • vérifiez que votre navigateur est à jour.
  • désactivez GreaseMonkey.

Si le problème persiste, vous pouvez le signaler sur ce sujet. Merci de donner le maximum de détails quand vous décrivez un bug :

  • Description du bug rencontré : mode de jeu, action effectuée, ...
  • Configuration de votre machine : navigateur, version de flash, ...
  • Capture d'écran si besoin.

Petites précisions :
Les administrateurs liront le sujet mais sans pour autant y passer leur vie. Afin de ne pas surcharger le sujet, les signalements en double et tout message hors sujet seront donc supprimés.
L'existence de ce sujet ne veut malheureusement pas dire que les administrateurs pourront corriger tous les bugs signalés...
