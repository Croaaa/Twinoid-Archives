Bonjour,

Je pense que pas mal de joueurs ici sont victime de lags sur les jeux Arkadeo. Je crée ce topic pour qu'on essaie de trouver une solution ensemble.

Je propose que chaque joueur vienne poster ici sa configuration en utilisant le format suivant :
• Je lague sur Arkadeo : Oui / Non
• OS : [Windows XP, Windows 7 ; Windows 8 ; Linux ; ... ]
• Navigateur : Internet Explorer, Mozilla Firefox, Google Chrome (+ version)
• Adobe Flash Player : [Version]
• Adobe Air : [Version]
• Carte Graphique :
• Processeur :

Pour connaître votre processeur et carte graphique avec Windows :
Démarrer > Exécuter > tapez "dxdiag". Une fenêtre d'infos sur DirectX s'ouvre, donnant diverses informations telles que votre mémoire RAM, votre processeur, votre carte graphique (onglet affichage).

Le but est de comparer les différentes configurations des joueurs, pour essayer de trouver celle qui permet de jouer dans les meilleures conditions. Je ne sais pas si on trouvera, ou même si quelque chose d'unique existe. Mais soit, on aura essayé.

Je commence :
• Je lague sur Arkadeo : Oui
• OS : Windows 7
• Navigateur : Google Chrome
• Adobe Flash Player : 13.0.0.214
• Adobe Air : 13.0
• Carte Graphique : AMD Radeon HD 6520G
• Processeur : AMD A6-34200M APU avec 8 Go de RAM

Si vous pensez que d'autres paramètres doivent être connus car peuvent causer les lags, n'hésitez pas à en faire part !


Merci d'éviter le hors-sujet.

Psychopathe du classement

  • Adblock peut poser soucis aussi, étant très lourd [inutilement].
    Autant le désactiver sur arkadeo, y'a pas de pub de toute façon (oui bon y'a les scripts google analytics.. certes)
  • Autre chose qui peut aider, désactiver le javascript quand on joue
    Et éventuellement fermer les autres onglets/fenêtres (ça changera pas forcément quelque chose ça, dépendant de quel genre d'onglets sont ouverts).
  • Evitez à tout prix d'avoir plusieurs instances de flash (pubs flash, vidéos, etc.)
    Si vous voulez jouer en écoutant de la musique sur youtube, lisez plutôt la vidéo avec VLC (Open Media > Open Network sur la version anglaise, j'ai pas de version fr sous la main ).
  • Des jeux comme Mel ont l'air d'avoir des performances qui varient pas mal. Quand je viens de redémarrer mon navigateur, ça va en général beaucoup mieux puis se dégrade peu à peu.
    >> Redémarrez régulièrement votre navigateur si chez vous aussi les performances se dégradent au fil des parties.

J'ai fait quelques essais. Voici ce que j'en tire :

• Connexion par câble, bonne connexion d'ailleurs (pack pro), donc le problème ne devrait pas venir de là.

• Adblock : je ne l'ai pas installé.

• Désactiver le javascript : si je désactive le javascript, le jeu ne se lance pas. Donc je l'ai désactivé après avoir lancé le jeu (quelle gym !), et j'ai même désactivé le téléchargement d'images... Bref, ça lague encore. J'ai d'ailleurs désactivé TOUS mes plugins (sauf flash player).

• Navigateur redémarré : c'est fait, et même l'ordinateur. Ca lague encore.

• Plusieurs instances flash : je n'ai qu'Arkadeo qui est ouvert. Pas de Skype, pas d'autre onglet, aucun autre logiciel même pas bloc-notes.

Je pense que le problème vient de la version flash, du navigateur et de l'OS perso. Vous pouvez donner votre configuration de ce côté-là (formulaire du premier post) ?
