alexsifle

Ok ok, je viens de regarder un replay, en fait :
les "+" forment une suite géométrique : 100n (n étant le nombre de groupe qui explosent en même temps)
les "," forment une suite arithmétique : 100 + 80n (n étant le numéro du combo)

En reprenant mon exemple d'avant, ça fait :
4R, 4B, 4V = 400 + 720 + 1040 = 2160
4R, 4B + 4V = 400 + 180 x2 x 8 = 2880
4R + 4B + 4V = 100 x 3 x 12 = 3600

Quailard

Autre chose qui est intéressant de signaler, faire tomber une mûre dans un tuyau incrémente le combo "," mais pas le combo "+".

Par exemple, si j'appelle l'événement faire tomber une mûre dans un tuyau "M" :

4R + M, 4B = 4R, M + 4B = 100x4 + 180x4 = 1120.
4R, M, 4B = 100x4 + 260x4 = 1440.

Il n'est pas rare dans les parties de pouvoir faire un coup 4R + M, on aime bien jouer ces coups car on a l'impression de prendre un tuyau tout en faisant un combo, mais des fois on rate un coup légèrement meilleur qui est M, 4R et qui rapporte 320pts de plus.

edit : j'appelle les cailloux des mûres parce que je trouve que ça ressemble depuis le début à des mûres, même si c'est des cailloux
