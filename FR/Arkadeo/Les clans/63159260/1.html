Modification du titre d'un chef de clan

Il faut tricher un peu, le plus simple est d'exécuter ce script dans la console javascript de votre navigateur sur la page membres de votre groupe :

$('.groupMembers td > div:first-child').addClass('editable')

Le chef de clan sera alors éditable comme les autres.

Note : ça ne marche évidemment pas sur les autres clans

Voir avec google & compagnie si vous ne savez pas comment utiliser la console.
