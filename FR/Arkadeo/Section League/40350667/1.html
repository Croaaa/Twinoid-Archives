Salut,

Fais un effort sur l'orthographe stp...

Un MT-Pass est un code qui permet d'obtenir plus de parties, pour jouer plus longtemps. Ce code est payant (téléphone, sms, carte bancaire,...). Si tu es mineure il vaut donc mieux demander l'autorisation de tes parents.

Il y a quand même moyen de gagner des MT-Pass gratuitement :
- En participant à des animations sur les forums ;
- En jouant (longtemps) sur certains sites web comme Carapass ;
- En échangeant tes points Kadeo contre des MT-Pass.

PS : Ce serait sympa de poster ici si tu as encore une question.
