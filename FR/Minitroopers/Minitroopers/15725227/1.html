Bonjour à tous,

C'est agité dans de nombreux QG.
Un Championnat est annoncé : La Lutte des Deuxièmes Classes.

Montez vos armes, huilez-les, entraînez vos corps... c'est parti !

Le Championnat est réservé aux 16 premiers joueurs avec une Armée ayant entre 50 et 100 de puissance.
Le Championnat se fera en 4 Tours.

Déroulement du Championnat :

  • Phase 1 : Inscription. Vous postez le lien vers votre QG (le lien en /hq ) ici
  • Phase 2 : Premier Tour : je désigne les adversaires (choix aléatoire) - 3 jours max pour faire les matches et poster le lien du combat ici.
  • Phase 3 : Deuxième Tour : je désigne les adversaires (choix aléatoire) - 3 jours max pour faire les matches et poster le lien du combat ici.
  • Phase 4 : Troisième Tour : je désigne les adversaires (choix aléatoire) - 3 jours max pour faire les matches et poster le lien du combat ici.
  • Phase 5 : Quatrième Tour : je désigne les adversaires (choix aléatoire) - 3 jours max pour faire les matches et poster le lien du combat ici.

Une Victoire donne 3 points, une défaite 1 point, un match non joué donne 0 point.
En cas de Perfect (aucun Trooper mort), bonus de 1 point.

A la fin, ceux ayant le plus de points gagnent. En cas d'égalité, organisation de matches à mort pour désigner le vainqueur.

Des smileys exclusifs pour les gagnants sont à remporter.

Have fun.
