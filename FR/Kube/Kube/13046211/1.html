Il est possible que le serveur de jeu temps réel ne fonctionne pas actuellement, vérifiez avec d'autres joueurs qu'ils arrivent à y accéder.
Utilisateurs avancés : dans le cas où vous avez un pare-feu et la possibilité de le configurer, vous devez autoriser les connexions sortantes TCP sur le domaine play.[jeu].muxxu.com (par exemple play.kube.muxxu.com pour le jeu Kube) et sur le port 6667.
