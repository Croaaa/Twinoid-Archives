Voici plusieurs idées de Kubes

Certains des mégakubes proposés ici mélange différent kube. Les 3 étages ( 3 kubes de hauteurs ) seront représentés par un autre plan. Imaginé donc les trois étages superposés, l'étage 1 en bas.
Exemple :

Etage 1 : __ Etage 2 : __ Etage 3 :
/// ///
/// ///
/// ///

= ID XX
= ID XX
= ID XX



Séries - Kubes Espace !

Kube Météore
Trouver dans : les îles mirages, mais seulement, des îles mirages en hauteur ( vers les 20-30 kubes de hauteur ) !
Pour exploiter ces îles, il faudrait donc construire un pont en hauteur...
Exemple de kube météore + île mirage hauteur : Blanc = île mirage, noir = meteore.
Allure : un peu comme les laves refoidies mais plus 'ferreux', 'carbonisé'.

Kube gruyère

Trouver dans : coffre, dans les îles, comme les transparents.
Allure : Kube jaune-beige, comme un fromage. A des trous, à la façon des buissons mais moins dispersés/nombreux.
