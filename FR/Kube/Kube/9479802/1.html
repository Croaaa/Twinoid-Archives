C'est un bug du script Greasemonkey Kube bridge accompagnant KubeTools de Durrs.
Il n'a malheureusement pas trouvé pourquoi.

Bref, au choix :
-Désactiver ce script et recharger la page.
-Cliquer le + de la description s'il y a, ça débloque le bouton.
-Sélectionner du texte dans la description de zone, appuyer sur shift+tab pour mettre le focus (pointillés) sur le bon bouton, puis appuyer sur entrée pour l'activer.
