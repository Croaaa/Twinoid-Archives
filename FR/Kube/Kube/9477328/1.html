oui vous avez bien lu il est possible de jouer a kube avec votre manette de jeu favorite: la wiimote(oupas)
vous devez disposez:
d'un ordinateur (ça peut servir )
de bluetooth ou d'une clef bluetooth
du logiciel bluesoleil sinon vous pouvez le télecharger (bluesoleil est souvent fourni avec bluetooth)
du logiciel glovepie sinon vous pouvez le télécharger
d'une wiimote et d'un nunchuk (ça peut servir aussi)
un emméteur infrarouge (ex diode infrarouge voir même deux bougies (je n'ai pas testé avec les bougies) si vous n'en avez nous allons utilisez votre wii.

-ouvrez bluesoleil cliqué sur le soleil (rond rouge-orange) et appuyé en même temps sur les boutons 1 et 2 de votre wiimote (les leds de votre wiimote doivent clignoter)

-maintenant faites un clique droit sur votre manette qui est apparue a l'écran et cliqué sur "connect bluetooth human interface device" tout en appuyant sur les boutons 1 et 2 de la wiimote (les leds de votre wiimote doivent clignoter)
l'icône de votre manette est devenue vert sinon recommencer

-télécharger glovepie (vous le trouverez facilement grâce a un moteur de recherche) et "dézipez" le

-ouvrez glovepie aller dans file>open>et chercher le dossier "test.PIE"

- cliquez sur "run" et allez dans l'onglet value quand vous bougez la wiimote certaines valeurs doivent changer

-ouvrez un nouveau fichier (file>new)

-mettez votre emmeteur infrarouge dessous ou dessus votre écran si vous n'en avez pas prenez votre wii et metez la a coté de votre ordinateur alimenter là (branchez le cable électrique dans une prise électrique) et placez la sensor barre (barre que d'habitude vous placez dessus ou dessous votre télé) dessus ou dessous votre écran d'ordinateur


-dans l'onglet unamed copier coller ce script
var.xtrim = 0
var.ytrim = 0
var.ztrim = 0


var.deadzone = 5
var.KITTspeed = 100 ms
var.rumble = false

var.accx = wiimote.RawForceX + var.xtrim
var.accy = wiimote.RawForceY + var.ytrim
var.accz = wiimote.RawForceZ + var.ztrim

if wiimote.dot1vis and wiimote.dot2vis then

if var.accy > -7 then
var.orientation = 0
elseif var.accy > -45 then
if var.accx < 0 then
var.orientation = 3
else
var.orientation = 1
endif
else
var.orientation = 2
endif

if var.leftpoint = 0 then
if var.orientation = 0 then
if wiimote.dot1x < wiimote.dot2x then
var.leftpoint = 1
else
var.leftpoint = 2
endif
endif
if var.orientation = 1 then
if wiimote.dot1y > wiimote.dot2y then
var.leftpoint = 1
else
var.leftpoint = 2
endif
endif
if var.orientation = 2 then
if wiimote.dot1x > wiimote.dot2x then
var.leftpoint = 1
else
var.leftpoint = 2
endif
endif
if var.orientation = 3 then
if wiimote.dot1y < wiimote.dot2y then
var.leftpoint = 1
else
var.leftpoint = 2
endif
endif
endif

if var.leftpoint = 1 then
var.fix1x = wiimote.dot1x
var.fix1y = wiimote.dot1y
var.fix2x = wiimote.dot2x
var.fix2y = wiimote.dot2y
else
var.fix1x = wiimote.dot2x
var.fix1y = wiimote.dot2y
var.fix2x = wiimote.dot1x
var.fix2y = wiimote.dot1y
endif

var.dx = var.fix2x - var.fix1x
var.dy = var.fix2y - var.fix1y
var.cx = (var.fix1x+var.fix2x)/1024.0 - 1
var.cy = (var.fix1y+var.fix2y)/1024.0 - .75

var.d = sqrt(var.dx*var.dx+var.dy*var.dy)

var.dx = var.dx / var.d
var.dy = var.dy / var.d

var.ox = -var.dy*var.cy-var.dx*var.cx;
var.oy = -var.dx*var.cy+var.dy*var.cx;

var.ax = (var.ox * screen.desktopwidth) + (screen.desktopwidth / 2)
var.ay = (-var.oy * screen.desktopwidth) + (screen.desktopheight / 2)




var.dx = var.ax - mouse.cursorposx
var.dy = var.ay - mouse.cursorposy

var.d = sqrt((var.dx*var.dx)+(var.dy*var.dy))

var.a = 180 / (200 + var.d * var.d * var.d * .001)

if var.d <= var.deadzone then var.a = 1

debug = var.d + " " + var.a

var.finalx = mouse.cursorposx * var.a + var.ax * (1 - var.a)
var.finaly = mouse.cursorposy * var.a + var.ay * (1 - var.a)


mouse.cursorposx = var.finalx
mouse.cursorposy = var.finaly

else

var.leftpoint = 0

endif

mouse.LeftButton=wiimote.A
key.Space=wiimote.b
key.U=wiimote.1
key.Escape=wiimote.Nunchuk.CButton
key.Shift=wiimote.Nunchuk.ZButton
key.Up = Wiimote.Nunchuk.JoyY < -0.4
key.left = Wiimote.Nunchuk.JoyX < -0.4
key.down = Wiimote.Nunchuk.JoyY > 0.4
key.Right = Wiimote.Nunchuk.JoyX > 0.4

-si quand vous bougez le joystic de votre nunchuk vous ne bougez pas cliquez sur la fenêttre de jeu

vous pouvez maintenant jouer a kube :yipee:

voici les commandes
joystic du nunchuk: se déplacer
bouton A de la wiimote : poser/prendre un kube
bouton B de la wiimote : sauter
bouton 1 de la wiimote : annuler la pose d'un kube (si vous n'avez pas bougé entre temps)
bouton C du nunchuc : revenir en mode main
bouton Z du nunchuk : avancer doucement mais sans pouvoir tomber
si ces commandes ne vous conviennent pas alors créez votre propre script ou demandez moi de vous en faire un

quand vous avez fini de jouer sauvegarder le fichier (file>save) et appelez le kube

pour recommencer a jouer ouvrez bluesoleil faite clique droit sur votre manette et cliquez sur "connect bluetooth human interface device" tout en appuyant sur les boutons 1 et 2 de la wiimote (les leds de votre wiimote doivent clignoter)
l'icône de votre manette est devenue vert sinon recommencer

puis ouvrez glovepie ouvrez le fichier kube.PIE (file>open et chercher le dossier kube.PIE)

cliquez sur "run" vous pouvez de nouveau jouer

si tout cela vous intéresse vous pouvez utiliser les scripts fourni avec glovepie:
wiidumm2 vous permet de jouer a la batterie avec votre wiimote
wiitar de jouer a la guitarre
wiilight ou wiiled est un script utilisant les leds de la wiimote (pratique pour apprendre a les utiliser dans vos scripts)
wiisaber trouvable sur le web permet de transformer votre wiimote en sabre laser

pour utiliser la wiimote comme souris utiliser wiimouse
ou encore pour les FPS wiimouserelative dans lequel la souris revient au centre dans l'écran tout le temps (pour ne pas sortir du cadre de jeu)

si vous trouvez que kube manque de musique pourquoi ne pas combiner le script wiitar ou encore wiidumm2 avec votre script kube

sachez aussi que l'on peut utiliser plusieurs wiimote il faudra alors écrire wiimote1 ou wiimote2 pour indiquer de quel manette il s'agit
