Amis kubiens débutants (comme moi), je vous présente DolFind, un chercheur de dolmen.

Vous êtes perdus et vous cherchez un dolmen dans le coin ? Seul soucis, si vous êtes en 250 ; -45, et qu'un dolmen est en 252 ; -44 (c'est à dire tout près) vous pouvez chercher des heures dans les listes dans le trouver...

La solution ? DolFind !

Sans plus attendre, le lien :
http://muxxu.blue112.eu/dolmens.php

Deux champs :
Domaine : Vous permet de dire jusqu'à combien vous voulez marcher pour trouver le dolmen. Exemple, si je met 5, le dolmen sera à maximum 5 cases de moi.
Coordoonées : Facile, l'endroit où vous êtes. Mettez par exemple : -250,200 si vous êtes à ces coordonnées.

Hope it will be usefull

--Blue112
