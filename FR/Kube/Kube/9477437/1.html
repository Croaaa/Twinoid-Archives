Bonsoir chers amis architectes et explorateurs du monde de Kube ! :youpi:

Je me lance dans une compilation : Je souhaite compiler toutes les îles de Kube découvertes avec leurs coordonnées.

Les données seront compilés dans le fichier classeur suivant : Ficher Kub'il

Le fichier est à télécharger et à ouvrir avec un logiciel tableur de genre : Microsoft Excel ou OpenOffice.

Je vous invite à lire la feuille de présentation ainsi que les autres.

L'intérêt de ce classeur est qu'il vous permettra de trouver les îles d'un type voulu où que vous soyez, exemple : Je suis en [-502][13] et je cherche une île violette ... oh une se situe en [-510][15], j'y vais de ce pas !

De plus, étant donné que le fait que lîle soit pillée ou non est compilé, ce fichier vous permettra de rechercher une île idéale pour par exemple creuser afin de trouver des kubes transparents. :niark:

Soit, ce sujet servira a communiquer les îles à compiler, vous devrez préciser si cette île ressemble à un gruyère ou non (au second sens du terme), toutes les données communiquées sont compilées.

Pour désigner les îles merci d'utiliser les noms spécifiés dans la troisième page du fichier, autrement les données communiquées ne seront pas compilées :charte:, toutefois les images des îles sont disponibles dans le ficheir pour éviter toute confusion.

Postez nombreux ! :youpi:
