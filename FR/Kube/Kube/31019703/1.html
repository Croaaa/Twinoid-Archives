Bonjour à la communauté.

Je suis ici pour vous proposer un script indiquant la date de délocation des zones Kubes sur lesquelles vous avez le droit de dormir (vos zones aussi, donc, c'est d'ailleurs pour ça qu'il a été fait à la base).
Grâce a lui, donc, vous pourrez savoir quand relouer vos zones.

/!\ Vous devez avoir GreaseMonkey ou TamperMonkey d'installé.

Le script se trouve ici : >> Déloc' <<

Voici la marche a suivre pour l'installer :

1. Se rendre à l'adresse du lien ci-dessus
2. Cliquer sur "Install" en haut de la page
3. Attendre 5 secondes puis cliquer sur "Installer"
4. Le script est installé avec succès (et pas de redémarrage nécessaire)

L'idée vient de lwxtz2004, et j'ai réalisé le script.

Si vous rencontrez des bugs, merci de m'en informer.

EDIT : Vous pouvez utiliser les script d'ayalti, sinon.
