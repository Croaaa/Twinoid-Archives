Je voudrais votre avis, j'aimerais créer un groupe de constructeur.

Les règles de ce groupe serais :
Uniquement des figures du jeu vidéo
Uniquement du pixel 3d
Le plus possible des kubes standard ( on ne met pas de kube or juste pour faire blingbling )
On utilise des zones sur les ponts pour les protéger ( ce qui n'est pas toujours évident )

Le groupe fonctionneras simplement, une personne peut proposer un projet ( de préférence avec au moins un début de plan ), les constructeurs peuvent ensuite voter pour le projet et si il y a suffisamment de monde, le projet est pris en charge par le groupe.

Le but est de donner sa chance à tous le monde, mais pour autant, ne pas ce disperser sur des dizaines de projet.

---------------------------------------- -

Je propose la création de ce groupe parce que je me rend compte de l'engouement des constructeurs pour les projet en pixel 3d.
Mais il y a manifestement un besoin d'organisation.

Si il y a suffisamment d'engouement pour cette idée, je lance le groupe.
(l'idée n'est pas de gagner le concours, mais si on peut faire un beau design ... )
