Il suffit de chercher un minimum dans les aides.
Voici ce qui est écrit dans la FAQ (premier lien de ce forum) :
Où trouver les cartes bancaires ?

Les cartes bancaires se trouvent dans les mails et certains fichiers (fichiers.doc), il s'agit souvent de documents de facturation, des reçus ou le listing d’une commandes.

En moyenne, il est possible d'en trouver deux par missions, parfois plus, parfois moins, de fois même vous n'en trouverez aucun. Pour trouver ces codes bancaires vous devrez lire le contenu des fichiers.

Cliquez sur un fichier pour son contenu s'affiche dans la colonne à droite de l'interface de jeu. Vous pourrez les repérer au fait que :

* ces documents sont les seuls à contenir un paragraphe de couleur rose
* un numéro de carte respecte toujours le format suivant : xxxx xxxx xxxx xx

Et sinon, pour tes questions, il faut utiliser le sujet questions/réponses (également l'un des premiers topics de ce forum...).
