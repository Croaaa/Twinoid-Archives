Voilà la liste complète des statistiques qui me paraissent intéressantes :


En fonction du nombre de mission remplies (Statistique / Missions remplies) :

- Suppression d'anti-virus
- Extraction de fichiers de valeur
- Extraction de ressources
- Traces effacées
- Systèmes corrompus
- Mots de passe cassés
- Systèmes crashés
- Cartes bancaires exploitées

En pourcentage :

- Missions sans échec : (Missions sans échec * 100) / Missions remplies
- PACKs rares revendus (en fonction des pack revendus ) : (PACKs rares revendus * 100) / PACKs revendus
- PACKs revendus (en fonction du total des packs récupérés: (PACKs revendus * 100) / Extraction de fichiers de valeur
- Balance entre corruption/destruction :

Corruption : ( Systèmes corrompus * 100 ) / ( Systèmes corrompus + Systèmes crashés )
Crash : ( Systèmes crashés * 100 ) / ( Systèmes corrompus + Systèmes crashés )

En secondes, minutes, heures : :P
- Temps de jeu gagné au total : Traces effacées * 30s
- Temps de jeu gagné par mission : (Traces effacées * 30s) / Missions remplies

Pas de statistique pour :

- Missions remplies
- Agent de haut niveau

D'autres stats qui peuvent être intéressantes mais je ne sais pas si c'est techniquement possible :

- Prime moyenne des missions
- Prime moyenne des missions pour un niveau
- Moyenne des ventes de données volées
- Moyenne des crédits récupérés sur les CB
- Type de mission préféré : Destruire un poste, supprimer des mails, récupérer un code sources, etc
ex : Votre type de mission favori est de voler des émissions de TV (25% de vos missions remplies)
