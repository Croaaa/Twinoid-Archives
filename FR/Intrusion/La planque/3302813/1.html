Dans le système, tu trouves souvent des Mot de passe de plusieurs utilisateurs du PC. Donc, je te conseille d'y faire un tour. Ce sont des fichiers qui ont un point d'exclamation rouge avec un code binaire derrière (1 ; 0).

En faite, c'est un fichier comme les fichiers qui te disent où sont les antivirus dans les dossiers, ou qui te permettent de gagner 30s, si tu les supprimes ( Un fichier log (?) )

Bref, bonne chance.
