Pour les mots de passe, on peut les trouver :
- dans les fichiers .log
- dans les fichiers password (souvent codés mais pas toujours)
- dans les e-mails
Ensuite il suffit de copier/coller le mdp sur les dossiers possédant un cadenas pour les casser.

Pour les cartes bancaires, on peut les trouver dans les e-mails ou bien les documents, elles sont repérables à l'écriture rosée comme ceci : http://imageshack.us/m/23/9748/61768858. png
Il suffit ensuite d'entrer les numéros trouvés ici.
