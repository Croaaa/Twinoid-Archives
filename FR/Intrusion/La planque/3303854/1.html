bon info j'ai etez dans la rubrique aide de http://support.motion-twin.com/
dans l'equel est expliquer comme suit (copie du texte) comment faire pour resoudre le probleme je site :

Comment activer/désactiver un chipset ?
Dans l'interface du jeu, vous de vous équiper d'un chipset, OU de choisir de ne pas vous équiper d'un chipset.

Dans la section Decks :
(la page où vous avez l'habitude de sélectionner les virus dont vous souhaitez vous équiper)
Dans le premier cadre, juste sous le menu se trouve "Choix du chipset actif", où figurent tout les chipset que vous avez achetés, ainsi qu'une option AUCUN, si vous souhaitez qu'aucun chipset ne soit équipé pour votre mission.

Il est également possible de choisir quel chipset équiper dans le premier tableau d'une mission. Cependant, dans ce tableau l'option AUCUN ne figure pas dans la liste. Donc, si vous souhaitez ne vous équiper d'aucun chipset, vous devrez opérer sur la section Decks, où cette option est disponible.

Bug connu :
Lorsque l'on ne possède qu'un seul chipset, celui-ci se trouve automatiquement équipé, sans possibilité de le retirer. Cela peut s'avérer problématique, si votre premier achat de chipset se porte pour le chipset Scout.
Solution :
Si vous cliquez sur le lien suivant : Choix de chipset : Aucun
(Vous devrez être connecté à votre compte Muxxu pour que cela fonctionne)
Cela aura pour effet de désactiver le chipset automatiquement équipé, et débloquera à l'avenir la possibilité de choix entre chipset --- et aucun chipset.
