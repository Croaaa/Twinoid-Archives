L'exécution de virus est plus rapide, l'entrée dans un dossier est plus rapide, la connexion est plus rapide. Mais c'est cher :


Version 1.0 → - 75000 → Version 2.0
Grande augmentation de vitesse (connexion en ?.?? seconde)

Version 2.0 → - 250000 → Version 3.0
Augmentation de vitesse moyenne (connexion en 0.25 seconde)

Version 3.0 → - 500000 → Version 4.0
Augmentation de vitesse minime (connexion en 0.20 seconde)


La 4.0 c'est vraiment pour ceux qui veulent tout avoir au max, parce que le rapport qualité/prix est vraiment nul (500 000 sous pour une soi-disant augmentation qui est invisible), pour la plupart 3.0 est largement suffisant.
