Bon, je sais très bien, ce jeu n'est pas fait pour être réaliste, blablabla.
Mais n'empêche. Un petit peu de bon sens, quand même...

Qu'on trouve les mots de passe des ordinateurs personnels dans les mails, les fichiers de logs, les fichiers passwd, logique.

Mais que le password d'une imprimante soit dans un mail disant "Au fait, ton mdp a été changé, c'est bidule maintenant". Comment dire... "Hé, salut, ma copine l'imprimante, c'est moi ton pote le grille-pain, tu sais pas quoi ? Ton pass a changé !"

Si on veut vraiment un mail qui indique un changement de mot de passe (pour une imprimante, ou un firewall, ou toute machine non "personnelle"...), il serait plus logique de le faire sous la forme d'un mail envoyé à une ou plusieurs personnes du réseau, par exemple :
Message à tous les utilisateurs de l'imprimante SPZ-5.
Le mot de passe de la machine a changé, c'est désormais ploum.
Merci de supprimer ce message dès que possible.

C'est très logique en même temps : root a envoyé ce message à tout le réseau. Pour s'en rappeler, Julien Natalis l'a imprimé (donc il est dans le fichier de l'imprimante). Puis comme il est zélé et consciencieux, il l'a supprimé de sa machine, comme les autres du réseau. Donc on le trouve pas sur les autres machines (sauf si on décide de le mettre sur une machine, auquel cas ça donne une forme toute faite de message).
