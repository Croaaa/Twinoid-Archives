Il est possible de terminer la plupart des ordinateur d'un réseau, voir de tous les terminer.
Ensuite, tu n'auras pas à chercher partout, car un mot de passe se trouve, soit sur l'ordinateur de la personne (dans les mails, dans un fichier password ou celui qui surveille ce qui se passe et donne 30 sec) soit dans un fichier "globalpassword", "masterpassword" ou les mots de passes de tous le réseau sont indiqués. Ils sont assez souvent cryptés au niveau 1.

Pour préciser un peu mon voisin.
Tu peux trouver les mots de passe :
• Dans les fichiers .log comme d'habitude
• Dans les Mails. Les mails contenant les MDP sont généralement court, du style : "Hey body, ton mot de passe c'est [...] maintenant ! N'oublie pas ..."
• Encore plus rare, mais ça m'est déjà arrivé, dans les fichiers "GlobalWord", "MasterPass. Ce genre de fichier se trouve dans les Imprimantes (Et j'ai entendu Base de données, je ne sais plus trop sur quel topic )
