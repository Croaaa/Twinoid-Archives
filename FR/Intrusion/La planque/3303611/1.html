2) Sur le forum
http://muxxu.com/forum/thread/59893
- Les fichiers password: Ils détiennent le mot de passe du terminal sur lequel vous êtes. Parfois, sur les imprimantes et bases de données, vous trouverez un fichier "global password" ou "master password" qui détient tous les mots de passe du réseau. Les mots de passe permettent d'aller dans les dossiers qui arborent une serrure. (ce sont les dossiers verrouillés, et seule l'obtention du mot de passe permet d'y aller.)

Soit dit en passant, les mots de passe peuvent se trouver dans les mails des PC, faites donc attention. Et aussi, il est inutile de copier les fichiers password dans l'optique de les réutiliser dans les missions suivantes: les mots de passe varient selon les missions.
