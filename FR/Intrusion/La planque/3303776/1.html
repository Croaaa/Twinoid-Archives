Je vais préciser un peu les mots de mon prédécesseur :
Les mots de passe se trouvent :
• Sois dans les fichiers .pass (Ils se nomment généralement "Authentification", "Passlogin", ...). Ces fichiers sont parfois cryptés (Et donc il faut utiliser un virus décodage).
• Sois dans un fichier .mail. Le fichier concernés a généralement cette forme la :
Hey mon pote, ton nouveau mot de passe c'est azur1689 ! N'oubliez pas.
• Parfois, il peut arriver que tu doivent aller dans une imprimante ou une base de données. Ces terminaux contiennent parfois des fichiers "MasterPassword', qui contiennent tous les Mots de passe de toute les terminaux !

Et pour finir, chaque terminal ne contient qu'un et un seul mot de passe (Utilisable plusieurs fois).

Concernant les marqueurs Shield. Au niveau 6, tu as la possibilité d'acheter le virus "Faille : Ouverture" qui supprime un marqueurs Shield d'un fichiers. Comme l'a dit plus haut, un marqueurs fait protéger 5 points de dégâts a un fichier.
Pour chaque Anti-Virus présents dans le terminal, tous les fichiers gagnent un marqueurs Shield (Comme un Anti-Virus Paladin, sauf que ca concerne tous les fichiers (Anti-Virus, Gardien, Coeur, ...)). A savoir que les Anti-Virus Shield s'octroie aussi à eux-même un marqueurs Shield. Il faut donc utiliser le Virus "Faille : Ouverture" pour enlever un marqueur.
Ce virus n'étant accesible qu'au niveau 6, je te laisse continuer les missions oranges
