Comment faire son fichier .hackrc

J'ai débuté le mode Uber-31337 récemment, j'ai pris longtemps a faire mon fichier .hackrc.
Le problème est qu'il y a beaucoup d'aide dessus, mais jamais de vrai démarche pour le créer.
Voici donc mon petit guide pour créer son fichier .hackrc.
Je prend l'exemple de mon fichier pour expliquer la démarche.
# ---- Commandes MOUVEMENTS ----
# ENTRER dans un dossier
alias open = cd
# RETOURNER au dossier precedent
alias out = cd ..
# QUITTER le système
alias exit = exit
# CIBLER un fichier
alias target = target
# ---- Commandes VIRUS ----
# COPIER un fichier
alias copy = copy
# CORRUPTEUR de fichier
alias control = corrup
# ATTAQUER par "degats legers"
alias attak = dmgm
# RESERVE de ressources
alias mana = mana1
# DECODER un fichier
alias lock = dec2
# ---- DIVERS ----
# EXTRAIRE un Pack
alias take = extrac

1) Les bases d'Alias et de la dièse (#)

Pour créer son fichier, il faut savoir un grande principe :
Pour créer un raccourci, il faut taper alias, suivi du nom du raccourci que l'on veut donner,
suivi d'un espace, d'un égale (=), d'un autre espace, et pour finir, le nom du raccourci globale du virus ou de la commande
alias [RaccourciQueL'OnVeutDonnerAuVirus] = [RaccourciGlobaleDuVirusOuDeLaCommande]
Exemple :
alias take = extrac
# "extrac" est la commande pour extraire un fichier PACK. Dans ce cas, quand je taperai "take" dans ma console, le fichier ciblé sera extrait

alias attak = dmgm
# "dmgm" est la commande pour utiliser "Dégats légers". Dans ce cas-ci, quand je taperai "attak", le fichier ciblé subira le
virus "dégats légers".
Jusqu'ici rien de bien difficile me direz-vous.
Une autre chose à apprendre, vous pouvez décorez votre fichier avec des commentaires.
Pour les utiliser, vous devez les mettre avec en début de ligne une dièse (#)
Exemple :
# COPIER un fichier
Vous pouvez les utilisez tant que vous voulez.

Sachez que tous le monde peut voir votre fichier, alors évitez de mettre des trucs choquants. Les termes doivent toujours respecter
le CGU.
Vous pouvez être prévoyants : Si jamais des petits nouveaux regardent votre fichier, aidez-les ! Faites en sorte
qu'il se retrouve dans votre fichier.
J'ai par exemple astucieusement mis des commentaires de titres, ... pour organiser mon fichier .hackrc.
Exemple :
# ---- Commandes MOUVEMENTS ---- [Titre, ce paragraphe contiendra tous ce qui est mouvement dans les dossiers du jeu]

# ENTRER dans un dossier [Ce que fait la fonction, ici, elle permet d'entrer dans un dossier]

# ---- Commandes VIRUS ---- [Rebelote, ici, les alias contiendront des raccourcis pour utiliser ses virus.]

# COPIER un fichier
Ainsi, les lecteurs pourront facilement se retrouver dans vos alias !

Voila, c'était toute la théorie à "apprendre". Dans la prochaine partie nous verrons comment faire votre fichier.

2) Créer mon fichier .hackrc !

"Mais que dois-je faire ?"

Dans l'ordre, On le verra, attends, oui, NON !

Ici, j'ai du apprendre plus ou moins de moi même. Mais, en exclusivité, je vais vous donner ce qu'il faut faire.
Le problème est qu'on a peur d'oublier quelques chose, et de faire une partie fausse.
Il faut surtout penser à 4 chose :
-cd, cd .. et exit (Que je regroupe en MOUVEMENTS")
-Ses virus
-target
-extraire

a) Les MOUVEMENTS

Personnellement, j'ai regroupé les 4 sous un titre, Je vais vous expliquer les 4 raccourcis.
"cd" : Ceci vous permet de pénétrer dans un dossier
Exemple :
cd mails
Ceci vous ouvrira le fichier "Mails"


"cd .." : L'inverse du premier, il vous permet de quitter le dossier
Exemple :
cd ..
Tout simplement, il vous renvoie un dossier plus haut

"exit" : Quand vous serez dans le premier écran (généralement avec un dossier "dossier" et un dossier "program files") du terminal,
vous pourrez utiliser cette commande pour revenir à l'écran des terminaux (A noter que vous ne pouvez choisir un terminal au clavier )

Bien sur, vous pouvez utiliser des "alias" pour simplifier ces commandes :
Exemple :

# ---- Commandes MOUVEMENTS ----
# ENTRER dans un dossier
alias open = cd
# RETOURNER au dossier précèdent
alias out = cd ..
# QUITTER le système
alias exit = exit

b) Les VIRUS

Rien de bien compliquer, pour savoir la commande, vous pouvez survoler avec la souris le virus dans l'onglet "Deck" pour savoir
le raccourcis globale du virus
Exemple :
dégats Légers = dmgm;
Corruption = corrup;
...
Essayer de prendre des raccourcis intuitifs, que vous ne soyez bloqué devant l'antivirus en sachant que mettre dans la barre de la console.

c) La commande TARGET [Personnellement, je l'ai rajouté dans MOUVEMENT car il y ressemble ]

C'est une chose très importante en mode Uber-31337. Pour cibler l'antivirus que vous voulez détruire, vous devez faire comme ceci :
Exemple :
target glue[...]
[...] = Seulement, généralement, les antivirus ont des noms très compliqués du style "glue123.anti-vir"
Alors la MT a simplifier la chose, a inventer la touche "TAB" (La touche avec 2 flèche opposées).
Quand vous mettez le nom d'un fichier, dossiers, ... Vous pouvez ne mettre qu'une partie et appuyer sur "TAb"
pour que le PC fasse tout seul le reste. Il va fouiller dans tous les dossiers, fichiers, présents et voir celui qui correspond
avec le début de ce que vous avez écrit
Exemple :
target glue[TAB]
# Le pc va automatiquement compléter avec :
target glue123.anti-vir
Avec ceci, ça vous prendra beaucouo moins de temps pour cibler vos fichiers.
"Mais, un instant, et si il y a 2 anti-virus repurgator dans le même dossiers ?"
Eh ben il faudra préciser. Il faudra écrire un début un peu plus pousser pour distinguer les 2 antivirus
Exemple :
Si dans le fichier, il y a :
"repurgator986.anti-vir"
"repurgator556.anti-vir"
# Eh ben il faudra indiquer :
target repurgator5
# Pour cibler le 2ème antivirus, effectivement, il n'y a qu'un virus qui commence par repurgator5
Si vous avez compris ça, vous avez compris une bonne partie du mode Uber-31337 !
Sachez mettre un bon raccourcis, car cette commande sera, probablement, la plus utilisée pour le mode Uber-31337

d) La commande EXTRAIRE

Rien de bien compliquée, mais ça peut arriver qu'on oublie de le mettre et qu'on ne sache que faire devant un fichier PACK !
La commande est tout simplement "extrac"
Exemple :
target archive236.pack
extrac
# Le fichier sera extrait (et + de ressource ou un fichier à vendre pour plus tard dans la bourse noire

Dernière chose à dire, vous pouvez utilisez plusieurs raccourcis pour une même commandes ou actions :
Exemple :
# CORROMPRE le fichier
alias control = corrup
alias ctrl = corrup
alias banquier = corrup

Et voila ! Si vous avez bien compris tout ce texte et que vous avez minutieusement fait votre fichier .hackrc, eh ben vous serez prêt
pour votre première mission uber-31337.

P.S. Ne vous inquiétez si vous ne réussissez pas votre première mission, ou que vous pataugiez au point de devoir retourner
dans votre fichier .hackrc. Le tout est de s'entraîner !
