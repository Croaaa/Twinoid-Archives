Les mots de passe se cachent dans :
Les mails
Les .logs (fichiers rouges à effacer pour gagner 30 secondes, vous connaissez ?)
Les fichiers .pass

Pensez à chercher un fichier global passwords (un fichier qui rassemble les mots de passe du réseau), ils se trouvent parfois dans les bases de données, dans les imprimantes ...
