4. Le Téléchargement d'anti-virus
Ceci demanderai un nouveau virus (débloquable après une action, un trophée, etc), à l'aide de ce virus, on pourrait télécharger un anti-virus (rien de compliqué, on prend le virus, -20/30 points de rescource, on clique sur l'anti-virus ciblé, on attend de 5sec à 30sec (selon l'anti-virus) et walà !).
A la fin de la mission, le registre affiche :
Code sources d'Antivirus :
Data/couveuse
Data/fantassin
Et c'est là que ça se complique...
Dans votre disque dur, vous avez accès à une ligne de code pour chaque anti-virus (et de belles lignes de codes !), et vous devez, à partir des codes, modifier la source pour obtenir un virus !
Ligne à compléter :
<Data=/(a)/vir/source/(b)>
(a) = 7 chiffres et lettres du code source (pas choisis au hasard)
(b) = 5 chiffres du code source (pas choisis au hasard)
Si vous complétez le code source correctement, vous pouvez acheter un virus inédit dans le HackStore !
Les + : Cela néscéciterai beaucoup de stratégie, d'analyse, et ça développerai le côté Hack
Les - : Je n'en vois pas, mais vous allez m'en trouver, j'en suis sûr...=P

5. L'espionnage
Ce nouveau type de mission serverait à infiltrer un réseau de PC en espionnant les fichiers à l'aide de Virus.
Bien évidemment, mission spéciale, virus spécial, le virus espion serait disponnible on-ne-sait comment dans le HackStore.
Une fois connecté, infiltrez-vous sur le PC concerné, et faîtes en sorte d'affecter le plus de fichier important (z'ai oublié le nom, je vous en informe peut-être aujourd'hui, ou demain) ainsi que le Coeur (donc neutralisation des gardiens) du réseau.
Et voilà, vous pouvez maintenant vous infiltrer autant de fois que vous voulez sur ce PC pour voler, piller, vous amuser (vous n'avez que trois minutes par infiltration, et ça demanderai des jetons)

J'éspère que ça vous à plû, à bientôt les zamis !
