puisqu'il est là, autant y répondre :
pour trouver un mot de passe, tu dois trouver les fichiers .pass (souvent codés)
sur ces fichiers .pass, tu trouvera une liste comme :
Jean Delarue : privus9 (révoqué)
Jean Delarue :Lucile24 (actif)
dans ce cas, c'est le mot de passe ou il est écrit (actif) qui est bon. mais parfois, rien n'est indiqué a coté d'un password, dans ce cas, c'est le mot de passe ou in n'y à rien d'écrit.

Si par contre, le fichier est codé, a toi d'utiliser le virus "décodeur V x.0"
