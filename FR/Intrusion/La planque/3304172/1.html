Les codes sont des mots de passe, on peut les trouver dans :
- les fichiers .log
- les fichiers password (souvent codés mais pas toujours)
- les e-mails
Ensuite il suffit de copier/coller le mdp correspondant sur les dossiers possédant un cadenas pour les casser.
Le bon mot de passe est repérable dans la liste par la balise "(actif)" si elle est présente, sinon c'est le mot de passe qui ne possède pas de balise "(révoqué)" qui est le bon.

Si le fichier contentant le mot de passe est codé, il faudra alors décoder le fichier à l'aide du virus décodage du niveau équivalent ou supérieur au codage et lancer le virus sur le fichier à décoder.
