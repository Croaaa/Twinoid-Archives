Oui je confirme, fichierspack.free.fr ne fait plus d'estimations quotidiennes. Ce serait un peu long à expliquer le pourquoi du comment.

Par contre, il peut malgré tout vous être utile pour estimer la valeur de vos fichiers.

Par exemple :
CODE SOURCE D'UN PROGRAMME
205
249 .. 176 .. 51

Donc "205" c'est le dernier prix du jour enregistré. Bref il n'est plus d’actualité.
Ce qui peut vous être utile, c'est les 3 valeurs juste en dessous :
249 .. 176 .. 51

  • La première, est le prix le plus élevé jamais enregistré. (Max)
  • La seconde, est le prix moyen.
  • La dernière, est le prix le plus bas jamais enregistré. (min)

Ce site a fonctionné correctement pendant plus de deux ans. Donc ces chiffres (max, moyen, min) se basent sur près de 700 relevés statistiques.

à votre niveau, il suffit de regarder le prix du jour dans le jeu, et de le comparer à ces max, moyen, min pour peser le pour et le contre entre vendre ou attendre.
