les mots de passe sont TOUJOURS présents sur un poste dans un dossier accessible :

- soit dans un fichier .pass généralement crypté
- soit dans un fichier .mail

c'est TOUJOURS un de ces deux cas. Les fichiers .pass pouvant être dans les dossiers système, ou dans les documents, et les mails pouvant être un peu n'importe où dans les documents.

les fichiers global password ainsi que les fichiers logs sont des BONUS, on peut toujours trouver le mot de passe d'un poste sans y recourir, juste en trouvant un fichier .pass ou bien le mail dans les dossiers accessibles du poste concerné.
