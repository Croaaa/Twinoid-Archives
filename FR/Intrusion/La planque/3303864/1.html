Les codes de carte bancaire peuvent se trouver dans les emails et les fichiers dos, il suffit de les consulter l'un après l'autre. Les messages concernant les cartes bancaires ressemblent à des bons de commande, il est facile de les distinguer des autres messages.
Quand tu en trouves un, soit tu copies le fichier, soit tu copies/colles le numéro de carte dans un bloc-notes. Dans les deux cas, une fois ta mission fini rends-toi dans Disque dur / Cartes bancaires volées pour valider ton code.
