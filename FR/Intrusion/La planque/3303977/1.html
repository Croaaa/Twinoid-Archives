Dans tous les postes qui comprennent des dossiers vérouillés par un mot de passe, ce mot de passe est TOUJOURS accessible d'une des deux manières suivantes:

1) un fichier "bloc note" se terminant par .pass est accessible (et peut être éventuellement crypté). Il se situe soit dans les dossiers système, soit dans les dossiers de fichiers personnels (souvent dans le sous dossier qui contient des documents textes)

2) sinon, c'est que le mot de passe est forcement dans un e mail
l'e mail peut se trouver dans les dossiers de fichiers personnels, très souvent dans les dossiers mails et archives mails, mais parfois dans d'autres dossiers (sauf celui contenant les programmes installés)


dans TOUTES les missions, vous pouvez accéder au mot de passe par une des méthodes ci dessus

cependant, dans certains cas, on peut obtenir le mot de passe plus rapidement en :
1) lisant le bas des fichiers .log d'un poste (si le mot de passe n'est pas indiqué dans le premier fichier .log d'un poste que vous trouvez, ne perdez pas votre temps ! il ne sera pas non plus indiqué dans les autres du même poste)
2) trouvant un fichier "global password" qui contient les mots de passe de tous les postes du réseau: toujours trouvable dans les bases de données, et parfois dans les imprimantes réseaux ou d'autres postes dans des cas très rares...
