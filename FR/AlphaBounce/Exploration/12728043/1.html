Cela a pas mal de conséquences.


En effet, tous les liens qui étaient disponibles dans les topics d'aide sont maintenant obsolètes. Il faut donc aller chercher chaque topic pour trouver son nouveau lien.
Cependant, il n'est pas possible d'utiliser la fonction Recherche pour retrouver ces topics, car seuls les topics créés après le passage des forums sur Twinoid sont trouvables via la recherche.
Il faut donc chercher les topics en se tapant les pages une par une ... Ce qui est assez fastidieux
C'est pourquoi j'en appelle à vous : si vous avez le courage de chercher certains anciens topics utiles, merci de poster le lien ici


Autre problème, le topic Sites & fichiers d'aide a perdu la quasi-totalité de ses messages ... Nous avons contacté les administrateurs pour savoir s'il était possible de les récupérer, mais si ce n'est pas possible il faudra refaire les messages nous-mêmes


Concernant les topics à retrouver, la plupart se trouvent dans ce topic.
Vous pouvez donc taper le nom du topic dans la recherche de votre navigateur (Ctrl+F) pour accélérer la recherche. Ca évite de devoir lire les noms de topic un par un.
