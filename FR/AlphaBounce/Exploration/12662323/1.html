Bon voila j'ai eu l'idée de créer ce topic un peu farfelu qui va servir à determiner le pourcentage de fréquence de chacun des bonus (sauf le U evidemment )

voici donc comment ça fonctionne

lorsque vous jouez un niveau poster ici :

toutes les capsules du niveau
les coordonnées du niveau

le pourcentage sera calculé grace a la formule suivante

X = (a : b) x 1000

a = Nombre de capsules "lettre" trouvées au total
b = Total des capsules trouvées
X = Pourcentage


Faisons un exemple

imaginons qu'on ait trouvé le "R" 24 fois et qu'on ait trouvé 1032 capsules au total ça donnera donc :

X = (24 : 1032) x 1000
X = 0,0232558 x 1000
X = 23,2558

Ca voudra dire qu'il y aura 23,256°% de chance que lorsqu'on trouve une capsule se sera un bonus "R"

Sachant que le poucentage sera arrondi au milliéme prés et exprimer en pour mille et non en pour cent afin d'obtenir plus de précision

ensuite tous les pourcentages des toutes les capsules sera réprésenté sous forme d'une barre. celle-ci sera visible dans le fichier que je créerai pour ce topic il y aura aussi les coordonnées des niveaux que vous avez postés c'est d'ailleurs pour ça que je vous demande les coordonnées du niveau que vous poster. plus simplement vous avez le droit de renseigner uniquement les niveaux donc les coordonnées ne sont pas dans le fichier car les coordonnées des niveaux situées dans le fichier sont des coordonnées deja testés donc deja renseignés sur ce topic

Malheuresement comme le symbole pour mille ne peut pas etre taper au clavier alors le pourcentage sera ecrit de cette maniére: X/1000 donc pour notre exemple ca donne : 23,256/1000

Eviter tout de meme de refaire de faire un post pour chaque niveau essayer de mettre plusieurs niveaux dans un seul post ce qui est pratique pour les EScorpiens qu'en ils font des long nettoyages cette régle est appliquée afin de ne pas surcharger le topic de messages
