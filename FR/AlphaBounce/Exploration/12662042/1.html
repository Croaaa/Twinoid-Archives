Bonjour

Le détenu Bab6 a fait un outil qui vous permettra de prendre des screenshots (captures d'écran) du jeu très facilement! L'outil ne prend que l'aire de jeu: 400x360 pixels.

Son nom : ABScreenMaker. Il est en téléchargement libre via le lien précédent, on en trouvera un aperçu ici. La dernière version en date est la v1.4.8 .

Un des buts d'ABScreenMaker est d'être entièrement compatible avec la MineMap.

Avant de pouvoir l'utiliser, assurez-vous d'avoir le .NET Framework 2.0 (ou plus), disponible ici. Cet élément est inclus d'origine dans Vista et 7.

Une fois que le .NET est installé, ABScreenMaker ne nécessite aucune installation.

Ce programme est compatible avec au moins:
- XP
- Vista
- 7
Sur les navigateurs:
- Internet Explorer 7
- Firefox 2 + 3 + 4
- Opera 9

Utilisation:
1. Téléchargez/installez le .NET framework si besoin.
2. Téléchargez ABScreenMaker.
3. Double-cliquez pour le lancer.
4. Allez dans "Paramètres", choisissez le dossier où iront les captures.
5. Quand vous jouez à Alphabounce, avec ABScreenMaker ouvert, faites Alt+Espace sur le clavier pour prendre la capture. Si le jeu semble se bloquer pour les contrôles clavier, essayez de rappuyer sur Alt. Il est préférable de prendre le screen dès le chargement du niveau pour voir toutes les briques intactes sur le screen.
6. Une fois de retour dans ABScreenMaker, entrez les coordonnées du niveau.
7. Si besoin, vous pouvez cocher la case "Bonus Provisions" qui agira comme un rappel.
8. Appuyez sur "Enregistrer" pour sauver l'image.

Le screen sera sauvé en tant que [X][Y].png dans le répertoire indiqué. En cochant la case du bonus, il prendra le nom [X][Y]P.png

Merci à Bab6 pour l'outil
