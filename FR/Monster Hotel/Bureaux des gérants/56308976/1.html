Après une petite recherche, j'ai trouvé la solution que voici :

Nouveauté de la version 49.0.2 de Firefox :

Asynchronous rendering of the Flash plugins is now enabled by default. This should improve performance and reduce crashes for sites that use the Flash plugin. ( Bug 1307108 )

J'ai donc désactivé le rendu asynchrone du plug-in Flash et cela fonctionne.

- Taper dans la barre d'adresse :
about:config
Cliquer sur "Je prends le risque"
- Rechercher :
dom.ipc.plugins.asyncdrawing.enabled
- Double cliquer dessus le résultat pour passer la valeur a "false"
C'est tout

Les jeux flash refonctionneront comme avant cette mise à jour de Firefox.
