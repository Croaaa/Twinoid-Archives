Non. Chrome/Chromium/Opera sont basés sur le même moteur de rendu (Webkit), mais pas Firefox (Gecko). Adobe a donc décidé de ne mettre à jour le plugin Flash que pour les premiers, et le plugin n'est de base pas utilisable par Firefox. Ce plugin est intégré aux navigateurs.

Ce que freshplayerplugin fait, c'est qu'il va extraire ce plugin du navigateur que tu auras installé. C'est pour ça qu'il faut en installer un, sinon il ne peut pas l'extraire (logique ). Il va ensuite l'adapter à Firefox pour que ce dernier puisse l'utiliser. Il n'y a pas d'installation détournée de Chrome dans Firefox, pas d'écrasement de quoi que ce soit, pas de panique : Firefox restera intègre, et Chrome boudera seul dans son coin sans rien faire.

Rockfaller Journey nécessite le lecteur Flash en version 12 ou supérieur, parce qu'il utilise des fonctionnalités disponibles seulement à partir de cette version. Grâce à cela, la carte graphique peut être utilisée sur la plupart des ordinateurs. Ce sera également le cas de nos prochains jeux qui disposeront d'une version Flash.

Sous Linux, comme ça a été indiqué, il faut donc utiliser Chrome (ou Chromium ou tout autre solution qui permet d'utiliser un lecteur flash récent).
Et quoi qu'il arrive on ne profite malheureusement pas de l'accélération matérielle avec le player Flash sous Linux, mais le jeu tourne plutôt bien malgré tout sur le CPU sur un ordinateur récent.
