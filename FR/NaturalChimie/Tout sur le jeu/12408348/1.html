Un peu plus d'infos sur la modification du classement général :

Vous l'avez vu (ou pas), le classement des meilleures alchimies dispose d'un nouveau filtre avec un bouton "quêtes".

Ce qui signifie que chaque joueur a désormais 2 "meilleurs scores" : un réalisé sans quête, l'autre réalisé pendant une quête qui modifie la partie.
Sachant que si vous avez une quête en cours mais que votre partie n'est pas modifiée, notamment si vous n'êtes pas sur le lieu demandé => pas de problème, votre socre pourra compter pour le classement normal.

Pour mettre cela en place, nous avons dû décider que tous les scores réalisés précédemment rentraient dans le classement "quête". Tout simplement parce que pour les anciens scores, nous n'avions pas l'information qui permet de savoir comment il a été réalisé

Cette solution est mise en place pour essayer de satisfaire le maximum d'alchimistes, j'espère qu'elle vous conviendra
