To be able to make the forbidden recipes, there are two solutions:

1- If it's a school recipe, it will be forbidden only if you're not a student of this school, but I don't suggest changing schools in this case, there's a much easier solution (see point 2 )

2- To be able to make a forbidden recipe and keep it, you have to get rid of the cauldron guardian. The only way to do so is by using one of the three following recipes: Inc Umbra, Rotavirus and Arcus Minor (Arcus Minor is a school recipe, the two other ones are Guilde recipes and therefore acessible to everyone). All three recipes belong to the "Inutilité publique" section.

After the recipe is correctly done, the guardian will be gone. Not forever, but still long enough for you to make a few forbidden recipes

Hope this help, and good luck

Vifargent
