Bonjour

Suite aux débats et démonstrations dans la marmite à blabla, je vous présente nos résultats :

Pour un seuil S de points nécessaires pour atteindre un niveau de réputation, il faut atteindre un niveau minimum N de point dans cette réputation pour ne pas redescendre de réputation après l'application d'un coefficient de perte P.
N est fonction de S et P tel que : N = S*P/(1-P)
Le coefficient de seuil C est donc C = P/(1-P)
Au 12/07/2011 :
P = 15% = 3/20
C = 15%/85% = (3/20)/(17/20) = 3/17

Les seuils des niveaux de réputations :
Sn = 60 = 60
Sbd = Sn + 470 = 60 + 470 = 530
Sosr = Sbd + 1340 = 530 + 1340 = 1870
Sabo = Sosr + 4200 = 1870 + 4200 = 6070
Schou = Sabo + 12600 = 6070 + 12600 = 18670

Les niveaux minimum à atteindre par réputations :
Nmbd = Sn * C = 60 * 3/17 = 10.59
Nmosr = Sbd * C = 530 * 3/17 = 93.53
Nmabo = Sosr * C = 1870 * 3/17 = 330
Nchou = Sabo * C = 6070 * 3/17 = 1071.18
