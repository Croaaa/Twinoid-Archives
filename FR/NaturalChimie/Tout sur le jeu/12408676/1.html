En attendant d'avoir une réponse du support, j'ai trouvé un petit hack qui permet de quand même afficher la quête qu'on veut sur l'écran des quêtes.

Pour celà, 3 étapes :
- Afficher la source de la page la (Ctrl+u ou Clic droit + afficher la source. Ça peut varier en fonction des navigateurs)
- Rechercher dans la source le titre de la quête, puis la chaine suivante : var content = document.getElementById (Crtl+f et coller le texte), le nombre qui suit immédiatement le "infos_" correspond à l'ID de la quête recherchée (dont la description se trouve juste au dessus).
- aller sur ce lien et remplacer XXXXXX par l'ID de votre quête.

J'espère avoir été assez clair, et que ça vous permettra d'avancer.

Réponse du support MT à utiliser si vous avez toujours les bugs

Bonjour,

Un fichier javascript corrompu du jeu a été téléchargé par votre ordinateur. Celui a remplacé par erreur le bon fichier pendant quelques heures sur nos serveurs. Si vous n'avez pas de cache sur votre navigateur ou si votre navigateur a eu besoin de recharger ce fichier, il a récupéré ce "mauvais" fichier, ce qui a généré cette erreur sur votre ordinateur.

Nous avons forcé le rechargement du fichier javascript correct pour tout le monde. Si cela ne fonctionne toujours pas de votre côté, vider le cache de votre navigateur corrigera le problème (procédure pour vider le cache : http://support.motion-twin.com/go/cache

).

Veuillez nous excuser pour ce contretemps.
