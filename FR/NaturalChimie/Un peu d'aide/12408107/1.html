Bonjour,

Un fichier javascript corrompu du jeu a été téléchargé par votre ordinateur. Celui a remplacé par erreur le bon fichier pendant quelques heures sur nos serveurs. Si vous n'avez pas de cache sur votre navigateur ou si votre navigateur a eu besoin de recharger ce fichier, il a récupéré ce "mauvais" fichier, ce qui a généré cette erreur sur votre ordinateur.

Nous avons forcé le rechargement du fichier javascript correct pour tout le monde. Si cela ne fonctionne toujours pas de votre côté, vider le cache de votre navigateur corrigera le problème (procédure pour vider le cache : http://support.motion-twin.com/go/cache ).

Veuillez nous excuser pour ce contretemps.
