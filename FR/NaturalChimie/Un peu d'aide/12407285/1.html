To get the triangles (we call them "pyrams"), you can:

- buy them with real € (get to the "banque" link)
- use a special recipe for that ("Calamus").
- if you're lucky, you can find them in a random box ("caisse surprise"): you buy those boxes (with )in shops around the world

Also, with the whistle, you can buy some stamps ("timbres") and send private messages, if you're stuck with some french instructions!
