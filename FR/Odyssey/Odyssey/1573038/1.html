Le ralentissement peut venir de Flash Player, qui à la longue peut faire ramer l'ordinateur. Quand cela se produit, plutôt que de relancer le navigateur, il suffit de faire planter le programme qui gère les plugs-in sous Firefox:

- Ouvrir le gestionnaire de tâches (CTRL+ALT+SUPPR)
- Ouvrir l'onglet "Processus"
- Rechercher le processus plugin-container.exe (si plusieurs portent ce nom, sélectionner celui qui utilise le plus de RAM)
- Cliquer sur "Arrêter le processus" et confirmer
- Réactualiser la page pour relancer le plug-in Flash Player.
