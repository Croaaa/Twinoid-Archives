Bien le bonjour ,cher visiteur ,bienvenue au C.R.V.I.P ,centre de repérage de VIP. Notre travail est simple : nous recherchons nous même les VIP dans les hotels ,ou ,si le directeur d'un hotel est gentil ,il noous envoie le nom de son VIP et ses caractérisstiques. A partir de ces informations ,notre agent gras-du-bide entre les donnés dans l'ordinateur ,établie une fiche d'identité ,qui se présente comme sa :

Nomd du VIP : (Le nom que vous voyer au dessus de sa tête ,par exemple ,Capitaze)
Effet ayant un impacte positif sur lui : (ce qu'il aime ,par exemple ,les odeurs)
Effet ayant un impacte négatif sur lui : (ce qu'il déteste ,par exemple ,l'humidité)
Histoire : (petit rp visant à montrer comment le VIP est devenu célèbre ,ses nombreux passsages dans les hotels ,...)

Si certaines de ces informations venait à manquer ,l'agent note quand même les autres informations.

Après cette petite introduction rp ,vous avez compris que vous devez poster une fiche par VIP ,la fiche du dessus ,pour montrer votre vip ,mais si une lde ces informations manquait ,le plus courant étant l'histoire ,car beaucoup de joueur ne pratique pas le rp ,s'est pas grave ,vous pouvez quand même marquez les autres informations.Bon ,je vous laisse commencer ,car je dois finir l'histoire de capitaze ,ce saleté de VIP mécontent !
