Essaye ceci :
Tape cette adresse URL (dans le champ blanc en haut de ton logiciel) : http://hotel.muxxu.com/ mais ne clique pas sur cet hypertexte (lien souligné en violet) recopie le ou fait un copier/coller (Ctrl+C (copier) Ctrl+V (coller))
Si ça ne marche pas, vérifie que http://hotel/muxxu.com/ ne soit pas enregistré dans les sites interdits. Pour ce faire :
Clique sur Outils, Options Internet. Dans l'onglet Sécurité, clique sur Site Sensibles, puis sur sites. Dans la fen*etre qui apparaît, vérifie si http://hotel.muxxu.com/ n'est pas inscrit dans le deuxième champ blanc (le grand) Si oui, toujours dans le même champ clique une fois sur http://hotel/muxxu.com/ (il s'affiche en bleu ) puis sur Supprimer. Ensuite essaye d'accéder à Croquemotel. Si apprès cette opération cela ne marche toujours pas, ou que http://hotel.muxxu.com/ n'est pas inscrit dans la liste des sites non autorisés, essaye avec un autre navigateur web (tel que Mozzila Firefox ou Google Chrome). Si cela ne marche toujours pas, contacte le support Twinoid.

(Chemin pour supprimer les sites non autorisés : Outils/options internet/sécurité/sites sensibles/sites/supprimer/ )
Ps : La prochaine fois, pose ta question sur le Questions/Réponses situé en haut du forum de Croquemotel.
