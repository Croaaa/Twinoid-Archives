Le 25 novembre 2010
ElGuigo
Vous voulez savoir où vous mettez les pieds et ne pas tomber sur de mauvaises suprises, FeverScreen est fait pour vous !

Cette application permet d'uploader les screens des iles, voir les chemins possible et le nombre de monstres présents

Pour poster vos screens, il faut ajouter le compte FeverScreen à vos amis et vous rentre sur la page suivante : http://muxxu.com/a/feverscreen.

Si vous n'avez pas FeverScreen dans vos amis, vous pouvez consulter la carte ici : FeverScreen

PS : J'accepte les demandes des joueurs ayant plus de 500 points.
