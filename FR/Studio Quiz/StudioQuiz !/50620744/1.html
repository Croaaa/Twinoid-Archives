Les joueurs composant cette poule sont :

Quelques rappels sur les phases de poule :

- Les parties devront se jouer dans la version "classique" du salon Mégaquiz ;

- Chacun des participants ci-dessus doit affronter une fois tous les autres (soit 4 parties par joueur) ;

- Pour les horaires des rencontres, pensez à utiliser des sites comme horlogeparlante, afin de bien vous synchroniser pour arriver dans le salon ;
Remarque : il est préférable d'utiliser la Messagerie Privée pour fixer l'horaire exact, afin d'éviter la présence de "perturbateurs".

- À l'issue de chacune de ces rencontres, n'oubliez pas d'indiquer les résultats (score y compris) sur ce topic (avec si possible une capture d'écran) ;

- Les délais pour jouer les matchs sont précisés ci-dessous.


Rappelons pour finir les enjeux de ces phases de poule (pour ceux qui auraient oublié ) :

À l'issue de ces rencontres, les deux joueurs victorieux de la poule seront sélectionnés pour les huitièmes de finale !
cf. le post de Marfygne sur le topic principal du tournoi pour le système de classement retenu.


Délais maximum pour jouer les matchs :

1 match avant le jeudi ​3 décembre au soir
2 matchs avant le dimanche ​6 décembre au soir
3 matchs avant le mercredi ​9 décembre au soir
4 matchs avant le samedi ​12 décembre au soir

Je rappelle qu'il s'agit d'un délai maximum, il est donc préférable de jouer ses matchs dès que possible.


Pour conclure, je vous souhaite à toutes et à tous un agréable tournoi, et que le meilleur gagne

Miltiade n'est pas très loin derrière, suivi de Rock4ever un peu distancée : ça m'arrange. Le principal est bien sûr de gagner définitivement ma qualification pour les phases finales. Le match se poursuit, et une nouvelle question sans réponse de ma part.
