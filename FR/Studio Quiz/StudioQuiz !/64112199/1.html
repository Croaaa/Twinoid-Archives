Bonjour à tous,

Afin de palier aux nombreux bugs de "Studio Quiz" n'ayant jamais été corrigé et plaçant l'expérience de jeu proche de 0.

Je propose une alternative à "Studio Quiz" que j'ai développé de A à Z en reprenant ce qui fait l'essence du jeu d'origine car quand je ne trouve pas un logiciel qui me convient, je le développe moi-même.

Je propose donc si vous le désirez de vous inscrire sur : Direct Quiz

Pour l'inscription, c'est simple, vous choisissez juste un pseudo, et une adresse e-mail.
Le prénom et le nom étant facultatif à l'inscription.

Ensuite il vous suffira de vous rendre sur l'onglet "jouer une partie" pour rejoindre le salon de quiz.
Pour le moment, il y a un seul salon de 8 places maximum, j'envisage de créer d'autres salons en fonction de l'évolution.

En espérant vous voir nombreux sur Direct Quiz pour bien s'amuser

Le code de la distance de Levenshtein :
Cette implémentation semble donner des résultats corrects:
function levenshtein(a, b){
if(!a || !b) return (a || b).length;
var m = [];
for(var i = 0; i <= b.length; i++){
m[i] = [i];
if(i === 0) continue;
for(var j = 0; j <= a.length; j++){
m[0][j] = j;
if(j === 0) continue;
m[i][j] = b.charAt(i - 1) == a.charAt(j - 1) ? m[i - 1][j - 1] : Math.min(
m[i-1][j-1] + 1,
m[i][j-1] + 1,
m[i-1][j] + 1
);
}
}
return m[b.length][a.length];
}
