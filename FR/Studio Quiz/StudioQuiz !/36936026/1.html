Salut à tous !

C'est parti pour la phase de poule du tournoi XIII de Studio Quiz !

Vous pouvez consulter les informations sur le tournoi dans son ensemble ici (encore merci à tous les participants !).


Voici donc le topic de la poule 8, pour ce tournoi se déroulant exclusivement dans le salon Mégaquiz !


Dans ce sujet, les participants de la poule pourront échanger, prévoir les matchs, commenter, etc.
Le topic restant bien sûr ouvert à d'éventuels spectateurs, supporters et autres pom-pom girls...


Mais passons sans plus attendre aux choses sérieuses, avec la liste des membres de la poule, fraîchement tirés au sort !

Les 6 joueurs de cette 8e poule sont donc :

- 2bruych
- bubblegum
- Julien94
- Teebeau
- tekpafman
- Xlilly


Quelques rappels sur les phases de poule :

- Les parties devront se jouer en salon Mégaquiz.

- Chacun des participants ci-dessus doit affronter une fois tous les autres (soit 5 parties par joueur).

- Pour les horaires des rencontres, pensez à utiliser des sites comme horlogeparlante, afin de bien vous synchroniser pour arriver dans le salon.
Remarque : il est préférable d'utiliser la Messagerie Privée pour fixer l'horaire exact, afin d'éviter la présence de "perturbateurs".

- À l'issue de chacune de ces rencontres, n'oubliez pas d'indiquer les résultats (score y compris) sur ce topic (avec éventuellement une capture d'écran).

- Un délai d'un match minimum tous les 2/3 jours serait l'idéal, afin de ne pas dépasser deux semaines de phase de poule.


Rappelons pour finir les enjeux de ces phases de poule (pour ceux qui auraient oublié ) :

À l'issue de ces rencontres, les deux joueurs les mieux classés de la poule seront sélectionnés pour les huitièmes de finale !


Bon jeu à tous !
