Quelques rappels sur les phases de poule :

- Les parties devront se jouer en salon Mégaquiz.

- Chacun des participants ci-dessus doit affronter une fois tous les autres (soit 5 parties par joueur).

- Pour les horaires des rencontres, pensez à utiliser des sites comme horlogeparlante, afin de bien vous synchroniser pour arriver dans le salon.
Remarque : il est préférable d'utiliser la Messagerie Privée pour fixer l'horaire exact, afin d'éviter la présence de "perturbateurs".

- À l'issue de chacune de ces rencontres, n'oubliez pas d'indiquer les résultats (score y compris) sur ce topic (avec si possible une capture d'écran).

- Un délai d'un match minimum tous les 2/3 jours serait l'idéal, afin de ne pas dépasser deux semaines de phase de poule.


Rappelons pour finir les enjeux de ces phases de poule (pour ceux qui auraient oublié ) :

À l'issue de ces rencontres, les deux joueurs victorieux de la poule seront sélectionnés pour les huitièmes de finale !
cf. topic principal du tournoi pour le système de classement retenu.
