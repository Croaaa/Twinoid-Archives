Quelques rappels sur les phases de poule :

- Les parties devront se jouer en salon Mégaquiz.

- Chacun des participants ci-dessus doit affronter une fois tous les autres (soit 5 parties par joueur).

- À l'issue de chacune de ces rencontres, n'oubliez pas d'indiquer les résultats (score y compris) sur ce topic** (avec si possible une capture d'écran, sait-on jamais).

- Les délais seront suffisamment larges pour que chaque joueur doive faire un match tous les 2 ou 3 jours contre un de ses adversaires à priori. Evidemment, vous avez parfaitement le droit de faire tous vos matchs à la suite ! Si vos adversaires sont présents, autant en profiter ("un tiens vaut mieux que deux tu l'auras" dit toujours ma grand-mère).


Une fois tous les matchs joués, les deux meilleurs de la poule seront sélectionnés pour disputer les phases suivantes (comprendre : pour se faire humilier par ayalti ).
Tout le système de points est expliqué dans le topic principal, mais la base est très simple : 3 points pour une victoire, 1 point pour un match nul, 0 pour une défaite.
