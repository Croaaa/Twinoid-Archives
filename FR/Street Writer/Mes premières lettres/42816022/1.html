Les points précalculés affichés sur le plateau sont parfois faux :
En gras les lettres déjà sur le plateau. 1er exemple
MAILLEE : 7 + 7² = 56pt
+NE : 2 + 1² = 3pt
+10pt cachés
= 69pt, le compte est bon.

Ajout d'un N
+NEN : 3 + 1²= 4pt (mot imaginaire)
= 73pt, mais le jeu affiche un improbable 171.
2ème exemple
ETIRADE : 7 + 1² = 8pt (mot imaginaire)
Mais le jeu n'affiche que 3.
Le point commun : une lettre placée entre 2 existantes.
