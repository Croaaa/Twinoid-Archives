Bonjour, je propose ce topic pour tous ceux qui utilisent un smartphone ou une tablette.
Un topic explique une astuce comment jouer aux jeux en Flash Player.

Astuce (iOS : iPhone, iPad et iPod)

Il faut malheureusement acheter un des deux applications que je connais...

Photon Flash Web Browser for iPhone (3.59€)
Photon Flash Web Browser for iPad (4.49€)

Ces deux applications utilisent en streaming et ce sont les seuls meilleurs applications à mon goût sur la performance et la fluidité.

Astuce (Android)

Il y a plusieurs moyens de jouer sur Android et certaines machines ne sont pas compatibles avec Flash Player.

Déconseillé
Ne pas télécharger le SDK Flash Player sur n'importe quel site non-confiance. Vous risquerez d'endommager une machine. Comme si vous installer un logiciel malveillant !

Conseillé
Vérifiez sur le site Adobe pour télécharger le Flash Player via le navigateur web d'une machine ou vérifiez dans le paramètre d'une machine si il existe une option pour activer le Flash Player.

Si le Flash Player n'est pas compatible avec la machine ...

Des applications existent avec le Flash Player sur Google Play Store.

Bonjour

Tu es sur quel OS ? Modèle de smartphone ?
As-tu essayer de paramètrer dans Photon ? (En cliquant sur Diagnostiquer)

Mon paramètre pour iPhone 6 Plus :
- Vitesse : 6
- Mode : Jeu
- Port : 1-2-4 (Port 3 : Echec)
- Moteur Photon : V4
- Accès Local : OFF

Le diagnostic sert à vérifier le résultat de chaque Port et voir si un des ports est passer ou échec.

Et de bien paramètrer le fuseau horaire dont Guadeloupe est : GMT -4
Pour France : GMT +1
