Bon, temps pis ! En tout cas je le ré-ouvre ! Vous l'avez deviné, c'est un tuto sur les lignes de codes de Dino-RPG.

ex: simulation de titre ou enregistrement de map

Pour faire une simulation de titre, allez sur votre page de Maître faite clique droit code source de la page.

Prenez un ligne de code vers le milieu où il y a écrit : "http://data.dinorpg.com/swf/titles.swf? v=0".

Ensuite, prenez la ligne de code qui ressemble a ça : "title=Maître&amp;sub=votrepseudo". Enlevez le "amp;" : devant "title", mot en vert, devant "sub", mot en brun.

Voilà !!
