Un collège Ubuntu

Moi aussi je suis dessus

1. Telecharge tes polices sur un site du genre Dafont.com, une fenêtre apparait normalement faire 'enregistrer le fichier'

2. Une fois ta police telechargée, va dans le dossier ou elle est contenue. Le fichier devrait normalement être enregistré en en ".zip" (sauf dans certainc cas ou c'est directement en ".ttf" dans ce cas là tu oublis cette étape). Pour le dézipper c'est simple, clique droit sur le fichier puis "extraire ici", un nouveau fichier devrait apparaitre. Comme ceci

3. Il va maintenant falloir placer la police dans le dossier 'font' de ton ordinateur fait "Ctrl+l" pour afficher la barre de frappe puis inscrit "/usr/share/fonts" appuis sur entrée. Copie ta police (clique droit / "Copier" ) puis colle là dans le dossier ouvert auparavant. Il est préférable de crée un sous dossier pour les polices ".ttf"

Voilà, en espérant avoir était clair :/
