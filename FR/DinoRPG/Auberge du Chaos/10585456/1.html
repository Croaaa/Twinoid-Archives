Facile, suffit de suivre la même méthode pendant un combat

Vous recherchez la ligne qui ressemble à cela :

http://data.dinorpg.com/swf/fight.swf?v= 13 ","fight"...

Juste après v=13 vous mettez un & et vous chercher un code qui se trouve sur la ligne dans dessous (en faite, 2 ligne en dessous, mais vous n'allez pas chipotez!! )
qui ressemble à cela :

so.addParam("FlashVars","data=oy8%3A_mbo ttomi20...

Vous copiez à partir de "data=..." (sans les guillemets bien sur) jusqu'à la fin, le code est très long (astuce : utiliser shift pour ne pas vous y perdre).

Et voilà, vous pouvez admirer un fabuleux combat!!!

Dans le même style, faite pareille pour les dinozs se trouvant dans la boutique!!

Recherchez la ligne qui ressemble à cela :

http://data.dinorpg.com/swf/dino.swf?v=8 ","sheet2",190,165,8,"#fce3bb

De la même façon que pour les autres animations flash, retirez la partie après le v=8 et ajouter un & à la place...
Puis rajouter la ligne de code ressemblant à ça :

flip=1&data=30A5bmlzUfOR7kbQ&chk =43813

à la suite de votre lien et vous aurez une plus grande image.

Note : si vous enlevez le code "flip=1&" l'image sera inverser, (flip = retourner).

Bon, sachez que ce petit jeu marche pour énormément de chose, à essayer sur toutes les animations flash, de n'importe quel site motion twin (y'a des chances que sa marche) et voir sur d'autre site (mais y'a peu de chance que ça marche!!)
