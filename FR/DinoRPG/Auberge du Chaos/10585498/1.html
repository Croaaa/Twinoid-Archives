Les Bases pour créer un site sont le Html ET le Css.

Html pour écrire le texte de ton site.
Css pour le design de ton site.

Après je te conseille d'essayer d'apprendre le Php et savoir te servir de Mysql. Ainsi tu pourras faire beaucoup plus de choses.

Si il fallait faire des comparaisons entre ces langages et un humain, ça serait :

Html => Notre Identité (Prénom, nom, ... ect)
Css => Notre couleur de cheveux, peau, taille ...
Mysql => Cerveau
Php => Yeux, bouche ...

Donc je résumé, l'html c'est le texte. Le Css pour le design, le Mysql qui grâce au Php peut enregistrer des informations sur le visiteurs.

Puis si tu as envie le Javascript qui permet une interaction entre tes visiteurs et ton site. Mais bon, on peut s'en passer.
