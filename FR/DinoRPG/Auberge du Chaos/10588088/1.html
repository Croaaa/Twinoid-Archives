Il y a plusieurs moyens de jouer sur Android et certains machines ne sont pas compatibles avec Flash Player.

:tid_exclaim: **Déconseillé** :tid_exclaim:
Ne pas télécharger le SDK Flash Player sur n'importe quel site non-confiance. Vous risquerez d'endommager une machine. Comme si vous installer un logiciel malveillant !

:tid_idea: **Conseillé** :tid_idea:
Vérifiez sur le site Adobe pour télécharger le Flash Player via le navigateur web d'une machine ou vérifiez dans le paramètre d'une machine si il existe une option pour activer le Flash Player.

Si le Flash Player n'est pas là dans la machine ou n'est pas compatible...
