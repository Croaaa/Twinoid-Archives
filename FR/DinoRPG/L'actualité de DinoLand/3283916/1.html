il faut deplacer le client (depuis sa chambre ou la file d'attente) vers le labo

les autres clients de la file d attente ou placés dans les chambres perdent un point de satisfaction a ce moment la, tu peux construire 2 instituts pour "sacrifier" les clients par paires, l'abomination apporte 40% de progression de recherche et tu touches le point pour le labo quand un institut arrive a 100%

enfin une recherche tres avancee te permet d'obtenir (en gros a 5-10%) un objet de deco lors de "l'examen" d'un de tes clients dans cet institut.

pour finir, il vaut mieux placer ces questions dans le topic questions & reponses a l'avenir.
