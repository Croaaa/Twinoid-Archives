Leçon 1 (Durée de lecture, moins de 2 minutes)


Démêlons une fois pour toute les se/ce sa/ça ses/ces/s'est/c'est et autres s/c de ce genre.


La règle de base est simple: "s" est possessif et "c" démonstratif.


Exemples et explications:


se construire un hôtel -> on pourrait remplacer le "se" par "me" -> me construire un hôtel
ce client -> on pourrait remplacer par celui-là, ce client là
sa voisine -> on pourrait remplacer le "sa" par "ma" -> ma voisine
ça = cela -> Comment ça va? pourrait se lire (en exagérant un peu ) Comment cela va t-il?
ses clients -> on pourrait remplacer par mes clients
ces clients -> ceux-ci -> pas les siens (qui lui appartiennent) mais ceux que je présente, de qui je parle, etc
s'est -> il s'est fâché -> possessif + verbe être -> s'être fâcher
c'est -> démonstratif + verbe être -> celui-là est

Leçon 3 (Durée de lecture, moins de trente secondes)


Les verbes en er. Comment démêler les er/é/ez. La solution est très simple: On n'a qu'à remplacer le verbe par battre ou mordre.


battre = er ; battu = é ; battez = ez ;


Exemples:


Il a donné (battu) un client à son binôme.
Donnez (mordez) un client à votre binôme.
Le rôle du binôme est de donner (battre) un client.

Leçon 4 (donnée par Anana123)


Les est/et/ai/aie


1) est/es tiennent pour le verbe être. respectivement conjugué à la 3ème et la 2ème personne du singulier.
par exemple : tu es beau. ton hôtel est joli.
pour ne pas les confondre avec les suivants vous pouvez remplacer par semble :
tu sembles beau. ton hôtel semble joli.


2) et : on le met pour ajouter quelquechose :
par exemple mon client aime le voisinage et le bruit.
pour ne pas le confondre on peut remplacer par mais aussi
mon client aime le voisinage mais aussi le bruit.


3) ai : verbe avoir conjugué au présent simple
j'ai faim. pour le reconnaître on peut le remplacer par avais.


4) aie : qui est aussi le verbe avoir, mais au subjonctif. Autrement dit il y a toujours que devant :
il est possible que j'aie perdu.
pour le reconnaitre on remplace le verbe avoir par le verbe faire :
il est possible que je fasse et pas il est possible que je fais

Leçon 5: Accord du participe passé: première partie

Utilisé avec l’auxiliaire être, le participe passé s'accorde en genre et en nombre avec le sujet du verbe:

Julien est allé au cinéma.
Julie est allée au cinéma.
Ils sont allés au cinéma.
Elles sont allées au cinéma.


Deuxième partie:
Utilisé avec l'auxiliaire avoir, le participe passé s'accorde avec le C.O.D. si le C.O.D. est placé avant l'auxiliaire.


J'ai acheté des carottes => J'ai acheté quoi? des carottes= le C.O.D. "des carottes" est placé après l'auxiliaire, donc on n'accorde pas.


Les carottes que j'ai achetées étaient bonnes. Le C.O.D. "des carottes" est placé avant l'auxiliaire, donc on accorde le participe passé en genre et en nombre avec le C.O.D.



P.S. J'ai utilisé ici le passé composé, pour plus de lisibilité, mais cette règle fonctionne aussi, bien entendu, avec les autres temps composés de l'indicatif: le plus que parfait, le passé antérieur et le futur antérieur.
