MISE AU JOUR 2.1 DE L'ANIMATION
- Nouvel objet : les jetons ()
Les jetons peuvent s'obtenir en trouvant un code CAT (Chasse Aux Trésors).
Les jetons servent à :
- Acheter des points
- Acheter des clients
- Acheter un bonus pour jouer à l'animation
- Passer un niveau
Tarifs :
- Achat de points : 1 jeton par points
- Achats de clients : 50 jetons par clients
- Achat d'un bonus : prix variés (voir ci-dessous)
- Passer un niveau : 1 000 jetons par niveaux
- Indice pour le joueur (privé) : 50 jetons par indices
- Indice public (pour tout le monde) : 35 jetons par indices
Bonus :
- Double pelle : vous pouvez donner 2 cases par jour : 100 jetons /semaine
- Gains doublés : 75 jetons / semaine

  • Fonctionnnement :
    Lorsque vous trouvez un code CAT, envoyez-le par mp à Eolienne. Vous serez informés des jetons que vous avez gagné.
    Pour acheter : Envoyez votre achat à Eolienne par mp. Votre compte sera débité et vous recevrez votre achat. Pour savoir combien vous avez de jeton, envoyez un mp à Eolienne.
    Trouver un code CAT :
    -Sur forumactif.com
    -Sur mon nexus
    -Sur ce topic
    -Ect...
