gogoman2
* Allez dans la page que vous voulez capturer. Appuyez sur le bouton "IMPR ÉCRAN" de votre clavier (en haut à droite).
* Ouvrez le logiciel PAINT de votre ordinateur (généralement dans le menu Démarrer, sous Accessoires). Appuyez sur CTRL + V pour coller l'image.
* Sauvegardez l'image. Pour cela, cliquez sur FICHIER > ENREGISTRER. Tapez un nom et choissisez JPEG dans le type de fichier. Gardez l'image dans un endroit dont vous vous en souvendrez.

Plus de détails sur cette opération à cette adresse (site externe).

Mettre en ligne l'image

* Allez sur : http://imageshack.us
* Cliquez sur "BROWSE" ou "EXAMINER".
* Cherchez votre image et sélectionnez-la.
* Cliquez sur le bouton HOST IT! ou START UPLOAD!
* Copiez ensuite le url qui apparaitra dans un cadre. L'url aura plus ou moins le format suivant : "http://img40.imageshack.us/...nomdimage .jpg"

Tu le saurais si tu avais lu la FAQ.
