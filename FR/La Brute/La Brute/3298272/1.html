Les combats ne sont pas interactifs, le joueur les regarde sans diriger le personnage. Le jeu est basé sur le hasard et la chance, avec quelques choix pour le joueur : quelle brute combattre (parmi 8 proposées), le choix entre deux options quand on change de niveau, etc...

Merci d'utiliser le topic dédié, plutôt que d'en créer un.

Pour les questions/réponses, utilise plutôt le topic Questions/réponses 6 (il faut numéroter vos questions, par exemple Q36 si la précédente était n°35). Il est préférable de ne pas multiplier les topics, cela réduit la lisibilité du forum, donc le confort, les topics dédiés aident à regrouper les infos.
