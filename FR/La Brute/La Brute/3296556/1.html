Mais qu'est-ce que la Chaine Pirate?
C'est une chaine inconnue des autorités(ou pas...) qui diffusent des "émissions" montrant une dizaines de combats.

Qui c'est le PDG?
Scipion épaulé de DonDarkan et exceptionnellement de 6mxw9.

Comment La Chaine se fournit-elle?
Les PDG peuvent fournir leur combat, en trouver par hasard sur certains topics du forum ou sur des chans IRC mais surtout... sur les combats envoyés par les téléspectateurs!

Hein ? Comment ça, on peux envoyer des combats ?
Oui, à cette adresse. Il faut mettre le lien complet du match(avec le ?chk) pour que cela soit valide.

Sur quels critères les combats sont-ils sélectionnés ?
Cela dépend, y'a les combats qui font rire, d'autres qui sont bons via les capacités ou certains enchainements.

Où la Chaine diffuse t-elle?
D...euh nan, elle diffuse ici.

Combien de combats sont diffusés par émission?
Entre 10 et 20.

A quoi correspondent les catégories des combats?
Epic Fail : Un bon gros échec d'un truc qui aurait du faire mal.
Familiers : Combat dans lequel les familiers ont un role important.
Barbare: Quelles puissances dans les coups!
Suspens: Un beau retournement de situation ou un combat égal qui dure
Combos: Enchainements de capacités ou autres.
Capacités: Les Capacités sont importantes dans ces combats.
Rapide: Quelles vitesse d'exécutions!
Exploit: Quoi? Un petit level qui explose une brute plus haute que lui?
Divers: Le reste
Bonus Track: Combats "bonus", souvent avec les brutes des PDG de La Chaine Pirate

Vous pouvez commenter les émissions:
Vous avez aimé une émission? Signalez le en cliquant sur "Je veux commenter cette émission".

Les commentaires sont pré-modéré, c'est à dire qu'ils doivent être validé par les PDG. Tout commentaires écrits en sms sera refusé.

Combien d'émissions diffusés à ce jour?
14.

1) Télécharger GreaseMonkey:
Il faut avoir Firefox puis aller télécharger GreaseMonkey
https://addons.mozilla.org/fr/firefox/ad don/748/ (on m'excusera mais la balise lien ne fonctionne pas avec les https (pensez à supprimer l'espace de l'adresse).

2)Installer le Script Chaine Pirate:
/!\ Attention, les scripts GreaseMonkey peuvent être dangereux, n'installez que les scripts sur des sites dont vous avez entièrement confiance. Certains scripts peuvent vous voler des informations confidentielles sur vous(noms, codes de CB, password...).
Le script Chaine Pirate est fiable, et vous pouvez de toute facon visualiser le code source pour vérifier.
Installer le script Chaine Pirate
On notera que le script a été codé par Kill0u.

3)Utiliser le script:
Faites un combat, attendez la fin.
A coté du bouton "Continuer" il y a un bouton "Pirater". Cliquez dessus et le combat sera ajouté(un message de confirmation apparaitra).
Cliquez sur continuer et faites d'autres combats.
