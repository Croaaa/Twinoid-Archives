Il s'agit d'une récompense, dans chaque jeu de muxxu, il est possible d'en obtenir. Il y a 3 types de récompenses :
-Trophée (ou picto) : ce sont ces petits symboles
-Les titres
-Les suffixes.

Ces récompenses influent sur le nombre de point qui figure sur ton profil.

Tu peux utiliser les trophée que tu as gagné dans presque tous les espaces d'expression : forum, MP, message du profil, commentaires, groupes...

Ces trois types de récompenses te permettent également d'éditer un titre honorifique qui figurera sous ton avatar sous ce modèle :
Trophée + Titre + suffixe
Par exemple (le mien) :
+ Collecteur + sans accroc
Trophée caféjeux + Titre gagné dans intrusion + suffixe gagné dans labrute

Pour en savoir plus sur les récompenses de Muxxu :
