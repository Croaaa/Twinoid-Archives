Après la fonctionalité est dispo, tu peux choisir d'entrer un niveau de commerce ou une distance.

Dans la partie Données : Niveau de commerce de départ
Clic sur le bouton pour entrer un niveau de commerce
Clic sur le drapeau pour entrer une distance (le niveau de commerce initial est automatiquement calculé)

Le niveau initial, c'est le carré de la distance en négatif :

d = distance
c = d x d x -1

Ou résumé :

c = -d²
