Dans l'historique du monde, on voit parfois indiqué que tel ou tel seigneur est mort avec une icône de tête de mort.

Genre : Le Comte xxx est mort ! Trucville est désormais une ville Barbare...

Mais pour certains décès, il est indiqué :
Le Royaume du Seigneur xxxx s'est effondré : Trucville est désormais une ville Barbare.
précédé d'une icône représentant un château détruit (le même que lorsqu'une capitale est conquise et le royaume correspondant détruit.


Ma question est donc celle-ci : pourquoi cette différence de traitement entre les différents décès ?

Le 28 août 2013
Drange

Le premier est mort de mort naturelle ou par poison.
Le second est mort de famine suite a une période prolongée d'inactivité.
Le résultat est le même, ils sont morts tous les deux.

Le second type de mort est important pour éviter le blocage des cartes par des joueurs qui pour diverses raisons n'ont pas continué leur partie.
