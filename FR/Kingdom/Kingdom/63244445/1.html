Bonjour à tous,


Je vous présente l'extension KCL.


Fonctionnalités principales
- Affichage des seuils de nourriture et d'or.
- Gestion des titres féminins.
- Panthéon amélioré, avec de nouveaux classements et filtres.
- Intégration d'un simulateur de commerce.
- Ajout d'informations sur la page de gestion.
- Actions sur les généraux depuis la page de gestion.
- Amélioration de la page de transfert d'unités.
- Planification de déplacements point par point depuis la carte.
- Filtrages des historiques.
- Correction de petits bugs graphiques.


Vous trouverez le détail complet des améliorations depuis la page "Informations" qui apparaîtra dans la barre Kingdom.


Installation
Pour installer le script, vous devrez disposer de Tampermonkey dans votre navigateur (Chrome ou Firefox). Malheureusement, le script ne fonctionne qu'à moitié sur Greasemonkey.


Le script est disponible sur le lien suivant : https://openuserjs.org/scripts/Clanke...


Une fois installé, il vous suffit de charger une page Kingdom.

Cette possibilité a toujours existé. Le contrôleur du général est envoyé caché via une commande HTTP (tout comme le transfert de unités entre le général et le territoire). Ce n'est pas visible à l'extérieur. Mais vous pouvez simplement le construire. La commande est:


kingdom.muxxu.com/general/123456/move?p=step2:step3:step4;c=step1


123456 = Le ID de votre général (vous pouvez le voir dans le http)
c=step1 = Le premier territoire où votre général aller (ID de la zone)
p=step2:step3:step4....:stepXX : Une liste des domaines dans lesquels le général devrait ensuite se rendre (IDs de la zones)


mais finalement, il n’a qu’un avantage théorique. vous en avez à peine besoin efficacement.

J'ai essayé de tenir compte de vos commentaires :

  • Talsi : "Pour les titres au féminin, c'est un peu dommage que ça change pour soi mais pas pour les autres joueuses." Pas évident vu la diversité des changements et comme vous l'avez dit, cela pose un problème de performance/charge du serveur MT d'aller chercher l'info à chaque fois. J'ai donc opté pour une liste intra-extension et ai tenté d'adapter les différentes occurrences dans le jeu. N'hésitez pas à poster si vous trouvez des incohérences.
    Note importante: les titres sont adaptés pour les joueuses référencées dans l'extension. Pour compléter la liste, il faudra poster un message ici pour que je vous ajoute, ou poster les liens vers les pages de profil Kingdom à ajouter.
    Dernière note: la page de mort n'est pas adaptée pour le moment, car j'ai besoin de l'avoir moi-même pour travailler dessus.
  • Talsi : "Est-ce qu'un tri des lignes d'un panthéon suivant un critère serait envisageable ?" et remarque de simoons " j'imagine que ça demanderait de lire toutes les pages du Panthéon, donc pas vraiment imaginable en temps réel (mais sans doute mis à jour tous les mois ?) ". Une mise à jour mensuelle aurait impliqué une action périodique de ma part, ou un code automatique de chargement des pages (que n'aurait pas apprécié MT). J'essaie de garder l'extension auto-suffisante. Chaque joueur peut maintenant lui même enregistrer les scores du panthéon (en parcourant simplement les pages) et ensuite profiter du nouveau panthéon KCL, qui fait remonter pas mal d'info intéressantes (je sens que ça va vous plaire ).
  • simoons : "Ce qui pourrait etre interessant aussi, ca peut etre un tracking de stats pour enfin pouvoir evaluer l'influence de ceci ou cela sur la mort, la force de la faillite, la revolte, le descente de commerce dans les negatifs, etc." Je suis bien d'accord, mais ça m'a l'air d'un boulot tout à fait conséquent : enregistrement des différents types d'event (à travers ts les browsers du joueur?), récupération, centralisation dans un site tiers. A réfléchir. En tout cas, mon emploi du temps ne me permet pas de m'attaquer à cela dans les deux mois qui viennent.
  • Talsi : "Le seuil pour le blé, ce serait bien que tu le mettes aussi dans la page de la capitale." Les seuils ont été activés dans la page de la capitale.

Détail de la mise à jour :

  • Gestion/Onglet vassaux : collecter des taxes rouvrira cet onglet après le rechargement de la page.
  • Gestion : correction du bug affichant "NaN" dans les scores si le tableau de diplomatie est présent dans la page.
  • Capitale : coloration de l'icône du stock de nourriture en cas de situation d'abondance.
  • Capitale : coloration de l'icône du stock d'or en cas d'âge d'or.
  • Capitale : extension réappliquée lorsqu'un tour a été joué.
  • Transfert d'unités : cliquer sur l'icône d'un type d'unités les déplacera toutes de l'autre côté.
  • Adaptation des titres pour les joueuses (page de gestion, carte, classement, pantheon, ...).
  • Panthéon : affichage d'une pastille de couleur pour indiquer l'ancienneté du score.
  • Panthéon : mise en évidence du lien du joueur (en bleu) s'il s'agit du joueur actuel.
  • Panthéon : les pages peuvent être parcourues à l'aide des flèches gauche/droite du clavier.
  • Panthéon : parcourir une page du panthéon enregistrera les scores dans l'extension.
  • Panthéon KCL : les scores enregistrés seront affichés avec de nouvelles options de filtrage (année, âge) et de classement (gloire, commerce, armée, âge, ...). Étant donné que les données sont locales, parcourir ce panthéon est beaucoup plus rapide.

Rendez-vous dans la page "Informations" du plugin pour retrouver la liste complète des fonctionnalités.
