Voilà la solution ! J'avais glissé dans la première phrase une aide : les fautes d'orthographe : « d/c/o/c/e » en tapant ces lettres sur google on nous renvoie sur ce site : http://www.dcode.fr/ qui peut être une aide très précieuse pour la suite. Enfin bref on pouvait aussi faire sans et tout décrypter à la main .
Le premier texte n'est pas crypter mais est tiré d'un article wikipédia : http://fr.wikipedia.org/wiki/C%C3%A9s... Le texte wikipédia a été modifier pour vous faire voir les mots suivants : « césars des lettres », « code » et le chiffre 4. Après réflexion et/ou recherche on retombe sur un code très très célèbre : le code césar ( http://fr.wikipedia.org/wiki/Chiffrem... )qui consiste à décaler d'un certain nombre les lettres de l'alphabet. Ici le nombre était le chiffre 4. En utilisant cette méthode de décryptage on obtenait une traduction du second texte :


LE TEXTE SUIVANT EST CODE SUIVANT UN CRYPTAGE BIEN CONNU UTILISER AU COURS DES SIECLES. LA CLE DE DECHIFFRAGE DE CE TEXTE EST : "BONNECHANCE".


Après quelques recherches il était assez simple de comprendre que le cryptage utilisant une clé était le « chiffre de vignere » : http://fr.wikipedia.org/wiki/Chiffre_...

Il suffisait ensuite de décrypter le second texte avec la clé « BONNECHANCE ». Le texte nous donnait :


BOF, LA PREMIERE ENIGME ETAIT BIEN TROP FACILE... LE RESTE DU CODE T'ATTEND DE PIEDS FERMES ! (AH BON CA A DES PIEDS UN CODE ?!) : LE ONZE DECEMBRE DE L'ANNEE DE LA FIN DU MONDE, LE MAURE DE VENISE A POSTE UN MESSAGE SUR SON PROPRE NEXUS. IL SERA LE PHARE DE TES PAS.


Pour comprendre le message il fallait savoir qui était le maure de venise. Là, google est ton ami mais normalement tout le monde connait la pièce de théâtre de Shakespeare intitulée « Othello » ou le "Maure de Venise" . Ainsi il suffisait de chercher dans mon nexus le message en question :http://twinoid.com/ev/19213819

Après analyse du message il fallait trouver le lien sur le premier smile : http://textup.fr/36484ow.

L'albatros de Baudelaire est un poème très célèbre, il fallait l'utiliser comme clé du dernier texte (avec vignere bien sur) .
Après décryptage du dernier texte on obtient :


ARF, LA TU COMMENCES A M'IMPRESSIONNER. POUR LA DERNIERE EGNIME : CEST UN CHAMP DE BATAILLE SANS BATAILLE, UN ROI SANS COURONNE , DES SOLDATS SANS ARMES , UN MONDE EN NOIR ET BLANC, QUI SUIS JE ? ENVOIE MOI LA REPONSE PAR MP ET TU SERAS LEGAGNANT DE CETTE PETITE ANIMATION
