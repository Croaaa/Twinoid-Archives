Voici les deux cas présents lorsqu'un lieu est coupé de la capitale :

- commerce positif :
le commerce ne compte plus dans la balance commerciale mais le salaire des soldats compte quand meme

- commerce négatif :
le commerce compte toujours dans la balance commerciale de même que le salaire des soldats.

Il est donc préférable d'avoir un commerce positif sur tous les lieux, même ceux qui ne sont pas reliés.
Le must étant quand meme de garder tous les lieux reliés.
