Je vais vous présenter un script créer par Flagoul, un grand merci à lui et à tous ceux qui nous ont aider à faire ce script : Tyler_Durden, Sirios, Moloko-Plus.


Ce script appelé "Kingdom +" vous permet d'optimiser l'utilisation du simulateur créer par Smoggstudio. Ce script est génial, mais pour mieux vous faire comprendre de quoi il s'agit voilà quelques screens shots :
http://img11.hostingpics.net/pics/753...
http://img11.hostingpics.net/pics/760...
http://img11.hostingpics.net/pics/472...


Voilà le lien du script : https://dl.dropbox.com/u/43094257/kin...


Comment l'installer ? :
Sous chrome: normalement il va télécharger le script. Il faut enregistrer le script quelque part (dans téléchargements par exemple), ou regarder à quel endroit il l'a téléchargé s'il le fait automatiquement.
Ensuite, aller dans le menu de chrome en haut à droite, Outils -> Extensions. Ensuite tu arrive sur la fenêtre des plugins.
Il suffit de glisser-déposer le script sur cette fenêtre depuis l'endroit où tu as le script, et c'est bon
Sous firefox il suffit de télécharger Greasemonkey et de cliquer sur le lien.
