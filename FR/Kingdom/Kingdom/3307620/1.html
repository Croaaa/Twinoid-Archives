Pour tout problème technique, voici ce qui est indiqué dans l'aide du support :
Avant toute chose, essayez d'isoler les causes du problème :

* désactivez tout firewall ou logiciel de protection internet,
* désactivez l'extension Grease Monkey si vous l'avez installée sur Firefox (aucun support ne pourra vous être fourni si vous utilisez ce type d'extension),
* désactivez les logiciels destinés à bloquer la publicité.

Vérifiez que vous utilisez la dernière version de votre navigateur :

* Firefox : http://www.getfirefox.com
* Opera : http://www.opera.com
* Chrome : http://www.google.com/chrome
* Internet Explorer : http://www.microsoft.com/france/windows/ internet-explorer/

Vérifiez que TOUS vos composants (plugins) sont à jour dans votre navigateur et notamment le lecteur Flash :
http://www.mozilla.com/plugincheck/

Videz le cache de votre navigateur :
http://support.motion-twin.com/help?s=ca che

Si rien n'y fait, vous pouvez signaler votre problème directement dans la section ci-dessous :

Rapports de bugs

Donnez le maximum de détails sur le problème que vous rencontrez !
Donc une fois les vérifications d'usage faites, si tu as toujours le même problème, il faut le signaler au support, car eux seuls pourront t'aider
