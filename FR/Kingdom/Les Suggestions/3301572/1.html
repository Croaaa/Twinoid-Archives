Voici une idée qui m'est venue hier, dans un combat ou, sur 60 soldats, 40 étaient blessés et sont repartis se faire massacrer... L'idée que j'ai est donc de créer un "bâtiment portable" : le dispensaire. Il coûterait assez cher du fait de son utilité :

-50 lin pour la toile
-300 d'or pour acheter les médicaments (eh oui c'était cher ! )
-1 citoyen ? Il pourrait jouer le rôle de médecin
-150 de nourriture pour ce villageois

Le dispensaire rendrait un point de vie a toutes les unités blessées par quart d'heure.

Il existerait aussi une variante pour les machines : le chariot (pas trouvé de nom...)
Il couterait quand a lui :

-25 de fer pour les réparations diverses
-25 chevaux pour le faire bouger ( pas génial niveau RP mais il faut équiliber...)
-300 de bois pour les réparations diverses
-Encore un fois, un citoyens jouant au bricoleur
-150 de nourriture pour nourrir le citoyen

Il rendrait aux machines (balistes/catapultes) 1 point de vie par quart d'heure au même titre que son cousin le dispensaire

Note : Si le général meurt ou fuit, les "bâtiments portables" sont détruits

A vos avis !
