Bonjour,
Je vous présente une version d'un concours de Rap,d'R'N'B et de Hip-Hot Hop.

Chaque participant s'inscrira(si il le veut)et choisira entre ces trois disciplines.

Pour le Rap :
Un texte d'une création personnelle ou une vidéo.

Pour le R'N'B :
Une mise en scène d'un texte plus une musique de fond ou un instrument.

Pour le Hip-Hop :
Un texte sera difficile donc une petite vidéo de 5 min.

Récompenses :
1er : 3 clients C.M + 3 Recrues + 3 clics et un code de 60(A voir.....)
2ème : 3 clients C.M + 3 Recrues + 3 clics.
3ème : 1 client C.M + 2 Recrue + 2 clics.
Les Récompenses seront données par moi ou des participants généreux.
Je ne commence l'Animation qu'a 5 joueurs par discipline.

Personne n'est obligé de participer,cette anim' demandera des spectateurs pour juger.
