  • 1 Les prérequis
    • Pour avoir la mise à jour gratuite : Windows 7, 8 ou 8.1 ;
    • Processeur de 1 GHz ou plus ;
    • Pour les système 32 bits :
    • 1 Go de RAM ;
    • 16 Go de libre sur le disque dur.
    • Pour les système 64 bits :
    • 2 Go de RAM ;
    • 20 Go de libre sur le disque dur.
    • DirectX 9 ou version plus ;
    • 800x600 de résolution minimum.
      .
  • 1bis Pour les possesseur de Windows 8 uniquement
    • Ouvrez Windows Store
    • Cliquez sur la mise à jour vers Windows 8.1
    • Appuyez sur télécharger et suivre les instruction
    • Si la mise à jour est introuvable, passez à l'épape 3, puis recommencer à cette étape.
      .
  • 2 Réservez votre license Windows 10 gratuite (Windows 7, 8 ou 8.1)
    • Cliquer sur le logo blanc de windows dans la barre des tâches.
    • La fenêtre "Obtenir Windows 10" s'ouvre.
    • Et réserver votre mise à niveau gratuite.
    • Vous pourrez revenir à cette fenêtre pour vérifier que tout votre système est compatible avec Windows 10 ; si vous voyez "Nous vérifions actuellement que votre PC et compatible avec Windows 10" alors ne forcez pas l'installation.
    • Si vous ne voyez pas le logo blanc de Windows, passez à l'étape 3 puis recommencer à cette étape.
      .
  • 3 Faites toutes les mises possibles
    • Lancez "Windows Update".
    • Vérifiez que le service est actif dans "Modifier les paramètres".
    • Recherchez les mises à jour et installez les toutes.
    • Si des messages d'erreurs apparaissent mais qu'au moins une mise à jour a fonctionné, Redémarrez l'ordinateur et recommencez.
    • Si aucune mise à jour ne s'installe, Essayez la réparation automatique de Windows Update pour Windows 7 : http://go.microsoft.com/?linkid=9666880
      .
      Si à cette étape windows update vous propose la mise à jour vers Windows 10 : vous avez réussi !
      .
  • Autres méthodes d'installation (ces méthodes peuvent sauter des étapes de vérification du matériel...)
    Vous pouvez installer la mise à jour Windows 10 à partir d'une clé USB, un DVD que vous aurez gravé ou d'une copie sur le disque dur.
    • Récupérez le fichier ISO de la mise à jour :
      • Par torrent (en cours d'upload) :
        • Windows 10 Famille x32 ;
        • Windows 10 Famille x64.
      • Par l'utilitaire de Microsoft :
        • Utilitaire pour PC x32 (peut télécharger l'ISO x32 ou x64, dans toutes les langues et toutes les versions de Windows 10)
        • Utilitaire pour PC x64 (peut télécharger l'ISO x32 ou x64, dans toutes les langues et toutes les versions de Windows 10)
    • Pour créer une clé USB bootable : utilisez l'utilitaire RUFUS.
    • Pour créer un DVD bootable : utilisez le fichier ISO "depuis une image disque" dans le logiciel de votre graveur.
    • Pour faire la mise à jour depuis votre disque dur : Extraire les fichiers de l'ISO dans un dossier avec 7Zip (par exemple) et lancer le fichier setup.exe
