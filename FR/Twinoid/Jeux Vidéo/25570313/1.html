Warframe, qu'est-ce que c'est ?

Warframe est un jeu gratuit, en mode third-person-shooter qui se joue en coop' ( on peut aussi, bien sûr, jouer en solo. Les " équipes " sont de 4 joueurs maximum. ).

Le principe est simple : vous contrôlez un personnage, appelé Tenno ( les Tenno sont les rescapés de la Galaxie, leurs ennemis principaux sont les Grineer, qui vont bientôt découvrir qu'ils ne sont pas morts ), votre but est de mener à bien diverses missions ( seul ou à plusieurs ) à travers les les planètes du système solaire ( vous commencez sur Mercure ).

Au début du jeu, vous devez choisir entre 3 Warframes ( = armures ) qui vous seront proposés :

- Excalibur ( recommandé aux nouveaux joueurs )
- Loki ( une bonne manipulation des touches et des armes est nécessaire pour maîtriser avec aisance ce personnage )
- Ash ( ce dernier préfère les combats corps à corps ).

Vous commencerez avec 3 armes de départ :

- Un rifle ( MK1-Braton )
- Un shotgun ( Lato )
- Une melee ( Skana )

Au fur et à mesure du jeu, vous gagnerez des crédits en menant à bien :

- Des missions ( solo ou coop', mais très souvent solo )
- Des voids, ... qui vous feront gagner environ 25000 crédits par partie.

D'abord celles des 3 principaux ( au début du jeu ), et c'est les caractéristiques de départ c.à.d. qu'au fil du jeu ( en jouant quoi ), vous les améliorerez en les faisant passer aux grades supérieurs ( de même qu'avec les armes ) :

- Excalibur :

  • Vitesse de sprint de 1.0 ( assez lent mais ça va encore )
  • Vie de 100 ( quand un ennemi vous tire dessus, vous ne perdez pas directement de la vie, c'est vos " boucliers " qui encaissent )
  • Bouclier de 100 ( les boucliers sont les 1ers à prendre les coups quand un ennemi vous touche, et ils se rechargent automatiquement, cependant la vie ( = santé ) ne se recharge pas, il faudra donc frapper des " conteneurs de stockage " qui contiennent de tout : énergie, santé, ... ).

- Loki :

  • Vitesse de sprint de 1.6
  • Vie de 75 ( moins que Excalibur )
  • Bouclier de 75 ( idem )

Ce personnage nécessite donc une bonne maîtrise du jeu ^^


- Ash :

  • Vitesse de sprint de 1.0
  • Vie de 150 ( perso je suis à 160 avec mon Excalibur après 2 jours de jeu, donc Ash est vraiment pas mal à ce niveau-là )
  • Bouclier de 100

Voilà j'pense avoir dit le principal ^^
Cependant vous pouvez obtenir de nouveaux Warframes, mais cela vous coûtera des platines ( monnaie du jeu qui est payante ), en général 220 platines.


Aussi, en réalisant des missions, vous obtiendrez des " mods " que vous pourrez fusionner avec des " coeurs de fusion " ( bah vi ) et qui vous permettront d'avoir des ptits bonus ( vitesse, ... ).


En outre, aussi dans les missions, vous obtiendrez peut-être des schémas, de la ferrite, ... qui vous permettront de vous créer un nouveau Warframe ( c'est assez coûteux et les objets nécessaires sont assez complexes à trouver ), de même pour les armes . Vous pouvez aussi acheter des schémas ( généralement 30.000 crédits ).
