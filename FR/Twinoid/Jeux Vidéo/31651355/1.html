Bonjour,
Je souhaiterai vous présentez dragon ball dreams.

Dbdreams est un jeux dans le style 2fight pour se qui connaisse.

Donc dans se jeux vous devez affronter d'autres joueurs pour exp vos différent personnage pour faire leur Mode histoire, Quêtes, Compétitions ou guerre des clans.

Les fight se déroule jusqu'a la mort d'un des deux personnages
le monde des mort n'existe pas dans se jeux donc faut pouvez affronter autant d'adversaire que vous le souhaiter (dans la limite de combat par Heure).

Pour augmenter la puissance puissance de vos persos vous pouvez changer leur tenue/transfo si ils en ont exemple: goku a pour transfo kaioken, super saïyen 1,2,3,4 etc...
Vous pouvez les obtenir via mode histoire, Quêtes, compétitions, boutiques ou par les Dragon ball.

Les différent Mode Histoire du jeux: goku, gohan ado, gohan adulte, vegeta, trunks du futur, piccolo, raditz et nappa actuellement dispo et d'autres arriveront avec le temps.

Venez essayer
