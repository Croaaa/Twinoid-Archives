LIEN DU DOCUMENT

Bonjour a tous.
Suite a des demandes de pseudos sur plusieurs sujet de ce sous forum, je vous propose, un document regroupant l'ensemble des pseudos des joueurs de Twinoid sur chacun des jeux.

Pour vous inscrire :
  • Vous devez cliquer sur 'TABLEAU PRINCIPAL' (en bas)(normalement fait automatiquement), et indiquer votre pseudo Twinoid, et votre pseudo sur chacun des jeux auxquels vous jouez.
  • Vous devez impérativement vous placer a la fin du tableau et me laisser faire le tri le jour où je passe (formules semi-automatiques, toussa toussa).
  • Si vous le souhaitez, vous pourrez ensuite vous rendre sur les pages des jeux auxquels vous jouez (si elle existent), afin de rentrer des informations complémentaires. Toutes les informations que vous avez déjà remplis seront présente.
    Si vous souhaitez modifier une information écrite automatiquement, vous ne pourrez pas, il faudra la changer dans le tableau principal.
  • Afin de rester dans le tableau principal, vous devrez vous connecter régulièrement (une ou deux fois par an) afin de remettre un 'o' dans la colonne la plus a gauche du tableau.
    Ceci est fait afin d''éliminer' les inactifs depuis plusieurs mois/années.


Et c'est tout.



Si vous souhaitez remplacer 'Jeu X' par le jeu de votre choix, ou si vous souhaitez ajouter une nouvelle page, faites la demande sur ce sujet.
Je vous encourage dans ce cas a ajouter un lien vers ce sujet, ou vers la page présentation du document, a la fin du premier post du sujet du jeu sur Twinoïd.


Le document a été crée initialement sur lol, je l'ai modifié pour qu'il puisse devenir plus général.
