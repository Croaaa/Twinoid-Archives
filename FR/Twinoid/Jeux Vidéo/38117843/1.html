Salut, j'ai cherché après un topic Garry's Mod et malheureusement, personne n'en n'as crée, donc j'ai décidé de le faire !

Donc, voici les infos du jeu :

Garry's Mod ( surnommé GMod ) est un jeu de type sandbox qui est apparu en 2006 sur Steam, la puissance de ce jeu est qu'il nous laisse faire TOUT CE QUE LE JOUEUR VEUT ! Et quand je dis tout, c'est TOUT !

Il n'est que sur PC, il ne requiert pas un grand PC de compétition, donc ( presque ) tout le monde peut y jouer.

D'ailleurs, voici un petit bonus : La config minimum !
Système d'exploitation : Windows® Vista/XP/2000
Processeur : Processeur 1.7 GHz
Mémoire Vive : 512 Mo de RAM
Affichage : Carte graphique compatible DirectX® 8.1 (Nécessite le support SSE)
Disque dur : 150 Mo d'espace disque disponible
Autres : Connexion internet
Il tourne sur le fameux moteur de Valve : Source, qui est un bon moteur, ce qui est aussi un atout !

Son prix est de 10€, donc ce n'est pas très cher, ce qui est un atout, malheureusement, si vous n'avez pas CS:S, ajoutez le à votre panier, car Garry's Mod à besoin en grande partie de CS:S pour les armes, les textures, les personnages, etc..

Si vous voulez mon avis sur ce jeu :

- Prix pas trop cher, même si CS:S est recommandé pour y jouer, ce qui fait un total de 20€, l'équivalent d'un jeu PS3 D'occasion, ça va quoi, surtout que les deux jeux sont bons...

- Les graphismes ne vont pas vous caresser la rétine certes, mais c'est pas le truc le plus lait au monde non plus.

- La communauté Workshop est juste magnifique, il y a beaucoup de packs d'armes, de voitures, de personnages, de maps, et même de modes ! D'ailleurs, vous pouvez en créer aussi !

- D'ailleurs en parlant de modes, je vous donne quelque noms pour bien commencer sur GMod :

- Trouble in Terrorist Town
- Murder
- Prop Hunt
- Hide and seek
- Stop it, Slender !
- Escape Pedobear
- Jailbreak
- etc...

Si jamais quelqu'un veut faire une partie avec tout les joueurs de Twino, je pense que ça peut se faire

Bref, tout ça pour dire que ce jeu est superbe !

Dans Murder tu as 3 types de joueur :

Le Murderer : Il doit tout simplement tuer tout le monde avec un couteau, il doit éviter de se faire repérer, sinon il risque de se faire tirer dessus direct.


Le Bystander : A.K.A le civil, il n'as pas d'arme, il ne peut rien faire contre le murderer, son seul truc, c'est crier le nom du Murderer comme un attardé.

Le " Sheriff " : Il a un revolver qui tire une seul balle et qui doit être rechargé à chaque coup, Il doit découvrir qui est le murderer, grâce aux Bystanders, pour pouvoir le tuer, si le sheriff tue un innocent, il perds son arme aux profits des Bystanders, qui pourront le prendre.

J'espère que c'est clair.
