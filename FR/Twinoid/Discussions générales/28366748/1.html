DavfLes règles des forums de groupe sont les même CGU que sur toute la plateforme Twinoid.
Mais tant qu'il y a pas de signalement, les propos interne peuvent rester hors-charte ?
Oui, mais attention : il suffit que quelqu'un signale un message (même un an après) pour que les modérateurs du jeu aient accès à toute la conversation. Cela peut arriver si un autre message est signalé dans la même discussion ou si ce message est signalé plus tard par un nouveau membre du groupe, par exemple.

Un modérateur pourrait donc voir ces messages et décider de sanctionner s'il s'agit d'abus graves. En pratique, ce genre de modération longtemps après les faits n'aura probablement lieu que pour les cas extrêmes. Normalement, les modérateurs font preuve de discrétion et ne vont pas fouiller sans raison dans le forums non-officiels. Mais cela peut arriver. Un message qui enfreint gravement les CGU (surtout si cela enfreint aussi la loi) peut donc être une bombe à retardement pour son auteur.
