La version alpha ( "A" en grec )est la base du jeu, non jouable par les joueurs.

La version bêta ( "B" en grec ) est une version à peu près finie, ouverte à un certain nombre de joueurs ( par exemple l'admin met le code dans une annonce et les 100 pemiers à l'utiliser ont l'accès ). L'objectif de cette version bêta est de repérer les bugs pour que les admins les corrigent.

Le jeu est ensuite ouvert à tous.

Maintenant ça suffit les questions sur la bêta
