Il est aussi possible de jouer directement depuis Chrome ou Firefox grâce à TheToto.

Pour ça, il faut utiliser ce script dans Tampermonkey (chrome, firefox, les autres).

C'est assez rudimentaire pour l'instant, par exemple, il n'est pas possible de ce noyer et les cubes invisibles ne sont pas invisibles.
Mais ça permet de jouer avec un vrai navigateur :3

Les contrôles :

  • Ce déplacer avec ZQSD ou les touches fléchés
  • Déplacer l'angle de vue avec clic gauche (maintenir) + souris (glisser).
  • Sauter avec la touche Espace
  • Prendre un cube avec clic gauche
  • Poser un cube avec clic droit

Visuellement, ça donne ça pour le jeu et ça pour la carte.
