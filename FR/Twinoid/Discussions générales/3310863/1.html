Pour ceux qui savent quoi en faire (et uniquement ceux là, pour les autres il y a du boulot avant ) :
Le script : Muxxu MSG Center

__________________________________
Pour les autres :

Il s'agit d'un script à ajouter à votre navigateur, permettant d'ajouter des fonctionnalités à une page web.
Ce script ajout la suppression/archivage/blacklistage/etc. multiple (des cases à cocher apparaissent pour effectuer une action) des messages privés de Muxxu.

Il devrait fonctionner pour tous les navigateurs (donc tous sauf Internet Explorer), mais seules les instructions pour Firefox seront données. (en cherchant "installer un userscript sur [nom du navigateur]", vous devriez pouvoir vous débrouiller).

Première étape : Installer greasemonkey
Greasemonkey est une extension populaire de firefox, permettant l'installation d'UserScripts et donc la modification d'un site (modification effective "chez vous" et les éventuels autres utilisateurs du script).

J'vais pas réécrire un tutoriel existant, surtout que ça ne serait pas plus clair :
Tutoriel : installation de greasemonkey

Deuxième étape : relancer firefox et installer le script
Une fois firefox relancé (et donc à priori Greasemonkey installé, le script peut être à son tour installé.
Allez sur le lien suivant et cliquez sur "installer" (soit une barre s'affiche en haut sous la barre d'adresse, soit une fenêtre s'ouvre, soit greasemonkey n'est pas correctement installé et vous devez retourner au tuto) :
Le script : Muxxu MSG Center

Allez sur votre messagerie Muxxu pour voir la différence, et amusez vous
