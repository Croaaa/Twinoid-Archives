Les applications ont été uniquement autorisées pour améliorer le confort des jeux Muxxu.

Cela est clairement précisé dans le Guide des Applications qui a été lu par les développeurs : http://muxxu.com/app/help, je cite :
les applications ont pour but de compléter ou d'améliorer le confort des jeux Muxxu, tout autre utilisation est exclue

Les règles étaient donc bien connues depuis le début, les développeurs ne peuvent s'en prendre qu'à eux même s'ils ne les ont pas respectées. Nous n'avons en tout cas a aucun moment été contactés via le support pour demander si les jeux étaient autorisés.

Nous réfléchissons néanmoins à une ouverture de la plate forme Muxxu à d'autres développeurs, mais cela ne se ferra pas au moyen d'applications.
