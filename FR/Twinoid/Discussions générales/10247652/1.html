Bonsoir,

La balise Gloire (visible ici par exemple) qui change de design selon le jeu a été crée pour être une récompense pour avoir réussi un des Grands Buts d'un jeu.

Par exemple sur Hordes c'est les joueurs ayant été les premiers au classements des villes lors de la dernière Saison.

Je suggère l'ajout de cette balise sur les autres jeux :

  • KadoKado : Pour avoir été le meilleur joueur de la dernière période
  • CroqueMotel : Pour avoir X étoiles
  • Labrute : Pour avoir une Brute Légende Brutale
  • StudioQuiz : Pour ceux étant dans le top classement Gloire
  • Odyssey : Pour avoir fini l'histoire des 16 personnages
  • Snake : Pour avoir complété l'encyclopéfruit
  • Fever : Pour avoir changé d'Archipel
  • TowerQuest : Pas d'idée
  • Alphabounce : Pour avoir retrouver la Terre
  • NaturalChimie2 : Pour les gagnants de la dernière Lutte des Classes
  • Kingdom : Pour avoir été Empereur
  • DinoRPG : Pas d'idée
  • Intrusion : Atteindre le level 9 ? Vainqueur challenge Platine ?
  • Majority : Victoire x X ?
  • Kube : Pour ceux ayant le titre Cartographe
  • MB2 : Pour avoir vaincu TourneBoule une fois

Voili voilou.
