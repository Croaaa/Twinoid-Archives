:mmax: Présentation de l'outil ...
A l'heure où de nombreuses suggestions fleurissent sur les forums de Muxxu, un outil existe. Que ce soit pour égayer vos messages avec de nouveaux smileys, pour vous faciliter la vie sous kingdom, intrusion ou kube, ou encore pour accéder plus rapidement à vos pages préférées : MuxxuMax est fait pour vous !
Compatibilité
MuxxuMax est compatible avec les navigateurs suivants :
- Firefox 3.x
La prochaine version de MuxxuMax sera compatible avec les navigateurs suivants :
- Opéra 10
- Chrome
Téléchargement et Installation
Veuillez vous rendre sur la page suivante pour des instructions détaillées
Informations Importantes
- La version publique actuelle de MuxxuMax est la version 1.55
- La version 1.5x de MuxxuMax n'est plus développée
- Une version 2.00 est en cours de développement et de tests et devrait sortir avant la fin de l'été
- L'utilisation d'une version de test sans autorisation provoquera votre bannissement des serveurs de MuxxuMax
- MuxxuMax n'est pas libre de droits.
Aidez-moi !
N'hésitez pas à consulter le Mode d'emploi
Si vous rencontrez un bug, merci de me contacter en utilisant le lien présent sur la page d'accueil de Muxxu.com

Comme tout outil externe, MuxxuMax n'est lié ni à Muxxu.com, ni à MotionTwin.
Il s'agit d'un outil développé par les joueurs, pour les joueurs.
