Pour activer un script sur un site précis sur Opéra :

  • Télécharger le script et l'enregistrer dans un dossier sur son PC.
  • Ouvrir le site où l'on désire activer le script sur Opéra.
  • Clic droit sur la page > Éditer les préférences du site...
  • Onglet Scripts > Choisir le dossier où est enregistré le script tout en bas de la fenêtre et valider.

Après, forcément, il faut que le script soit compatible avec Opéra (ce n'est pas le cas du DGM, par exemple). Je veux bien croire que HMUpdater est compatible s'ils le disent, mais quand j'ai testé, les pages de Hordes ne se chargeaient plus
