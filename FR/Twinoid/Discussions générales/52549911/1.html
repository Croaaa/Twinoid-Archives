Essaye de vider ton cache :

  • localement : sur la page concernée par le problème, utiliser une des combinaison de touche suivante (varie suivant le navigateur et le système d'exploitation) : ctrl + f5, ctrl + shift/maj + R (ça actualise en même temps)
  • globalement : voir dans les options du navigateur, dans Firefox c'est dans Options > Vie privée > Effacer l'historique récent > Détails > décocher tout sauf Cache > Effacer maintenant (et actualiser les pages concernées ensuite)
