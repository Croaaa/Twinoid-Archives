C) La Destruction/Passation des Pouvoirs
A tout moments, le créateur peut choisir soit de :

- Détruire le groupe si le Créateur est tous seul
- Donner ce groupe à un autre membre : si le nombre de membres est supérieur à 1.
Dans ce cas, le membre choisi deviendra Créateur est obtiendra tous les droit qui y confère.
L'ancien Créateur se voit donc attribuer le rôle le plus à Droite dans la page "Administration"
