Pour ajouter une image à une page de groupe, tu cliques sur "image" pendant que tu modifies une page. Tu choisis l'image sur ton ordinateur et une espèce de code ressemblant à cela @muxxu/5/a/0431bc79_154382.jpg@ apparaîtra.
C'est ton image. Tu n'as plus qu'à enregistrer les modifications de la page pour y voir ton image.

Si tu veux montrer une image dans la page discussion d'un groupe (ne fonctionne qu'en nouveau message, pas en commentaire), tu fais la même chose, sauf qu'au lieu d'enregistrer la page modifiée, tu copies et colles le code de l'image dans la page discussion.

Pour le Logo, le fond et la bannière, il suffit d'aller dans les options correspondantes, de cliquer sur "choisir" et de sélectionner sur ton ordinateur l'image que tu souhaite utiliser.

Voilà, j'espère avoir répondu à ta question (et avoir bien expliqué aussi), si tu as une question, n'hésite pas.
