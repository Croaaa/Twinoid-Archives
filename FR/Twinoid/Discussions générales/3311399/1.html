Je vais quand même t'en trouver 2:
Les animations et les groupes.

Pour ce qui est des animations, l'organisateur peut avoir besoin de contacter tous les participants pour donner une information particulière. Il a toutefois la possibilité de passer par le topic de son anim' pour prévenir les gens et contacter les retardataires au cas par cas.

Pour les groupes, il pourrait en effet être pratique de pouvoir contacter tout le monde d'un coup, surtout depuis que Twinoïd limite franchement le passage des gens sur le fluxx. La solution serait ici de créer une option "Message de Groupe" utilisable seulement par les gérants dudit groupe. Cette option permettrait d'envoyer un message identique à tous les membres du groupe (ou à une partie via un système de case à cocher) avec comme destinataire le nom du groupe. Cette fonction subirait les mêmes limitations que pour un message privé quant au nombre d'annonce générale pouvant être envoyé à la suite.
