Voila un petit topic rapide afin que les joueurs les moins à l'aise avec l'informatique puissent aisément arranger leur message personnel qui est modifiable via le profil juste sous l'avatar.
Sachez que votre message personnel utilise les mêmes balises que les messages du forum
A savoir :

-[i)]votre texte[/i)] pour mettre le texte en italique. (Enlever bien sur les parenthèses ")" après les i qui ne sont là que pour vous permettre de visualiser le code.)

-[g)]votre texte [/g)] pour mettre en gras. (encore une fois n'oubliez pas d'enlever les parenthèses comme précédemment.)

-[lien=l'adresse du site/url commençant par http://]votre texte[/lien] pour avoir une phrase de votre choix à la place de l'url parce que c'est plus jolis quand même vous ne trouvez pas ?

-[cite)]votre texte[/cite)] pour faire une "citation".(La aussi pensez à enlever les parenthèses ")" qui sont utiliser ici juste pour visualiser le code.)

-[spoil)]votre texte[/spoil)] pour mettre un texte en "spoil".(voyez le résultat plus bas et n'oubliez pas encore une fois de supprimer les parenthèses.)


Et pour vous montrer le résultat voila des exemples, dans l'ordre :
-Italique
-gras
-votre texte
-la citation :votre citation ici
- le "spoil" :Votre texte ici
