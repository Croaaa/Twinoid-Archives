Oui, le droit s'applique bien évidemment. On pourrait se poser la question pour un site dont les serveurs se situent sur un autre territoire :

S'il s'agit d'un site dont les serveurs sont situés hors de France, la France peut empêcher ses citoyens d'accéder au site. Cependant, elle n'a aucun droit quand aux serveurs eux-mêmes. Elle aura besoin de l'accord du pays étranger en question pour faire quoi que ce soit.

Exemple : pour Twinoid, toutes les lois françaises s'appliquent. Pour être accessible en Allemagne, Twinoid doit être conforme aux lois françaises ET allemandes.

Juste pour info, les lois y sont applicables, mais Motion-Twin n'est pas considéré comme responsable des propos que vous y posté.

Pour le contenu posté par les utilisateurs ( forum, mur, avatar ... ) deux cas sont possibles :

- Il y a une modération avant que le contenu ne soit affiché ( pré-modération ) : Motion-twin est considéré comme éditeur de ce contenu et est donc responsable de TOUT ce qui est posté.

- Il n'y a qu'une modération a postériori ( cas actuel ) : Motion-Twin n'est pas responsable et est légalement considéré comme simple hébergeur de ce contenu.
