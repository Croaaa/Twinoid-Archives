Bonjour à tous,

L'arrivée des groupes Twinoid à été très appréciée des utilisateurs. Cependant, je pense que des améliorations sont à faire...

  • Rôles : Pouvoir avoir un bouton "ajouter un rôle" pour avoir la possibilité d'en avoir plus.
  • Rôles personnalisés : Quelques fois, on veut ajouter un droit à un membre de notre groupe sans créer de rôle officiel non plus. Ceci n'est paspossible pour le moment, ce serait quand même pas mal.
  • Forum : Pouvoir avoir son propre BBCode. L'administrateur pourrait ainsi créer un BBCode avec deux champs : le champ "balise BBCode" et le champ "correspondance en HTML / CSS" (voir plus bas pour plus d'informations).
  • Recherche : Pouvoir avoir (dans la liste des groupes) un bouton "tous les groupes" pour afficher à l'utilisateur tous les groupes triés par ordre alphabétique. Ce serait bien aussi d'avoir un hamp "Rechercher un groupe".
  • Mise en forme sur le nexus : Pouvoir avoir les balises [mod][/mod] et [annonce][/annonce] sur le nexus.

 
Exemple de BBCode personnalisé :
Nécéssite des connaissance en HTML / CSS

Je veux créer un BBCode (une mise en forme sur le forums) personnalisé : un simple cadre vert ayant pour balise [cadrevert][/cadrevert] :

Code : BBCode 

[cadrevert]{TEXT}[/cadrevert]

Code : HTML + CSS 

<table><tr><td style="background_color:green;">{TEXT}</td></tr></table>

Et en + pour le BBCode : Pouvoir décider si le BBCode personnalisé apparaîtra sur le champ d'écriture dans un forum / nexus et les permissions d'utilisation du BBCode.

Voilà ! J'ai encore plein d'autres suggestiosn pur les groupes mais répondes déjà à celles-ci ce sera déjà pas mal, on verra après
