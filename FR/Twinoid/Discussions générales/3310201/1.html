Bonjour, cette étude/sondage a pour but de cerner la manière de jouer des joueurs. Vous n'êtes pas obligé de répondre à tout le sondage, vous pouvez répondre seulement qu'à une partie ou plusieurs mais cela ne sera pas bien long.

Partie A : Kingdom :
Question 1 : Vous voulez être Empereur/Roi pour :
a) le prestige d'avoir réussit ce défi
b) les trophées

Question 2 : Vous gaspillez :
a) Pour les récompenses
b) involontairement
c) vous ne gaspillez pas

Question 3 : Vous vous ruez sur les lieux de ressources :
a) Pour avoir de meilleurs unités
b) Pour avoir les récompenses de récolte de ressources

Question 4 : A la venue des Cartes de Groupe vous vous êtes dis :
a) Chic on va pouvoir jouer entre potes
b) Chic cela va être plus facile pour les récompenses de gaspillage/récolte

Question 5 : Si vous ne jouez plus à Kingdom c'est à cause de :
a) Vous êtes entièrement lassé
b) Vous avez toutes les récompenses

Partie B : Labrute :
Question 1 : Vous choisissez vos adversaires :
a) Au hasard
b) De manière à tenter d'affronter 3 animaux pour la récompense
c) De manière à gagner(en regardant les caractéristiques)

Question 2 : Vous choisissez les carac/armes/capacités en fonction :
a) de manière à avoir les récompenses (barbare, brute, pacificateur, ninja, sans accroc...)
b) De façon à avoir une bonne brute

Question 3 : Vous achetez de nouvelles brutes pour :
a) Avoir plus de chances d'avoir les récompenses
b) Jouer plus
c) Vous ne pouvez/voulez pas acheter d'autres brutes

Partie C : Majority :
Question 1 : Vous faites des pour :
a) Quand ça vous fait vraiment rire
b) Pour avoir le trophée + les statistiques

Question 2 : Avez vous déjà plusieurs fois rejoins volontairement des parties sans pouvoir les jouer uniquement pour avancer dans l'obtention de la récompense "parties jouées x200" :
a) Oui
b) Non

Question 3 :Vous voulez gagner pour :
a) Avoir le prestige d'être vainqueur
b) Pour avoir toutes les récompenses liées aux victoires

Partie D : Kube
Question 1 : Votre trophée vous l'avez eu :
a) En passant une après-midi/journée à gaiment vous noyez
b) De façon naturelle en se noyant normalement de temps en temps
c) Vous ne l'avez pas

Question 2 : Votre trophées vous l'avez eu :
a) En passant une après-midi/journée à gaiment vous téléporter
b) De façon naturelle en vous téléportant normalement de temps en temps
c) Vous ne l'avez pas

Question 3 : Vous avez acheté l'appareil photo pour :
a) Faire des photos du paysage
b) Avoir le trophée
c) Vous ne l'avez pas

Question 4 : Vous essayez d'avoir le plus de Kubes différents de touchés possible pour :
a) Complétez votre "Kubopédie" ()
b) Avoir les récompenses

Partie E : Intrusion
Question 1 : Vous avez joué à Intrusion :
a) Uniquement pour les récompenses
b) Pour le jeu

Question 2 : Vous jouez en mode Uber-eleet pour :
a) Le plaisir de jouer différemment
b) Avoir les récompenses associées
c) Vous ne jouez pas en mode Uber-eleet (trop dur ou autre)

Question 3 : Vous avez arrêtez de jouer à Intrusion :
a) Une fois après avoir eu toutes les récompenses
b) Parce que vous en aviez marre
c) Vous y jouez encore
d) Vous y jouez encore pour avoir toutes les récompenses

Partie F : Snake :
Question 1 : Quand vous trouvez un violon une flûte ou une trompette vous vous dites :
a) Youpi-banane mon score va être merveilleux
b) Youpi-banane, ça va faire du bien à mon encyclo pour avoir les images

Question 2 : Vous jouez :
a) Pour le score
b) Pour avoir les images des fruits

Question 3 : Vous jouez à la loterie :
a) Pour avoir une carte précise
b) Avoir la récompense
c) Vous ne jouez pas à la loterie

Partie G : Fever :
Question 1 : Utilisez vous vos uniquement pour avoir les récompenses ?
a) oui
b) non

Question 2 : Cherchez vous les objets "consommable" uniquement pour leur récompense ?
a) Oui
b) Non

Question 3 : Vous cherchez les Objets éternels pour :
a) Leur effet
b) Avoir leur image en récompense

C'est tout
Répondez juste pour la forme suivante :
A)
1) a
2) b
etc
