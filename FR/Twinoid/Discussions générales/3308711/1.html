Pour organiser/référencer un événement Muxxu sur WikiMuxx :

- Vous m'envoyez un petit MP dont le sujet commence par la balise [crea-event]
- Dans ce MP vous décrivez un peu votre événement (jeu concerné, principe, durée, nombre de participants, besoin ou non des clans ... )
- Si vous voulez que des trophées WikiMuxx puissent être gagnés durant votre event vous devez me les envoyez au format 16*16 en PNG ou GIF, le nombre de trophées doit resté raisonnable.

Si votre projet est accepté (ce qui ne seras pas si difficile) je vous donnerez les droits nécessaires pour gérer votre événements (pensez à vous créer un compte sur WikiMuxx avant ! )
