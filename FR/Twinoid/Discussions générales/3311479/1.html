Bonjour et bienvenue dans l'émission "CoDeCode !"

Je vous rappelle le concept de l'émission, si vous êtes nouveau et que vous venez de nous rejoindre ! Nos chers rédacteurs que l'on salue et qui travaillent en coulissent vous créent un code, et, depuis leur pupitre, les candidats devront essayer tous les moyens existants sur Terre pour pouvoir tirer quelque chose de ce code !

Ce code, ça peut être du texte à déchiffrer, des chiffres à transformer, ou un truc à faire avec un dessin ! En tous cas, c'est fun, et parfois piégeur !

Aujourd'hui, je vais vous présenter une énigme qui peut vous faire mal aux yeux aux premiers abords ! Mais cette énigme rapportera la somme rondelette de 60 jetons à celui qui en trouve la réponse ! 60 jetons !
Cette somme sera distribuée sous forme d'un code SMS Pass au premier qui postera la réponse exacte sur ce sujet même !

Attention, il faut être très rapides !

Tout le monde a le droit de participer, même les modérateurs ! Eh oui, car cette année, les Animuxxions vous sont intégralement offertes par ... moi-même

Bon. Maintenant, passons au vif de l'émission. Aujourd'hui, le code qui va être offert à nos candidats. Ce n'est pas comme la semaine dernière un code à clé incompréhensible doublé d'un code agissant avec des factorielles. Rappelez-vous qu'il avait noyé les candidats dans les chiffres ! Non ... Cette semaine, je vous propose ...

Ceci !

Attention, le temps vous est compté, car seul le premier gagnera 60 ! Faites vite !!

Bonne chance !
