Tu ne peux pas lui envoyer des que tu as sur son compte. Mais tu peux aller ici et en acheter par SMS, téléphone fixe, par internet... Avec ces moyens-là, tu reçois un code (contrairement au paiement par carte bancaire, qui envoie la monnaie directement sur ton compte). Il te suffit de copier le code et de l'envoyer à ton ami, qui ira au même endroit pour le valider.

Evidemment, si tu fais ça, toi, tu ne dois pas entrer le code dans le cadre "Noter ici le code reçu". Sinon c'est toi qui recevras les .
