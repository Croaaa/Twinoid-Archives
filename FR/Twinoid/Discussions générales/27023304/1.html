Un nouveau script pour Twinoid arrive, en attendant les modifications tant attendues

Nexus Cleaner vous permet de filtrer votre Nexus selon vos désirs.


Au menu : filtrage par application, par type de message (récompenses (images, trophée, titre, etc.), statuts, évènements (objectifs bonus, morts, etc.), etc.), par joueur, etc.
Un description plus poussée sera ajoutée.. un jour

Quelques screenshots qui seront plus parlants :
- Aperçu du Nexus modifié : [1]
- Gestionnaire de mises à jour : [1]
- Configuration des filtres : [1] [2]


Vous pouvez télécharger la dernière version dès maintenant.
Le script a été testé sous Firefox avec Greasemonkey (version récente nécessaire à cause d'indexedDB), et devrait fonctionner sous Chrome avec Tampermonkey et sous Opera.

Vous pouvez également proposer ici-même des filtres universels que je pourrais ajouter aux prochaines versions.
