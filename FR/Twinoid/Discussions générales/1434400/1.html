Etant étudiant graphique.

Il est possible de baisser le poids de l'image (ko) en utilisant le logiciel Photoshop CS sans rédimensionner l'image.

Exemple : 600x600 pèse 66.3Ko. Si le système demande un avatar de maximum 50 Ko.

Sous photoshop, je met l'image original puis Enregistrer pour le web.... La fenêtre s'ouvre pour configurer :
Choisir le type d'image :
- Gif : 164.2Ko (c'est trop)
- JPEG : 48.1Ko (Et hop, résultat parfait)
- PNG-8 : 138,9Ko (c'est trop)
- PNG-24 : 279.3Ko (c'est trop)

Donc je sélectionne JPEG pour convertir le poids d'image.
Mais si le site demande 30Ko alors comment faire ? Je sélectionne JPEG qui convertira 48.1Ko en qualité Elevé
Qualité Moyenne : 24.85Ko
Qualité Faible : 16.85

Si le système demande 10Ko alors gardes la qualité Faible puis de rédimensionner un peu jusqu'à -10Ko

Voilà, j'espère que ça vous apprend sous Photoshop

Spiky : Si tu parles d'ici, il y a pas une limite

Une image de 100x100 pixels ou carrée est recommandée. Les formats de fichiers suivants sont supportés : JPEG, GIF et PNG.

Les contenus hors-charte sont interdits (injurieux, adulte, non libre de droits...) : merci de consulter le règlement (lien en bas du site) en cas de doute.

Mon image est de 1284x1600 avec 181 Ko alors ça passe et il n'y a pas de limite car le système redimensionne automatique.
Sinon l'autre site ailleur, il n'y a pas de système redimensionner auto donc tu fais comme ce que j'ai expliqué sous Photoshop
