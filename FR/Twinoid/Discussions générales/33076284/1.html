Edit raté. Je voulais dire "rentabiliweb".

En fait twinoid ne prend pas direct les code neosurf. Il prend des MT-Pass (parce que c'est eux-mêmes), ou des codes via rentabiliweb. Neosurf s'utilise via rentabiliweb. D'où cette étape supplémentaire.

Mais bon, c'est pas si compliqué que ça. Suffit de suivre les instructions. C'est juste que, par réflexe, les gens ne le font en général pas, tellement sûr que puisqu'il faut saisir un code et qu'il y a un code sur la carte, ça se passe d'explication.

Donc faut cliquer sur "obtenir un code grace à ma carte neosurf" (etape 2). Et ensuite tu taperas le code obtenu.
