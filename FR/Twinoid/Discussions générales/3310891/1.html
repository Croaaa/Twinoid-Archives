Le Flux XML 'user' de Muxxu renvoie dans les données un attribut "lang" qui contient "fr" (français) ou bien "es" (espagnol) ou "en" (english).

Il suffit de récupérer la valeur du "lang" pour connaitre la langue de l'utilisateur et lui afficher l'application dans la bonne langue (c'est ce que fait Durss pour le Kube-Builder )
