Méhou!

J'ai reconstruis un nouveau labyrinthe sur les ruines du précédent. Pour ceux qui aiment les dédales, les balades trépidantes, les pièges... cette aventure est faite pour vous !

Monsieur Règles du jeuUn but : récolter le plus de jetons  ; celui qui en aura trouvé le + gagnera un bisou un MT-Pass !

Pour cela, vous aurez le droit chaque jour à 1 action + 1 déplacement (dans cet ordre, et par MP svp!)) pas d'inscription nécessaire

Cette action, c'est vous qui l'imaginez ; exemple : « Aujourd'hui, je fouille le sol de la salle  »
Cette action peut vous faire gagner des jetons, ou bien des avantages.

Le déplacement permet d'évoluer dans le labyrinthe, mais vous avez le droit de rester où vous êtes (je ne vous cache pas que vous n'irez pas très loin mais c'est vous qui voyez ) Il vous suffira de me dire une des directions : Est, Ouest, Sud ou Nord. Bien entendu, comme j'aurais pris le soin de vous mettre un bandeau sur les yeux, vous n'y verrez rien du tout ! ( interdiction de l'enlever sous peine d'élimination, nan mais c'est vrai quoi, faut pas déconner hein ! )

Des sont disséminés dans le labyrinthe, parmi quelques pièges, je vous laisse les découvrir par vous-même !
Si vous mourrez , votre partie s'arrête, mais vous gardez les amassés jusque là.

Encore un petit détail : j'ai égaré un épluche-légume rose fluo (oui oui, celui de la version 1 de mon labyrinthe!), si vous le choppez, vous éplucherez tous les participants sur votre passage ! Ils mourront dans d'atroces souffrances.

Si vous êtes absent un jour, pas de rattrapage, ni la veille, ni le lendemain, c'est comme ça, et il n'y aura aucune dérogation. Je me connecterais régulièrement pour vous répondre au mieux chaque jour.

La partie commence demain et chaque personne peut commencer dès qu'elle le veut. Le plus tôt est quand même le mieux pour récolter les jetons... Elle se terminera quand toutes les salles du labyrinthe auront été découvertes au moins une fois.

En attendant demain, si vous avez des questions (j'ai pu oublier un détail), je vous invite à les poser, je mettrais à jour les règles si des précisions doivent être ajoutées.

Méhou+
