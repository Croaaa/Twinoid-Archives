Bienvenue dans le jeu “A la conquête du Monde Gourmet”, tiré d’un de mes mangas préférés : “Toriko”.

.

Dans ce jeu, vous incarnez un personnage qui devra augmenter en puissance et évoluer dans un monde fantastique rempli de monstre, pour la plupart assez étranges mais délicieux, pour pouvoir atteindre le Monde Gourmet et trouver “GOD”, l’aliment ulltime.

.

Personnages :

Toriko :
Type : Polyvalent, corps à corps
Attaque de base : Sun Ren Kugi Punchi, Fork, Knife
Vie : 600 pv de base

Bonus : Aucun
Malus : Aucun


Zebra :
Type : Puissance, attaque à distance
Attaque de base : Machine Gun Voice, Voice Bazooka, Thunder Noise
Vie : 850 pv

Bonus : Grâce à son ouïe surdéveloppé, il peut éviter certaine attaque presque imparables.
Malus : On peut lui faire croire énormément de choses.

Coco :
Type : Défense, attaque à distance
Attaque de base : Poison Ripple, Poison puissance maximale, Détection
Vie : 600 pv

Bonus : Étant un homme-poison, Coco est immunisé contre toutes les sortes de poison.
Malus : Aucun


Sani :
Type : Polyvalent, attaque à distance
Attaque de base : Frill Ganchi, Hair Lock, Hair Clutch
Vie : 600 pv

Bonus: Aucun
Malus : Il refuse d’approcher des choses trop “dégueulasses”.

GT Robot / Nitro :
Type : Puissance
Attaque de base : Poing Mixeur, Incision, Frappe Éclaire
Vie : 700 pv

Bonus: Aucun
Malus : Étant d’une nature très agressive, ils attaquent tout ce qui bouge (amis comme ennemis)

.
