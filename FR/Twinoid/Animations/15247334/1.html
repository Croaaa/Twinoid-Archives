Bienvenue pour cette première édition du jeu "Pariez, c'est gagné !

Durant une saison (de football), vous allez devoir pariez sur des matchs.

Je vais choisir un match au Hazard...euh...hasard.
Ex : Bayern Munich - Werder Brême

Vous allez devoir miser sur le match.
Ex : Bayern Munich 1-2 Werder Brême

Si vous pariez sur la bonne équipe, vous gagnez 2 points. ☼
Si le score que vous avez indiqué est exactement le même vous gagnez 5 points. ☼
Si vous vous trompez dans votre pronostic, vous perdez 1 point. ☼
FACULTATIF : Vous pouvez, si vous le désirez, parier sur le ou les buteurs de la rencontre !
Si vous gagné le pari des buteurs, vous remporté 10 Points !! ☼
A vos risques et périls, car si vous faites une erreur vous perdez 1 point. ☼

Ce symbole représente les points que vous gagnez >>> ☼

La rencontre sera exposée dans la discussion du groupe : La tribune.
Vous allez devoir donner votre pronostic sur la Discussion.

[ATTENTION] : Je prends votre premier pronostic donc si vous faites une erreur lorsque vous écrivez, tant pis pour vous...
