Cette édition sera un peu spéciale puisque c'est les vacances des fêtes et que c'est une édition en duo. Donc avant de me massacrer de "Je m'inscris" Vous devrez vous trouvez un coéquipier, après vous pourrez dire "Je m'inscris avec..."

Le Master Forumeur, MF ou encore LMF, est un jeu sous forme de compétition par vote pour élire le meilleur forumeur de Twinoïd. (le plus présent, le plus actif, le meilleur des jeux ou le plus connu) Vous devez donc voter pour votre équipe préférée pour chaque match. Si vous ne savez pas pour qui voter lors d'un match, vous votez "blanc".

Les règles:


8 équipes de 2 joueurs devant se trouver un nom d'équipe en lien avec Noël
Tournoi sous forme de double élimination
Match de 11 points (La première équipe qui obtient 21 votes gagne)
Logiciel pour faire les matchs: The Hat, pour le tableau: Toornee
Je participe et je cherche quelqu'un
Pour demander un coéquipier, vous pouvez le faire ici, mais pour le nom d'équipe en MP svp ^^.
Publicité seulement pour se trouver un coéquipier, pour le reste non.
Évidemment, on ne vote pas pour soi et vous pouvez voter pour les autres matchs une fois par jour.


Merci de le faire part ici quand une équipe avec le nom est formé, l'édition commencera quand on aura nos 8 équipes.

Bonne édition
