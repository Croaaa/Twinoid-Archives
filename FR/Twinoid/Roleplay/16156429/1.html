I / But :

Je vous présente un énorme petit concours. Le But est simple, il aura uniquement pour but d'élire le meilleur rôliste du Forum. Il n'y aura aucun gain surtout parce que je suis un MotherFucking radin de la mort par manque de moyen parce que je suis pauvre. Jugez le gagnant selon les critères que vous voulez : Activité / Talent / Connerie BornToKillYou on t'attend sur la scène !.

II / Déroulement

Les votes se feront par MP, quand je donnerai le top départ ici-même.
Pour les inscription, c'est ici.
La grande innovation de ce concours c'est que vous aurez le droit de vanter votre mérite et j'avoue qu'à l'heure actuelle, j'ai peur ... ici-même, sur ce RP. Tout message que je trouverai trop abusif, sera masqué et ne devra pas être pris en compte.
Ironie, séduction, arguments, vous avez le droit d'utiliser tous les outils nécessaires pour séduire les électeurs.
Les votes devront être anonymes. N'allez pas dire "Ouesh mec j'ai voté pour toi parce que tu m'as donné des Dragibus hier ! Aight !"
Le RP est éventuellement autorisé, pour les inspirés.
Heureusement pour vous j'ai gardé le meilleur pour la fin. Lors des inscription je ferais de petits tableux de matchs. Chacun de ces matchs sera voté, et l'électeur devra voter pour celui qu'il souhaite monter en grade. J'annoncerai les gros avantages.

Maintenant laissez-moi vous poser une question : Pour les Matchs entre Rôlistes, vous préfèrez afficher les vote ici, ou via MP ?
