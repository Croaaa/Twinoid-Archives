Les classes et les metier influanceron vos stats !

Puis m'envoyer votre fiche , je vous rendrai vos stats !

Les actions posibles :

,,, : Attaquer un monstre ( 2 PA )
,,, : Attaquer quelqu'un ( 3 PA )
,,, : Réparer son arme ( 2 PA ( Action posible que pour les Forgerons ))
,,, : Vendre ses objets ( 1 PA ( Action posible que pour les Marchand ))
,,, : Assembler des runes ( 3 PA ( Action posible que par les Alchimistes ))
,,, : Scruter les environs ( 1 PA ( Action posible que par les Eclaireurs ))
,,, : Crée un poison ( 2 PA ( Action posible que par les Etudiant en poison )
,,, : Donner des premiers soins ( 1 PA ( Action posible que par les medecins )
,,, : Ecouter ce qu'on dit ( 0 PA )
,,, : Voir les magasins ( 0 PA )
,,, : Se déplacer ( 1 PA )
,,, : Crée / Rejoindre un groupe ( 0 PA )

Vous envoyer vos actions a faire ( EN COMPTANT QUE VOUS AVEZ QUE 10 PA CHAQUE JOUR ) par mp a moi !

Je vous envoye les resultat ou les posibilité d'attaque ou de magasins ect ... ET JE COMPTE LES PA ! par mp aussi !

Vous commencer sur une carte comme pour horde ( 0/0 ; 0/-1 ect . ) puis je vous indique oû vous etes qui est la ect ..

Quand vous avez plus de PA vous ne pouvvez plus rien faire aujourd'hui !

Pour monter de nivaux aux niveau 1 vous devez avoir 10 Pts d'xp !

Si je me rend compte que vous trichez je vous déinscrit du " jeu " !

Le " chat " sera parler ici meme sur ce post !

Option de groupe :

,,, : Distribution d'xp : Egale / Par niveau
,,, : Renvoyer un joueur du groupe
,,, : Teleporter un membre du groupe prés de soi
,,, : Dissoudre le groupe
,,, : Donné le rôle de chef a un membre du groupe
