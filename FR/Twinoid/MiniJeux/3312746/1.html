Ptibiscuit
Alors, enfaite : Le HTML concerne la mise en page d'un site.
Le CSS concerne les styles de textes : Centrer, italiquer, grasser, ...

Tu peux te passer de fichier CSS Normalement, tu mets
<link rel="stylesheet" media="screen" type="text/css" title="Design" href="[Nom du fichier]" />
Pour lier un fichier CSS a ton fichier HTML.

Mais quand tu as un petit site et que tu n'as pas peur de réécrire des choses quelques fois, tu peux mettre le contenu de ton fichier CSS au début de ton fichier HTML.
Bref, pour mettre un fond dans ta page, tu peux caller ceci dans les balises <head> :
body {
background-image: url([Ton image]);
}
Pour mettre une image, tu mets :
<img src="[Ton image]" width="[X]" height="[X]" />

Mais pour tout, je te conseille les tutoriels de Site du Zéro, ils t'apprendront les bases du HTML, après, à toi de lire des bibliothèques de balises pour connaître toutes les fonctions du HTML/CSS !
