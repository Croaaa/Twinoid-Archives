Bon je vais commencer avec un jeux se nommant mushihimesama .

Ce jeu ne doit rien evoquer à 95% d'entre vous ce jeu est quasiment inconnu en france

alors on commence avec les informations un peut technique

Genre : shoot em up

Developeur : cave

Editeurs : Mircrosoft ( xbox live arcade )
Taito ( ps2 )
Ami ( arcade )

Plateforme : Ps2,Arcade,Xbox360 ( live )

Ce jeux est tout d'abord sorti sur arcade ou il fut considéré comme un des meilleurs jeux de ce genre
Il fut porté sur Ps2 ou il est connu pour etre un jeu de collection introuvable et un des 4 shoot em up de la consol
Le jeu est egalement sorti dans une version allégé ( bien plus facile ) sur le xboxlive .

Il fait partis des meilleurs shoot em up pour avoir une jouabilité d'enfer une bande son chanté cultissime .
Mais le point le plus atrayant de ce jeu est ces 3 niveaux de difficulté ( normal ultra god ) ou ( original maniac et ultra )

Le seul mode de jeu interessant est le plus difficile pour trois raison
-le contenu du jeu est entier
-il y a un score
- la difficulté est un défi meme pour les meilleurs joueur du monde

Le jeu est d'une difficulté sans nom en faite meme en mode moyen le jeu est dans le top 10 des jeux les plus difficile et possede 2 des 10 boss les plus difficiles .

Je vous laisse juger par vous meme ce jeu est génial et je le conseil vivement sur emulateur par exemple ( etant donné que le jeu est introuvable sur Ps2 ) ou sur le xbox live ( je croit que il n'est cependant pas disponible sur le xbox live francais ou un truc du genre ... )
