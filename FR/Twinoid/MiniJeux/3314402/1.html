Je n'avais pas fait attention à ce message de nico99 :
J'ai essayé sur Gimp et la moitié de mes images sont quasi-transparentes. Il devient donc impossible de faire quoi que ce soit avec... Je voulais ajouter mon nom dessus, pour votre information.
C'est normal : une image GIF animée est généralement sauvegardée de manière optimisée en n'incluant que les différences d'une phase de l'animation à la suivante. Le reste de l'image est transparent pour que les pixels de l'étape précédente soient visibles.

La solution : Filtres -> Animation -> Désoptimiser : chaque calque contiendra alors une copie complète de l'image.

La procédure habituelle pour modifier une animation est :
Fichier -> Ouvrir
Filtres -> Animation -> Désoptimiser
(optionnel si on veut modifier les couleurs) Image -> Mode -> RVB
Modification des calques de l'animation : c'est ici qu'on travaille...
(optionnel si on a modifié les couleurs) Image -> Mode -> Couleurs indexées
Filtres -> Animation -> Optimiser ("différence" ou "pour GIF" suivant les cas)
Fichier -> Enregistrer sous...
Pour les changements mineurs, il n'est pas nécessaire de désoptimiser : on peut travailler directement sur les calques transparents si on sait ce qu'on fait. Mais pour des changements plus importants, il vaut mieux faire comme décrit ci-dessus.
