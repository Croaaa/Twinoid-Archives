Vincenzo
Prenons une équation du second degré.

x²+x+1 = 0
x² = -x-1
x = -1-1/x

On va remplacer x dans l'équation de départ.

x²-1-1/x+1 = 0
x²-1/x = 0
x² = 1/x

x^3 = 1

La seule solution réelle de cette équation est x=1

Remplaçons x par 1 dans l'équation de départ ...

1²+1+1 = 0
3 = 0
0 = 3

2 + 2 = 2 + 2 + 3

2 + 2 = 7

Vous vous y attendiez, n'est-ce pas ? 7, c'est aussi le nombre de jours de ban, hein !
