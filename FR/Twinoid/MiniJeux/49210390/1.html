Ce sujet est pour organiser un Flashmob.

Flashmob: C'est un évènement dans lequel tous les participants se coordinent pour faire une même action.

Le jeu se joue en cycles de cinq jours.

Comment jouer?

Vous dites quelle personne vous souhaitez voir participer en vous servant du bouton "Joueur".
Il peut s'agir d'une personne ou plus.
Ensuite, vous lancez un dé de 100. Le score le plus haut gagne.
Vous dites quelle action vous souhaitez faire.

NE FAITES QUE DE BONNES ACTIONS, ou vous serez disqualifié.

Qui doit participer au Flashmob ?
Tous les gens qui ont participé au jeu pendant le cycle de cinq jours.

Où faire le Flashmob ?
Au cinquième jour, je créerai un message privé de masse incluant tous les participants.

Si vous vous comportez mal dans le message, vous serez disqualifié du jeu définitivement.

Je posterai ici les actions des gagnants.
