Salut alors les règles de ce jeux:
- le joueurs ayant marquer un message sur ce topique le plus proche de minuit mais pas après minuit gagne des points.
- Si l'ont marque 1 message et que l'ont est avec le message le plus proche de minuit ont gagne 10 points.
- Si l'ont marque plus d'un message et que l'ont est avec le message le plus proche de minuit ont gagne des points.
(-1 point pour chaque message sans compter le premier)
- Le but est d'arriver a 100 points
- Interdit de marque plus de 10 message par jour
