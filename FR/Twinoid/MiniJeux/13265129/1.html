Bienvenue dans le jeu "Le Tigre !"

Dans ce jeu, votre but sera de capturer le tigre en premier (il y aura 5 participants).

Le Plateau du Jeu.

Le jeu se passe sur un plateau Dino-RPG, que voici:

Les 0, 1 et 2 de fin correspondent à 10, 11 et 12.
Légendes:
: Case vide.
: Tigre.
: Avatar d'une équipe, selon ce qu'a choisi le joueur.

Le Fonctionnement du Jeu.

Chaque jour, chacune des équipes (et le MJ aussi) lancera 1d6.
Si le dé fait 1 pour un Joueur: Le Joueur se rapproche d'une case vers le Tigre.
Si le dé fait 2: Le Joueur se déplace d'une case vers l'est.
Si le dé fait 3: Le Joueur se déplace de deux cases vers le nord.
Si le dé fait 4: Le Joueur se déplace de deux cases où il veut.
Si le dé fait 5: Le Joueur se déplace d'une case vers l'ouest.
Si le dé fait 6: Le Joueur ne se déplace pas.

Si le dé fait 1 pour le Tigre (MJ): Se déplace d'une case vers le sud.
Si le dé fait 2: Se déplace d'une case vers le nord.
Si le dé fait 3: Se déplace d'une case vers l'ouest.
Si le dé fait 4: Se déplace d'une case vers l'est.
Si le dé fait 5: Ne se déplace pas.
Si le dé fait 6: Se déplace d'une case vers le nord.

Pour gagner la partie, un des cinq joueurs doit être sur la même case que le Tigre. A ce moment, il lance un dé 4. Si il fait 1, 2 ou 3, il gagne la partie. Si il fait 4, le Tigre lui échappe et se déplace de deux cases.
