But : Ne pas obtenir le nombre que j'ai lancé

Le Joueur : Contient 3 vie depuis le commencement du jeu, dois posté tout les jour 3 D100

Le D100 : Il se trouve dans Gadgets RP et c'est le 4ème Gadget

Un petit + : la lettre permettra au joueur qui le désire de reprendre un vie par exemple la lettre et 0 je lance et je tombe sur 0, je regagne une vie, se moyen est possible quand cas de survie

Exemple : Le joueur lance 3 fois le D100 et je tombe sur X* - X - X
Le nombre du jour et X Le joueur perd 3 vie

X* : Pour vous valeur inconnue

Merci de joué fair-play avec les autre joueur et admettre que l'on a perdu un vie
