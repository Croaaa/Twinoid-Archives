Salut les Muxxiens !
Je reviens avec mon new's jeu : Elixir ! ( Gages, Asmodée éditions )
Pour animer un peu plus le cœur des Muxxiens actifs et oublier les kikoos ...
Pour revenir a nos moutons ( ) voici les règles :

Le but du jeu est simple, vous commencez la partie avec 4 sorts de 3 trouvailles.
Un sort peut être joué en donnant les trouvailles qu'il lui faut ...
Vous avez 1 sort qui se joue avec 1 trouvaille, 1 avec 2, 1 avec 3 & 1 avec 4;
Évidement plus le sort coûte de trouvailles, mieux il est !
Chaque jour, je vous donne un hasard 1 trouvaille qui figure parmi elles :
- Nez de troll, sachet de poudre d'escampette, canine de vampire, mesure de peur bleue, cheveux d'ange, grain de folie, herbe qui chante, langue de dragon, œuf d'homme-lézard, goutte de rosée mauve, dose de bonne humeur, teinture de pépin de citrouille, barbe d'ogre, once de poussière d'étoile, plume de griffon.
Pour les sorts vous verrez ( héhé )
le premier joueur qui à joué tout ses sorts à gagner, vous pouvez jouez un sort n'importe quand via MP me.
Les MP justement, chaque joueur doit m'envoyez un message : " [Jeu] Elixir [Saison 1] " c'est dessus que je vous dit vos sorts ( Gardez les secrets ), vos trouvailles et que vous m'informer des sorts lancés, c'est uniquement moi qui
dit les sorts lancés.
RP :
( Malheureusement, c'est beaucoup de travail pour moi et je ne peux accepter que 6 joueurs ! )
Donc vous incarnez 6 sorciers qui parlent des potins autour d'une table ronde ...
Voila, donc pour jouer il faut avoir une bonne écriture ( Anti kikoos ) et être très actifs.
A vos MP !
