Voici le jeu de contagion de zombies. Le but de ce jeu est pour les humains de tuer tous les zombies et pour les zombies, de tuer les humains. Il y aura le tableau suivant:











Il doit être updaté à chaque tour. Un tour se finit lorsqu'un citoyen, puis un zombie ont fini leur tour. L'ordre est la suivante: Un citoyen, Un zombie et tout le monde peut rejouer mais pas deux citoyen en ligne ou des zombies. Si vous voulez conquérir un point avec personne dessus, ca ne prend rien. Si vous voulez prendre un point avec un adversaire, vous devez lancer un d4 et si vous avez 4,vous le tuer et prenez sa place.
