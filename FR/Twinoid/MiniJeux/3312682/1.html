Hixe
Si tu trouves une image sur Internet: Clic droit => "Enregistrer sous" => Choisis un dossier où l'enregistrer (et souviens-en toi).
Lorsque tu as une image: "Mon profil" => Clique sur ton avatar vide => Tu arrives ici => "1. Avatar :" => "Choisir" => Sélectionne le fichier que tu veux en tant qu'avatar et clique sur "Ouvrir" => Enfin, sélectionne "Valider"

Voilà, tu as un nouvel avatar.
