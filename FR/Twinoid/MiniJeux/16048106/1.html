Voix OffDans des temps très ancien de l'ancien temps , un jeu aussi vieux que le monde et aussi pourri que Twilight fit son apparition sur les terre hostiles du .... Lavabo!

Salut tout le monde c'est Darknosis et je vais vous faire découvrir un jeu de chance et d'aventure nommé Bactoria !

Pour savoir les règles c'est plus bas

But du jeu:
Proliférer dans le lavabo et devenir le roi!

Pour cela des magasins seront à votre disposition

Comment sa marche ? :
Vous ramasserez des globules qui serviront d'argent pour survivre et achetez diverses trucs ...

Comment je crée mon perso ? :
Alors c'est un systeme spéciale d'abord vous m'était un dé de 4 pour savoir quelle sorte de bactérie vous est :

.Si vous tombez sur 1 : Vous êtes un microbe maigre (+2 VIT ;-3 DEF )
.Si vous tombez sur 2 : Vous êtes un microbe simple (5 points partout )
.Si vous tombez sur 3: Vous êtes un microbe siamoi (-2 VIT mais les attaques ont des chances pour faire le double de dommages )
.Si vous tombez sur 4 : vous êtes un microbe berzerk (+4 FOR +2 DEF - 2 VIT )

Vous aurez une compétence en début de partie (Aléatoire)

Et le système de combat ? :

Simple ! Lancez un Dé 4 pour déterminé le nombres d'ennemis (ils s'adaptent à votre niveau)

vous infligez des dégâts avec le dé de 4 et vous avez votre taux de réussite

1 raté...; 2 touché. ;3 critique!(+2 de dégats); 4 ACHEVEMENT! (x2 dégats)

Caractéristique :

Maigre: 5 FOR / 7 VIT / 2 DEF / 5 INT

Simple : 5 FOR / 5 VIT / 5 DEF / 5 INT

Siamoi : 5 FOR / 3 VIT / 6 DEF / 5 INT

Berzerk : 9 FOR / 3 VIT / 7 DEF / 5 INT

Voilà c'est fait n'oubliez pas de cubez et de me demandez des renseignement par MP

ALLEZ salut et bon jeu !
