Lorsque tu récupères /me, as-tu spécifié les champs que tu souhaites obtenir de historyHeroes ?

Voici un exemple fonctionnel pour ma part :

me?fields=historyHeroes.fields(charId,date,deathCycle,deathId,deathLocation,epitaph,group,heroId,log,rank,season,shipId,skillList,triumph,user,wasMush)

Intégration avec la librairie :

$mushApi = new MushAPI($authTwinoid->getToken());
$currentUser = $mushApi->getMe(historyHeroes.fields(charId,date,deathCycle,deathId,deathLocation,epitaph,group,heroId,log,rank,season,shipId,skillList,triumph,user,wasMush));

D'accord ! Non j'avais pas préciser les fields, je pensais que par défaut il renvoyait au moins un truc ^^"
Merci beaucoup ! ( D'ailleurs je savais pas qu'on pouvait préciser les fields comme ça aussi !! )

Du coup ca donne ça :

MushAPI.class

public function getMe($fields="creationDate,freeGroupTicket,hero.fields(id),historyHeroes.fields(charId),historyShips,id,mushXp,myGroups,paidGroupTicket,xp" )

public function getHistoryHero(){
$history_fields = "charId,date,deathCycle,deathId,deathLocation,epitaph,group,heroId,log,rank,season,shipId,skillList,triumph,user,wasMush";

return $this->getMe($fields="historyHeroes.fields($history_fields)" );
}

Ca fonctionne parfaitement, encore merci !
