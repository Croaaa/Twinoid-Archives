Salut, tu développes sous Linux ? (si c'est le cas je peux pas trop t'aider ^^) Mais si tu es sous Windows tu peux utiliser ce GUI là, il est assez simple.

Sinon pour les commandes tu auras principalement besoin de :

clone pour faire une copie du dépôt sur ta machine
pull pour rapparier les modifications faites sur le dépôt vers ta machine (git pull origin master si tu veux pull depuis la branche master)
commit pour enregistrer tes modifs
push pour envoyer tes modif vers le dépôt
