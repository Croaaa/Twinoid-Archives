On peut cacher un lien dans un message.
  • Avec une espace insécable comme nom de lien : Ici -> <- Là
  • Avec un smiley comme nom de lien : 


On peut cacher du texte dans un message qui ne sera visible qu'en le citant :
  • En interrompant une adresse avec une espace : Lien
  • En ajoutant un "=Truc" à une balise [] qui n'en prends normalement pas : Caché
  • Dans le pseudo d'une balise pseudo-ID : warp
