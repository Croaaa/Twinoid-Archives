Malheureusement, les solutions proposées jusqu'ici sont incorrectes ou très inefficaces. Il ne faut pas utiliser ImageIO.read() pour lire l'image car tout ce qui se trouve dans un fichier JAR doit normalement être utilisé à l'aide de getResource().

Le premier lien (java.fr) donne la bonne solution, en faisant :
ImageIcon image = new ImageIcon(this.getClass().getResource(nomImage));
setIcon(image);

Attention : le répertoire contenant les images se nomme "images", pas "img" ou autre chose.

Voir aussi la documentation de Java (en anglais) qui explique différentes manières d'organiser les ressources images pour qu'elles soient accessibles par getResource() (bien que cette documentation soit un peu vieille et n'explique pas vraiment comment bien utiliser les packages).

Je répète : Il ne faut pas utiliser ImageIO.read() pour lire l'image car tout ce qui se trouve dans un fichier JAR doit normalement être utilisé à l'aide de getResource().

Dans ce cas-ci, tu ne sembles pas utiliser de package donc tu devrais pouvoir accéder à l'image à partir de getResource("image.png") ou getResource("images/image.png") dans un "new Image(...)" ou "new ImageIcon(...)".

Par exemple (non testé) :

java.net.URL imageURL = Panneau.class.getResource("images/image.png");
if (imageURL != null) {
 ImageIcon blablabla = new ImageIcon(imageURL);
}

Remarque : je te conseille de lire un peu la documentation de getResource() et aussi de trouver un tuto concernant les packages. Il n'est pas obligatoire d'utiliser un package mais cela permet de mieux organiser son code.

Ce n'est pas toujours marrant de lire la documentation surtout quand on veut s'amuser à coder des trucs qui fonctionnent tout de suite, mais souvent il vaut mieux investir une heure ou deux à lire pour bien comprendre tout, au lieu de procéder par essais et erreurs et obtenir quelque chose qui fonctionne plus ou moins mais sans qu'on comprenne vraiment pourquoi. Parce qu'une fois qu'on a compris, c'est beaucoup plus facile de trouver et de corriger des erreurs ou d'utiliser cette connaissance pour des projets plus complexes.

Pour convertir ton code actuel en package, il suffit d’insérer en début de chaque fichier java quelque chose comme "package org.elektro;" et de déplacer les fichiers dans un répertoire /src/org/elektro/... Il y a d'autres détails à régler mais je te laisse le soin de les découvrir.
