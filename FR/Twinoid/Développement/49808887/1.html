En regardant vite fais sur ton profil, je vois un moyen de différencier les dinozs qui sont congelés de ceux qui ne le sont pas.
De là, il sera possible de les masquer.

Je n'ai malheureusement pas internet le soir, donc difficile pour moi de bricoler cela.

Mais pour les intéressés :
Il y a une balise <embed>, avec un attribut "flashvars". Dans cet attribut, il y a la donnée "&status=congel". Il suffit de faire un script cherchant cette info et de masquer le div parent lorsqu'on la trouve.

Exemple :

<embed 
type="application/x-shockwave-flash" 
src="http://data.dinorpg.com/swf/31/dino.swf" 
style="undefined" 
id="dino_1984439" 
name="dino_1984439" 
bgcolor="#fce3bb" 
quality="high" 
allowscriptaccess="always" 
flashvars="data=BAChtNHi7d68u000&amp;chk=105045486&amp;damages=0&amp;status=congel" 
menu="false" 
wmode="transparent" 
height="148"
width="170">

Pour récupérer le flashvars

dataToTest = document.getElementsByTagName("EMBED")[i].getAttribute("flashvars")

Avec i, un nombre de 0 au nombre d'élement de la liste de Dinos.
Pour faire le test :

 patt = new RegExp("&status=congel")

Et le test :

 patt.test(dataToTest)

Si true, tu caches, sinon rien.

Désolé, de pas pouvoir te sortir le script exact fonctionnel, mais c'est possible à faire. Je laisse un motivé bricoler ça
