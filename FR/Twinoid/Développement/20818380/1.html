Procédure à suivre pour demander l'ajout dans l'annuaire d'une application Hordes :

La procédure de validation pour les applications Hordes est maintenant la même que pour la validation des applications Twinoid, vous devez donc contacter la guilde de l'Architectoire en passant par la page "Contacter un groupe officiel" et fournir l'id de votre application sur TwinOid.


L'ancienne API n'étant plus maintenue, plus aucune modification ne sera faite pour les applications dépendants de celle-ci.

Pour toutes demandes supplémentaires ( news, hébergement ... ) ou questions, vous pouvez également passer par la guilde si cela concerne un site développé autour des jeux de Motion-Twin.

Bon courage pour vos développements d'applications,
La Guilde de l'Architectoire.
