Pour les caractéristiques du jeu, c'est comme Pokémon :

  • déplacement case par case avec des touches de directions
  • terrains créés case par case :

Exemple :

0000000000
0111111110
0122222210
0122222210
0122222210
0111111110
0000000000

0 = noir ; 1= arbre ; 2 = chemin

  • chaque terrain est accompagnée d'une carte de mouvement (1 on peut aller sur la case, 0 on ne peut pas)
  • chaque terrain est accompagnée d'une carte de script (chaque case peut avoir un script qui est appelé lorsque le joueur va sur la case)
  • chaque terrain peut appeller un script lorsque le joueur y entre
