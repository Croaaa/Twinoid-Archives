Une alternative parmi tant d'autres :

Fais ton plan en deux exemplaires :
- Un pour l'affichage.
- Un second qui a chaque pièce remplie avec une couleur qui lui est propre.
(avec des calques sous toshop/gimp pour pouvoir facilement éditer et réexporter)

Ces deux plans doivent être exactement à la même échelle (encore que y'a moyen de "downscale" le plan coloré, mais ça ajoute de la complexité pour pas grand chose) et avoir les murs qui coïncident, donc.

Le principe : quand la souris se déplace en (x,y) sur le plan affiché, tu regardes le pixel en (x,y) sur le plan coloré. La couleur de ce pixel => la pièce, plus qu'à afficher les infos correspondantes comme tu veux / où tu veux.

Objectif : Réaliser une "map" de l'espace, qui affiche à partir d'un point (la position actuelle en [X;Y]) affiche toutes les cases autour dans un périmètre de Z cases).
(Z étant assez variable)
Et afficher le tout sous la forme d'un <table> html avec des <tr> et <td> classiques.

J'ai créé un algorithme qui me liste toutes les cases nécessaires (c'est ok, j'ai bien toutes les cases).
Mais mon algo a 2 soucis : Les cases ne sont pas "dans l'ordre" (+ l'origine n'est pas affiché).
Image concrète montrant le soucis : ici

Si vous pouviez m'aider à trier tout ça

Algo qui liste toutes les cases :

function generateSquare($origin, $size)
 {
 //$origin = array(X, Y);
 $boxes = array();
 for($z = $size; $z > 0; $z--)
 {
 //On va chercher les bords du carré
 $A = array( $origin[0]- $z, $origin[1] + $z);
 $B = array( $origin[0] + $z, $origin[1] + $z);
 $C = array( $origin[0] + $z, $origin[1] - $z);
 $D = array( $origin[0] - $z, $origin[1] - $z);
 
 //On génére les segments AB, BC, CD, AD
 //AB - Y = celui de A et B. X appartient [Xa, Xb]
 for($x = $A[0]; $x <= $B[0]; $x++)
 {
 $boxes[$x][$A[1]] = 1;
 }
 
 //BC - X = celui de B et C, Y appartient [Yc, Yb]
 for($y = $C[1]; $y <= $B[1]; $y++)
 {
 $boxes[$B[0]][$y] = 1;
 }
 
 //CD - Y = celui de C ou D. X appartient [Xd, Xc]
 for($x = $D[0]; $x <= $C[0]; $x++)
 {
 $boxes[$x][$C[1]] = 1;
 }
 
 //AD - X = celui de A et D. Y appartient [Yd, Ya]
 for($y = $D[0]; $y <= $C[0]; $y++)
 {
 $boxes[$A[0]][$y] = 1;
 }

 //On fait la même chose sur le carré de taille Z-1, au final on a toutes les cases
 }
 return $boxes;
 }
