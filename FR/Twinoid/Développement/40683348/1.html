J'ai pas regardé le code en détail, mais pour savoir si un caractère est une lettre tu peux regarder le code ascii de ce caractère et vérifier s'il correspond à une lettre.

Apparemment, en python ça peut se faire avec

ord('a')

qui te renvoie le code en décimal.

Sachant que les lettres majuscules sont entre 65 (A) et 90 (Z), et les minuscules entre 97 (a) et 122 (z).

La méthode "isalpha" est là pour ça. Et a l'avantage d'être portable (elle ne dépend pas du codage ; même si je suppose que le code ascii n'est pas sur le point d'être remis en cause un jour, puisque UTF8 l'inclus)

"toto".isalpha() répond vrai
"bla13bla".isalpha() répond faux
