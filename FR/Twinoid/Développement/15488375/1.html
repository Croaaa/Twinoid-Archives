A 2 0 0 1 1 0
B 0 2 0 1 0 1
C 0 0 2 0 1 1

donc 1 chance sur 6 d'avoir 2 zombies,
2 chances sur 6 d'avoir 1 zombies,
3 chances sur 6 d'avoir 0 zombie.

L'algorithme que j'ai créé est basé la dessus. Ca n'est que des stats.
