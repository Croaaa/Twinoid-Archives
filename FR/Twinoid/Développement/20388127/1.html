Petits conseils perso pour éviter les soucis présentés par Nadawoo. Tout le monde connait les accolades pour les conditions, les boucles, etc ... Mais sans un IDE adéquat, on en arrive rapidement à se perdre dans son propre code (sans même parler des gens qui n'indantent pas leur code).

Petite astuce pour ne plus jamais se tromper, il suffit de changer de syntaxe :

if(...) {
...
} else {
....
}
devient
if(...):
...
else:
...
endif;

De même,

foreach(...):
....
endforeach;
switch(...):
...
endswitch;
while(...):
....
endwhile;

Ca marche avec pratiquement tout. C'est un conseil que je donne à tous mes élèves pour simplement pouvoir se repérer dans votre code plus facilement. Je ne garde les accolades que pour les fonctions et les classes.
