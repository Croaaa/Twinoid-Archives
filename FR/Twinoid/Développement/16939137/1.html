Bonjour a tous,

Je viens d'ouvrir l'un de mes cartons qui traîne ( et j'en ai un bon paquet ) pour retrouver le projet d'un application mi-RP/mi-classement pour Croque Motel.

Le principe :
Une sorte de guide du routard version CroqueMotel, chaque joueur peut rédiger un petit descriptif complet pour son hôtel et via CroqueMonster, il est possible de laisser un avis par le biais de ses monstres. Avec quelques distinctions déblocables ( ce qui rajoute quelques objectifs dans CroqueMotel )


Mon but serait de laisser une certaine liberté pour la rédaction du descriptif, de fait il s'agit d'un contenteditable mais ne sont normalement accepté quelques balises et tout injection de script est censé être tronqué.

C'est justement pour ça que j'aurais besoin d'un peu d'aide, je voudrais que vous tentiez de faire des injections dans cette description.

Pour pouvoir y accéder, il faut que je sois dans votre liste d'amis TwinOid ( si ce n'est pas le cas, vous pouvez laisser un message ici ) et après vous êtes connecté sur un jeu Muxxu que vous accédiez a Cette adresse.

Le design va être légèrement revu, et un bug fait qu'actuellement certaines description sont raccourcies ( pas encore réussi a trouver les raisons de ce bug, si vous avez des idées sur ce qui peut causer ça ... )

Merci d'avance et bonne journée
