Le probléme ne venais pas du Json mais du script php qui le générais.

Si tu a un probléme avec le Json n'hésite pas a demander (sur ce sujet) je te répondrais volontiers.

Voici un article sur le json (en anglais) : ecma pdf json syntaxe

Pour exploiter simplement du json en php :

<?php
$json = <<<JSON
{
 "objet": {
 "item1": "content-1",
 "item2": "content-2"
 },
 "array": [
 "valeur1",
 "valeur2",
 "valeur3",
 "valeur4"
 ]
}
JSON;
$json = json_decode($json);
echo "<pre>";
print_r($json);
echo "</pre>";
echo "<hr>";
$objet = $json->objet;
echo "Object content :<br>";
echo $objet->item1.'<br>'.$objet->item2.'<br>';
echo "Array content :<br>";
foreach($json->array as $array) {
 echo $array."<br>";
}
?>

Voila voila, j’espère couvrir les éventuels question que tu te posais avant de tomber sur ce sujet.
