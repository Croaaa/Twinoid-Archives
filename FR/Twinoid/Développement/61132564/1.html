Ce programme est certainement fait avec de bonnes intentions, mais la réalisation ne respecte pas les CGU de Motion Twin concernant l'utilisation de programme externes.
Ne sont tolérés que les users scripts et les sites annexes (soumis à leurs propre règles).

Qui plus est, nous rappelons aux personnes ayant déjà téléchargé ce programme qu'installer un logiciel développé par un tiers n'est pas sans risques. Un script malveillant pourrait prendre le contrôle de votre ordinateur et pourrait avoir accès à vos mots de passe, à vos données personnelles, à tous les sites que vous visitez, etc. Soyez donc très prudent avant d'installer un tel script.

Si on en croit Virus Total ce programme est détecté comme pas clean par plusieurs anti-virus. Ce sont surement des faux positifs, mais le risque potentiel que présente ce programme pour nos utilisateurs nous oblige à le supprimer des forums. Nous avons donc supprimé les liens pour le télécharger. En aucun cas ce retour ne peut venir de l'hébergeur. C'est plus probablement un soucis dans le code et/ou le compilateur sur l'ordinateur de l'auteur (ordinateur déjà compromis ?)
Nous notons aussi qu'un simple fichier Google Calc ferait exactement la même chose, sans aucun risque pour l'utilisateur.

Nous ne pouvons que vous suggérez de contacter l'architectoire pour voir avec eux ce qui est possible de faire.
