Comment faire un copier coller sur son ordinateur?

Dans ce deuxième cours de la série Bien organiser son ordinateur, je vous propose 2 techniques pour apprendre à effectuer des copier coller de vos fichiers et dossiers.

Objectifs

Savoir copier un fichier ou un dossier
Savoir coller le fichier ou le dossier où l’on souhaite
Faire un copier coller : L’essentiel
Je vous présente deux techniques pour faire un copier coller. Voici un résumé de ces deux techniques, à vous de choisir celle qui vous convient le mieux. Dans la suite du cours, je développe ces deux techniques sur un exemple concret.

Copier Coller : raccourcis clavier
Sélectionnez avec un clic gauche le fichier à copier,
Pressez en même temps les touches Ctrl et C pour copier,
Allez à l’emplacement où vous désirez placer la copie,
Appuyez simultanément sur les touches Ctrl et V pour coller,
La copie de l’élément (fichier ou dossier) apparaît

Copier Coller : menu contextuel

A quoi ça sert de copier coller des documents ?
Pour organiser son ordinateur, il est important de connaître les techniques de copier/coller. Cette manipulation peut se faire aussi bien sur les fichiers (photos, documents…) que sur les dossiers. Les techniques que je vous présente dans ce cours peuvent même être appliquées dans divers logiciels :

un traitement de texte, comme word, open office(logiciel gratuit) ou le bloc note. Vous pourrez copier et coller du texte, dupliquer rapidement un paragraphe ou une phrase pour agrémenter la mise en forme ou revoir la mise en page de vos documents.
Dans votre navigateur internet, vous avez possibilité de copier du texte pour ensuite le coller dans un traitement de texte.
Ça marche également dans les tableurs comme excel dans lequel vous pouvez effectuer le collage de cellules.
La technique du coller vous sera utile lors de la capture d’écran pour déposer votre capture dans un logiciel tel que Paint…
Dans la pratique, le copier coller vous permet d’obtenir une copie exacte d’un élément de votre ordinateur. Cela peut être intéressant par exemple si vous souhaitez appliquer une correction ou réduire la taille d’une photo avant de l’envoyer à un ami par mail. Dans ce cas, vous copiez la photo dans le dossier de votre choix, vous manipulez cette photo et vous conservez la photo d’origine dans son emplacement initial, soit sur le disque dur ou la carte sd de votre appareil photo numérique.

Autre cas d’usage, vous pouvez vous constituer des modèle de documents, lorsque vous souhaitez écrire un nouveau document vous copiez le modèle d’origine dans un dossier et vous le modifiez par la suite. Vous conservez ainsi le modèle d’origine qui pourra vous resservir dans le futur.

Ces techniques vont également avoir une grande utilité lorsque nous allons aborder les notions de sauvegarde de données.

La technique du copier/coller s’applique dans l’explorateur de fichiers mais aussi sur votre bureau Windows.

Le copier/coller : cas pratique
Pour illustrer ce tutoriel, je vais m’appuyer sur un exemple concret sur lequel vous pourrez facilement vous inspirer pour gérer tous vos fichiers et vos dossiers.

Je souhaite copier une image qui se nomme Paysage1 dans un dossier A envoyer afin de pouvoir la réduire pour la transférer par mail à mes amis. Une fois que j’aurai envoyé mon mail, je souhaite copier mon dossier A envoyer dans un dossier qui se nomme A effacer.

Je dispose donc pour cet exemple de 3 éléments :

Un fichier image qui se nomme Paysage1, il s’agit d’un fichier au format JPG.
Un dossier A envoyer qui est vide au début de l’exercice.
Un dossier A effacer qui est également vide au début de l’exercice.
La première étape va consister à copier le fichier Paysage1 dans le dossier A envoyer.

La seconde étape consiste à copier le dossier A envoyer et son contenu (fichier Paysage1) dans le dossier A effacer.

Voyons de suite comment effectuer ces actions.

Publicité
Comment faire un copier coller ?
Il existe plusieurs moyens d’effectuer un copier/coller sur un ordinateur. Dans ce cours je vous en présente deux qui ont l’avantage d’être universels, ces techniques peuvent être réalisées quelque soit la version de Windows.

La première technique utilise les raccourcis clavier et sera appliquée pour réaliser la première étape de mon exemple.

Pour réaliser la deuxième étape de l’exemple, j’utilise la technique du menu contextuel que nous avons déjà abordé dans le cours sur la création de dossier.

Libre à vous de choisir la technique dans laquelle vous vous sentez plus à l’aise, le résultat sera le même quelle que soit la technique choisie. Vous pouvez même mixer les deux techniques, cela n’a pas d’importance.

Copier Coller : Les raccourcis clavier
Le but est donc de copier notre fichier image Paysage1 dans le dossier A envoyer.

Etape 1 : Sélectionner le fichier que vous souhaitez copier
Faîtes glisser le curseur de votre souris puis double cliquez sur l’icône du dossier qui contient le fichier que vous souhaitez copier. Sélectionnez le fichier en cliquant une fois dessus avec le bouton gauche de la souris. Le fichier doit apparaître en surbrillance.

comment faire copier coller
Sélection du fichier à copier

Etape 2 : Dire à l’ordinateur que vous souhaitez copier ce fichier
Après avoir cliqué sur le fichier, il faut taper simultanément sur les touches ctrl et C de votre clavier d’ordinateur. Pour faire cette manipulation, il faut procéder de la sorte :

Appuyez d’abord, avec un premier doigt, sur la touche ctrl.
Vous devez garder cette touche enfoncée pendant que vous appuyez, avec un deuxième doigt, sur la touche C de votre clavier.
Ensuite, il suffit de relâcher les deux touches pour que l’ordinateur garde en mémoire dans le presse papier, le fichier que vous avez copié. La vidéo du cours qui se trouve au début de cet article vous montre comment effectuer cette action.
Etape 3 : Coller le fichier dans le dossier souhaité
Avant toute chose, vous devez vous rendre à l’endroit où vous souhaitez coller le fichier. Pour notre exemple, j’ouvre donc le dossier A envoyer.

Pour coller le fichier, il me suffit d’appuyer en même temps sur les touches Ctrl et V de mon clavier de la même façon que le Ctrl et C pour copier.

Une fois cette action réalisé le fichier est collé et apparaît dans son nouvel emplacement, ici dans le dossier A envoyer.

Copier Coller : menu contextuel
Le but est donc de copier le dossier A envoyer et son contenu dans le dossier A effacer.

Etape 1 : Sélectionner le dossier que vous souhaitez copier
Dans un premier temps, il faut placer le pointeur de la souris et cliquer avec le bouton gauche sur l’icône du dossier à copier. Le dossier apparaît en surbrillance. Cette étape est optionnelle car vous pouvez directement faire l’action présentée dans l’étape suivante.

Etape 2 : Dire à l’ordinateur que vous souhaitez copier ce fichier
Faîtes un clic droit sur le dossier que vous souhaitez copier, le menu contextuel apparaît. Choisissez l’option Copier dans ce menu. L’ordinateur garde en mémoire le dossier que vous désirez copier.

copy
Menu contextuel – Copier

Etape 3 : Coller le dossier dans le dossier souhaité
Accédez à l’endroit où vous souhaitez coller le dossier. Pour notre exemple, j’ouvre donc le dossier A effacer.

Pour activer le collage du fichier, il me suffit de faire un clic droit sur une zone vide de mon dossier, le menu contextuel apparaît. Choisissez l’option coller dans ce menu. Le dossier et son contenu est ajouté dans l’emplacement désiré.

faire un copier coller
Menu contextuel – Coller

Le temps de copie d’un fichier ou d’un dossier dépend essentiellement de sa taille. Pour les dossiers, le nombre d’éléments peut également faire varier le temps de copie.

Résultat
La copie de document n’efface pas le document d’origine, ce type d’action correspond à la technique du déplacer un fichier ou un dossier que nous verrons dans notre prochain cours.

Ainsi avec les manipulations que j’ai réalisées dans ce cours j’ai à présent:

Le fichier image d’origine Paysage1.
Le dossier A envoyer d’origine qui contient désormais une copie du fichier image Paysage1.
Le dossier A effacer d’origine qui contient désormais la copie du dossier A envoyer qui contient lui-même une copie du fichier image Paysage1.
Ces 2 techniques devrait vous suffire pour démarrer l’organisation de votre ordinateur. Une autre technique est disponible à partir de la barre d’outils de l’explorateur de fichiers. Je ne l’ai pas présentée ici car elle me semble moins pratique et elle ne s’applique que dans l’explorateur de fichiers.
