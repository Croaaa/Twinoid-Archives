Les débordements

Tous les zombis n'attaquent pas forcément la ville. Certains peuvent errer sans but. Pour calculer le nombre de zombies maximum qui attaqueront les maisons, il convient d'utiliser la formule suivante : Nb de Zombies qui attaquent = X * Nb de personnes en ville

où X varie selon le jour où l'attaque se produit

De J6 à J14 : X = Jour + 1
De J15 à J18 : X = Jour + 4
De J19 à J23 : X = Jour + 5
A partir de J24 : X = Jour + 6

En mode chaos, Nb de personnes en ville est égal à 10 même si le nombre de citoyen est inférieur.

Exemple : attaque de 1485 zombis J12 avec une défense de 1088 dans une ville de 21 survivants (doit être inférieur à 25). 397 zombis sont censés passer. Nb de zombis qui attaquent : 13*21 = 273. Donc il devrait y avoir 124 zombis errant.
