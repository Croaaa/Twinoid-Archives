Bonjour,

Si le nombre de zombie est supérieur au défense du joueur :
Le joueur meurt


Si le nombre de zombie est inférieur ou égal au défense du joueur :
Ou le joueur devient terrorisé
Ou le joueur a des chances d'avoir le message indiquant qu'il échappe à la terreur.

Le message évoque des "bibelots" à serrer contre soi, les joueurs pensent que la décoration augmente les chances de l'obtenir, mais des cas furent remontés d'avoir ce message sans objets de décoration.
D'autres joueurs ont évoqués les hypothèse suivantes :
- Les objets, n'importe lesquels permettraient d'augmenter les chances.
- Ranger son habitat augmenteraient les chances.

Mais aucune de ces hypothèses ne fut certifiées.

Enfin, je rappelle que Twinpédia est un Wiki par les joueurs pour les joueurs et n'a rien d'officiel, il contient donc des erreurs
