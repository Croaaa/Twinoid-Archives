Twinpédia indique ceci pour les zombies errant

Tous les zombies n'attaquent pas forcément la ville. Certains peuvent errer sans but. Pour calculer le nombre de zombies maximum qui attaqueront les maisons, il convient d'utiliser la formule suivante : Nb de Zombies qui attaquent = X * Nb de personnes en ville

où X varie selon le jour où l'attaque se produit

De J6 à J14 : X = Jour + 1
De J15 à J18 : X = Jour + 4
De J19 à J23 : X = Jour + 5
A partir de J24 : X = Jour + 6

En mode chaos, Nb de personnes en ville est égal à 10 même si le nombre de citoyen est inférieur.
