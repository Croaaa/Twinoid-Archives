Bonjour,

Pour clarifier les états :

Si le statut "désaltéré" est présent dans la barre d'état :
Rien => Rien le lendemain
Soif => Soif le lendemain
Déshydraté => Mort le lendemain

Si le statut "désaltéré" n'est pas présent dans la barre d'état :
Rien => Soif le lendemain
Soif => Déshydraté le lendemain
Déshydraté => Mort le lendemain

Précisions complémentaires :
L'utilisation d'un hydratone fait passer d'un état de soif à celui précédent mais ne remet pas le compteur de PA à 0.
CAD si vous êtes en soif, que vous faites 10 déplacements, prendre l'hydratone vous passera en "normal", le fait d'avancer de 1 PA vous repassera immédiatement en soif.

Le pouvoir Vaincre La Mort (VLM) bloque l'évolution des états uniquement durant l'attaque. Marcher dans le désert modifie vos états.
