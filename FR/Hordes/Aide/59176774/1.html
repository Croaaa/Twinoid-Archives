Salut !

Le terme "amélioration" a été mal utilisé, ce n'est utilisable que dans les travaux de maison d'après twinpedia
Permet d'avoir tous les jours 6 Points de Chantiers (6 Icone PC) utilisables, comme leur nom l'indique, dans les chantiers mais aussi pour les travaux de maison (coin sieste, cuisine, etc). Par contre, ils ne sont pas utilisables dans les améliorations de maison (tente, taudis, etc).
