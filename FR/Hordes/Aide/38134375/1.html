Bonjour,

Actuellement, le support pour les problèmes techniques n’est plus très utilisé à cause de la lourdeur de son fonctionnement, il reste lu mais plus de manière quotidienne et efficace.

De ce fait, il est généralement plus efficace de remonter un problème sur la section Aide pour avoir une réponse rapide et espérer un correctif plus prompt.

Merci de lire entièrement ce sujet avec les différentes procédures.

Bug ou suspicions de bug dans le jeu.
Bug ou suspicions de bug hors ville.
Problème en rapport avec l’achat de JH
Une coquille ou faute d’orthographe.


Avant de remonter un problème, merci de suivre ces consignes :

  • Désactiver l’ensemble de vos scripts externes :
    TamperMonkey (Chrome), GreaseMonkey (Firefox), Stylish.
  • Appuyer sur CTRL + F5 ou SHIFT + F5
    Permet de recharger la page en vidant le cache.

  • En cas de bug, ou suspicion bug dans le jeu :
    • Ouvrez un nouveau sujet sur la section Aide.
    • Utilisez le tag optionnel “bug”.
    • Détaillez le mieux possible votre problème et la situation :
      • Nom de la ville et son ID.
      • Description détaillée du problème.

  • En cas bug ou suspicions de bug hors ville (Coalition, Aide, etc) :
    • Ouvrez un nouveau sujet sur la section Aide.
    • Utilisez le tag optionnel “bug”.
    • Détaillez le mieux possible votre problème.

  • En cas de problème lors d’un achat de JH :
    • Pour ce genre de problème il faut aller sur le support.
      Contact => [PAIEMENTS] J'ai besoin d'aide avec un paiement => Choix de la méthode de payement.
    • Description détaillée de votre problème.

  • Si vous trouvez une coquille ou une faute d’orthographe :
    • Veuillez la remonter sur ce sujet : http://twd.io/e/TpLh0g


Attention :
Les sujets ouverts avec le Tag Optionnel “Bugs” seront considérés comme des rapports de bugs.
Les Fakes seront très sévèrement punis.
Il est préférable de laisser les Oracles répondre.
