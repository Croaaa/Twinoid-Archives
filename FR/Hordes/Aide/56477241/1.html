Bonjour,

Oui et non. Il y a deux "flux de données" :

  • Le flux XML : ancien flux de données pas maintenu à jour.
  • L'API hordes : nouveau flux de données comportant des erreurs/bugs.

Il existe dans les options des VPs la possibilité de bloquer l'API hordes.
Cependant le flux XML, lui reste disponible.

Or par exemple BBH fonctionne avec le flux XML et ne sera pas bloqué.

A voir directement avec le Dev de BBH si il est possible de bloquer les MAJs d'une ville.
