J'arrive avec un peu de retard.

De manière générale, les problèmes, si vous êtes le seul à le subir, ne proviennent pas du jeu mais oude votre navigateur ou de vous.

En cas de problèmes, effectuez toujours les étapes suivantes :

  • Désactiver les scripts externes GreaseMonkey / TamperMonkey / Stylish.
  • Recharger la page en vidant les caches : CTRL + F5 et/ou SHIFT + F5.
  • Vérifier que Flash est à jour.
  • Vérifier que le javascript est activé.
  • Vider les caches du navigateur.

Si vous avez toujours un problème après ces étapes :

  • Vérifier si cela fonctionne via un autre navigateur.
  • Indiquez le sur cette section. Il est possible que le problème soit plus "important" et "général".
