explicationsTapez about:plugins dans la barre d’adresse en haut de la fenêtre du navigateur Chrome.

Cliquez sur Détails dans l’angle supérieur droit de la page.

Dans la page qui s’affiche, recherchez le module externeintégré dans la liste Flash (ou Shockwave Flash) et cliquez sur le bouton de désactivation associé. Pour identifier le module externe intégré, reportez-vous au tableau des noms de modules externes ci-dessus.

Dans la page qui s’affiche, recherchez le module externe système dans la liste Flash (ou Shockwave Flash) et cliquez sur le bouton d’activation associé. Pour identifier le module externe système, reportez-vous au tableau des noms de modules externes ci-dessus.

Fermez toutes les fenêtres Chrome et relancez le navigateur.
Chrome utilise le module externe système installé. Pour confirmer la version de Flash Player en cours d’exécution dans votre navigateur, consultez l’aide de Flash Player.
