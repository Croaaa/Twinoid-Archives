Il faut bloquer les éléments ayant une classe (CSS) "tid_block.tid_newGame".

Voici un exemple utilisant l'extension Stylish :

@namespace url(http://www.w3.org/1999/xhtml);

@-moz-document domain("twinoid.com"), domain("mush.vg"), domain("www.hordes.fr"), domain("www.naturalchimie.com"), domain("www.dinorpg.com"), domain("hotel.muxxu.com"), domain("kube.muxxu.com") {
 span.tid_block.tid_newGame {
 display: none !important;
 }
}
