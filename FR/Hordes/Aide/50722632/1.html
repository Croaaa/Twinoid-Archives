En sachant tout de même que comme l'a dit lechanceux, il faut l'état désaltéré pour conserver son état de soif.

Rappel général :
Changement d'état après l'attaque :
Normal => soif
Soif => déshydraté
Déshydraté => mort

Normal + désaltéré => normal
Soif + désaltéré => soif
Déshydraté + désaltéré => mort !
___

Changement d'état tous les 11 pa effectués dans le désert
Normal + 11 pa => soif
Soif + 11 pa => déshydraté
Déshydraté + 11 pa => mort
___

En commençant la journée en soif, il faudra donc faire 22 pa pour mourir.
S'il n'a pas bu aujourd'hui il commencera la journée déshydraté, donc mourra après seulement 11 pa.
