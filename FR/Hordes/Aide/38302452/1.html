Bonsoir l'ami,

Une ration d'eau empoisonnée ne peut pas être utilisée comme une ration d'eau saine, ce sont deux objets complètement différents avec la même icône, en fait elle peut simplement tuer.

De ce fait, il existe plusieurs moyens pour les distinguer.

Par exemple :

Il n'est pas possible de faire boire une ration empoisonnée par escorte.
Il n'est pas possible d'assembler une BAO ou PAO avec une ration d'eau empoisonnée
Il n'est pas possible d'utiliser une ration d'eau empoisonnée avec la bonbonne.
Il n'est pas possible de l'utiliser pour des chantiers et elle n'est pas cumulable en banque
Il n'est pas possible de l'utiliser pour sortir un cadavre. Il n'est pas possible de mettre une ration d'eau empoisonnée dans le puit

Certes, cette liste n'est pas exhaustive, mais je crois que tu as compris le principe
