Bonjour a vous, survivants !

Dans cette sugg je vous propose de pouvoir investir la banque c'est a dire de pouvoir se cacher pendant une nuit dedans.

C'est une action que tout les habitants pourraient faire.

C'est action n'est faisable qu'une fois par ville (comme une AH donc mais pour tout le monde.)

Elle permettrait de se cacher dans la banque après la nuit grâce a une action disponible "se cacher dans la banque".

Cet action serait considéré comme nuisible et donnerait un picto "Parasite" ainsi que les titres :

20 : -Cafard
50 : -le lâche
75 : -Peureux est mon deuxième prénom
100:-ne prenez rien chez moi !

Quiconque entrerait dans la banque avant l'attaque interromprait cette action (le citoyen retourne chez lui et vis comme les autres.)

Les avantages :

Vous avez 4 fois plus de défense par rapport a votre maison.
Les vols dans la banque ne sont pas signalés (vous êtes DANS la banque )
La banque devient votre coffre

Les désavantages :

Vous êtes 2 fois plus attaqués (votre odeur + celle des objets rapportés)
Utilisable 1 seule fois
Les objets de votre maison peuvent être volés, bien que vous soyez en ville.

Voila ce sera tout !

Si vous avez des questions dites les et j'accepte toutes les critiques CONSTRUCTIVES

Good luck for the night
