Bonsoir,

Une petite sugg pour rééquilibrer certaines choses et prévenir un risque de limitation possible !

Le but est très simple, hordes compte deux types d’âme :

-Les âmes fortes ( héros)
-Les âmes faibles (non héros)

Un joueur mourant dans le désert donne obligatoirement une âme faible qui faut ensuite affiner pour qu'elle puisse être utilisée dans les chantiers.

Le but est de cette suggestion est simple. Quand un héros meurt dans le désert l’âme que l'on retrouve est obligatoirement forte, il est impossible de la rendre faible.

Lorsqu'un habitant meurt dans le désert, l’âme est obligatoirement faible (comme actuellement)

Le but :

Une ville lambda classique n'a que faire des points de classement, dans beaucoup de cas elle n'aura de toute façon aucune chance d'y accéder, sans chaman la possibilité de faire des chantiers temporaires défensifs est nul. En revanche les métas coalitions chassant le classement auront besoin de points supplémentaires et on sait qu'actuellement les jump en pandé se font avec une très grande majorité de héros pour ne pas dire du 100% héros pour certain.

Cette contrainte supplémentaire obligera les métas à prendre en ville avec eux des non héros au lieu de se limiter à un jump 100% héros pour chasser la légende.
