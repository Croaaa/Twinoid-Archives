Hordepédia c'est quoi ?
Vous retrouverez toutes les explications sur le premier post de ce sujet.

 

Ces explications données, commençons.
Ce sujet servira au rassemblement des différentes données sur l'historique des regroupements de joueurs. Les Jambons, les Saucissons, Alpha Hordes, la Main Noire, les Chèvres, l'Interméta, la Junky Church, les Pim's, les Rhô, les CMK, les Bandes Décimées, les Viendez Comme Vous Êtes, les Architectes,...

L'histoire de hordes est jalonnée de regroupements de différentes formes, avec différents buts et différentes formes de sélections ou d'adhésion.

Je ne prétends pas pouvoir parler de tous les regroupements, le travail titanesque serait surtout inutile.
Mais la plupart des groupes cités ci-dessus ont marqué d'une manière ou d'une autre les joueurs.

Comme pour le sujet des events, je pense qu'un listing des différents regroupements à aborder dans l'historique de hordes est une première étape obligatoire avant de réellement écrire les différentes descriptions de ces regroupements.

Informations clés à détenir :
Nom du regroupement
Dates de création / Mort ?
But du jeu (opti / pourrisseurs / écoles / pictophilie / merveilles / ...)
Créateur(s)

Dans un second temps, il faudra pousser la recherche pour retrouver un contexte, des joueurs clés dans l'histoire du regroupement, des villes liées marquantes (je pense au CDL2 pour l'interméta, à la première explosion des FDA,...)
