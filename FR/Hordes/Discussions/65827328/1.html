à défaut de trouver votre bonheur avec la solution de tchekof :

C'est possible de farfouiller au pif sur la wayback machine :

https://web.archive.org/web/*/imgup.motion-twin.com/hordes/*

1) Sélectionnez le filtre [url] et ça vous donne une liste de toutes les images recensées.

2) Pour repérer lesquelles sont toujours disponibles sur le serveur MT, ouvrez la console et tapez-y ce code :

$("td.url a").each(function(){ var url = $(this).html(); $(this).before('<img src="'+url+'" width="50" height="50">'); })

3) ça ajoute une image devant chaque url de la liste, si elle n'est pas cassée, c'est qu'elle est disponible.

4) Parcourez la liste, refaites cette commande dans la console à chaque page jusqu'à trouver votre bonheur. Copiez l'url de l'image si c'est le cas.

5) Voyez le topic d'Eliam pour mettre à jour votre avatar

Près de 3000 images sont recencées. Vous pouvez orienter la recherche à twinoid (~8000) :

https://web.archive.org/web/*/imgup.motion-twin.com/twinoid/*

ou l'élargir à tout le serveur imgup (+10000) :

https://web.archive.org/web/*/imgup.motion-twin.com/*

La wayback machine bloque à 10000. Si vous spécifiez le sous-repertoire, vous aurez davantage de résultats : dinorpg, twinoid, muxxu, ....
