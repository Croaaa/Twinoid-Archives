En fait c'est plus ou moins déjà le cas.

Les application externes fonctionnent avec des informations envoyées par les serveurs de la MT.
Anciennement, ces informations étaient envoyées via un "Flux XML", ce sont l'ensemble des applications disponibles dans l'annuaire.
Ce fonctionnement + l'ancienne charte faisait qu'il y avait ce "disclaimer", cette page demandant confirmation.

Depuis, un nouveau système existe, l'API Twinoid/Hordes.
Ce système permet de ne passer qu'une seule fois via le "disclaimer" pour confirmer qu'on souhaite accéder à l'application.
Exemple pour BBH (pas accessible à tout le monde encore) :
- Connexion via Twinoid
- Validation de l'application
- Gestion des applications autorisées

Pour voir les applications existantes (tous jeux confondus) :
http://twinoid.com/developers/browse


Quand les applications seront passées via cette nouvelle API, il y aura plus besoin de passer via l'annuaire.
Il suffira d'aller sur le site en question, faire "connexion" et hop vous serez connecté.
