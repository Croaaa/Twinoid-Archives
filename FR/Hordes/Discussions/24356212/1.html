Combien de fois avez vous été banni par des trolls ? Ceci est énervant, pour le moins. Voici ce que je propose pour les en empêcher :

1. Si le citoyen n'est pas encore banni :

Ceci rajouterait à côté des dénonciations un bouton :
Bouton :Mais... C'est faux !
En passant la souris dessus, on verrait apparaitre :

Message :Cette accusation vous semble fausse... Vous pouvez la raturer
Si vous cliquez ici, cette dénonciation est barrée et elle n'est plus prise en compte.

Les actions "Mais... c'est faux" sur les autres dénonciations de ce citoyen sont barrées est non utilisables. Voici leur message quand vous passez la souris:
Message :Vous avez déjà raturé une dénonciation de ce citoyen ! En raturer plus serait suspect !
On pourrait même imaginer un avantage héros qui permettrait d'en raturer deux... La deuxième donnerait alors pour la deuxième :
Message :Les citoyens ont grande confiance en vous. Vous pouvez raturer une deuxième fois, mais attention, pas une troisième !

2. Ci un citoyen est déjà banni :

Les boutons "Mais... C'est faux !" seraient barrés. En passant la souris dessus :
Message :Ce citoyen est déjà banni... C'est trop tard.

Mais ci ce citoyen est aux portes de la ville, il dispose de l'action :
Nom de l'action :Plaider sa cause !
Cette action, activable une fois par journée et valable pour la journée tant que le citoyen reste aux portes, ajouterait sur la page du citoyen banni ce message :

Message :Ce citoyen a plaidé sa cause ! Vous pouvez décider de le réhabiliter...

Alors, le bouton mais... C'est faux serait remplacer par un bouton "réhabiliter" avec cette description :
Message :En cliquant ici, vous raturez cette dénonciation... Si toutes les dénonciations sont raturées, le citoyen est réhabilité !

Ce message barrerait les autres actions "réhabiliter" (même message et même possibilité pour les héros), ainsi que cette dénonciation. Une fois toutes les dénonciations raturées, le citoyen est réhabilité (il n'est plus banni) et gagne le picto "réhabilité" x1.

Précisions :

1. On ne pourra dénoncer un citoyen qu'une fois par partie, même s'il est réhabilité, d'autres personnes devront les bannir.

2. Le picto ne mène à aucun titre car ce serait trop avantageux pour les métas. (Se faire bannir/réhabiliter 2-3 fois par ville...)

Qu'en dites vous ?
