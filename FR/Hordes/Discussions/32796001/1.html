Bonjour à tous,

je trouvais très dur de trouver des paracétoine ou des bandages alors j'ai eu une idée: Le désinfectant Chapik 100ml

Description:
Ce petit désinfectant vous permettra d'éloigner les vilaines bactéries de vos gros bobos. Il pique un peu, allez on serre les fesses, ça ira mieux après... Bizarre quand même ces taches de sang sur la bonbonne...

Utilité:

Le désinfectant Chapik 100ml permet de désinfecter une blessure et qu'elle ne s'infecte pas, mais attention la plaie est toujours ouverte. Le désinfectant a 3 charges, il en perd une à chaque utilisation (une seule utilisation par jour par personne).

Effet:

90% de chance de désinfecter la plaie. Obtention du statut "désinfecté(e)" qui empêche l'obtention du statut "infecté(e)" si une blessure est présente durant l’attaque de la nuit (perte de l'état "désinfecté(e) après l’attaque).

Description de l'action:

  • "Vous pulvérisez le désinfectant sur votre plaie, Aïe Aïe Aïe ça pique, mais ça ira mieux après vous verrez"
    ____________________________________________________________ Vous êtes maintenant désinfecté(e)

5% de chance d'infecter la plaie immédiatement. Obtention de l'état "infecté(e)".

Description de l'action:

  • Vous pulvérisez le désinfectant sur votre plaie mais vous vous rendez compte avec horreur que du sang sort du spray! Très surement celui d'un citoyen infecté...
    ____________________________________________________________
    Vous êtes maintenant "infecté(e)".

5% de chance de soigner définitivement la blessure. Perte de l'état "Blessé(e)" uniquement si la blessure n'est pas infectée.

Description de l'action:

  • Vous pulvérisez le désinfectant sur votre plaie. Après quelques instants de douleur intense, vous saignez et la plaie commence à se refermer, mais qu'avait t-il dans ce désinfectant?
    ____________________________________________________________
    Vous êtes guéri(e).
    ____________________________________________________________
    Vous êtes maintenant "convalescent(e)"

Cas particulier:

20% de chance de soigner l'infection. Perte de l'état "infecté(e) uniquement si l’infection vient d'une blessure.

Description de l'action:

  • Vous vous sentez mal à quoi bon se désinfecter? Mais bnon autant tenter; après quelque minutes après vous être désinfecter, vos boutons rouges se révulsent, vous allez beaucoup mieux d'un coup, mais qu'avait-il dans ce désinfectant? Mauvaise nouvelle: votre blessure vous fais toujours aussi mal...


Merci pour avoir lu cette SUGG, pour vos commentaires et si vous trouvez cette suggestion bien, j'espère qu'elle sera intégrée au jeu
