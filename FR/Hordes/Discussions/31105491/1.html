Dinasty1Nom : La Caravane (Pleine/Vide)
Type : Véhicule à recharge (à pousseurs ) (Hé hé)
Chances de réussir le déplacement : 80 %
Casse : Incassable
Description : "Vous avez rafistolé cette chose, et apparemment ça avance avec du zombie, donc..."
Utilisation Primaire : Dans le désert, vous avancez de 2 cases en 2 cases, tant qu'il n'y a pas d'obstacles (Bâtiment) De plus, vous pouvez transporter d'autre citoyens, si ils se sont mis en escorte, il y a un maximum de 12 personnes dans La Caravane. (Même si vous n'êtes pas Héros, vous pouvez transporter des gens). Bonus en camping comme une toile de tente
Recharge(s) : 3 zombies pour 2 cases ben ouais, c'est eux qui poussent la caravane, quand on a des mulets on fait dans l'écolo. les 3 zz sont consommés après le déplacmement. vous l'aurez compris.. pas de zombies = pas de pousseurs. Possibilité d'y atteler des citoyens bannis si le bureau des esclavagistes est construit
Fabrication : 1 moteur cassé + 3 rustines + 2 vis et écrous + 2 tentes + 2 portières + 2 morceaux de contreplaqué + 1 gros coffre en métal + 2 tas de cartons + 4 roues + 1 guitare + 45 (La Caravane se construit depuis le chantier, et peut être abandonnée dans, mais sera toujours là le lendemain)
Pictogrammes :
camping des flots bleus : La Caravane construite une fois.
Raphael : La Caravane construite cinq fois.
Bamboleo : La Caravane construite dix fois.
Gipsy King : La Caravane construite quinze fois.//
