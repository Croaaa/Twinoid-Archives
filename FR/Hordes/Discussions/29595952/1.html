Le retour du micropur et du jerrycan dans le désert!

Le principe est simple, on ne peut pas utiliser le micropur avec le jerrycan dans le désert. Pour cause, le jerrycan est souillé et nous n'avons pas de gourde vide pour mettre l'eau propre dedans.

Alors voici la solution!

Puisque le micropur produit entre 1 et 3 rations d'eau, il faudrait donc un contenant qui puisse en contenir autant. Mais existe-t-il? Oui!

Je propose l'action suivante:

Assembler Micropur dans le désert
Vous mettez le micropur dans le jerrycan, et vous versez le contenu dans une bonbonne d'eau vide!

Résultat:
Bonbonne d'eau vide devient bonbonne avec 1, 2 ou 3 rations, déterminer comme pour le puits.

À vos -1
