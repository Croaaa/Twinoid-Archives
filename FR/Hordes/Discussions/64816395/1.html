Salut à tous,

J'ouvre ce sujet pour parler (encore) de MyHordes. Avec un pavé, encore
Ce sujet est là pour les questions que vous avez peut-être mais également pour nous permettre de vous donner les informations que nous avons vis à vis des transferts. Les screens fournis sont allemands mais ce sera bien évidement traduit dans la version publique
Pour ceux qui ne savent pas ce qu'est MyHordes, je vous invite à lire le spoiler ci-dessous

MyHordes est un projet intégré dans le projet Eternal Twin. Le but global est de sauvegarder tout les jeux possibles de Motion Twin et de les rendre accessible sans Flash (qui à la fin de l'année 2020 ne sera plus mis à jour). Ces jeux qui au fur et à mesure de leur baisse de fréquentation risquent donc de fermer.
Et donc, MyHordes c'est la version sans Flash de Hordes (et de ses versions allemandes / espagnoles & anglaises) car on assemble toutes les versions en une
Le jeu est actuellement disponible en alpha ici, il manque encore des fonctionnalités du jeu d'origine mais on y travaille Et tu peut voir un topic relant les questions de la communauté, créé pour l'occasion ici



Via l'API de Twinoid, nous pouvons obtenir pour le moment :

Points d'âme
Tout les pictos
La liste des villes auxquelles vous avez participé
Les titres et icônes débloqués



Grâce à ces informations, nous pouvons ainsi débloquer :

Les bonus spéciaux donnés à ceux qui ont les pictos Témoin de l'Armageddon / Témoin de la Grande Contamination (le système qui donne ces bonus est déjà en place normalement).

L'expérience d'Héros approximative, basée pour le moment sur l'addition de tout les pictos métier. Cela ne représente pas le vrai montant mais pour le moment c'est ce que nous pouvons faire de manière sûre, cela peut représenter une perte de 5 à 20% des jours passés, mais nous travaillons sur une méthode de calculs qui permettrait d'avoir environ 1% de perte seulement

Pour les écrits RP, nous ne pouvons pas avoir la liste des écrits que vous avez débloqués. Cependant on peut débloquer un nombre d'écrit aléatoire correspondant au nombre de picto RP parmis la liste des textes ou bien vous aurez la possibilité de choisir les textes que vous débloquerez (le nombre d'écrit débloqués ne changera pas et correspondra au nombre de picto RP)



Concernant, le fonctionnement du système de transfert il sera composé pour résumer comme ceci :

Sur MyHordes, un bouton sera accessible dans les paramètres de votre âme pour faire le transfert.

Une fois que vous aurez cliqué sur ce bouton, vous pourrez transferer une âme à partir de DV, D2N, Hordes et ZN. Les transferts d'âme ne seront pas bloqués mais limités (details un peu plus bas). Une page sur le site de Twinoid vous mettra un message de type "Voulez-vous partager vos données avec l'application MyHordes ?" avec le choix "Oui / Non", cela nécessite donc que vous soyez connecté sur votre compte Twinoid. Si vous acceptez, votre compte Twinoid sera associé à votre compte MyHordes et ne pourra plus être dissocié après, si vous refusez cela annule l'importation et aucune donnée n'est envoyée sur MyHordes.

Vous aurez une vue globale des données transferées et si vous confirmez, les données brutes seront ensuite enregistrées dans votre profil MyHordes, donc fondamentalement aucune information ne peut être perdue ou alterée après le transfert.

Cela fusionnera les villes passées de toutes vos âmes si vous transferez plusieurs versions du jeu.

Si vous transferez plusieurs versions du jeu, vous devrez choisir les pictos & points d'âme de quelle version garder, une seule version du jeu pourra être selectionnée pour les pictos & points d'âme, et ne sera pas modifiable ensuite.

L'avantage de cela, c'est que puisque nous stockons les données brutes données par MT via l'API, toutes les données sont toujours en miroir sur MH donc même si MT s'arrête, rien n'est perdu à partir du moment où vous avez fait le transfert avant.

Si vous refaites une importation de votre âme plus tard, cela mettra à jour ces données. Par exemple après le transfert vous avez gagné 100 points d'âme et des pictos dans le jeu d'origine, ils seront mis également sur MyHordes seulement si vous refaites l'importation, les données stockées par MyHordes ne peuvent être mis à jour automatiquement, cela se fait uniquement manuellement quand vous cliquez sur le bouton d'importation.

Une fois l'importation faite, une bannière sera affichée sur votre âme qui dira "Ce joueur a transferé son profil depuis le jeu d'origine", pour différencier les personnes qui ont fait le transfert et ceux qui ne l'ont pas fait.

Concernant les pictos spéciaux comme par exemple "Champion du Choc des héros", "Organisation du Choc", "Choc des héros", "Festival du Jeu Vidéo 2010" : ils seront bien transferés lors de l'importation, cependant comme nous n'avons pas encore créés ces pictos seulement le nom et le nombre seront gardés, donc l'icone et la description n'apparaitront pas tant que nous n'avont pas recréés ces pictos spéciaux.
Quand nous auront recréé ces pictos spéciaux, vous devrez re-importer votre profil, comme les dernières données seront stockées sur le serveur de MyHordes, ces pictos redeviendront normaux même si Twinoid ou le jeu d'origine est hors-service. Si vous transferez des pictos qui ne sont pas encore créés dans MyHordes, un message vous le dira. Pour l'instant, voilà les pictos créés sur MyHordes.



Pour les classements actuels de saison, nous n'avons pas actuellement un truc fiable pour pouvoir décerner les pictos "Ville légendaire !" / Ville classée" / "Ville participante" ainsi que le réveil et la balsie glorieuse. Donc pour le moment on ne peut rien affirmer.


Je pense avoir fait le tour d'un peu tout vis à vis du transfert d'âme, si vous avez des questions ou des suggestions n'hésitez pas
