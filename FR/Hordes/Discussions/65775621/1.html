Peuple Hordien, bonsoir,

Suite à l'annonce de mon confrère Eliam concernant la possibilité de remettre des avatars déjà existants, lui et moi même avons réalisé une compilation de tous les avatars disponibles via l'historique des villes BBH.

cliquez ici pour les détails de la technique permettant de re-mettre un avatar

Cette compilation est disponible via le lien suivant :

tchekof.github.io/twinoid/XXXXXX.html

Où il vous faudra substituer XXXXXX par votre id twinoid (id trouvable dans l'url de votre profil twinoid)

Exemple avec l'id 1730724

Une fois rendu sur la page correspondant à votre id twinoid, vous y trouverez la liste de vos avatars passés encore disponibles sur BBH.

Il vous sera aussi possible de consulter l'historique des avatars d'une personne tierce, pour porter l'un d'entre eux si votre cœur le désire, ou pour humilier une personne que vous n’appréciez guère en lui rappelant les goûts discutables qu'elle arborait jadis.
