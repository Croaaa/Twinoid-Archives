Pour expliquer les 13 citoyens morts, il faut imaginer un algorithme.
Soit N = 151, le nombre de zombies restant.
X => numéro de matricule du citoyen.
A => entier.

Tant que N > 0;
Choisir un entier A comprit entre 0 et N;
A => X; A zombies viennent tuer le citoyen X
N - A => N; Le nombre de zombies diminue, ce qui est normal.
X + 1 => X; Passons à un autre citoyen
Fin Tant que;

Au final, on peut avoir 3 citoyens morts comme on peut en avoir une bonne dizaine. Je n'ai pas pris en compte les améliorations des habitations, c'est juste une approximation de ce qui arrive durant l'attaque, lorsque la veille ne suffit plus.
