Bien le bonjour Citoyen/toyenne


Depuis peu je me suis mise dans la tête de fonder un petit groupement de Coalitions, une Alliance de jumpeurs prêts a s'entraider plutôt qu'a s'entretuer.
C'est pourquoi je vous informe qu'a partir de ce jour L'Abris 2,4 est ouvert
Petit site sans prétentions qui a pour objectif de faciliter la communication entre différentes Coalitions liée par un même but, survivre en outre monde, jumper ensembles.



3 règles
3 sections
3 coalitions(dans un premier temps)


Règles
1 - Respecter votre prochain, je ne m'étendrais pas là dessus, tous le monde sais se qu'il doit faire pour maintenir un bonne entente.
2 - Celui ou celle qui souhaite intégrer l'alliance devra impérativement se présenter/postuler (classique ou Rp) selon son gout via la section 'Arche d'Eva.
3 - Pas d'autre règles restrictives, le site est ouvert a tous.


Sections
Arrow Aides de jeu {Liens utiles} [Survivre en territoire zombie] :Astuces/liens utiles/ expériences personnelles/abréviations.
Arrow Présentation {Candidature} [L'Arche d'Eva] :Tout est dans le titre, ici vous vous présentez et/ou postulez, (classique ou Rp).
Arrow Coalitions {Salons privés} [Citoyens de l'Abri 2,4] : Chaque coalition aura a sa disposition un salon privé
~Le Grand Dôme: salon privé de l'alliance, accessible a toute les membres de l'alliance.


Coalitions
Je n'intégrerais que deux nouvelles coalitions a l'alliance dans un premier temps.
La Coalition qui souhaite intégrer l'Alliance, enverra un émissaire postuler pour celle-ci, qu'il soit le fondateur ou pas. chaque membres se présentera ensuite, condition sine qua non pour que je valide la demande.
Le Citoyen seul qui souhaite intégrer une de nos coalitions devra faire sa candidature/présentation. Si une place est libre dans l'une des coalitions et que sont fondateur le souhaite, le candidat y sera intégré.
Le Citoyen (fondateur) qui souhaite intégrer l'alliance dans l'optique de fonder une coalition dans la foulé, se présentation/postule (candidature soignée). Si il est pris au sain de l'Alliance il est tenu de fonder sa coalition dans la semaine et de recruter un joueur dans les deux semaines qui suivent sont intégration.

---

Membres de l'alliance, n'hésitez pas a envoyer les joueurs que vous rencontré sans coalition sur le forum, poussez les a postuler si vous les estimez, quand bien même votre Coalition est complète, faite leurs savoir qu'il y a peut être une place dans une Coalition alliée.
