Toutes les actions dans le jeu actualisent le compteur de présence : se connecter, actualiser une page, effectuer des actions en ville ou dans le désert, etc.

Par contre, l'envoi de messages via la messagerie Twinoid ou l'utilisation des forums ne comptent pas, donc il est possible qu'un joueur soit renseigné comme étant absent depuis longtemps alors qu'il est actif sur le forum.
Il faut noter aussi que l'heure de déconnexion est approximative : suivant la manière dont le joueur quitte le jeu, il peut y avoir plusieurs minutes de différence entre l'heure indiquée et l'heure où le joueur est parti.
