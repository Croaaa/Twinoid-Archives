Bonjour,

J'ouvre ce sujet concernant les Scripts valables pour hordes de manière générale.
Aussi bien les scripts Stylish, GreaseMonkey, Tampermonkey ou autres.

Il y a souvent des questions sur ce sujet en Section Aide, lieu non adapté pour cela.

Ce sujet regroupera les scripts et les questions à leur sujet pour agir comme un "Support".
J'éditerai au fur et à mesure le premier message pour ajouter les nouveaux scripts.


Script Stylish

  • Script Stylish :
    Des scripts qui ne modifient que l'apparence et ne rajoutent pas de fonctionnalité.
    Ils ne doivent pas supprimer de fonctionnalités.

    Il est préférable de les installer avec l'extension stylish :
    Firefox et Chrome.

    Les scripts de développeurs différents ne sont pas forcément compatibles.
    L'icône de Stylish dans la barre d'extension de votre navigateur permet d'activer ou non les scripts. Veuillez les tester un par un si vous avez des soucis.


    Script HAT : Hordes Âme Twinodienne.
    Modifie légèrement l'apparence de l'âme en gardant son aspect "Twinoïd".
    Pas compatible avec le script HAA.
    Voir comparatif : http://uppix.net/yni9Fv.png
    Développeur : Davf. Support sur Firefox et Chrome.


    Script HAA : Hordes Âme Ancienne.
    Modifie l'apparence de l'âme pour lui rendre un aspect d'avant twinoïd.
    Pas compatible avec le script HAT.
    Voir comparatif : http://uppix.net/J5G48L.png
    Développeur : Davf. Support sur Firefox et Chrome.


    Script HFA : Hordes Forum Ancien.
    Modifie l'apparence des forums twinoid pour leur rendre une apparence hordienne.
    Voir comparatif : http://uppix.net/cEjA4E.png
    Développeur : Davf. Support sur Firefox et Chrome.


    Script HBA : Hordes Balises Anciennes.
    Modifie l'apparence des balises des forums pour retrouver les anciennes balises.
    Voir comparatif : http://uppix.net/BckJY4.png
    Développeur : Davf. Support sur Firefox et Chrome.


    Script Hordify Forum : Hordify Forum.
    Met les éléments Twinoid du forum (boutons, balise RP) aux couleurs de Hordes.
    Voir comparatif : En attente
    Développeur : Nadawoo. Pas de support.


    Script Hordify Âme: Hordify Âme.
    Donne un aspect hordien à l'âme et au "passeport Twinoid"
    Voir comparatif : En attente
    Développeur : Nadawoo. Pas de support.



Script GreaseMonkey, TamperMonkey et autres script JS

  • Script GreaseMonkey, TamperMonkey et autre:
    Des scripts qui peuvent modifier l'apparence, rajouter des éléments et des fonctionnalités.
    Ils ne doivent pas supprimer de fonctionnalités.

    Il est préférable de les installer avec l'extension stylish :
    GreaseMonkey sur Firefox et TamperMonkey sur Chrome.

    Certains scripts ne nécessite pas ses extensions, cela sera précisé.
