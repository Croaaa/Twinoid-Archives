Voici un petit bilan de ce qui a été vérifié et ce qui reste à vérifier sur cette ville test et d'autres villes qui ont réalisés des tests pour moi.


Les âmes :

  • Ramassage :
    • Un chaman peut en porter autant qu'il le veut.
    • Un non chaman ne peut en transporter qu'une seule.
    • Un Apprivoiseur :
      • Non chaman : peut envoyer une âme avec le chien.
      • Chaman : peut envoyer autant d'âme qu'il le veut avec le chien.
  • Visibilité sur la minimap :
    • Invisible si porté par un joueur.
    • Visible si reposé au sol.
    • Visible si elle se trouve dans une cache d'un banni.
    • Visible si elle se trouve dans une ruine.
    • Toujours présente en bas à droite du "carré".
      (C'est le fait d'être au sol qui semble les afficher sur la carte).
  • Évolutions :
    • Le fait de la déposer / reposer n'empêche pas la transformation.
    • Une âme errante dans la banque peut évoluer, mais se dédouble (Une errante et une torturée)
    • Une âme errante dans une ruine peut évoluer.
    • Une âme errante dans le sac peut évoluer.
    • Une âme errante dans le coffre d'un joueur vivant peut évoluer.
    • Une âme errante dans le coffre d'un joueur mort : Inconnu.
    • Une âme errante dans une cache de banni : Inconnu.
  • Effets et purification :
    • Les âmes torturées augmentent l'estimation de l'attaque.
    • La purification des âmes torturées refait baisser l'estimation.
    • Différence entre les âmes étant devenus torturées le jour de leur purification et les âmes plus vieille ?

Les pouvoirs du chaman :

  • La potion :
    • Peut échouer.
    • Lors d'une seconde prise, un message indique si le joueur était déjà protégé ou non.
    • Ne peut pas servir pour arroser un cadavre.
  • La protection :
    • Est fiable à 100%, si la potion a marché.
    • Se garde plusieurs jours.
    • Se garde toute le long de la ville : Inconnu.
  • Les soins :
    • Peuvent être réaliser qu'en ville.
    • Peuvent soigner : Infecté, terrorisé, drogué et ivre.
      • Le soin réalisé semble aléatoire.
    • Le soin est impossible à réaliser si :
      • Le chaman possède l'état qu'il souhaite soigner : inconnu.
      • Le chaman possède un des états soignable : inconnu.




Bannissement et Chaman / âmes :

  • Les âmes ne sont pas récupéré lors d'un bannissement.
  • Un chaman Banni ne peut plus faire de soins.
  • Un chaman qui devient banni ne perd pas son statut de chaman.
  • Un banni peut être désigné comme chaman.




Autres :

  • Goules :
    • Aucun GamePlay particulier avec les âmes.
    • Un goule mangeant un joueur, puis utilisant le vaccin, ne gagne pas le JH associé à sa crime. Dans le cas où elle redevient goule à nouveau et remange un joueur, elle gagne ??.
