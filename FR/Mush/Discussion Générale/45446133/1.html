C'est une bonne idée de convertir le script Stylish en script Grease/Tampermonkey, mais pas la peine de te compliquer la tâche : il suffit d'ajouter un élément style contenant le CSS proposé dans l'en-tête. À vue de nez, je dirais quelque chose comme ça :

var cssChat = document.createElement("style");
cssChat.innerHTML = ".tid_editorContent{ visibility: visible !important; }";
document.head.appendChild(cssChat);

Ce serait plus simple, et surtout ça résoudrait le problème automatiquement, sans avoir à cliquer sur un bouton Normalement ça marche et ça reste quand le jeu se rafraîchit.
