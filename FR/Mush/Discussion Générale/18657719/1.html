Pour trouver le "chk perso" affichez le code source de la page.
Tapez [F3] pour faire une recherche.
Recherchez ce texte :

<div id="tid_chk"

Sur la ligne trouvée votre chk perso est indiqué entre les guillemets à l'attribut data-value

Exemple :

<div id="tid_chk" class="tid_display:none;" data-value="012ab"></div>

Dans cet exemple le chk est 012ab

Attention
Sachez bien qu'il s'agit d'une bidouille d'URL, ce qui est tout à fait déconseillé.
(et interdit par les CGU)
Il vaut mieux attendre que la page soit réparée pour gérer les niveaux comme il est prévu de le faire. à vous de voir si vraiment le passage d'un niveau vous est indispensable en ce moment même dans votre partie en cours.

Il faut entrer une url de ce type :
http:// mush.twinoid.com/levelUp?hid=y&chk=xxxx

(enlever l'espace en http:// et le reste de l'url).

y c'est le numéro du perso, 0 pour Jin Su , 1 pour Frieda, etc....

Pour le chk tu va dans l'onglet Compte tu clic droit sur la page et tu affiche le code source de la page.
Ensuite CTRL + F et tu cherche le mot "chk" jusqu'à trouver une ligne

<div id="tid_chk" class="tid_display:none;" data-value="999999999">

Le chiffre derrière data-value sera ton xxxx dans l'url, ici c'est 999999999.
