Tous ces petits mots de fin de vaisseau, tous ces titres gagnés à la sueur de votre front, tous les souvenirs de ces moments improbables et fantastiques qui vont se perdre comme des larmes dans la pluie… Si seulement il existait un moyen d’en garder trace…

Allez, fini la poésie, j’ai encore fait un script. Promis c’est le dernier. Ça vous colle un bouton sur la page « Mon compte → Ma fiche », sous les voyages et castings, et ça vous pond une page autonome comme ça que vous enregistrez sur votre ordi (et après vous en faites ce que vous voulez).


Installation
1. Comme d’habitude, il vous faut Tampermonkey (Firefox, Chrome) ; le script ne fonctionne pas avec Greasemonkey.
2. Installez le script depuis ce lien.


Le script génère deux fichiers (v1.3+) :

  • une page HTML pour afficher votre historique ;
  • un fichier JSON qui pourra être utilisé ultérieurement avec eMush.

Détails

  • Le pourcentage d’expérience est recalculé pour être correct.
  • Malgré mes efforts, impossible de récupérer la description des gains.
  • Le personnage qui est premier au classement du vaisseau n’a jamais sa mort indiquée sur la page de fin, donc impossible de la récupérer non plus, même si en contexte c’est évident.
  • Si un machin affiche une infobulle, vous pouvez cliquer sur le machin pour fixer l’infobulle. (C’est pas clair ?)
