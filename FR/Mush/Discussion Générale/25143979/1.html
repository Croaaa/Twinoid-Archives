Bonjour,

En cette ère où les castings s'approchent, je vous présente mon script GreaseMonkey Analyse Profile.

Que fait ce script ?
Il analyse le profil des joueurs en récupérant les informations de toutes les parties jouées par un joueur.
Il s'active sur le profil de n'importe quel joueur et rajoute les informations suivantes : voir sur le screen.

Installation du Script
Ce script ne fonctionne (pour une raison inconnue) avec Firefox et GreaseMonkey.
Installez GreaseMonkey puis rendez-vous ici. (Si votre navigateur ne propose pas d'installer le script, actualisez la page).

Objectifs du script :
Le but premier du script est pour aider les (futurs) recruteurs de casting pour cerner les profils des gens.
En bonus, il permet de voir votre type de profil (bonne survie moyenne ? Bon triomphe moyen ? Bon ratio de retours de Sol ? Bon ration de morts assassinés ? etc.)
-
Nota : Les parties de la bêta (avec comme mort "Pas d'infirmerie disponible" ne sont pas comptées. De un parce que cela faussait les morts, de deux parce que pas représentatif (pas la même communauté, pas les mêmes règles).

Have fun.
