ASTROPAD : TOPIC OFFICIEL


FONCTIONNALITÉS

L'Astropad permet de tenir un inventaire collectif, accessible depuis un onglet à droite des canaux privés. Chacun.e peut mettre à jour la salle dans laquelle iel se trouve en un clic, et modifier les données avant de les envoyer (c'est-à-dire ajouter ou enlever un objet, ou modifier ses propriétés).
Cet inventaire peut être partagé sur le chat et vu par toustes grâce à une carte externe du vaisseau.
Les spécialistes (Médecin, Botaniste, Cuistot…) peuvent s'iels le souhaitent partager les effets nutritifs qu'iels voient ; ces effets seront visibles même si un.e non-spécialiste met une salle à jour.
(Exemple : Ian botaniste partage les effets d'une Bottine en baie Icarus. Si Raluca indique qu'une Bottine est présente au Nexus, les effets donnés par Ian seront rattachés à cette Bottine aussi, même si Raluca n'en voit pas les effets.)
Note : L'AstroPad ne permet en aucun cas de détecter un.e Mush.


Installation :
- Sous Firefox, installez l'addon Tampermonkey pour Firefox.
- Sous Chrome/Chromium, installez l'addon Tampermonkey pour Chrome.
Installez ensuite le script en cliquant ici.


Version actuelle : 0.28.11

Si vous utilisez encore la version de Sunsky, il est vivement conseillé de passer à celle-ci afin que vous puissiez bénéficier des nouvelles fonctionnalités et des corrections de bugs. Pour cela, réinstallez simplement le script.


Avant de remonter un bug, vérifiez que vous êtes sous la bonne version de l'AstroPad.
Cliquez sur l'icône Tampermonkey, puis ouvrez le Tableau de bord.

Le numéro de version doit correspondre à celui mentionné ci-dessus. Si l'AstroPad apparaît plusieurs fois, supprimez les versions obsolètes.
