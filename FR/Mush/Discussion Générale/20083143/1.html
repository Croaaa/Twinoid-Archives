Bonjour à tous

Pour cette nouvelle année, je vous propose un petit script utilisateur pour mush.
Un script utilisateur permet d’ajouter/modifier certaines fonctionnalités sur une page web.

Ce script va vous simplifier la vie pour certaines choses en ajoutant certains boutons.

Ce script est encore en phase de développement. Il peut y avoir encore plusieurs problèmes.

  • Vous arrive-t-il d'avoir un message non lu et de ne pas le trouver ? Eurêka !! Maintenant, en un clic, vous arriverez directement dessus.
  • Vous avez l'habitude de taper votre message sur un autre logiciel afin de pouvoir faire des retours à la ligne? Et bien maintenant, vous aurez directement une zone de texte sur la page mush. Avec un bouton pour mettre en gras ou en italique.

Mais cette zone de texte est magique en plus !! Elle peut se remplir toute seule. Ou du moins en cliquant sur un bouton.
Au moment opportun (enfin il n’est pas encore parfaitement opportun ce moment ), un bouton va apparaitre pour générer automatiquement un petit message. Ce qui vous permettra de gagner un certain temps et éventuellement un peu de lisibilité.

Et voici la liste des messages possibles :

  • Faire un inventaire de la pièce :
    Exemple :Dortoir beta : Savon
  • Lister les médicaments avec leurs effets.
    Exemple :Euh bah j’en ai pas sous la main mais plus ou moins ca
    Médicaments :
    Nom : +3 moral, guéri un truc, impérissable
    Nom2 : …    Le bouton apparait au labo, mais si vous ne pouvez pas lire les effets, il ne fonctionnera pas correctement.
  • Donner la description des planètes scannées.
    Exemple :Planètes :
    Mushepuna : Nord - 3 Fuel
    6/6 : Océan, Oxygène, Hydro-Carbure, Ruine, Marais, Océan
    (bonne planète au passage )
    Si vous avez scanné 2 planètes, les deux apparaîtront.
  • Lister les différents projets disponibles.
    Exemple :Projets :
    Agrandissement de la cale : L'icarus peut transporter 2 personnes supplémentaires.
    Distributeur pneumatique : La nourriture du Jardin est envoyée directement au Réfectoire via le Pneumatique.
    Réducteur de trainée : Le nombre d'assaillants qui vous rejoint après un voyage est réduit de 25%.
  • Lister les différents réglages du bios.
    Exemple :Terminal BIOS :
    Destruction denrées : Non
    Priorité CPU : Recherche
    Verrou équipage : Pilotage
    Annonces vocodées : Autorisées
    Signaler les décès : Oui
    Entrave DMZ-CorePeace : Non

Ces boutons apparaissent si vous êtes dans la bonne pièce. Mais pas forcément au bon moment il faut que je corrige un peu cela. Mais quoi qu’il en soit, ils ne fonctionneront que si vous avez les informations à l’écran. (Donc rien de "cheaté")

Prérequis :
Vous avez besoin d’un navigateur récent. Mais de ce côté-là, mush lui-même le demande donc il n’y aura pas trop de problèmes je pense.
Un addon pour votre navigateur permettant d’utiliser des scripts utilisateur (voir installation)
Je n’ai testé que sous les dernières versions de Firefox et Google Chrome. Je testerai sur d’autres navigateurs s’il y a vraiment besoin . Mais pourquoi en utiliser un autre ?

Installation :
Ce que je vous propose est un "script utilisateur".
Pour pouvoir l'exécuter il vous faut un addons (une fonctionnalité en plus) sur votre navigateur.
Si vous utilisez Firefox vous pouvez utiliser celui-ci : Greasemonkey
Si vous utilisez Google Chrome vous pouvez utiliser celui-ci : Tampermonkey
Vous devriez avoir un bouton 'installer' ou 'ajouter au navigateur' un truc comme ça.
Cliquez dessus, il y aura probablement un message de confirmation pour l’installer.
Pour être sûr que l’installation soit prise en compte, vous pouvez fermer et ré-ouvrir votre navigateur

Et enfin normalement, il ne reste plus qu'à cliquer sur le lien suivant, qui est le script en lui-même:
==> Ici le script <==
Là aussi, il devrait y avoir un petit message vous proposant de l’installer, validez et retournez sur mush.
Actualisez la page, si tout s’est bien passé, une petite barre devrait apparaitre en haut à gauche.
Cliquez dessus pour l’agrandir.
Les mises à jour devraient se faire automatiquement.

Utilisation :
Pour s’adapter aux petites résolutions/petits écrans, par défaut, le script se réduit automatiquement quand vous cliquez ailleurs. Vous pouvez, si vous le souhaitez, l’épingler afin qu’il ne se réduise qu’a votre demande, en cliquant sur le bord droit du script.

Voilà, maintenant si vous avez des questions , des critiques , des propositions … n’hésitez pas.

Installation :
Ce que je vous propose est un "script utilisateur".
Pour pouvoir l'exécuter il vous faut un addons (une fonctionnalité en plus) sur votre navigateur.
Si vous utilisez Firefox vous pouvez utiliser celui-ci : Greasemonkey
Si vous utilisez Google Chrome vous pouvez utiliser celui-ci : Tampermonkey
Vous devriez avoir un bouton 'installer' ou 'ajouter au navigateur' un truc comme ça.
Cliquez dessus, il y aura probablement un message de confirmation pour l’installer.
Pour être sûr que l’installation soit prise en compte, vous pouvez fermer et ré-ouvrir votre navigateur

Et enfin normalement, il ne reste plus qu'à cliquer sur le lien suivant, qui est le script en lui-même:
==> Ici le script <==
Là aussi, il devrait y avoir un petit message vous proposant de l’installer, validez et retournez sur mush.
Actualisez la page, si tout s’est bien passé, une petite barre devrait apparaitre en haut à gauche.
Cliquez dessus pour l’agrandir.
