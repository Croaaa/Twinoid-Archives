Je ne savais pas trop où mettre ce topic (vu que ce n'est pas vraiment un bug), mais voilà, j'ai relevé quelques fautes d'orthographe dans la FAQ, et je pense que ce serait pas mal de les corriger

D'autant que c'est probablement le premier endroit où va aller un nouveau venu sur mush, il faut faire bonne impression !

En voici quelques exemples que j'ai relevé sur http://mush.vg/help?cat=general&amp;s...

"L'activité sur les forum" => "L'activité sur les forums"

"L'équilibrage d'un personnages " => "L'équilibrage d'un personnage"

"les personnage cités seront revus" => "les personnages cités seront revus"

"Les phase de Beta" => "Les phases de Bêta"

"qui désire placer les gameplay avant la débauche graphique" => "qui désire placer le gameplay avant la débauche graphique"

"il nous ai nécessaire de prouver" => "il nous est nécessaire de prouver"

"Nous sommes également une très petite équipe qui désirons" => "Nous sommes également une très petite équipe qui désire"

"Cela est risqué car ca ne touchera" => "Cela est risqué car ça ne touchera"

"Les paiements sur un jeu abreuvent dont tout l'équipe" => "Les paiements sur un jeu abreuvent donc toute l'équipe"

"chaque éléments monétisable" => "chaque élément monétisable"

"Pourquoi n'acceptez vous" => "Pourquoi n'acceptez-vous"

"en dark vador peux nous valoi quelques ennuis" => "en dark vador peux nous valoir quelques ennuis"

"L'auto-concurrence" => "L'autoconcurrence"

"Les skins sont notre gagne pain" => "Les skins sont notre gagne-pain"
Il doit également y en avoir sur les autres pages, mais je n'ai pas pris le temps de regarder (mais je peux le faire si vous le souhaitez !)

Voilà voilà ! (Attention, je ne veux surtout pas paraître prétentieux ou quoi, j'ai juste pensé que ce serait mieux pour le jeu et l'équipe de dev)
