Comme vous le savez certainement, il est possible de faire certains trucs sympathiques sur le tchat,
On peut parler en gras, en italique rouge ou noir... ou d'autres...
Le gras se fait avec les : ** ... **
L'italique rouge : // ... //
L'italique normal : [cite] ... [/cite]

Bref...
Il est aussi possible, avec certaines touches de raccourci, de faire le smilies Twinoid (), les smilies Mush () ou tous les autres ().
Sachez que vous pouvez faire tous les smilies Mush, sans le : mush_

Mais, bon, le problème n'est pas là. Tout cela est écrit sur Twinpedia, mais vous n'y avez pas tous accès. Qu'importe, je vais vous aider...

Sachez que certains raccourcis, sont vraiment utiles !
En tapant : :pa: vous obtenez l'icone de PA. C'est cool, non ?
C'est plus facile à retenir et plus court !
