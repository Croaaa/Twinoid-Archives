Pour aider un peu...

Vous créez un fichier texte sur le bureau, vous le renommez en .vbs et vous tapez ça dedans :
message = InputBox("Tapez le texte de Frieda ici")
message = CStr(message)

message = replace(message, "c", "k")
message = replace(message, "g", "k")
message = replace(message, "v", "f")
message = replace(message, "s", "z")
message = replace(message, "z ", "s ")
message = replace(message, "z.", "s.")
message = replace(message, "z,", "s,")
message = replace(message, "j", "ch")

msgbox(message)

C'est très largement perfectible selon les affinités de chacun mais ça vous traduira le français en un frieda très basique

Si quelqu'un sait mettre le contenu de "message" dans le presse-papier, ce serait encore mieux.
