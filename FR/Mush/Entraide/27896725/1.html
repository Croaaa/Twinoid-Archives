La vraie question n'est pas de savoir si c'est RP ou pas, mais bien de savoir si cela risque de déséquilibrer le jeu ou pas.

Si une application d'inventaire partagé fait la même chose que ce qu'on peut faire via le tchat du jeu (ou le forum du jeu, suivant le type de jeu), cela ne pose généralement pas de soucis d’équilibrage. Cela offre un gain de temps mais ne change pas la nature du jeu. Mais il faut offrir les mêmes possibilités que ce qui est normalement possible dans le jeu lui-même : mettre à jour l'inventaire doit être un choix et pas une obligation (sur le tchat Mush, on peut lire un sujet d'inventaire sans être obligé d'y participer), on doit aussi pouvoir transmettre des informations erronées ou incomplètes si on en a envie (comme sur le tchat Mush), etc.

Voir aussi le brouillon de charte pour les scripts qui définit ce qui devrait être autorisé ou pas. Voir en particulier le point "5. Partage entre joueurs".
