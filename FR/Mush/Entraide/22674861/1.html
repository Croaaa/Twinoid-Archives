Bonjour,
comme vous le savez, plus on avance dans le jeu, plus les mushs sont mieux équipés en compétence. (tant mieux hihihi... )

Prenons par exemple la compétence mush "transfert":
(Je n'ai perso jamais utilisé transfert, mais à première vu)
- le mush doit poinçonner d'abord sa victime
- il lui suffit de faire l'action "transfert"
- ensuite le mush peut "embrouiller" les autres, etc...
- à noter que (selon certain dire), le mush récupère tous les canaux privés de sa victime

Bref ! Une plaie et un cauchemar pour les joueurs "humains".

J'ai cherché un moyen de palier à ce problème et j'en ai trouvé un !
C'est un procédé simple mais s'il est bien appliqué, il peut devenir une plaie pour les mushs qui aime se transférer.

Ma solution (roulement de tambour) ==> un code unique pour chacun.

Je m'explique:
1) Chun dés le début du jeu doit donner à chaque membre du vaisseau, sur canal privé, un code unique (Chun doit mettre les codes sur une feuille, comme une sorte de liste).
2) Les membres du vaisseau doivent ensuite mettre ce code sur un bout de papier (réel) et supprimer le canal privé (pour empêcher toute récupération de ce canal par un mush).

C'est là que le code devient important, chaque membre du vaisseau a donc un code qui lui est propre et que personne ne sais.
Vous avez bien deviné:
3) S'il s'avère qu'un mush s'est transféré ( ex: ==> ) (soupçon de transfert, etc...), Chun devra alors passer en revue les codes de chacun.
4) Le mush ne connaîtra pas le bon code et sera identifié (cela permettra aussi de savoir qui n'est pas le mush)

Voilà j'espère avoir été assez clair, je vous pris de me dire ce que vous en pensez, voir même de me proposer d'autres astuces.

(Ps: pardon pour les fautes d'orthographe)

- Voir la réaction des joueurs (si le joueur humain transféré se manifeste).
- Leur poser un ultimatum où ils devront s'entretuer (les humains doivent être coopératifs)

- Utiliser différente capacité "op" (diplomatie, pilote en mode fuite, s'appeler Terrence et être connecté souvent pour bouger + attaquer)
- Saboter le simulateur de gravité (et éventuellement des portes)
