Bon les calculs :

  • sans informaticien :
    • 25% d'utiliser 2+2pa ;
    • 23,4% d'utiliser 4+2pa ;
    • 20,1% d'utiliser 6+2pa ;
    • 15,3% d'utiliser 8+2pa ;
    • 9,8% d'utiliser 10+2pa ;
    • 4,8% d'utiliser 12+2pa ;
    • 1,4% d'utiliser 14+2pa :
    • 0,1% d'utiliser 16+2pa.

Soit une moyenne de 5,64+2pa.

  • avec informaticien :
    • 50% d'utiliser 2+2pa ;
    • 31,3% d'utiliser 4+2pa ;
    • 14,6% d'utiliser 6+2pa ;
    • 4% d'utiliser 8+2pa ;
    • 0,1% d'utiliser 10+2pa.

Soit une moyenne de 3,46+2pa.

Du coup par exemple, sans informaticien, si tu possèdes 10 pa, tu aura 83,8% de réussir (25+23,4+20,1+15,3). Ce qui est confortable.
