Il y a d'origine 3 caméras au hasard dans les 5 stockages.
Plusieurs peuvent être "empilées" dans le même, ça se voit pas mais en enlevant "la" caméra il reste alors quand même une caméra installée.

La compétence Paranoïaque donne 2 caméras.

Pas d'autres caméras en dehors de ça.


Enlever coûte 1 ou 1 PA.
Installer coûte 1 ou 2 PA.
Actions toutes 2 salissantes.
On ne peut poser les caméras dans les étagères, mais si elles tombent suite à une mort ou inventaire plein ça les installe magiquement dans la pièce.

Il faut les placer :

  • La où les joueurs stationnent, pour éviter les poinçonnages faciles.
  • La où les joueurs passent beaucoup, pour éviter les extractions de spores faciles.

Donc les 2 dortoirs, le pont, l'infirmerie, les couloirs, mais aussi le réfectoire, les 2 baies...

Une myco-alarme peut remplacer une caméra, mais :

  • Ça sert à prendre la main dans le sac au lieu de dissuader.
    Donc sort risque que le mush surpris passe en mode dawa.
  • Il faut les cacher pour être vraiment efficaces.
    C'est rarement possible en dortoirs, et Eleesha observatrice risque de les trouver sans le vouloir.
