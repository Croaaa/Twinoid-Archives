Salut à tous !

Je viens vous proposer de reprendre un ancien jeu que j'animais : Contagion

Au vu de son ancienneté, j'ai pris la décision de remettre tous les scores à zéro pour que tout le monde reparte sur la même base.

Je rappelle que c'est un jeu et non une animation : pour l'instant, aucune récompense n'est prévue

C'est reparti, avec une MAJ des règles et une nouvelle intro RP !

INTRO

Vous avez finalement atterri sur Eden. Sur place, vous découvrez malgré vous la situation : la planète a été colonisée par les humains ! Ils sont très nombreux, votre situation est tendue : vous allez devoir maîtriser la situation rapidement.
Mais ce n'est pas tout : les humains ont parmi eux une enquêtrice redoutable, Eleesha Williams, qui sera constamment sur vos traces ! Prenez garde, ou elle vous supprimera si elle découvre votre secret.

REGLES

Comment jouer ?

Première chose : dans chacun de vos messages, vous ne pouvez faire qu'une seule action.
Votre but est de contaminer les humains. Ceux-ci sont pour le moment 500, mais leurs nombre peut évoluer.
Pour qu'un humain soit contaminé, il doit avoir été poinçonné 3 fois. Vous ne pouvez poinçonner qu'une fois par action.
Pour poinçonner, il vous faut un spore. Afin d'obtenir un spore, il faut l'extirper. Pour cela, tirez un dé 4 et dites dans le même message quel nombre vous pensez faire. SI votre prévision et votre dé sont le même chiffre, vous avez réussi à extirper. Sinon, c'est raté, il vous faut réessayer.
A la fin de chaque action, lancez un dé 100. Si vous faites 1, l'enquêtrice Eleesha vous aura démasqué et tué. Votre nombre de points sera alors divisé par deux, et tous vos spores/objets/argent disparaîtront. Si vous faites 100, votre action est doublée : si vous avez poinçonné, l'humain recevra deux poinçons (de votre part) ; si vous avez contaminé (le 3ème spore), un autre humain sera contaminé d'un seul coup ; si vous aviez extirpé, vous recevrez deux spores ; si vous avez raté une extirpation, rien ne se passe.
Vous pouvez stocker 4 spores maximum !


FONCTIONNEMENT

Une action par message
Pas de double post (même si vous vous êtes trompé de dé ou autre, vous attendez)
Pour réaliser une action, vous devez attendre que deux personnes aient réalisé une action depuis votre dernière action.
Le nom des humains que vous poinçonnez sera un numéros : H1, H2... etc jusqu'à 500 (pour le moment ^^)
Une fois qu'un humain a reçu un spore (de joueur X par exemple), les autres mushs ne peuvent pas le poinçonner ! Il lui est réservé !
Vous ne pouvez contaminer qu'un humain à la fois !

SCORES

1 extirpation réussie : 1 point
1 poinçonnage : 1 point
Mettre le 3ème spore sur un humain et ainsi le contaminer : le score du poinçonnage + 1 point (cela fait donc 2 points en tout)
