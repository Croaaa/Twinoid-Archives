AIDE TECHNIQUE

Le navigateur conseillé par Kadokado est Firefox.

Pour mettre le plugin Flash à jour, allez à cette adresse

Comment mettre un lien ? :

- Ouvrez la page que vous voulez mettre en lien : copiez l’url (ctrl+c) de la page qui commence par http:// …
- Revenez sur le forum : tapez votre message et cliquez sur lien.
- Une fenêtre s’ouvre : collez (ctrl+v) l’url de la page à laquelle vous renvoyez.
- Cliquez sur ok : la même fenêtre se rouvre avec l’adresse en grisé. Vous effacez l’adresse et la remplacez par le titre de votre choix.
- Cliquez sur ok.

Comment faire une capture d'écran ? :

Sous Windows :
Lorsque vous avez sur votre écran ce que vous voulez nous montrer, il faut que vous appuyiez sur le bouton 'IMPR ECRAN' ou 'PRT SCRN' de votre clavier.

Puis, vous :
- ouvrez PAINT (menu démarrer / tous les programmes / accessoires / Paint).
- allez dans Edition / coller.
- enregistrez votre image dans votre ordinateur (choisissez le format .jpg pour n'avoir aucun problème pour l'héberger).

Sous Mac :
- Pour une capture totale de l'écran : Pomme + Majuscule + 3
- Lorsque vous avez sur votre écran ce que vous voulez nous montrer, il suffit d'appuyer sur pomme + majuscule + 4 et la souris va se transformer en curseur. Glissez le curseur pour former un rectangle, puis relâchez : votre image est enregistrée sur le bureau.

Comment héberger une capture d'écran sur votre pc ? :

- Rendez-vous sur imageshack.
- Cliquez sur Parcourir puis choisissez votre fichier qui se trouve dans votre ordinateur.
- Cliquez sur HOST-IT.
- Copiez le dernier lien qui sortira nommé direct link to image.

Comment vider son cache ? :

- Sous Internet Explorer :
Outils => supprimer l'historique => supprimer les fichiers.

- Sous Firefox 1
Outils => options => onglet Vie privée => onglet Cache => Vider le cache.

- Sous Firefox 2
¤ Outils => options => onglet Avancé => onglet Réseau => Cache => Nettoyer maintenant.
ou:
¤ Effacer mes traces => cocher "cache".

Comment accéder à son historique ? :

La commande directe est Ctrl+H.

- Sous Internet Explorer :
Affichage => Volet d'exploration => Historique.

- Sous Firefox :
Historique => Afficher le panneau latéral.
