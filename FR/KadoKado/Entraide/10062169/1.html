Réponse du support :

Bonjour,

Les dernières modifications de KadoKado ont dû être faites dans un délai très court, puisqu'elles ont été réalisées peu avant le départ définitif de Yota - administrateur du jeu. A ce moment il fallait se concentrer sur du développement simple et nécessaire.

Pour l'instant, Yota étant parti, nous ne pouvons plus faire de modification de ce type - car rien n'est prévu dans le jeu pour récolter ce type d'information. Ce n'est pas inenvisageable, mais ce ne sera probablement pas pour les mois à venir. Nous transmettrons toutefois l'idée.
