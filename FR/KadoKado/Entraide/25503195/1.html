C'est moi qui poste, mais ce message a été écrit avec l'aide d'oubangui. Nous parlons donc ensemble.

Le problème de pause sur les jeux est combattu. Désormais, un message d'erreur devrait apparaître à la fin de la partie si vous utilisez la pause.

Merci de nous contacter le plus rapidement possible par MP sur la messagerie des modérateurs dans les cas suivants :
- Si vous parvenez encore à enregistrer des parties avec la pause ;
- Si vous jouez normalement (sans tenter de tricher) et que le message d'erreur apparaît à la fin de la partie ;
- Si un autre problème survient (les admins vont sûrement bricoler un peu).

A chaque fois que vous remontez un problème, précisez :
- Le(s) jeu(x) concerné(s) ;
- La version du navigateur (Firefox 21.0, Internet Explorer 9, etc) ;
- La version du lecteur Flash (Flash 11.7.700.202 par exemple) ;
- Votre système d'exploitation (Windows XP, Windows 7, Linux,...) ;
- Toute autre information que vous jugez utile de nous donner.

Merci !
Oubi-moe'.

Bonjour,

Nous (oubangui et moejul) avons fait le choix de vous donner quelques infos concernant le problème de triche. Nous estimons qu'en tant que joueurs, vous avez parfaitement le droit de savoir ce qui est fait pour tenter d'y remédier.


Le problème est connu, des solutions sont étudiées
Nous avons remonté les problèmes aux administrateurs. Nous leur avons expliqué en détails et listé les jeux principalement concernés, ainsi que des vidéos et autres éléments pouvant les aider.

Par ailleurs, nous avons évoqué différentes solutions avec la MT :
- Désactiver le clic droit sur les jeux ;
- Installer un système de replay comme sur Arkadeo ;
- Interdire l'accès aux jeux via Mozilla Firefox ;
- La suppression momentanée des jeux concernés ;
- etc.


Pourquoi on attend si longtemps ?
Plusieurs raisons expliquent l'apparent manque de réaction :

- Il faut savoir qu'il y a déjà quelques années, les admins ayant codé KadoKado (yota,...) ont quitté la Motion-Twin. Seuls quelques autres admins ont été formés pour pouvoir s'occuper de KadoKado. Cependant, ces admins sont en vacances longues (plusieurs mois). Ceux qui sont présents actuellement sont un peu là comme intérimaires. C'est-à-dire qu'il est plus difficile pour eux d'agir surtout pour un problème de cette ampleur.

- Le code du site est trèèèèès vieux. Tout bon développeur sait à quel point il est difficile de replonger dans les vieilles lignes de code d'un site pour mettre celui-ci à jour.

- Le code du site n'était pas adapté à certaines solutions proposées. Il n'a pas été pensé pour cela. Par exemple, il est techniquement impossible de faire un système de replays comme sur Arkadeo, car cela n'a pas été incorporé dans le code du site. S'ils devaient le faire, ils devraient recoder tout KadoKado, en gros. Vous direz "pourquoi pas ?", mais non.

- Certaines solutions émises sont difficiles à mettre sur pied. Par exemple, désactiver le clic droit dans les jeux peut paraître simple, mais cela revient à modifier le code de chaque jeu pour y interdire le clic droit. De plus, il faudrait sûrement modifier le programme incluant les jeux (la fenêtre autour des jeux), pour interdire ce clic droit. Et pour rappel, les admins présents ici sont intérimaires et ne pourront pas faire ce genre de modifications.


Conclusion
Nous souhaitons toujours trouver des solutions, même momentanées (nous attendons notamment une réponse pour une solution proposée).

Concernant la suppression de jeux, nous avons bien insisté pour qu'ils n'appliquent ça qu'en tout dernier recours, et encore. Nous ne voulons pas voir disparaître des jeux ; cela priverait l'ensemble des joueurs à cause de quelques tricheurs. Bref, on n'en est pas là en tout cas, on y réfléchira si ça doit être envisagé.

Donc en bref, ce message a pour but d'éclaircir quelques points :

- vous sembl(i)ez penser que nous nous en fichons et ne faisons rien. Pourtant avant d'être des "modos", nous sommes des joueurs, et comme vous dégoûtés par la triche, donc mobilisés pour imaginer des solutions réalisables.

- vous sembl(i)ez penser que la MT ne sait pas et/ou ne bouge pas. Ce n'est pas la priorité première de tous les administrateurs, mais cela ne signifie pas qu'ils ne feront rien.

En attendant, si quelques tricheurs parmi les membres actifs du forum pouvaient prendre conscience des conséquences que peuvent provoquer leur méthode de jeu, ce serait sympa, et ça nous aiderait, nous, le site, les admins, et surtout tous les joueurs !

PS 2 : Merci à la MT de s'être occupé de ça !
