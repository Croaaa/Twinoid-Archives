Motion-Twin utilise :
-Ses propres codes (MTPass)
-Des codes issus de Rentabiliweb (Speedcode, GetPass...)

Mais pas ceux venant de Allopass.

Dans le doute tu peut toujours tenter de les entrer en section banque : quel que soit le type de code que tu déclare entrer, s'il est de ceux acceptés son type réel est automatiquement reconnu et le montant approprié de crédité.
