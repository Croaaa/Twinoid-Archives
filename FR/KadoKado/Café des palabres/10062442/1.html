Une fois les duels choisis "jeux, joueurs, arbitres, semaine" voici la marche à suivre pour rendre ce topic agréable et facile d'utilisation

Les arbitres, et eux seuls, seront autorisés à poster ici.
Seuls les messages ayant la forme demandée seront laissés dans ce sujet.

Avant que le duel ne commence, l'arbitre devra poster ici deux choses.

>> Tout d'abord copier le message qui suit et le compléter.
---------- ---------- -------- message ----------- ---------- ----------
Duel entre Joueur 1 et Joueur 2

Arbitre : ?????????

Jeux : Jeu A - Jeu B - Jeu C

Leurs scores :
Joueur 1
Jeu A - 0 | Jeu B - 0 | Jeu C - 0
Jeu A - 0 | Jeu B - 0 | Jeu C - 0
Joueur 2

Gagnant : ...

Le sujet : [Duel] Joueur 1 vs Joueur 2

Date du duel : du ??? mois au ??? mois
---------- ---------- -------- message ----------- ---------- ----------

Le message sera édité avec les scores et le gagnant
si en plus l'arbitre envoie les scores par MP ainsi que le lien du sujet à fermer, c'est top

>> la date du duel; sans elle il sera impossible de programmer le duel !

Aux arbitres, n'oubliez pas les règles qui suivent :
- un joueur doit attendre une semaine avant de refaire un duel.
- les duels sont joués dans un sujet spécifique dont le titre sera annoncé sous cette forme : [Duel] Joueur1 vs Joueur2
- les duels sont limités à 5 par semaine.
- les duels débutent le lundi et finissent le dimanche (horaires à votre choix)

Pour chercher un duel, ou poster un message différent de ce qui est demandé ici, c'est par ici
Et une nouveauté : La note de l'arbitre !
Note [de 0 à 10] qui prendra en compte la qualité du suivi des jeux, la présentation des joueurs, l'originalité.. etc...
Les 2 duellistes auront donc à noter leur arbitre
