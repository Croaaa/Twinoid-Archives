Oyé, Oyé en ce jour magnifique je vous présente mon jeu du wistiti qui conciste à faire une suite de mots qui ont toujours un rapport l'un après l'autre .

Je vais vous faire un petit exemple parce que je sais dans vos têtes vous vous dites "Oulala c'est quoi ce jeu ?"

Exemple:
Mr Dupont écrit : Chien
Mdme Dupont répond donc : Os ( car le chien aime l'os )
Le fils Dupont suggère : Humain ( car nous somme constitué d'os )
La fille Dupont se dit : Humanité ( rapport à l'humain )
Le bébé Dupont tout en bavant répond : Solidarité ( car pour être humaniste il faut être solidaire envers ceux qui n'ont rien )

Donc voilà je pense que vous avez compris, n'hésitez pas à me poser des questions si vous n'avez toujours pas compris
