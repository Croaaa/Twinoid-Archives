Règle du jeu:

- Il y aura des équipes de 3 joueurs (j'equilibrerai les équipes ou je ferai un tirage au sort )

- Quand 2 équipes s'affrontent les matchs se font en 1 VS 1. (exemple et descriptif ci-dessous)

1ere Manche:
J1 VS D1 jeu : Piou piou
J2 VS D2 jeu : Travoltax
J3 VS D3 jeu : Manda

2eme Manche:
J1 VS D2 jeu : Manda
J2 VS D3 jeu : Piou piou
J3 VS D1 jeu : Travoltax

3eme Manche
J1 Vs D3 jeu : Travoltax
J2 VS D1 jeu : Manda
J3 VS D2 jeu : Piou piou

- L'équipe gagnante est celle qui aura gagné 2 jeux ex : si l'équipe D gagne à Piou piou et Manda, elle est déclarée gagnante

- la manche dure 1 Journée donc un match dure 3 jours

- Au début ce sera sous forme de groupes de 4 équipes. Les 2 premières sont qualifiées
Ensuite ce sera les 1/8 de finale ou les 1/4 de finale selon le nombre d'équipes.

- Une victoire vaut 3 points et une défaite vaut 0 point

- Les jeux seront tirés au sort

- L'équipe qui gagne la finale gagnera 3 codes (1 par joueur)

- L'équipe gagnante devra s'affronter elle-même pour déclaré le GRAND joueur gagnant qui recevra 1 code

- le joueur le plus combatif se verra remettre un code ( l'explication de comment est élu le joueur le plus combatif sera donné à la fin du tournoi pour éviter les triches)

INSCRIPTIONS

- Si voulez vous inscrire envoyez moi un

- Le nombre de joueurs inscrits devra être un multiple de 3

Ouverture des inscriptions : OUVERT
Fermeture des inscriptions : Quand je reviens de vacances

IMPORTANT

Donnez moi toutes les tentatives de jeu que vous avez faites

Merci à Oubangui

Le tournoi débutera au début du mois d'Aout car je pars en vacances Samedi ou dimanche prochain MCI.
