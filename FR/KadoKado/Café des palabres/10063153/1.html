PRINCIPE :
Un plateau de jeu (voir post suivant) avec un jeu Kadokado par case + des cases spéciales cachées.
Tous les 2 jours, on avance d'un certain nombre de cases suivant le score qu'on a fait (au jeu qui correspond à la case sur laquelle on se trouve)
NB : en cas de duel le tour dure trois jours : un jour pour le duel + deux jours pour le tour

1 case
2 cases
4 cases
6 cases

De plus pour chaque case :

- Le joueur ayant le meilleur score avance de 3 cases supplémentaires
- Le 2e de 2 cases
- Le 3e de 1 case

On peut donc avancer de 1 à 9 cases par tour !
Exemple :
Joueur A : Kaskade2 (+6) / 1er (+3) + 9 cases
Joueur B : Kaskade2 (+6) / 2ème (+2) + 8 cases
Joueur C : Kaskade2 ( +4) / 3ème (+1) + 5 cases
Joueur D : Kaskade2 (+1) / 4ème (NonClassé) + 1 cases
Joueur E : Kavern (+2) / 1er (+3) + 5 cases
Joueur F : Kanji (+6) / 1er (+3) + 9 cases

CASES SPECIALES
Les cases spéciales sont cachées, de sorte que vous ne sachiez à l'avance s'il s'agit d'un bonus ou d'un malus . Voici la liste :

Hop Hop Hop :
Avance du nombre de cases que tu as avancé pour atteindre cette case.

Joker :
Tu gagnes un joker ; puis avances d'une case
Les jokers permettent d'avancer ou reculer d'UNE case à n'importe quel moment de la partie
Il n'est toutefois pas possible de l'utiliser sur une case spéciale

Échange :
Échange ta place avec la personne moins avancée que toi dans la partie.
Cette personne joue sur la case précédente

Duel :
Tu affrontes la personne la plus avancée dans le jeu sur son jeu favori. Si tu gagnes tu prends sa place ; elle, joue sur la case suivante.
Si tu es premier, tu joues sur ton jeu favori.

Stop :
Tu passe un tour
Le tour suivant tu joues sur ton jeu préféré

Oust :
Tu recules de 3 à 6 cases au choix

REGLES :
Les scores devront absolument être postés sur ce topic avant le temps imparti (même pas 5minutes de retard). De plus, lorsque vous posterez vos scores , faites apparaître la couleur de vos étoiles.

Les personnes n'ayant pas joué seront éliminées. Toutefois, si vous vous savez absent ponctuellement et que vous êtes très motivé, prévenez moi par MP
Les choix des jeux (pour les cases spéciales) se font par MP, je les dévoilerai une fois tous les jeux choisis
Vous pouvez d'ors et déjà envoyer le jeu commençant par K ainsi que votre jeu favori par
