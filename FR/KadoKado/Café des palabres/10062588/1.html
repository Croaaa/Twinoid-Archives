Bonjour,

A partir du lundi 5 avril, la phase d'attaques/missions changera pour s'adapter à la phase de missions.

Vous devrez respecter le planning suivant pour lancer vos attaques :

Le lundi : attaques sur Bactery avant 12h, sur Invasion à partir de 12h
Le mardi : attaques sur Iron chouquette avant 12h, sur Cooking Lili à partir de 12h
Le mercredi : attaques sur Drakhan avant 12h, sur Tout-Caen à partir de 12h
Le jeudi : attaques sur Manda avant 12h, sur Kaskade 2 à partir de 12h
Le vendredi : attaques sur Kanji Gaiden avant 12h, sur Xian-Xiang après 12h
Le samedi : attaques sur Digestomax avant 12h, sur Synapses à partir de 12h
Le dimanche : attaques sur Happy Pti Tank avant 12h, sur Judo Commando à partir de 12h

Par exemple : un clan ne pourra défier un autre clan que sur le jeu Bactery le lundi matin puis sur Cooking Lili l'après midi.

Les jeux autorisés changeront chaque semaine et à chaque période d'attaques/défenses.

Ces modifications permettront à tous les clans de s'affronter sur les mêmes jeux. Ce qui entrainera moins d'écarts dans le classement.

En parallèle les clans passeront à la fin de cette période à 20 membres. Ceci afin de limiter les inactifs et permettre une meilleure homogénéité dans les clans.

Nous espérons que vous apprécierez ces changements
