J'ai déjà eu ce problème.
Il suffit de désactiver le mode protégé sous Windows (source) :

  • Ouvrez le bloc-note (en tant qu'administrateur à partir de Windows 7)
  • Dans ouvrir (ctrl+O), tapez ceci suivant votre version de Windows dans la barre du nom de fichier et cliquez sur ouvrir :

Windows 32bit: C:\windows\system32\macromed\flash
Windows 64bit: C:\windows\syswow64\macromed\flash

  • Double-cliquez sur mms.cfg
  • Ajoutez cette ligne à la fin du fichier :
    ProtectedMode=0
    Note : Si le fichier n'existe pas, il faut le créer et y mettre cette unique ligne.

Le problème avec cette solution, c'est qu'elle désactive une protection qui est prévue pour empêcher que d’éventuels problèmes de Flash contaminent l'ordinateur.

Pour le moment, les deux options possibles sont toutes les deux assez gênantes :

  • Revenir à la version 11.2 de Flash. Cette version n'est pas parfaite et certains bugs peuvent causer des problèmes, mais généralement elle fonctionne bien sous Firefox et Chrome.
  • Désactiver le mode protégé. (EDIT pour Celler : en fait si, désactiver le mode protégé permet bien de résoudre les problèmes de ralentissement et d'affichage incorrect. C'est une mauvaise façon de résoudre ces problèmes parce que cela ne fait que contourner les bugs de Flash au lieu de les résoudre, mais cela fonctionne quand même.)

Les versions 11.3 et 11.4 de Flash semblent causer un bon nombre de problèmes, aussi bien sur Firefox que sur Chrome.

J'ignore qui s'occupe du contrôle de qualité chez Adobe, mais c'est assez désastreux...
