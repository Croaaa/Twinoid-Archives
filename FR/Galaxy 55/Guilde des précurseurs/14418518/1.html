Super chez

Téléchargez CCleaner pour vider les caches de votre navigateur web et d'adobe flash player.

Sinon il faut ajouter la ligne suivante dans mms.cfg :
ProtectedMode=0
Sous Windows 32 bits, le fichier ce trouve dans C:\Windows\System32\Macromed\Flash
En 64 bits, il se trouve dans C:\Windows\SysWOW64\Macromed\Flash
Sur Mac et Linux, faites une recherche sur internet.

Note : Sur Linux, la dernière version de flash est la 11.2. Inutile de chercher ce fichier car cette méthode ne fonctionne pas sur cette version.
