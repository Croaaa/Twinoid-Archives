Amis enseignants, N'en avez vous pas assez de voir les objets confisqués aux élèves qui s'entassent inlassablement dans votre bureau ?

Grace à Class Manager et ses 4304 armoires intégrées, ce temps est maintenant finis. Vous allez enfin pouvoir montrer aux yeux de tous à quel point vous êtes un prof intransigeant ... et soigneux !

Class Manager c'est quoi ?

C'est un petit script constitué de différents modules permettant de vous simplifier la vie ... ou simplement d'ajouter des petits plus sympathiques :


Rangement des collections :

Met un peu d'ordre dans la liste des objets.

Onglet "quêtes" :

Ajoute un onglet sur le profil affichant la progression dans la collection des séries d'objets débloquant des récompenses.

James Norray Facts :

Affiche, de façon aléatoire, un des facts officiel du site James Norray Facts ( rédigés par hiko )


Comment ça marche ?

Pour utiliser Class Manager vous devez disposer des navigateurs Google Chrome ou Mozilla Firefox :

Pour FirefoxVous devez disposer de la dernière version de Greasemonkey.

Pour ChromeVous devez disposer de la dernière version de Tampermonkey. Il est possible que le script fonctionne sans, mais je ne pourrais garantir l'absence de bugs.

Attention :

Les scripts Greasemonkey pour Firefox et Chrome sont utiles aussi bien sur les jeux de Motion-Twin que sur d'autres sites, mais un auteur malintentionné pourrait s'en servir pour voler vos mots de passe ou faire des actions sur les sites à votre insu, vérifiez toujours l'auteur du script ainsi que les sites sur lesquels celui-ci est actif ( affichés avant l'installation )

Un fois Greasemonkey/Tampermonkey installé, vous pouvez :
Installer Class Manager

Vous pourrez par la suite le configurer via le lien qui sera présent en bas de page, à coté de l'onglet "aide" de Teacher Story ( lien ne fonctionnant pas si vous devez choisir une mission ), et vous serez automatiquement avertis lorsqu'une mise à jour sera disponible.


N'hésitez pas à me faire part de vos retours, questions et suggestions sur ce sujet.
