Esta tema es una continuación al tema 189312 Monstruos


Esta es una herramienta que revela simplemente la cantidad de monstruos por isla, las ubicaciones de los Igniks y las islas mas pobladas de mosntruos


Monstruario Avanzado


El código de colores significa:
Violeta: representa las 74 islas mas poblabas, cada una con 39 a 47 monstruos
Violeta claro: representa las 33 islas siguientes con mas monstruos, cada una con 38 monstruos
Naranja: ubicación de los 77 Igniks disponibles en el juego (en el mundo de Gonkrogme)


Adicionalmente, en la hoja "Analisis" hay un contador que muestra cuantos monstruos totales existen en Gonkrogme y cuantas islas hay.


La primera vez que se ejecute el script (presionar botón calcular), puede que pida permisos de ejecución (presionar allow), luego cada vez que se marque una isla se debe presionar el botón para calcular los valores (puedo tardar unos segundos).


Es necesario que las islas despejadas se marquen con el color verde claro 1 (light green 1 en inglés), ya que el script toma ese color para identificar las islas despejadas.


Esta parte es todo gracias a thisdotfabio que creo el script que realiza el cálculo.
