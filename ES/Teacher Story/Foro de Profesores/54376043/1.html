Pupil Manager Creador : Watilin
Idioma : Francés
Sinopsis : Gestor para el envío de alumnos a contactos.
Tema oficial : http://twd.io/e/MEDW0w (Foro francés)
Cómo instalarlo
Mozilla Firefox Descargar GreaseMonkey
Google Chrome y Opera Descargar TamperMonkey
Una vez completada la descarga, instalar el script: Pupil_Manager.user.jsFUNCIONES (v1.3, 25/04/16):
Lista de contactos.
Enviar con un clic.
Ordenar contactos.

USO:
Si tienes alumnos para enviar, en la página de Tu casa aparece un nuevo botón con el nombre de Pupil Manager a la derecha del de enviar alumnos. Al pulsarlo aparece una ventana similar a esta:
https://raw.githubusercontent.com/Wat...

Simplemente busca el contacto al que quieras mandar seleccionando las páginas u ordenando por las columnas y haz clic en "Envoyer" en la línea del contacto para mandarle.
Si sale "Ok !" el alumno se habrá enviado con éxito, en caso contrario te informará del error ocurrido (por ejemplo si ya tiene un alumno tuyo, está lleno,...)
