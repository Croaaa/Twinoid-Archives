supongo que quieres decir el efecto de cada habilidad que dan los alumno

supongo que esa si la podemos hacer

Mudanza: te permite mover un pupitre 1 espacio hacia adelante ( solo funciona si hay un espacio adelante donde se pueda adelantar el pupitre) se puede usar cada 2 turnos

Alzheimer : da una una nueva serie de opciones ( hace que vuelva a girar las habilidades usables ese turno ) se puede usar cada 3 turnos

Copión : aumenta la nota del alumno 0.5 + si es derrotado . se puede usar cada 5 turnos

Cosquillas: quita todos los puntos de aburrimiento al alumno y le otorga el estado gruñon ( + 1 a los daños del alumno . el alumno ataca mas a menudo ) se puede usar cada 3 turnos

¡Shhhhhh! : los alumnos del pupitre pierden 2 estados negativos 1 ganan 1 . se puede usar cada 2 turnos

Codazo : el alumno pierde 1 . se puede usar cada 2 turnos

Electroshock : el alumno pierde todos los estados críticos. se puede usar cada 2 turnos

Expulsión : el alumno objetivo se sale de la clase . la clsae puede terminar sin el . se puede usar cada 5 turnos.

Explicaciones: el alumno pierde 1 . se puede usar cada 4 turnos

Dardo tranquilizante: el alumno recibe el estado eclenque ( el aprendizaje que recibe el alumno se reducen en 1 ; a cambio te inmuniza de los daños por parte del alumno ) . se puede usar cada 4 turnos

Mazo: el alumno recibe el estado migraña ( el alumno no hará nada durante 4 turnos o hsata que reciba daños de aprendizaje ) . se puede usar cada 3 turnos

Cerebrito : otorga a un alumno el estado muy atento ( su próximo aprendizaje se aumentara en 2 . se puede usar cada 4 turnos .

Meditación : le otorga a un alumno 1 , y te da a ti 2 puntos de autocontrol. se puede usar cada 3 turnos.

No es gracioso : elimina el estado muerto de risa. se puede usar cada 4 turnos

Acomodador: desplaza a los alumnos a los pupitres que estén vacíos mas adelante. se puede usar cada 2 turnos

Alivio: te da 1 punto de autocontrol durante 3 turnos . se puede usar cada 5 turnos

Campana: despierta a los alumnos dormidos y les da el estado atento. se puede usar cada 3 turnos

Sinergia: el alumno pierde 2 , si es derrotado por este efecto ganas 1 puntos extra de experiencia . se puede usar cada 4 turnos

Transferencia : el alumno pierde sus y se reparte la mitad a cada alumno del pupitre ( ejemplo : si un alumno tiene 4 , y ahi 2 en el pupitre se reparten 2/2) . se puede usar cada 2 turnos

Valium: el alumno recibe el estado inofensivo ( los ataques del alumno no te quitan autocontrol ) . se puede usar cada 4 turnos

Voluntad de acero: te da 1 punto de autocontrol . se puede usar cada 3 turnos
