Saludos a todos

Abro este post porque busco socios para un proyecto que estoy actualmente realizando y que creo que es unico por sus caracteristicas.

Este proyecto en cuestion se esta realizando en un planeta arenas calientes de tamano mediano y basicamente trata de seguir tanto como se pueda la linea temporal inicial de Galaxy55. La idea es construir una base cientifica en este planeta con 32 residencias divididas en tres niveles e ir expandiendonos a otros planetas pero siempre siguiendo ciertas bases fisicas y esteticas ( nada de construir edificio con arena, hojas o cosas asi)

En principio, el proyecto esta abierto solo para jugadores expertos que juegen desde nivel Alpha1 en adelante pero repito, en este proyecto se busca conseguir acabados que se asemejen a la realidad.

Se esta estudiando la posibiladad de remuneraciones para aquellos que colaboren con el proyecto.
