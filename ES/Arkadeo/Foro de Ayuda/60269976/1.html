Antes la configuración de flash permitía cambiar la calidad de renderizado de los gráficos para mejorar el rendimiento, pero hace un tiempo que ya no veo esta opción (creo que ahora se ajusta de manera automática)

Para poder elegir la calidad de renderizado tendrás que recurrir a extensiones de terceros. En esta página [en inglés] puedes encontrar una extensión para Chrome (Flash Render Quality changer), y un script de Greasemonkey para Firefox.
