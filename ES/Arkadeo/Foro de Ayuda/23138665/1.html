La bajada de fps en Mel es debida a una actualización de seguridad de Flash para protegerse del Meltdown/Spectre (el fallo de los procesadores Intel, AMD y ARM)

Por el momento la única forma de evitar este problema es hacer un bypass a la seguridad modificando el archivo de configuración del navegador, con lo que se dejaría de estar protegido contra este fallo. Estoy pendiente de si sale alguna otra solución

Para los que quieran correr el riesgo en Windows 10:
Edge, IE, Firefox
Si el sistema es de 64 bits, abrir el archivo (protegido) C:\Windows\SysWOW64\Macromed\Flash\mms.cfg
Si el sistema es de 32 bits, abrir el archivo (protegido) C:\Windows\System\Macromed\Flash\mms.cfg
En ambos casos, agregar la linea:
TimerJitterMicroseconds=0
Reiniciar el equipo.
 Chrome
Abrir el archivo (protegido) C:\Users\USERNAME\AppData\Local\Google\Chrome\User Data\Default\Pepper Data\Shockwave Flash\System\mms.cfg
Agregar la linea:
TimerJitterMicroseconds=0
Reiniciar el navegador. Repito : ESTO ANULA LA ACTUALIZACIÓN DE SEGURIDAD.
No es recomendable hacerlo.

Otra opción para quienes no quieran modificar el archivo de configuración es instalar una versión anterior a Flash Player 30 desde las versiones archivadas para desarrolladores del sitio de Adobe, aunque igualmente no se tendría protección contra Meltdown/Spectre.
Básicamente el problema es que para evitar los ataques cronometrados basados en el susodicho error, entre otras modificaciones en la versión 30 se incluyo una ralentización intencional en ciertas funciones internas de Flash. Esto es lo que provoca el problema con Mel.

Debido a que es algo causado intencionalmente desde el propio Flash, las únicas opciones posibles son modificar el archivo de configuración para deshabilitar la ralentización (http://twd.io/e/qh5x0g/160), instalar una versión anterior a la incorporación de dichas modificaciones, o esperar a que el equipo de Adobe encuentre otra solución y la incorpore en futuras versiones.

Resumiendo: O quitas la protección o hay que aguantarse con la ralentización de Mel
