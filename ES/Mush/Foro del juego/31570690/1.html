GreaseMonkey y TamperMonkey no son hacks, son aplicaciones diseñadas para agregar scripts a las páginas webs, siguiendo una lista de requisitos (si es de tal dominio, si acepta todos los subdominios, si es una URL específica, etc.).

Generalmente, estos programas son usados por scripters y moders para diseñar mejoras y utilidades, incluso para juegos web. Un ejemplo de un script para estos programas es RadarZ, el actualizador automático del Atlas para Zombinoia (claro, RadarZ es una aplicación de chrome, pero tranquilamente podría haberse echo como script).

Algunos script pueden usarse para "hacer trampa" en un juego, aprovechando vulnerabilidades. Habría que ver que es lo que hace el Script.

Estuve leyendo un poco el código de fuente del script, y creo que es un "enhancer" para el Inventario, El Chat y El Mapa... es decir, un potenciamiento, que te organiza las cosas según los datos que recibe. Repito, no sé lo que hace, pero podría hacer algo del estilo: Si entras y sales de una sala, te registra en el mapa quienes estaban ahí hasta que se vallan, Te marca eventos importantes en una sala, o soluciona el tema de hacer dos clicks para mandar un mensaje... Claro, que podría hacer otras cosas, estas son solo un ejemplo que se me ocurrió a mí de lo que podría hacer.

En resumen... No, no es un Hack, es un script que te ayuda, posiblemente organizando la información que recibes.
