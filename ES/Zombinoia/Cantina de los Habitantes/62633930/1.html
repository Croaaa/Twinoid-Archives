Será un poco largo:
Las computadores funcionan con puertas logicas hechas de transistores, con las que puedes acceder a informacion en una base de datos, modificarla, etc. De normal, seria muy tedioso tener que ser tu quien fuese manipulando la activacion de las puertas logicas manualmente. Por eso se crearon codigos de programacion, los cuales se encargan de actuar sobre los transistores, y actuan segun los comandos que tu escribas o segun las acciones que activen esos comandos.
Pues las API son como configuraciones que sirven de sustituto de muchas lineas de codigo.
Un ejemplo: En vez de decirle a la maquina que hay que hacer con las puertas logicas para sumar dos cifras cada vez, tu creas una API que haga esa tarea, y la usas cada vez que quieres sumar.

TL;DR: Las API son programas ya creados que hacen mucho mas facil la creacion de programas mas complejos.

Con la API de zombinoia, seria muchisimo mas sencillo crear un juego como el zombinoia, ya que no se habria de empezar desde 0 o usando decompiladores para realizar ingenieria inversa.
