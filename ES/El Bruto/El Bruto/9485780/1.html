Para poner una imagen como enlace primero debes subir dicha imagen (por medio del botón Imagen al crear una página). Te aparecerá el código para la imagen, debes copiar dicho código. Luego clicka el botón Enlace, introduce el enlace al que quieres dirigir y luego en Ingresa el texto de tu enlace debes pegar el código de la imagen que copiaste antes.

Sería algo así:

[lien=Aquí va el enlace] Aquí va el código de la imagen[/lien]


Las páginas de los grupos usan un formato HTML.
