Se pueden crear aplicaciones para Twinoid, pero tienes que saber programarlas (el protocolo OAuth 2 para dialogar con la plataforma y el lenguaje que prefieras para tu aplicación) Además requiere que las hagas en tu propio sitio web o en algún subdominio del que seas Administrador.

En http://twinoid.com/developers tienes el enlace para solicitar autorización y claves de interacción, la documentación necesaria (en francés) y la API que puedes utilizar. Además algunos juegos tienen también su propia API específica.
