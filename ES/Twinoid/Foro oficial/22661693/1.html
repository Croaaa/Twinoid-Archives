Una solucion al problema, para quien le apetezca:
-Usando Firefox:

  • Instalar el complemento de Greaseymonkey
  • Usar este script que he creado para arreglar el error.

-Como usar el script:
Abris el script con el bloc de notas.
Cuando esta instalado Greaseymonkey os saldrá una carita, pinchais y elegis "New User script..."
Ahora poneis en name: Barra Twinoid, y le dais a aceptar, se os abrirá en un editor u os dejará selecionar uno (por ejemplo el bloc de notas).
Ahora solo copiar el contenido del script (tal cual esta) y guardais.
Actualizais la paguina de twinoid (o del juego en el que esteis) y listo.
