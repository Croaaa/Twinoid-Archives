En Html5 no mucho, crear Divs, usar <section> <article> <footer> , poner ID y Class.

En Css:

Se usar la propiedad de Display (La use para un menú desplegable).

Transitions (Para una de esas galerías Sliders).

Posicionar los elementos. (Position: Absolute/Relative/Fixed)

Se usar los border y el border radius.

Poner fondos de colores (Con RGBa para usar transparencias) o Imagenes.

Cambiar los tamaños (Width y Height) y usar porcentajes o cal() para esos valores.

Usar fuentes perzonalisadas.

Se usar el :hover y el :checked.

No es mucho, pero puedo ayudar en algo .
