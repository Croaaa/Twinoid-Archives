Aprovechando que puedo crear tantos temas como quiera para mi rol creo este que realmente me va a ser de utilidad.

En este post se harán los tiros de dados que se deban tirar durante el rol.

Cada vez que se requiera de tirar dados yo voy a colocar un post con el objetivo de el/los dado/s. Si se requiere un dado específico voy a colocar el nombre del dado, en caso de ser combate no lo haré, por las diferentes formas de atacar que tienen. También pondré el nombre de el/los usuario/s que deben tirar el dado.

Para que su dado sea tomado en cuenta deben citar el post respectivo a su objetivo.

En una situación de combate se debe declarar el nombre de la acción que realizan, mencionar el objetivo (si tiene) tirar el dado y anotar finalmente la posibilidad de acierto que su personaje tiene en dicha acción (Con el acierto general basta para establecer si hubo un acierto o no, pero agradecería si colocan la probabilidad de efecto y crítico)

Para revisar los porcentajes recuerden utilizar su hoja de cálculo personal si disponen de mejoras en dicha acción por las habilidades de su personaje (Y también mencionen que tienen mejoras así yo las reviso y no me olvido c:) Si no utilicen esta hoja de cálculo como referencia.

ESTE TEMA NO ES PARA NADA MAS No existe razón para escribir en este tema, solo yo escribo y ustedes tiran dados, y escriben solo lo necesario.

Si tienen algo que decir respecto a este tema lo deben hacer en el post de dudas. Por favor, eviten decir cualquier cosa en este post. Solo tiren el dado, así es mas fácil de leer y mas organizado para todos, porque este tema también les servirá a ustedes para revisar los dados de sus compañeros.
