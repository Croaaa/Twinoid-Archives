Este es un foro de ROL, y como tal, sólo se tolerará ROL en él. Si quieres hacer amigos, hablar de tu gato o crear algún juego existen foros como Intereses Twinoides o Juguemos en el foro, que se apropian mucho más a ello.

Para que un rol funcione se necesita responsabilidad por parte de todos los usuarios o de lo contrario termina siendo un caos. Es por eso que debéis evitar por todos los medios dos cosas:

Abrir temas que nada tienen que ver con rolear.
Hacer spam o foro-chat, si dos usuarios no se ponen de acuerdo en algo, aunque sea referente a un rol, es preferible que se comuniquen por privado a que desorganicen todo el tema con su discusión.



Algunas normas a tener en cuenta:


Para postear, o " Regla del ABCA "
Para evitar saturar los hilos y mantener el juego en un ritmo cómodo para todos sus participantes, deberá esperarse a que hayan posteado al menos dos usuarios antes de poder postear de nuevo en un hilo. Esto es:

Post User A
Post User B
Post User A
Tal ejemplo está prohibido.


Post User A
Post User B
Post User C
Post User A
En este ejemplo se es correcto.



No hacer "godmodding"
Godmodding es:

  • Hacer al personaje "invencible"
  • Hacer al personaje exageradamante poderoso,
    típico "te-mato-de-un-golpe"
  • Controlar las acciones de los demás jugadores.

Tampoco hacer "Meta-juego"
  • Usar información obtenida fuera de personaje dentro del juego.

Por lo demás, cada juego se regirá por sus propias instrucciones que deberán estar en el primer post del hilo, siempre y cuando estas no contravengan el Reglamento General del Foro.
