Para desbloquear niveles superiores de palacio tienes que subir de título. Por ejemplo el nivel 3 de palacio se desbloquea cuando eres Vizconde.

Para desbloquear los niveles superiores del resto de edificios tienes que construir el palacio a ese nivel. Por ejemplo para construir granja a nivel 3 necesitas palacio a nivel 3.

Algunos edificios requieren de otros edificios para ser desbloqueados. Por ejemplo Caldero necesita que primero hayas construido Herrería.
