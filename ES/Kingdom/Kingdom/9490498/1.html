La puntuación del panteón se calcula con esta fórmula, la cual saqué de los foros franceses :

La puntuación base es el título conseguido multiplicado por 100 (100 para Señor, 200 para Barón, etc, hasta 900 por Emperador). Multiplicamos esta puntuación base por la dificultad de mapa (principiante = 1, normal = 2, media = 3).
Luego añadimos:
- El tamaño del mayor ejército dividido entre 3,
- La gloria a la hora de morir multiplicada por 2,
- Y la reputación máxima multiplicada por 3.


Ejemplo:
Mapa principiante = 1
Título Vizconde = 300
Ejército = 188/3 = 62
Gloria = 1*2 = 2
Reputación = 8*3 = 24
