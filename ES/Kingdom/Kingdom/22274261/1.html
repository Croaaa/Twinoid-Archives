Guía para novatos -> ¿Cómo se calcula el comercio?


Ahí tienes una calculadora que te indica cuantos giros necesitas para que el comercio sea 0, rentable (se paguen solas las unidades) y estable (comercio máximo con ese número de unidades) Además de un gráfico con la evolución a lo largo del tiempo. Está en francés pero es muy sencilla de usar.


Por ejemplo con -104 de comercio y 180 soldados necesitas 304 giros para que el comercio del territorio sea 0 y 1.676 para que sea rentable. Con 208 soldados necesitarías 263 giros para que el comercio sea 0 y 1.849 para que sea rentable.


Tantea un poco el número de unidades hasta que encuentres un equilibrio entre giros y soldados que te convenza.
