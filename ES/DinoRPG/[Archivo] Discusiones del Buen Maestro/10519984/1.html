1)Caparazón->Enredadera -> Estadio Primario
2)Caparazón; Salvajería -> Enredadera; Refuerzos Korgon -> Acróbata; Estadio Primario
3)Caparazón; Resistencia; Salvajería -> Enredadera; Crecimiento; Refuerzos Korgon
*-*-*-*-*-*-*-*-*-*-*-*
En (5%):
Mutación -> Saco Ventral -> Sumo
*-*-*-*-*-*-*-*-*-*-*-*
En (15%):

1)Celeridad -> Golpe Doble -> Plan de carrera
2)Celeridad -> Golpe Doble; Ataque-rayo -> Plan de carrera; Zancadilla
3)Celeridad; Inteligencia -> Golpe Doble; Primeros Auxilios; -> Adrenalina -> Sobrecarga(+ )
*-*-*-*-*-*-*-*-*-*-*-*
En (30%):

1)Vuelo; Estrategia -> Recolección; Análisis -> Especialista; Ojo de lince; Nube Tóxica -> Aliento fétido (Esta opción puede resultar útil para el clan)

2)Vuelo; Agilidad; Estrategia -> Análisis; Esquiva -> Especialista; Elasticidad; Disco Vacum -> Agujero negro

3)Vuelo; Agilidad; Estrategia -> Análisis; Recolector; Esquiva -> Especialista; Elasticidad; Nube Tóxica; Disco Vacum -> Agujero negro; Aliento Fétido

4)Vuelo; Agilidad; Estrategia; Mistral -> Esquiva; Análisis; Tornado -> Forma Vaporosa; Especialista; Elasticidad; Disco Vacum -> Agujero Negro.

Nota: El teroz al ser sus elementos principales + y tener mayor % en , es más conveniente ir por Autocontrol que por soplo de vida
*-*-*-*-*-*-*-*-*-*-*-*
En (40%):

1)Garras en llamas; Cólera -> Aliento Ardiento; Sangre Caliente; Cazador de Gupión; Furia -> Aura incandecente; Siesta; Combustión -> Meteoros; Autocontrol; Rio de lava; Corazón Ardiente; Bola de fuego; Venganza
