Seigol
A ver, tu puedes escoger las competencias que quieras.
El único requisito es que antes la hayas desbloqueado.
Es decir, para conseguir Líder, antes COMO MíNIMO tienes que tener Salvajería > Tenacidad > Carisma

Eso no quiere decir que no puedas tener Salvajería > Tenacidad / Refuerzos korgon > Carisma / Acróbata, y después Líder.

Si quieres, también puedes coger Salvajería / Caparazón > Tenacidad / Enredadera / Refuerzos korgon > Carisma / ..... > Líder

Si quisieras, incluso podrías coger todas las competencias.

En resumen: Que cojas una rama, no implica que tengas que seguir solo esa rama. Puedes coger las que quieras. Eso si, para poder coger algunas competencias, tienes que haber cogido otras anteriormente. En caso de Líder, antes de coger Líder, tendrías que haber cogido las que tu dijiste.

**** Para que sepas que significan los símbolos, te coloco un miniglosario de simbolos xD

> Se usa para referirse a "desnloquear". Cuando se dice "Caparazón > Enredadera" quiere decir que se coge caparazón, se desbloquea, y despues de coge enredadera.

/ Se usa para separar competencias dentro de un mismo desbloqueo. Cuando se dice "enredadera / refuerzos korgon" quiere decir que coges primero una y despues otra, pero ya tienes ambas desbloqueadas.

Por si acaso, también te diré que las competencias se pueden desbloquear en grupo. Es decir: Si coges Salvajería, después podrás o bien coger Caparazón o Resistencia, o desbloquear Refuerzos Korgon y Tenacidad. En caso de que decidas coger Caparazón o Resistencia, o las dos, después podrás desbloquear todas las competencias de segundo nivel de una sola vez. Así tendrás más competencias con el mismo número de caídas.
