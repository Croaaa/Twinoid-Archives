Para empezar, cuando compartes la conexión de Internet con alguien, el modem o ruoter, les otorga una IP privada diferente, pero la IP Pública es la misma para los equipos conectados, y es la que se usa para identificarlos cuando se conectan a páginas WEB, ésta IP puede ser observada con diferentes páginas que te ofrecen el servicio de mostrar tu IP Pública.

¿Puedo compartir el ordenador con un amigo o hermano para jugar a Dinorpg?
