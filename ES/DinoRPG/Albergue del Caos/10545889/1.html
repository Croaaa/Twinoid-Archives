En realidad hay muchos, para la edición de imágenes, puedes usar Paint, al menos funciona para dar pequeños retoques a las imágenes que capturas.

Si quieres algo mejor, yo uso GIMP, es un buen programa que es gratuito, y tienes algunoes efectos para animación, pero su mayor virtud esta en la capacidad de usar capas, tranparencias, y algunos Script que te ayudan a hacer logotipos, sombras y efectos de luz. Pero suele ser un poco complicado para los principiantes, pera nada que no se soluciones con práctica.

Para la animación en GIF, hay muchos programas, el más simple es GifAnimator, igual es gratuito y no necesita instalación, aunque para lo único que te sirve es para crear GIF con imágenes ya existentes y creadas en otro programa.

El que uso, se llama Animation Shop 3, es excelente, ya que te permite crear varios efectos, y transiciones, y es muy intuitivo, sin embargo, es Shareware, por lo cual solo tienes 30 días de uso gratuito.

Los Morph los hago con SmartMorph, es un programa excelente para hacer este tipo de efectos, además de ser gratuito, el único pero, es que el Morp resultante es AVI, el cual con el programa anterior lo puedes convertir a GIF. Hay una nueva versión pero ya no es freeware. Así que es posible que te cueste un poco de trabajo encontrarlo.

Otro que uso es el AVD Video Processor 7.7, este no tiene grandes cosas para crear GIF, pero tiene la virtud de tomar una vídeo MPG o AVI y extraer los Frames(cuadros) para crear un imagen animada en GIF el cual lo puedes editar y retocar a tu antojo con programas más especializados.

Espero les ayude esta información. Saludos.
