para insctibires tienen que rellenar este "formulario" ( campos para entrenador)

nombre del dino
poscion de juego
dorsal

para patrocinadores

nombre del sponsor (patrocinio).

habra un patrocinador principal y otros dos de secundarios.el principal sera el primero que se apunte

aleineacion : 4-3-3

para inscribir a sus dinos (2 maximo) tienen que llenar el siguiente formulario:

dino:
poscicion: (potero, defensa...)
dorsal: (1,2,3,4...)
