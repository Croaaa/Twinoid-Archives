Un solo programa no suele ser suficiente para crear Gif atractivos, solía usar varios programas para hacer diferentes efectos.

Gimp: Para limpiar imágenes y crear diferentes frames para las animaciones, también para retocar generar el Gif definitivo.

SmartMorph: Un programa útil para hacer morphing (transformar una imagen en otra).

AVD Video Processor: Muy útil para extraer frames de un vídeo y transformarlo en Gif.

Animation Shop 3: Agregar unos efectos, hacer transisiones de frame a frame y para generar el Gif definitivo.

Hay otros que pueden ayudar mucho, como el Photoshop y el Flash, pero la verdad, no son las herramientas las que crean un buen Gif, es la imaginación y dedicación.

Ya te di los nombres de los programas que llegue a usar, ahora te toca buscar con San Google para encontrarlos.

Saludos.
