However, the stats are still present in the HTML source code. You can view them by executing one line of javascript or tampering with the CSS.


Google Chrome (MAC): Press CMD+Alt+i to get the web developer toolbar. Choose the Tab "Console" and paste this line (press enter afterwards):

var c="stat";document.getElementsByClassName(c)[0].style.display="block";
This line of code looks for all elements with the class name "stat" and changes the display value of the first match to "block" (instead of "none" which is the default).
