Say, all the planets (ITT chunk data) are stored on the main server, doubt it would ever exceed several TB if there are even 100 000 planets in the universe.

Whenever a player descends upon a planet, he downloads the actual map of the planet from the server and "hosts" the server of that actual planet. Other players landing on that planet will join the server hosted by the first lander. Thus, any edits made to the planet will be dealt clientside.
