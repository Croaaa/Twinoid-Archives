Galaxy55 App
How it all started...
The Galaxy55 app is a little project I started some time ago. I wanted to create something that has everything together. The wiki, the irc and the game. I got some positive feedback, so I decided to continue it.

What is the app...
Like stated, the app has all information bundled in one application. There is co-operation between the wiki and the app, so all information that is not hardcoded in the system is always up to date.
The IRC Channel is also integrated, along with the game itself (as of now only a link to the site).

Features...
IRC Channel
The game itself
Information on all the kubes (provided by the wiki)
Integrated webbrowser (thanks to the DJ Project)

Requirements..
32-bit Java JRE6
Windows
Mac and Linux aren't tested at this moment. If someone does test it, please post the results here.
