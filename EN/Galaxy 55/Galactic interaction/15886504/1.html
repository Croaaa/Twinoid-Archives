When you go to this page, does it tell you that you have Flash version 11,3,31,230 ?

If you have that version of Flash, then it will trigger this error #1009 repeatedly. This seems to be the version of Flash bundled with the current version of Chrome.

You have four options:

  • If you are not using Chrome, upgrade your Flash player.
  • If you are using Chrome but you also have another browser (Firefox, Opera, IE), try to play Galaxy 55 from that other browser.
  • If you are using Chrome and do not want to switch to another browser, then follow these instructions:
    • Install the latest Flash player.
    • In Chrome, type "about:plugins" or "chrome://plugins" in the address bar. This will show you the list of installed plugins.
    • Disable (deactivate) the built-in Flash player (11,3,31,230).
    • Close and restart Chrome.
    • Go to this page and verify that you are now using the latest version of Flash.
  • If you are using Chrome and do not want to follow the steps described above, then just wait without playing Galaxy 55. In a few weeks or months, Chrome should update itself and include a newer version of Flash without this bug.
