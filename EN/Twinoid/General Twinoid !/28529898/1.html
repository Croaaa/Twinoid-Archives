coro

Here's some of the changes that deepnight has posted elsewhere, along with some that I noticed on my own:

  • easier access to game forums via the left menu
  • easier access to user search
  • follow/reply buttons have been switched
  • the notepad is now a pre-made button, ready to be added to your shortcut bar
  • on PCs/Macs, the bar is 9px shorter
  • shortcuts are now left aligned
  • built-in support for separators in the shortcut bar (still uses up a spot in your shortcuts)
  • Twinoid facts now appear in the left menu (new ones load each time the menu is opened)
  • groups appear in user profiles (thanks Miudo)
  • the number of new notifications appears in the title bar
