It is a Muxxu thing.

While Twinoid will allow you to play multiple language games at once (it just keeps swapping your "default language" whenever you go to another language), Muxxu only lets you play one language version at a time and denies any language version that is not your currently selected language version.

There is a language setting on everyone's Twinoid account. I would call this the default (or preferred) language. This setting is able to be changed, but there are strict limits on how often you can change it (similar to how you can't change your username whenever you want)

The Twinoid bar will set a cookie that tells twinoid.com what language to display. This would be the active (or "currently selected") language. As Ghost said, going to a game can change this temporary setting, if the game is in a different language than what you had it set to

Now that those things are explained, I shall explain the various ways games handle language, and how this effects and is affected by the two language settings


First, you have the "single language" game. While the game may be available in different languages, a single language game will have separate servers for each language. Otherwise, you have games like Naturalchimie 2, where the game is available in its only variety, French. Since MT designed Twinoid to be fair towards multilingual people, single language games do not restrict access based on your default language. Single language games will change your active language to match its available language

Second, you have the "multilingual" game. This game will have a single user base, and all officially-supported languages will get the same content. When you first log into a multilingual game, it will set the local language to be your default language. This local language will not be affected by the active language. In addition, the active language will change to match the local language
Some multilingual games have visible buttons to adjust the local language. Games made before Twinoid have their own system to manage the local language, while newer games respond to a "lang" query in any URL

Finally, you have the worst system of them all, Muxxu! Muxxu itself is like a multilingual game, where there is a single user base. However, unlike the multilingual games I described above, Muxxu gates users from games that don't match their default language. Another difference is that there is no easy way to adjust the local language. Due to how Muxxu used to be French-only, MT did not offer a way to change your Muxxu account's language setting. Whatever option you had at account creation was permanently set. Now that it's a part of Twinoid, Muxxu will now change its local language setting if your default language changes


Muxxu is an old(er) system that had multilingual capabilities bolted onto it, and MT didn't give any consideration to multilingual people (or people who would get annoyed at how the French players get more games). They likely designed Muxxu like that in order to prevent people from playing multilingual games in multiple languages (to get more free playtime)

I'm glad they were more lax with Twinoid, and I look forward to the Muxxu games being freed from their cage .-.
