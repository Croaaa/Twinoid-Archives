Since all the games that have been incorporated into the Twinoid system no longer allow new referrals, let's discuss how to fix the problem

I can think of several ways to do it, and here are my ideas, in order of perceived utility

  • 1) Change referral links to use your account's Twinoid ID number, and continue to use the old systems
    • Pro: Easiest to code
    • Con: Effort spent on a game's referral system doesn't give any direct benefits to multiple games, which is the main point of Twinoid
  • 2) Replace referral systems with a centralized system on Twinoid
    • Pros: Allows you to get benefits from every game a person decides to play; gives users a single place to get an overview of everything (probably)
    • Con: Since someone could start playing games that you don't (or don't like to) play, you would miss out on some referral bonuses
  • 3) Same as #2, but with a twist. Instead of earning bonuses in individual games, you would earn points that can be redeemed for bonuses in any game on Twinoid
    • Pros: Same as #2; also prevents the problem of "losing out" on benefits
    • Con: Potential chance for M-T to be "greedy" and price things in a way that isn't fair to users
