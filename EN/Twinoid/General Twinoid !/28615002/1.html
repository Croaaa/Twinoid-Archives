Version 3 of Twinoid is here! This latest version incorporates many new features and upgrades designed to bring the site interface righ up to date and to make Twinoid.com accessible via smartphone.

We are listening to your feedback and will continue to make improvements in response to this.

New Twinoid Bar

With its resolutely modern re-design, the new bar organizes your information by subject.

In the left hand menu, you'll find network updates and rapid access to your games (and their forums). In the right hand menu, your account, options, and notifications.

You can add up to 15 custom shortcuts via the menu personalization page.

New login options

It is now possible to login to Twinoid using several external services (Google+, Facebook and Live). Current players can also activate this service on an existing Twinoid account.

There are several reasons for this: simpler access for new arrivals and increased security if you activate, for example, Google's 2-step identification process.

Groups

Replacing Muxxu groups, Twinoid groups are free and allow you to have private discussions with players with the same interests as you. We will continue to roll out new features for these communities in the coming months.

Twinoid API

External application developers can now access a brand new API which allows them to build applications for their favourite games.

Lots more features will be added to this as we go on.

Even more!

You'll find all the details about the update on this page : update 3.0
