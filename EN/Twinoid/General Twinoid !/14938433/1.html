Please read the General Rules, as these rules build upon them!This forum is the place for you to talk about Twinoid, or Motion-Twin in general. When posting a thread, make sure it's supposed to be in this forum!

The following lists are not comprehensive, please use your good judgement if your thread is about something not explicitly mentioned here. Things that are NOT allowed

Posts with profanity in themNo exceptions! Please practice some form of self-censorship. Threads about a game that has a forumIf a game has it's own forum, USE IT.

Threads that do not respect this rule will be locked or deleted. Threads about anything unrelated to TwinoidPlease use the Coffee Shop. If you accidentally post a thread like this in here, don't worry, a moderator should move your thread. Things that are allowed

Discussion threads about upcoming gamesOnly one thread per game. Suggestions to improve TwinoidYour feedback is appreciated! You are free to post constructive criticism or totally new ideas. Threads asking for help with a technical problemHowever, please use the technical support for serious issues! Discussion threads about Motion-Twin games that are not on TwinoidOnly one thread per game. If a sequel to the game you're discussing has a forum on Twinoid, please post a thread in the sequel's forum. (An example of this would be My Brute.)
