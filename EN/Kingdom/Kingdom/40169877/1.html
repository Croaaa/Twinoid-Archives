So I played with this. I don't have my excel file here on this system so I can't fine-tune it.

I made this python script

initial_trade = int(raw_input("initial trade: " ))
soldiers = int(raw_input("number of soldiers: " ))
number_of_turns = int(raw_input("number of turns: " ))
alfa = 0.2

for turns in range(0, number_of_turns):
trade = initial_trade + (2 * soldiers - 1 - initial_trade) * (1 - e ^ ( - alfa * turns)) / (1 + e ^ ( - alfa * turns))
print int(trade)

I believe only alfa needs to be tuned.
