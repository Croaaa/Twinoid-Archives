Kingdom works fine on Android 4 and iPhone with the right software. But it will make you look like a Pokemon Go player if you run around ;-)


On Android 4:

- Download and install Mozilla Firefox for Android from the Play Store.
- Install the old Flash Player 11.1 for Android 4.0 (11.1.115.81) which you can find on http://helpx.adobe.com/en/flash-playe...
This flash plugin is several years old, you should not use it to surf other websites other than kingdom or die2nite due to security issues.
- See two screenshots here just taken from my cell phone:
http://up.picr.de/28138848it.jpg
http://up.picr.de/28138849bj.jpg
I did not try on Android 5 and 6, but it may work with Dolphin browser which has in-built flash support (you need to enable flash in the options).


On Apple iPhone:
- Puffin Browser supports Flash (via a cloud based solution, but it works with Kingdom)
