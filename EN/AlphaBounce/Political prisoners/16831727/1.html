Hey everyone

Prisoner Bab6 has made a nifty tool which can help you take game screenshots very easily!

The tool takes only the game area: 400x360 pixels.

Its main goal is to make it easy to take screenshots of the game, particularly so they fit in my map (under construction ).

Here's a little picture of what the program looks like: ABScreenMaker

Before using it, you must have the .NET Framework 2.0 or above installed, abailable here. This comes included in Vista, so if you're running this OS, no need to check.

You can download Bab6's ABScreenMaker here.
*no installation required*
Current version: v1.4.8

This program is compatible for at least:
- Windows Me, XP, Vista, 7
- Internet Explorer 6, 7
- Firefox 2, 3, 4
- Opera 9

*How to use*:
1. Download/install the .NET framework if needed
2. Download the ABScreenMaker
3. Double-click to launch it
4. Go to "Param√®tres", select the folder where you want to save the screenshots. The checkboxes are for activating notifications when the screenshot is actually taken. Validate this.
5. Go to your favorite game: AlphaBounce, and start a level, do rendom stuff, etc. and press Alt+Space when you want the screenshot to be taken (if the game gets stuck, try pressing Alt again, but it shouldn't happen if you do the shorcut correctly). Usually, you'd want to do that right when the level starts.
6. Once you finish your game/pause/other, go to the ABScreenMaker window, enter your coordinates.
7. If you want, you can chect the "Bonus Provisions" box if there is one in the level as a reminder.
8. Press "Enregistrer" to save the picture.

The screenshot will be saved as [X][Y].png in hte chosen directory. If you have checked the bonus checkbox, the form will be [X][Y]P.png

Many thanks to Bab6 for this little tool

This is a translation of his original post + stuff I added.
