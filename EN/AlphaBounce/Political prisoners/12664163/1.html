I relaunch a very old project:

To build a very big image of the space background by assembling individuals screenshots.

The Karbonis belt is the most interesting part, but I cover all space.

State of the project (so far just started)


To extend it, I need your help:

1) Place the cursor on the settings icon to avoid having coordinates drawn.

2) Wait a time when the radar or mission items blinking is off.
One may also uncheck ""Visible movements" in the options.

3) Take a screenshot and save it in non-destructive file format with coordinates as the name.
png (Portable Network Graphics) is the preferred format, jpeg must be avoided.

4) Send them to me at bruno.mailly@gmail.com with "Alphabounce background" as the subject.
If they are hosted online just send the link or post it here.

Sub InsPicture()
curx = ActiveCell.Column
cury = ActiveCell.Row
MyFile = Dir("F:\AlphaBounce\*.png")
Do While MyFile <> ""
x = InStr(MyFile, "]")
y = InStr(x + 1, MyFile, "]")
y = Mid(MyFile, x + 2, y - x - 2)
y = Val(y)
x = Mid(MyFile, 2, x - 2)
x = Val(x)
If x > -127 And x < 127 Then
Cells(y + 576, x + 127).Select
If ActiveCell.Value = "" Then
c = Selection.Interior.ColorIndex
If c = xlNone Then c = 2
Selection.Font.ColorIndex = c
ActiveCell.FormulaR1C1 = "[" + Trim(Str(x)) + "][" + Trim(Str(y)) + "]"
ActiveSheet.Pictures.Insert("F :\AlphaBounce\" + MyFile).Select
Selection.ShapeRange.ScaleWidt h 0.3, msoFalse, msoScaleFromTopLeft
Selection.ShapeRange.ScaleHeig ht 0.3, msoFalse, msoScaleFromTopLeft
End If
End If
MyFile = Dir
Loop
Cells(cury, curx).Select
End Sub

I add 2 guidelines :

•When near nebulaes (bright zones in space), take screenshots closely spaced.
This is because they brighten the whole mini-map and cause tiling glitches.

•If possible group captures by series and include a list of screenshot coordinates in order done.
This eases my work a lot.
