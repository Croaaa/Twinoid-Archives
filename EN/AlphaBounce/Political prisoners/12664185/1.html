Sure, I just didn't took the time to incorporate the data from Lwxtz2004. The problem is merging.

I could add the information for "unknown cell tough".

AB Path Planner use the "last know good data" for a (X,Y) coordinates. While Lwxtz2004 use "many different rows".

I manually merged some of MacLambert, but it's hard task.

In AB Path Planner, you not only have coordinates for P and K. you get the coordinates for any kinds of bonus, like V, M, N, E, F, G, (the entire alphabet).

Example:

[8,18, '0, nothing, brown wall, MCM KMB EMCE'],

This location (8,18 ) has a K bonus.

I saw two folks using the following notation:

[8,18, '0, nothing, brown wall, /M/C/M/K/M/B/E/M/C/E/'],

Might be easier to add a search for that kind of notation.
