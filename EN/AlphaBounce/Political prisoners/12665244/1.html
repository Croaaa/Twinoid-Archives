It's the transformation power up. If you hold the mouse button down you will transform any block that isn't normal into a normal one. You can transform 1 block before the power is used up and has to recharge. It only takes a couple seconds to transform a block and recharge. Be careful when using it because it will also transform minerals. Also the light beam it shoots when transforming can be distracting.

For more information on anything you need, check out this site:

alphabounce.pbworks.com
