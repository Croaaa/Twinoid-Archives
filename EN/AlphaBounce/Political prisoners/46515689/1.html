Besides setting that option, you need to manually create a cookie, because somehow this feature got screwed up... Hope you have a clue about that (setting cookies)...

I'll try to explain it in an easy way for you:
- hit F12 while you're on the AB page. this will open the developer tools for you (should work in any browser)
- look for the console tab where you can run javascript code
- paste the following code there (this defines a function for setting a cookie):

function setCookie(cname, cvalue, exdays) {
var d = new Date();
d.setTime(d.getTime() + (exdays*24*60*60*1000));
var expires = "expires="+d.toUTCString();
document.cookie = cname + "=" + cvalue + "; " + expires;
}

- after you run that code (hit enter after you paste it, or some run button, depending on the browser), run the next line

setCookie('zoom',1,1000)

- that function call should create a cookie with name 'zoom', on the alphabounce domain, with value 1 that should be valid for 1000 days

That did the trick for me, hope you have the knowledge to pull it off and that it works for you as well .
