I understood with gem

The game will automatically try to recontact the server if there is a problem with the connection, so whatever happens, do not refresh the page. It will count as a loss if the server got the request for a new game (and therefore deducted your capsule).

The best thing to do is leave the game as it is, and try clicking on "my page" or "rankings", so that you can make sure your connection is working correctly.

Do not open another tab/page with alphabounce, or else you may also have a few problems

Also, some bugs do happen, and most players have lost 1 or 2 because of server/connection problems...there's nothing we can do about it and no refund has ever been made by the MT for that, sorry
