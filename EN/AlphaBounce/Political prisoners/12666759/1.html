Yes it happened to me and it is not a bug at all.

When You are playing levels, Your game doesn't require connection with the game server. Only if You do anything that needs to be saved like failing the level, completing the level, choosing another level, buying something at shop, completing missions, etc. - then game needs to send this information to main server which saves them.

So when You began to play, the message like "he started to play" was sent to the server . While opening Alphabounce again, another message was sent to the server, containing information that You have finished playing. But there wasn't information that You succeed, so the server counted it as failed level.

The same thing would happen to You if You just refreshed Your browser while playing a level.

It is kind of security, protecting the game creators from "stealing capsules". If there wasn't something like this, some people would open a lot of pages with Alphabounce and play without using their capsules.
