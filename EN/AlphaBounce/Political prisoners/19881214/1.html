I don't know if somebody uses "Let my browser zoom the game" feature ("my page" -> "edit my data"), but seems it's not working properly. You can't enable it on new computer or browser.
Workaround:
In any cookie editor (extension for chrome, for example) add cookie for domain "www.alphabounce.com" name it "zoom", set value "1" and some long expiration date. That's all. Hope it helps.

LeoLiss, I imagine there are quite a large number of people who use this feature. I know I would hate the game without it... There was another topic about this zoom problem, but I don't remember what the link was, but I can tell you the solution. You can create a bookmark with the following info:

name: whatever, doesn't really matter
location (this is important):

javascript:$hxClasses["js.Cookie"].set("zoom",1,Date.fromTime(Date.now().getTime()%20+%20DateTools.days(365*15)),"/",".alphabounce.com");

This way, anytime you have the problem, then you just access the bookmark and the problem should be fixed .
