Well, in fact the game is programmed in the Haxe programming language and then compiled into ActionScript or Flash bytecode
to be finally run by the Adobe Flash Player...

Haxe is a programming language that was developed by Motion-Twin including a library API to overcome compatibility issues between Flash 8- (AS2) and Flash 9+ (AS3) among others.

For instance, AS2 and AS3 are not compatible with each other...
So instead of writing this game twice
(one for Flash 8- and one for Flash 9+),
they decided to create their own programming language
to target both...

The Haxe creator is Nicolas Conasse,
one of the co-founder of Motion-Twin.
