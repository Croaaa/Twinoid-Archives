Edit the config file for Flash:
C:\WINDOWS\system32\Macromed\Flash\mms.cfg


Add these two lines to the end of the file:

EnableAllowList=1
AllowListUrlPattern=*://www.alphabounce.com/

I already had a mms.cfg file on my system. I made the changes after reading the administration guide at this site:
https://www.adobe.com/devnet/flashpla...

In the document (page it has alternate locations for the Flash folder for different Win types. If you are running 64-bit Windows in 32 bit mode (if your Windows is 64 bit and your browser is 32 bit) then the path to the config file is the following (the location in my first post is the only other location mentioned):
C:\Windows\SysWOW64\Macromed\Flash\mms.cfg

If you don't have this file, try creating it in Notepad and put the lines in my previous message into it. My file also has these lines in it to suppress the nagging to uninstall Flash:

AutoUpdateDisable=0
SilentAutoUpdateEnable=0
