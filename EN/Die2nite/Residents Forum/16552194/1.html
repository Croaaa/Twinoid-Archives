This thread is for discussing the idea that I originally posted in Workshop's "New mega Die2nite Event" thread, which can be found here.

The idea is basically inspired as bit of a throwback to S1 Style towns, albeit with the new game mechanics like BPs and Camping involved, mainly for a bit of fun but possibly for distinctions if Epoq would oblige (and if enough interest was generated)

The Event would be conducted in Small Maps (13 x 13) and in Hardcore mode, along with the only classes being allowed to be used being Scavenger, Guardian and Scout (+ Resident, obviously!) As well as there being an SP requirement of 300SP

The winning team would be the team which survives the longest period of time. The towns would be picked by an organiser from (Ideas on how to pick the organisers?) in a Live Draft.

Would you be interested in an event like this? Got any suggestions? Post here!
