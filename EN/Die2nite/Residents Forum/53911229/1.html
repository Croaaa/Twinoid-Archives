In 2012 (Season 5), during the London Olympic games, Motion Twin hosted (!) an event for Residents called The Apocalympics. Seven Private Towns were created and filled with 280 Residents. Each town had their Mayor, which was a special Resident designed to choose his/her team from the list of enrolled players.

I was one of those Mayors and that event resulted on me creating my own meta and lots of fun in the 4 following years.

Henceforth, I think its my duty to host a similar event now that Rio Olympic games draw near.

However we should first discuss how would we carry out some aspects of this event. Although I want to make it as close to what the first event of this type was, there are some things we might not be able to reproduce.

- 1: Number of towns and Town type: There are around 500 Residents alive in the current running towns. Around 120 of those are Small map players and might not represent the real number of long-term players we have.

I think that seven towns might be too much. Let's use 5 towns for now, but in case there are more interest we can increase that number.

As for the town type, that's up for polling:

LINK FOR THE POLLS

My thoughts on this is that Small Maps would be more interesting as they would really test which Residents are the best. The bigger the map, the higher are the oddities. The lack of camping sucks though.

- 2: Private Towns Issue: The former admin epoq created the 7 towns for the first event. He gave the password to access those towns to the Mayors and they coordinated the start of the event.

Admins are the only ones able to create a private town and leave it to create some more. Corvuses and other moderators have the ability to create Private Towns for the purpose of events as well, but once they create it, they are trapped inside. I consider this a bug and I also prefer not to count with Administration help if it can be avoided.

So I propose the use of Volunteer Heroes (with the Mayor ability to create Private towns), one for each town. Perhaps those guys can act as the Mayors and select their team of 39 Residents.
