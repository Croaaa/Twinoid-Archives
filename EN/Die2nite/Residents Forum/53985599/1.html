Corvus Epithumia

Dream is correct that this would be against the rules. Outside applications are allowed to pull information from Die2Nite only with some sort of user prompt. Whether this is logging in to an app, refreshing the app or pushing a button on the screen to send information to the app, that's all fine. Using an app to constantly refresh for the user would violate the terms of valid applications.

There aren't applications for D2N like sand on the beach. There are actually very few people writing them so it's safe to assume that, if applications like that should arise, it would be special cases.
