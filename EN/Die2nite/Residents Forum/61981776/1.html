var n=0;$('div.tid_stats div.tid_goal img[src*=r_j]' ).parent().find("div.tid_count span:first" ).each(function(){var t=$(this),d=+t.text()||0;n+=d;});n;

Firefox Bookmarklet:

javascript:void(n=0,$('div.tid_stats div.tid_goal img[src*=r_j]' ).parent().find("div.tid_count span:first" ).each(function(){var t=$(this),d=+t.text()||0;n+=d;})&&alert(n));



Useful to recruit heroic META players


Technical explanation:

In the DIV with class tid_stats
find the DIV with class tid_goal
then find the images with "r_j" in the URL image source link.

Those images are the heroic icons.

From that image, go one parent up,
then find the DIV with class tid_count,
in that DIV multiple SPAN exists with the same value,
so take the first one.

For all entry in this sets of now unique counter next to heroic images,
convert all those HTML element into TEXT
then into a Number or Zero if invalid,
add them up, then display the total sum.
