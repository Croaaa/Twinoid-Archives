Camp Site Upgrading from Dream:
This is for all who may not know how upgrading works

Optimally:

Each campsite can be upgraded 10 upgrades to reach maximum, and from those 10 upgrades of
previous day : 3 upgrades get lost and only 7 upgrades remain stored in this campsite for the next day.

Now about DO (Defensive Objects) Installs:
We can only install DOs only if the campsite has 9.9 or less upgrades and to make sure to reach the
maximum possible installing DOs always must come last after normal upgrades:

We have 2 possible variation for DO Installs:
• either making total 9 upgrades including the upgrades stored from previous days then install a
DO to reach 9 + 1.8 = 10.8 (this counts as 11 in the first calculator)
• or making total 8 upgrades including the upgrades stored from previous days then install 2 DO
in a row.
o first 8 upgrades + first DO which is 8 + 1.8 = 9.8 (this counts as 10 in the calculator)
o then install second DO which is 9.8 + 1.8 = 11.6 (this counts as 12 in the calculator)

NOW Upgrading next day after installing DOs in previous days:
• if you did 9 upgrades + installed 1 DO yesterday then campsite had 10.8 yesterday, and 3 get
lost the next day which means the campsite has now 10.8 - 3 = 7.8 stored.

SO from now on all you can do to reach the max is:
o no upgrades AT ALL
o first DO install to reach 7.8 + 1.8 = 9.6 then
o second DO install to reach 9.6 + 1.8 = 11.4
• if you did 8 upgrades + installed 2 x DOs yesterday then campsite had 11.6 yesterday, and 3
get lost the next day which means the campsite has now 11.6 - 3 = 8.6 stored.

SO from now on all you can do to reach the max is:
o 1 normal upgrade to reach 8.6 + 1 = 9.6
o 1 x DO install to reach 9.6 + 1.8 = 11.4

ALL remember the Order of camping
with others in same campsite:
• 1st Camper gets 0% penalty
• 2nd Camper gets 0% penalty
• 3rd Camper gets 10% penalty
• 4th Camper gets 30% penalty
• 5th Camper gets 50% penalty

This is pretty much all needed to know how to reach
the max of upgrades needed when penalties start to
reduce our chances to get 90%

NOW in this town we have to get the very optimal
from each campsite which means we CAN not and
MUST never install DOs when they are NOT needed
because as you can see each DO is worth of 9%
camping chances and each day you install a DO 1 % drops
from the maximum the campsite can reach.

So the best way to do it is to install DOs only
when you can no longer reach 90% without them.
