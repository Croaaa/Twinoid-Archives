What is it?

A user script that will show which towns another player shared an existence with you.

Cool. How I get it to work?

1) Install greasemonkey add-on for firefox or tampermonkey for Chrome ;

2) Install the script;

3) Go to your target soul page;

4) You'll notice a new lost soul animated icon right next to their soul points. Click it.

5) You'll be taken to Your Game History. This time you will notice some changes there.

6) Clicking the "Analyze!" button will check the towns for any match. some previous filtering was already made before you get to this page. After clicking the button the page will start checking each town. It can be a bit laggy here. Specially if you are Season 0-old and are checking another Season 0-old players. The analysis takes 1 minute for each 150 towns, approximately. It halts all navigation for that tab until it completes. I highly suggest you to save whatever you are doing as there is a chance your browser crash during this process.7) Once it completes checking you will see a list of towns (or empty if no matches were found) that you shared with that person.if you click the animated souls icons you will get that town opened in a new tab. Sometimes you need to refresh it so it shows the content.
