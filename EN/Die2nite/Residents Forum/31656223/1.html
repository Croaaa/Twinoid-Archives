You hear your companions moaning... The only thing you can see in front of you is a pretty small destroyed town, named Epoqalypse Tombstone. All of you run in the town, escaping the horde. You don't know how you guys will survive... You'll probably need to build some dangerous stuff to kill these zombies. Well, we have an architect with us... And a mad scientist... The combination of these two guys could make good stuff!

Welcome to Epoqalypse Tombstone, the very first D2N animation! The goal of this animation is to survive as long as possible. Each day, you'll be able to use à dice with 10 faces. The result of it will be used as a AP. These AP will be sent to the daily project, that will protect you from the night attack

So, each day, you can throw one ten face dice. The result of it will be decreased from the AP needed to build the daily projet.Each day, you'll need more AP to build the project, so you'll need to ask you friend to help!

I don't have any rewards for this animation, but I hope you'll enjoy and try to survive for a while

New projet at 24:00, so you have one day to do it
