Paste this in the F12 Dev console:

var n=0;$(".ghistory table.table td.season + td + td" ).each(function(){var t=$(this),d=+t.text()||0;n+=d;}),n;

Firefox Bookmarklet:

1705 days survived since season 5.

Technical explanation:

Find the HTML tag with class ghistory
inside find the TABLE with class table
in that table find the column TD with class season.

Those are all the seasons of game history.

For that season column go next adjacent TD column,
then go next adjacent TD column.

Those columns are now all the days survived in game history.

For all entry in that unique TD column set,
convert all those HTML element into TEXT
then into a Number or Zero if invalid,
add them up, then display the total sum.
