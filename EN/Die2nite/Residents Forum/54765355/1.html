I'm writing a userscript to enhance From Dusk Till Dawn (the commonly used map application). It's a great map but I and some others have had frustration over some of its features or lack thereof. My intention is mainly to add useful things and fix some visual problems. Some of these additions may even be integrated into (a future version of?) the site itself.

The script will need Tampermonkey or Greasemonkey to run, in a similar fashion to the Igloo script and others. It will only make frontend modifications (JavaScript and HTML), so it won't be able to store additional information, for example.

A couple of features have already been written. I will publish a "beta" version once I get a domain set up, so that the script can be updated. Stay tuned.
