Technical explanation:

Find the HTML tag with class ghistory
inside find the TABLE with class table
in that table find the column TD with class season.

Those are all the seasons of game history.

For that season column go next adjacent TD column,
then match those that contains the word "Lacerated" case sensitive.

Those columns are now all the Lacerated
town death in game history.

Display the number of such entries,
by calculating the length of the selected set,
since no further computation is required.
