Hey guys hopefully FDTD comes back to us, but in the meantime so we're not 100% blind we can use the Hordes map. To do so you'll need to do a bit of set up but I made this easy to follow guide that should help everyone get through it fast and easy.

and don't forget if you're using chrome you can right click on the page and select "Translate to English" so you can actually read the page. This guide walks you through as if you didn't convert to English though so keep that in mind

How to use BigBrother on D2N

1. First go to http://bbh.fred26.fr/

2. On this page click the “exit” button on the top right of the page.
https://imgur.com/lKMcuqf If you never logged into hordes before you should be able to skip this step

3. On this page click the button on the right that says “Conn. Autres Versions”
https://imgur.com/vKeb9Fv

4. This will pop out a few more options:
https://imgur.com/V9D6uEX

-Set “Serveur” to “Anglais”
-For “Cle” you need to go to the settings tab of you Die2Nite Soul page and copy and paste “Your Identifier For External Sites:” https://imgur.com/WQm9Qso
-“Nom” type in your user name

Once these 3 are correct click “connexion”

To easily access the map you’re going to want to connect it to your quick bar to do so follow the following:

1. Go to your twinoid profile and click “Bar shortcuts” on the top.
https://imgur.com/Ve1fK1i

2. On this page scroll all the to the bottom where you see a box that says “Enter page address”. https://imgur.com/F2g5eM5

Paste http://bbh.fred26.fr/ into this box and click add.

Afterward you will have this https://imgur.com/9i3Ejks



As a added side note I don't like the Hordes external map I think it's ugly and needlessly convoluted. I would pick FDTD over it any day.
