Cantine Cook

I hadn't made this public yet because it's simply not ready. I'm still working with BerZerg on some integration with DTD, and some other bugs are lurking about. I barely have the spare time these days to support one town during an alpha phase, let alone everyone haha! But this script is no secret. I'm aware of several metas using it...

How long it has been around? Not sure exactly, I had the first version done near the start of our hardcore town so whenever that was. It was buggy though and after the town died I just sorta left it for a bit, but have picked it up again recently and done some more fixing.

Are there any other tools? Sure. Do a search on userscripts.org and userstyles.org I have some bits on there and there's more by other people. Don't rely on Google though, this script won't be on there for a while until it starts getting posted outside of town forums and IRC channels.

But what does this actually do? It's more of a set of scripts that I want to add to over time. Currently it does the following:

Normal map updates
Hardcore map updates
Updates camping blueprints in MV/DTD, and topology in DTD
Shows the (local/server) time your AS will end
Makes your escort button flash, hopefully reminding you to set it
If you're a ghoul, it'll show your hunger % on the bar

I have lots more stuff planned but I want to get the current stuff working 100% first. I don't think any of this breaks MT's rules. It's not changing how you interact with the game, nor is it playing the game for you. It's definitely not breaking the XML automation rule. When it's ready I'll ask Epoq if it can be added to the directory, or remove stuff he's not happy with. Use it or not, it's up to you, but please don't expect any support until it's released properly!
