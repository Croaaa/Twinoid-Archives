If you want to live... Follow me!

If you use Rulesy plugin, make sure that the plugin is DISABLED,
and use FDTD manually to know if stuff is poisoned.

Why? The plugin reads the HTML tags which does not say if it is poisoned.

FDTD manual reads the XML feed which has different XML item_id,
so it knows if it is poisoned or not.

BTW, if someone on your square use to plugin too,
they must also disable the plugin; otherwise,
you won't know...

Unless you have griefer in towns, it's VERY rare to find poisoned food,
from personal experience, in explorable ruins, only suspect veggies can be poisoned. You rarely will find opened can in ruins, mostly lunch bag, biscuits, gums or closed can which are all 100% OK.

Finally, if manual FDTD reads that you have poisoned water + clean water, it will indicate 2 clean water...

If you have N poisoned and 1 clean water == N+1 clean water indicated by FDTD

If you have N poisoned and 0 clean water == N poisoned water
indicated by FDTD

Same with food.

If you have empty pistol, empty plastic bag or empty cooler,
then you can test water all at once.

Normally, you always test water inside the ruin and never
bring back untested water from within the ruin.

Rule #2, never test water inside the ruin if you don't have plenty of time and you know what you are doing, it's normally not worth it,
the chance of finding clear water in the ruin is extremely low.
