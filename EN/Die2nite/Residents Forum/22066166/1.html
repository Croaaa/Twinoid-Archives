In our single meta town in Season 6, we started off poorly. By all accounts, we should have come in third place of the top 3 towns doing their run at that time. Shortly after the town was done, a few of us were talking about Les Miserables and fodgy made the comment that we should write a Gigantic Fell parody to Les Mis. Since I’m not one to pass up a challenge, I decided she was right and set off to find the lyrics for some of the songs that we could fit in. After getting input from a few others, it ended up being quite the list. It’s worth noting that there’s a story arc across the songs (as there is with any musical). While each song can certainly stand on its own, it makes a better story if you read through the whole thing.

Huge thanks go to stevyn, Tetsab and fodgycakes for help writing these. If I had to do it all on my own, half of them wouldn’t have even been started and the rest would be half done or done very poorly.

Each song will be in its own post along with a youtube link to some not terrible recording of the song. If the parody name is different from the real name of the song, the real name is in parentheses.
