If you usually use firefox, and don't want to carry on using chrome, you can trick the site.

In Firefox, type "about:config" into the address bar

Click past any warnings, can't remember if FF has one....

Right-click on the page and go to New->String

Name: general.useragent.override

Value: "Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/535.19 (KHTML, like Gecko) Chrome/18.0.1025.142 Safari/535.19"

Your Firefox will now identify itself as Chrome and will get the right files.

This is only a temp fix. When things are working properly, you should go back and reset the key (in about:config, type general.useragent.override in the search box, then right-click, reset)

Can't guarantee this will work 100% for every part of the site as I'm dead, but everything I've looked at so far is ok.
