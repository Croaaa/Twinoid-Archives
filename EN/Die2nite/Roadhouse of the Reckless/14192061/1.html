So what is it all about you ask.
It's pretty much like Ultra-Hyper-Rare Blueprints thread, but this time we are finding ITEMS and describing possible use for them!

So the rules are pretty simple:
Roll 3 d100 by typing {d100 } {d100 } {d100 } without spaces
Pick corresponding words from each of 3 lists
Provide a description of your item. What is it? What does it look like? What it does? Use your imagination!
