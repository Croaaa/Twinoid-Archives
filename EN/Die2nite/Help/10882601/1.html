Figured that a lot of people still have issues with some of the bugs and don`t know how to deal with them, so it would be nice to put them in one thread for everyone to see, before they get fixed eventually. Here are some work arounds known currently (in pictures):

1) Accessing your soul to see distinctions and new points system (Clash participants)

It`s still possible to see your soul in Twinoid, it looks exactly the same as in "My Soul" tab

Soul via Twinoid - go to "My Twinoid" section, next point & click on the large icon/poster of die2nite to the right

2) Accessing coalition page (especially handy for MMJ purposes soon to be able to invite people/accept invitations, it also affects Clash participants)

Step 1 - go to gazette, next point & click on "Shamaniac Monday" to the right.

Step 2 - go to coalition tab, or wherever you want, besides actual soul page

Other work around is simply using THIS link
- go to coalition tab from here like in previous example.

3) Accessing register
Use mobile address for die2nite:

http://www.die2nite.com/m - next point & click on "registry" to see all actions being carried away in town.

Example Picture - note that you have to be in town to see register. If you`re outside, you`ll be able to see all actions from your current zone only of course.
