You can find most towns by searching "town`s name + map viewer" phrase via google - as long as there was at least one person in a town updating the map viewer at the time. For instance for the last season`s normal ranking winner, The Crepuscular Island:

Type the crepuscular island map viewer in google, usually the first result is what you`re looking for.

From there you can add "/day/x" (x being your desired day number) to the end of the link, for instance:
http://die2nite.gamerz.org.uk/town/31...

It`s possible to see all the info for a given day: map, bank, constructions, citizen`s list and all available statistics, making Map Viewer basically the ultimate spying tool.

The towns numbers are displayed in links.
