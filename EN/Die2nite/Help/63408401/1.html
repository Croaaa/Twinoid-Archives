On september 9, 2019
Tetsab

You can make them eat or drink if they haven't eaten or drank already as long as they have the allow access to rucksack option checked. This is unchecked by default so they would have to have checked it.

You can't make them drink booze or use drugs or drop select items.

You need to be a hero in order to escort others.
