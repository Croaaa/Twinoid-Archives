Your first idea has merit, but the second one could be a bit OP or unnecessary (what if I chose that for a few towns, then want to go back to playing as a full-class?). But there are a few details that need to be hashed out:

At what point would someone be able to chose a dual-class? Would it be a certain number of Soul Points, or a certain number of Hero Days accumulated?

How would each class be nerfed into a demi-class? Would it depend upon the two classes chosen, or would the same attributes be nerfed regardless?

How would distinctions for playing as each class be handed out? Or would they be scrapped altogether if playing dual-class?

There are probably other questions that would need to be answered, but this is a start.
