If you're using Android 4.4, the only option you have that I know of is to use the Dolphin browser. See http://forum.xda-developers.com/showt... and follow the directions at the top exactly.

If you're using pre 4.4 I know Safari works (not sure about the others), but you'll still need to download adobe flash from here: http://helpx.adobe.com/flash-player/k...

Search that page for "android" and you'll see the releases. Download one of the apk files to your phone and install.

Regardless of which browser you use, you can't view the map without some form of adobe flash.
