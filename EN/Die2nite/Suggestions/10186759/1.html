Basically a new game mode which is a bit of a throwback to pre-blueprints game style, although updated a bit to avoid the set in stone build orders and uninteresting gameplay of yesteryear.

Basically you get every building already unlocked (Like before), in their respective trees, but as you build some buildings, some paths are locked off, i.e. If you build, for arguments sake, the Butcher, you can't build the Veggie Plot.

Another Feature would be that, since there is only so much space in a town, you have spacial restrictions on what you can build; some buildings, like walls around town, take up none of this 'space', others may take up x amount of space, for example, Butcher may take up 3 (Insert units which i've not thought about here) out of a total area, like 500 (of these units i've not come up with) and once you've exceeded this, if you exceed it, you have to upgrade the current buildings (whole new idea that i've only just thought up 10 seconds ago) or demolish them to build new ones.

The Attacks would be kept in line with the current attack formulae in terms of how long towns last, maybe they use a slightly different set of algortihms, but towns shouldn't exceed 20 days before camping maximum, really.

They'd also have their own leaderboards.
