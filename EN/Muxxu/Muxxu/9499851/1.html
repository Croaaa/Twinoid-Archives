I think it is one account per server... Motion Twin games have this. So, you can have an account on the French server and one on the English server: obviously you need to speak both languages (or use web translation) as the servers tend to insist on using that servers language only in forums. The key is that there is no interaction between the servers: with two accounts on one server you can use both in combination, or pass resources between them (cheating), an account on one server cannot interact with the account on another.

Might be worth checking, if you are worried.
