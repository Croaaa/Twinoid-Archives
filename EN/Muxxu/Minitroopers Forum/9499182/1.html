I decided to share my strategy on making a lot of recruits!

Lets make it a fair battlefield for everybody!

First step: you should be able to change your IP address, dynamic IP address changes all the time compare to a static address so you need a dynamic IP address...to change a dynamic IP address what I do is turn off my router and when i turn it back on(wait for like 5-10secs before turning if on again) it will give me a new IP address. You can try it if you want to check if you have dynamic IP or a static IP

Second step: Removing your caches and cookies, or clearing history data of your browser, from my experience clearing/deleting it once or twice doesnt work...i usually clear it like 5 to 8 time for it to work...and you should be able to do it before the internet gets back, it usually takes like 1 to 2 mins for my router to get connected again

Third step: Create 2 armies then return to step one again(you can only make 2 armies per IP)...it should be unlimited and you will not have any problem doing it if you have a dynamic IP..

Find WAN IP
If you want to see your WAN IP click here... http://www.findmyip.com
only when this number changes can you create two more recruits.

IP lease expirey date/time
I'm still using old Windows XP, if you are too then this is how you find out when your IP lease expires (which should change you WAN IP for you).

1. Click Start
2. Click Run
3. Type in cmd and hit ok (this opens a Command Prompt, it's all black and serious)
4. In the Command Prompt Type ipconfig /all
5. Press Enter

This will give you the LAN IP info and maybe also your lease expirey date/time.
