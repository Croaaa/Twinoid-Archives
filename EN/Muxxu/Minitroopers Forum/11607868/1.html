Hi guys, I would like to suggest a new tourney format here...

RULES:

Match Format:
5 Sets of 1v1s. Each 1v1 is a Bo3.
Attacker to be determined by toss coin.

Match Points:

Match 1: (1 point)
Match 2: (1 point)
Match 3: (1 point)
Match 4: (1 point)
Match 5: (2 points, Ace Match)

These matchups will be determined at the start of the week. Match can end in either:

Team Win (3 points)
Team Lose (0 Points)
Team Draw (1 Point)

ROSTER format:

Each team has 5 members, with 3 substitutes allowed. Each team will submit lineups at the start of each week, and will have played all matches before Friday.

Rest of the rules are same as KLD rules, except that maybe we could make a mixture of 2 big boys and 3 "sixed" accounts...

League style...hmm...what do you think?
