Starting up a new "continuous championship" style game.

Rules are:

1) Only armies with SIX or fewer troopers allowed. Any army that gets a 7th trooper will be removed from the listing

2) To sign up add a link to your HQ page (tinyurl is allowed to help with forum formatting issues) and army name then playername afterwards. This is to make it as easy as possible for everyone else to see which army name to use to fight.
e.g. http://tinyurl.com/knife-fighters Knife Fighters / pickupsticks
LINK then ARMY NAME then / PLAYER NAME

3) To move up the ranking, beat the person above you then post a link to the replay here. Copy and paste the list, adjusting your position, and re-post it.

4) MAKE SURE you use the correct list from the last post in the thread. MAKE SURE nobody else has posted at the same time as you. If this happens, then edit your post to correct it. Anyone that messes up the list more than three times in a month will be penalized five places, because it's a real pain for everyone else.

5) Only fights where you sent the challenge (are on the left) are allowed. Only direct challenge fights are allowed. Fights that end in BAT1/BAT2/BAT3 or MISS1/MISS2/MISS3 are not eligible.

6) A player may choose to go "inactive" by marking their army with [REST]
e.g. [REST] http://tinyurl.com/knife-fighters Knife Fighters / pickupsticks
If a player directly above you is inactive and you can beat the next player then you may skip the inactive player.
e.g.1) player A
2) [REST] player B
3) player CPlayer C may choose to move up into first place if they can beat player A. They cannot move into 2nd place unless they beat player B in the usual way.

7) Only one army per player is allowed.

...that should be enough rules to keep things in order, but we all know how it works so stop reading and get fighting.
