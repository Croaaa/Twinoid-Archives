KlarkMorrigan
I'll just re-post this thought.If you get to pick unique skills each level-up with a fixed upgrade path (3 drawn, pick 1), you should pretty much run out of them at level 45. I've run into a NPT with duplicated skills, but it's an exception (bug?) rather than a pattern.
How I see it, there's rather a path and not a tree: each new level you get 3 picks, and at the same time those options are excluded from the pool. Which means that you shouldn't ever meet skills you discard again. Otherwise it would be possible to take repetitive skills. Or, if you build a tree and not a path with proportionally more combinations branching from every pick you take, is it so cost-efficient to store all those instances with a trooper instead of randomizing each level?

Aside from specs and starting skills you have 130 to pick in total. So the bigger each level's pool, the fewer levels you can have in total. In case of 5 skills per level in a pool, you get 28th level as maximum. To be honest, I didn't see any NPT with any higher. It doesn't make much sense making level pools any bigger than 3, though, since what wasn't offered can be offered later, and what's offered won't show up again.
