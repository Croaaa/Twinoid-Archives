Minitroopers League: test run
An experimental competition

It could be fun to have a league competition going as well as the knockout cup. Is anyone interested?

I'm proposing the following conditions.

Compeition to last for five weeks. Six teams per league, play everyone else twice (home and away match) with six matches set for each week. Week to run from Saturday to Saturday, but results must be posted to thread by Friday for any challenges.
One point awarded for a win.
Failure to send a challenge results in a default win for your opponent, scoring them one point.
The army with the most points at the end is the winner. In the event of a tie I will organize a playoff!

Signups to run for a week or so and then a random draw done for the groups of six. We may end up with more than one league if more than six people sign up! If we have a number of applicants that isn't divisible by 6 then the last to sign up will miss out this time around.

As I'll be administrating this competition my decision will be final in all matters of dispute. But I'm a reasonable person, so there.

IF YOU ARE INTERESTED THEN SIGN UP BELOW
Sign up posts should have this format:

Army Name:
Link to army HQ page:

Questions and discussion are encouraged too.

THE WAY IT WORKS.

The fixtures for each week for your league can be found in the posts on pages 4 and 5 of this thread. The week normally begins on the server reset at the start of Saturday however in this first week today is a "rest day" and the competition begins tomorrow.

For each fixture, the team listed first must send the challenge. The challenge can be sent at any time during the week. Only the first challenge sent counts. "Mission" battles do not count, you must choose to infiltrate the army by typing in the name of your opponent.

A link to the battle should be posted in this thread. It is helpful if you include a comment on who won. Put this in a spoiler tag if you want to keep the suspense going for the spectators.
Links must be posted here before Friday (e.g. the server reset one day before the next week begins) failure to do so will result in a default win for your opponent.

One point is scored for a win.
I will be keeping a league table for each division, and aim to post them at least once a week. In the event of a tie the player with the fewest default losses will win. If still level then a series of playoff matches will be arranged.

If we keep the same format and move into a second season then when the season is over the top two teams in each league will move up and the bottom two teams will be relegated and move down. If people are interested in playing a season longer than five weeks then... well, let's get the first season out of the way and see where we are.

Any questions please ask!
(as the administrator my decision is final in the event of any weird business going on)

Good luck to everyone Hope this competition is fun for all involved.

If you play the same person in two different competitions (e.g. league and cup) then make sure you post in the thread to explain which match is which.

You will need one match for EACH competition.

So you might post something like this:
I am challenging MyOpponent twice today. The first one is a cup match, the second one is a league match.

As long as you make it clear which is which there shouldn't be a problem. Your opponent can see when and in which order the matches were started in... make sure you post BEFORE you send the challenges!
