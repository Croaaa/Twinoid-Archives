The first set of 3 unlock at once, and you only need 8 points


Here is how the assistant system works (after spending like 300 ;n; ):
----
In between classes, you get new buttons to click on, with a picture of an assistant on each (for example: ). Hovering over a button shows you the effect of the particular assistant. Clicking on it will prompt you to pay 10 to get the assistant's help in the next class (James uses his cell phone to ask for help). You can only have one assistant in a class. I don't have confirmation from someone else, but I assume the effect of Bonus Assistant is a third option for an assistant. The buttons seem to change order between each class, so I guess someone without this skill only gets to pick between two random assistants
