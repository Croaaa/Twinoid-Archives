For maximum efficiency you should collect:
Razir + Ignik + Fulgor = 1 more each day
+ + Cammemberk = 1 more each day

But it really depends on your playing style. If you have problems with monsters find more hearts, blades, cammemberks, vol-au-vents. If three rainbows a day are still too less for you get the umbrella to get another one.
The fork is a very powerful tool, you have a 10%-chance to kill two hearts with one blow.
If you always clear every isle complete the dice is a very good tool for you because you have a chance to win a ice cube for a rainbow.
The Grimoire is a very powerful book you can ban a game you like.
If you want to open green chests you need a magic wand.
If you want to try out specific games you have to find game consoles, and the Chromax to use rainbows for playing.

There are many items in the game it really depends what player you are. You have to make your own choices but you should think as longer the distance is it is more difficult to get the item. Monster difficulty grows with distance from the start

Have fun with fever
