You can repair rooms instantly with repair kits, including rooms which are occupied, and you don't have to buy them as guests occasionally leave them behind when they depart. It's not the best use of them, admittedly - I like to save them up for when VIPs' rooms get damaged just before they are about to leave or to repair a room after using fireworks in it - but if you use the repair kits wisely, i.e. only use one if there isn't enough time before the reset for an employee to do the repair, it should help a little.

When time is really of the essense and you're down to the last few seconds before the reset, you can place a guest in a damaged room then use the repair kit on it while they are in there. If you manage to get it done before the reset, all well and good, but if not, at least you won't have lost the guest because you didn't find them a room in time.

Collecting all the objects left behind in a single click wouldn't necessarily be a good thing. For example, I often deliberately don't collect them unless I need to put someone in that room or I need the object to satisfy a guest. That way, when I get given a quest which includes picking up several forgotten items, I can complete that part of it fairly quickly instead of over a number of days. (I'm becoming convinced that MT's code is written in such a way that when you have a task to collect forgotten objects, fewer guests leave them behind!) It would be nice to be rid of that pop-up message though. The only time it tells you something you didn't already know is when you retrieve a wallet (and that's assuming the contents are variable, which I'm not sure they are).
