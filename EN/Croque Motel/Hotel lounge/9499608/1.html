The cost of these rooms goes up for each one you buy. The first is 200, the next 300 and +100 for each after that. The more you build , the less chance that you will break-even. The increase applies whatever type of room you build (Wet room 1 = 200, Wet room 2 = 300 AND Wet room 1 = 200, Disco 1 = 300)

The cost could also cover the cost of extending your hotel, but it should only be considered if you are building a hotel bigger than the size needed to bequeath. Each of these rooms will count towards the required size

Another consideration for whether it will break-even is how long you will be using it. If you buy them early, you have more chance of getting a profit from it
