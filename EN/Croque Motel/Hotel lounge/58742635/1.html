The game and server are located in France. There is a setting you can select to use your own time. To use your own country time go to you user name,
1. double click you will see My Profile, to the right my Account settings,
2. scroll down to Options, select Options
3. to the right you will see General Options
4. you will see server time zone
5. uncheck server time zone.
6. scroll to the bottom of page.
7. select save changes.
8. you will now be using the time zone from computer.
