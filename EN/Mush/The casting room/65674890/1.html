Yeah sure! Everyone is invited, let's see, this is what I have done to fix the flash problem and be able to play the game.

So the first thing is to find an old version of a browser that supports flash, in my case I'm using "Firefox version 78.8.0 esr"

You can find any old version of Firefox on this link
https://ftp.mozilla.org/pub/firefox/r...

And... well... the flash player is harder, I found this youtube video
https://www.youtube.com/watch?v=i5Qlf...
And the files worked... First you uninstall flash (in case you had some old version) and install it again depending which browser you will use. For me Firefox is working perfectly fine.

Oh, important thing... as soon as you install firefox you have to go to settings and disable the updates to avoid going to the newest firefox version that doesn't support flash player.

But being honest it was worth, the game is amazing, pretty sad there are so little people still playing, but the other games still are amazing, more now that you can play all you want since they put everything free.

Okay, hope to see you in the casting and hope we can gather enough people to actually play a full ship game.
