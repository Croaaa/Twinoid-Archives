So! I have decided to work on putting the Daedalus in FTL as a playable ship using this Superluminal Modding Tool, and figured this would be a good place to talk about it, ask for advice, and distribute it if/when I've finished it.
EDIT: Download it here!
NEW VERSION:
-Artillery messed with. Tweaked in a manner to make it less a pain for everyone involved. Artillery reduced to a triple boss laser, with a second one in the main weapons because it apparently only does one damage per shot.
-Added additional weapon slot
-Halved starting power to reflect five fewer systems.
HOW TO USE:
1. Download this mod manager. Put it anywhere you want. Somewhere easily accessible, preferably.
2. Once the manager knows where your game's files are, put the .ftl file in the Mods folder in the Mod Manager's folders.
3. Run the Mod Manager, and patch the .ftl file. If you don't want to run the mod when you play, you can return the game to vanilla FTL by patching none of the available files.
Known Issues

-Weird graphical bug resulting in some room borders being thinner in spots. As far as I can tell this is purely graphical.
-No art aside from a shield. Purely graphical.
-ARTILLERY:
The meter at the bottom of the screen says that it takes 50 seconds for the lasers to fire. It actually takes the normal time for a defense laser.

Additional notes
1. The ship is as accurate as possible, complete with six basic lasers in each turrets. To balance this somewhat, there are only two other weapons slots and no shields. The turrets are unaimable and will fire automatically upon random rooms. EDIT: Note that it has not yet been rearranged to reflect the single artillery system.
2. The ship has exactly enough power for you to have the six guns, the oxygen, and the engines running. No more. Also, the oxygen doesn't start out powered, so you'll probably want to change that.
Credits
Thank you Vince for coming up with a more compact design for the ship that is presumably the only reason it's working right now, everyone else who posted in this thread with feedback and advice, and kartoFlane of the FTL modding forums, the designer of Superluminal, who helped me pick through my first modding effort without ruining everything ever.
Tell me if you find any bugs!

2. Once the manager knows where your game's files are, put the .ftl file in the Mods folder in the Mod Manager's folders.
3. Run the Mod Manager, and patch the .ftl file. If you don't want to run the mod when you play, you can return the game to vanilla FTL by patching none of the available files.
Known Issues
-No mounts. None. I'm not entirely sure why but whenever I start messing around with mounts the the game refuses to accept the mod, even if the mounts are removed. Possibly due to the fact that I don't know what I'm doing when I edit the XML file.
