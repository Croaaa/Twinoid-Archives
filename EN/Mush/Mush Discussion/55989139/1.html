Peanutz
Mais pourquoi est-il aussi mÃ©chant ?

I don't remember exactly, but there are only a few possible injuries : broken finger, broken nose, bruised shoulder, broken foot (I think). They don't all need to be healed, for instance broken nose has almost no consequence, and bruised shoulder is only a problem for pilots since it affects targeting.

Broken fingers do cause actions to cost 1 AP more, but not all of them. Actions that can use special points (...) don't cost more, even if you don't have the skill. For instance, self-surgery uses one healing point for nurses, and it still costs 4 AP, not 5, for everyone.

Injuries themselves are not such a big problem, especially on slow-filling ships, where you have time to do surgery. To me, the main problem is that players need to learn how to deal with injuries. For example, I have seen way too many people ask for surgery just for a broken nose, or for a bruised shoulder that did not incapacitate them at all. On the other hand, sometimes people walk around with a broken finger or foot, which is totally absurd.

As for sepsis if surgery fails, it takes several cycles before it can kill someone, and as always in the game, communication is the key. If you catch sepsis, you can ask for someone else to heal you instead of using 3 AP more on self-healing.
