This is spillover from the 'Gather current bugs here' thread, the OP became too long to post.

Translation issues:

◾Some messages generated by merchant ships are not translated to English.

◾Kaladaan rebel base (day 5) correctly gives 6 morale but creates a morale gain message claiming 3 morale was gained.

◾The Genius skill's tool tip names it Technician.

◾One bit of 'over translation': The knife instant kill message that begins "So are you," is confusing (causing players to suspect the killer is mush, when a mush has been killed) and should be changed to "Et tu,"

◾On the endpage, pushing someone out the door shows up as Vous avez flanqué [character] à la porte. instead of "you've pushed someone out the door". The 'you've accessed the FM Radio' is also untranslated, as well as 'you've completed a trade' and 'You talked with Andie'.

◾The Forum 'nexus' link for each casting group's page, example http://mush.twinoid.com/g/rp-or-die/n... , shows that group's members but not other viewers a line of untranslated text: "Vous n'avez pas le droit de poster sur le nexus du groupe. Demandez l'autorisation au chef de groupe."

◾Research 'Tesla SUP2X' states that it increases the number of turret charges by 2, actually increases them by 4. (Translation error versus bug?)

◾Both Phagocyte and Fungal kitchen skills tooltip describe the action as covert, but those actions are both invisible (Translation error versus bug?)

◾Nightmarish skill description states the action costs 0 AP, actual cost 2 AP (Translation error versus bug?)

◾Pilgred's repair states it will allow the coffe machine to produce coffee every odd cycle, actually produces coffee every cycle (Translation error versus bug?)

◾Vending Machine 'Targeting Assist' states it improves in-flight and turret targeting by 15%, actual improvement appears to be 25% (Pilots go to 99% chance to hit from 80%) (Translation error versus bug?)

◾Trapper skill description is misleading: "For 1 Action Point, you can booby trap a shelf unit with a Spore. If someone touches the Unit, they will become Infected. If they are Icon Mush, the spore is lost." Please replace with: "For 1 Action Point you can booby trap this room with a Spore. If a human gets an item or uses equipment here they will become Infected. If a mush does, the spore will be lost."

◾The Transfer skill tooltip claims the skill is covert, but it functions invisibly.

◾The research Patuline Scrambler description is misleading: "The infected can no longer communicate on the Mush channel." Please replace with: "Communication on the Mush channel is no longer possible."

◾Distinction Natamist contains a typo: "Represents ofstroyed with the Natamy Rifle" Please replace with: "Represents mush destroyed with the Natamy Rifle" (If it is mush, and not all kills?)

◾Drill description is misleading: "Expedition : Doubles the gain form "Hydrocarbon" events" Please replace with: "Expedition : Doubles all fuel gains" (Translation error versus bug?)

◾The :exp: icon, which previously appeared as a pair of binoculars, now shows as a broken picture link followed by the word "Expé"

◾The motivator skill states that 2 AP can be spent to give 3 MP to everyone in the room; it actually costs 2 AP to give 2 MP to everyone in the room. (Translation error versus bug?)

◾When the Mad Kube is solved, it creates two identical roomlog messages about glory gained for solving it (the glory is only given once).

◾Description of Broken foot injury refers to "food" instead of "foot"

◾The skill 'Confident' is misspelled, correct to 'Confidant'

◾The description of the oxygen tank contains the word "contnue", it should be spelled "continue"

◾The title gained from playing Ian, "She likes a little, a lot..." does not really mean much in English. Instead, "She loves me, she loves me not..." is a 'game' played with flowers, and would make sense in connection to Botanist Ian.

◾Derek's bio page is entirely in French

◾The mush skills Ninja and Bacterophiliac contain visible <em>tags

◾Diplomat's Cease Fire ability states it lasts until the user leaves the room. Should state it lasts until the user leaves the room or 3 cycles have passed.

◾On the forum, the 'new messages' message is not translated: "# nouvelles résponses" and the castings page countdown to launch is incompletely translated: "effectués à 1 H 30."
