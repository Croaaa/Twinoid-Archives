All those sweet end words, all those titles you worked so hard to get, all the memories of those improbable and wonderful moments, soon lost to time like tears in rain… If only you could keep them all forever…

Okay, I’m done being melancholic, here’s one last script of mine. It adds a nice button on your “My account → My file” tab, under the trips and castings sections, which will create a standalone page like this one that you save on your computer.


How to install
1. As always with userscripts, you’ll need Tampermonkey (Firefox, Chrome); for some reason, it doesn’t work with Greasemonkey.
2. Click here to install the script.


N.B.

  • The experience bar displays your true progress, recalculated by the script.
  • Sadly, I couldn’t find any way to fetch the description of your wins.
  • The character who ranks first in a ship doesn’t have their death cause listed, so I can’t fetch that either, but it should be obvious in context anyway.
  • Tooltips can be pinned by clicking on the thingy that sourced the tooltip. (Does that make sense?)
