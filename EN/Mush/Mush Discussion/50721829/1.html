After many sleepless nights, I am happy to present you the new Mush visual theme for the community to enjoy. This is my little gift to you all. Spread the word!

MUSH CHRISTMAS EDITION
Screenshot: http://imgbox.com/CFM0WWMR

If you are familiar with the other theme I created, you will notice a lot of improvement and bigger changes have taken place. I wanted to do this for a very long time, but since the Idea was to change everything possible, I couldnt make myself invest the time in figuring out the things I needed. It changed when it struck me that RP or DIE! is going to have a special Christmas theme. So I thought it was the right time to make it happen.

Installation process

Google Chrome:
1. Install Stylish Chrome extension by following this link:
https://chrome.google.com/webstore/de...
2. Install the theme by following this link and pressing install: https://userstyles.org/styles/121345/

Mozilla Firefox:
1. Install Stylish Firefox extension by following this link:
https://addons.mozilla.org/en-US/fire...
2. Install the theme by following this link and pressing install: https://userstyles.org/styles/121345/

Comments, opinions are welcome. If theres any visual inconsistencies make a screenshot.
