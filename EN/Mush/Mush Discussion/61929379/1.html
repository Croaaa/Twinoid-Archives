Misterme7

I've had that happen to me, though I also had some other programs running that were slowing down my PC. You have anything else running? Mush tends to eat up a lot of space.

Also, how long has the ship been running? Every time the game refreshes, which is every action, the game refreshes, which means reloading every message ever sent. The more messages, the longer this takes. It starts to be noticeable as a ship drags on. I'd advise using the Ctrl+W plug-in if you aren't already if your ship has gone on for a while. It has a utility to unload older messages, which will save time.
