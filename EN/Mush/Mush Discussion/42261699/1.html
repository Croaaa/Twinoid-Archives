Nope, only 1 account per physical person. (meaning an ip adress can host more than one account if they are for different persons).

If accounts with same ip adress (+some other requirements it seem) happen to be on same ship, they get a "multi teamster" status preventing them from launching new games and have to contact support.

If you want to play on more than one ship at the same time, that is a possible choice by playing with your same one account on one or all of the other servers.

Note that every server has its own language. If you wish to play on the French server, be prepared to write and read French (I have done this using Google translate. It was effortful, but possible).

Each server is completely separate, and you are allowed to play on each one with your same account.
