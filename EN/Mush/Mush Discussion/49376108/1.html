Small(Mush) v1.5

Small(Mush) also works on computers as a substitute for most Flash functions and has a Computer Mode in its settings.
If you already have Greasemonkey/Tampermonkey:
One-click install!



This is a script which will greatly improve your mobile Mush experience. And if you can't have Flash, don't worry… you can still play!
(I have to warn you, however, that without Flash, you cannot see the Flash minimap, nor see nor aim for the hunters from a patrol ship or a turret (you'll just shoot at random). I'm sorry, pilots, that's out of my control. But check the FAQ!)

I want it to be clear that this is a script, not a mobile app. It works a bit like a frame in which the page will fit. This means two things:

  • any bug you experience while the script is active must be retested with the script deactivated. If it is related to the script, report it on this topic or send me a PM (I do speak English and will answer you). Do not report it anywhere else.
  • this is not a mobile version of Mush. It is a player-made script, I am not a Motion Twin dev. So, as usual, be careful when you install a script, etc. Please read the Script Charter (with which this script fully complies) for more details.

So far so good? Great, let's install it. It may not be as easy as a two-click computer script installation, but you shouldn't need more than about seven for this.

Everything is explained in this page . Don't panic, it is actually very easy, all members of the beta team made it through. And have a look at the FAQ, there is useful information inside.

If you don’t have Dolphin Browser on your phone, it's up to your browser to make the script work. Firefox, Chrome and Safari are known to be good at this, on both Android and iPhones, but the Windows Phone browsers all seem to refuse to execute it. I cannot do anything about this and I'm terribly sorry.


Check the settings in the top right, some options are quite interesting!

It's exactly like in Ctrl+W (the code is from Ctrl+W, slightly adapted). In Mush, when you scroll down the chat, more messages get loaded, and all of them are reloaded at every action, that's one of the slowest parts of reloading. This option "wipes" the message cache* so that (until you scroll down again) you're left with only the most recent topics, not the three-day-old ones.
*Technically, there's no message cache: the number of messages to be loaded is stored server-side and linked to your current session ID. This is what I wipe, hence the hard reload (Ctrl+W hides it in an iframe, I don't, but it works exactly the same way).
