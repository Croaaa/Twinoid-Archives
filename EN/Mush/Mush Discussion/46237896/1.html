I've been playing mush for awhile now and sticking to non-casting ships since it's a very casual game for me. I've noticed that if someone counts action points, the mush are usually in a pretty hopeless state. If no one is counting action points, a mush who lies low can almost always win the ship. I feel as if counting action points is just a little bit too strong, especially when it's done by an Eleesha or Chao.

A way to nerf that and also give people the ability to be more active would I think be if action point gains were semi-random. For example, every cycle you normally get 1 action point as a base, maybe if there was a 20% to 33% chance to get 2? That would make counting people's action points useful but not as determinative as it is on a really active ship.

Well, the problem Loketta states is quite easy to fix.
- 20% chance to every player to get extra ap on cycle change
- 20% chance to every player to lose one ap on cycle change
