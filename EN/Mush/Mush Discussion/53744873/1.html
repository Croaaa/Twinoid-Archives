ASTROPAD: OFFICIAL TOPIC

FUNCTIONALITIES

The Astropad allows the crew to manage a collective inventory, available from a tab next to the private channels. Anyone can update the room they're in with a click, with the option to modify the data before sending it (that is, adding or removing a object, or changing its properties).
This inventory can be shared on the chat as text, and seen by all as a map of the ship.
Specialists (i.e., Medic, Botanist, Chef…) may share the food effects they see if they want to; these effects are visible even if a non-specialist updates the room
(For example: Botanist Ian shares the effects of a Bottine in the Icarus bay. If Raluca updates the Nexus with a Bottine, the effects shared by Ian will be displayed next to it, even though Raluca cannot see its effects herself.)
Important note: The Astropad cannot reveal a mush in any way.


How to install it:
- Firefox users: install the Greasemonkey addon.
- Chrome users: install the Tampermonkey addon.
Now you may install the script by clicking here.


Current version: 0.28.6

If you are using an old version of the AstroPad, you should update to this one so that you get all the bugfixes and the new functionalities. To do so, just reinstall the script.
