This is the list of all glory wins or losses that can be edited by casting admins. Some of them are specific to casting ships.

Individual human glory
CYCLE_HUMAN : each cycle alive
CYCLE_HUMAN_SLEEPY : each cycle with the inactive status
DAEDALUS_DEFENDER : killing a hunter
PREGNANCY : getting pregnant (for the mother)
NICE_SURGERY : successful surgery (random)
MUSHICIDE : killing a mush
EDEN_COMPUTED : calculating the coordinates of Eden
EDEN_ENGINEERS : for technicians when reaching Eden
EDEN_BIOLOGISTS : for biologists for reaching Eden

Collective human glory
SOL_CONTACT : initial contact with Sol
REBEL_WOLF : D2 rebel base
NEW_PLANET : arriving in orbit
ALL_PREGNANT : when a woman gets pregnant
RESEARCH_SMALL : 3-glory research (when finished and when going to Sol/Eden)
RESEARCH_STANDARD : 6-glory research
RESEARCH_BRILLIANT : Retro-fungal serum
RETURN_TO_SOL : return to Sol
SOL_MUSH_INTRUDER : for each mush alive on Sol
PREGNANT_IN_EDEN : for each pregnant woman on Eden
EDEN_ALIEN_PLANT : for each type of alien plant
EDEN_MUSH_INTRUDER : for each mush alive on Eden
EDEN_CAT : if Schrödinger is alive on Eden
EDEN_MUSH_CAT : if Schrödinger is infected
EDEN_NO_CAT : if Schrödinger is dead
EDEN_ONE_MAN : for each crewmember alive
EDEN_SEXY : if members of both sexes are represented
EDEN_MICROBES : for each disease

Casting ships only
CM_USE_EXTINGUISHER : extinguishing a fire
CM_REPAIR_OBJECT : successful repairs
CM_REPAIR_HULL : successful hull repairs
CM_MUSH_VACCINATED : curing a mush
CM_ALIEN_DOWN : winning a fight during an expo
CM_PILGRED : completing Pilgred (collective)


Characters' personal glory
PSYCHOPAT : Chao (killing someone)
PLANET_FINDER : Frieda (arriving on orbit)
PLANET_SEARCHER : Frieda (finding a planet)
PLANET_SEARCHER_2 : Frieda (finding a planet - decreasing chances)
FAST_FORWARD : Jin Su (moving the ship to a planet)
FAST_FORWARD_2 : Jin Su (moving the ship to a planet - decreasing chances)
SAVIOR : Jin Su (reaching Eden)
CHUN_LIVES : Chun (each day change)
PRECIOUS_BODY : Chun (each research requiring her presence in the lab)
REMEDY : Chun (reaching Eden)
PILGRED_MOTHER : Raluca (every 20% of Pilgred completion)
HUNTER_NEMESIS : Roland (killing 2 hunters)
HUNTER_NEMESIS_2 : Roland (killing 2 hunters - decreasing chances)
MUSH_FEAR : Gioele (when a mush dies)
TR_ANATHEM : Gioele (when a mush Pariah dies)
LOOKING_FOR_KIVANC : Paola (contact with Kivanç Terzi)
MAGELLAN_ARK : Kuan Ti (each project completed)
ROBOTIC_GRAAL : Terrence (upgrading a drone)
PRETTY_COOK : Stephen (each fights during expos)
AMBITIOUS : Stephen (getting a title)
REBEL_CONTACT : Eleesha (each rebel base decoded)
EXPLORATOR : Hua (going on an expo)
ANDIE_FATE : Andie (for each trade)
NATURALIST : Ian (planting an alien fruit)
EDEN_ALIEN_PLANT_PLUS : Ian (for each alien plant on Eden)
ALIEN_FRIEND : Janice (encounters with living beings on expos)
MUSH_SPECIALIST : Finola (for each anti-mush research)
LOVER : Derek (for each reciprocal flirt)


Mush glory
MUSH_INITIAL_BONUS : when being converted
CYCLE_MUSH : each cycle (including after their death)
CYCLE_MUSH_LATE : initial bonus - 3 per human cycle when you are converted
INFECT : infecting a human
NEW_MUSH : converting a new mush
HUMANOCIDE : killing a human
CHUN_DEAD : when Chun dies
MUSH_VICTORY : assassination by Neron
SOL_MUSH_INVASION : going to Sol
EDEN_MUSH_INVASION : going to Eden

Casting ships only
CM_SABOTAGE : sabotaging
CM_ALL_MUSH_HUMANICIDE : when a human is killed (collective)
CM_ALL_NEW_MUSH : when a human becomes mush (collective)
CM_DAEDALUS_EXPLODE : when the ship is destroyed


Mush and human
EXPEDITION : expeditions
STAR_MAP_1 : finding the first first starmap (collective)
STAR_MAP_N : finding a starmap (members of the expo)
KUBE_SOLVED : solving the kube
ALIEN_SCIENCE : being traded
