Dear players,

There must be people among you, who wished Mush had a similar atmosphere to the Alien movies, or just wanted it to generally be darker to ease up the stress on their eyes during the night time. This topic, and the style I'm about to present you is for this kind of people!

I was making the theme mainly for myself, but as I progressed further and further, I started to feel that it was turning into something great that others could also enjoy.

A screenshot is better than a thousand words, so if you are curious about how your Mush game can look like, here it is: http://i.imgur.com/5BpFTrr.jpg

I called this theme -

Somber Cosmos

As you see I have changed the graphics of the design elements and removed some things that were annoying, like useless tips, or banner over the character picture - the things that you will never miss, and some might appreciate. (Especially Bronze members.) Full change log and more screens can be found on the page of the theme: http://userstyles.org/styles/98520/so...

Installation process

Google Chrome:
1. Install Stylish Chrome extension by following this link:
https://chrome.google.com/webstore/de...
2. Install the theme by following this link and pressing install: http://userstyles.org/styles/98520/so...

Mozilla Firefox:
1. Install Stylish Firefox extension by following this link:
https://addons.mozilla.org/en-US/fire...
2. Install the theme by following this link and pressing install: http://userstyles.org/styles/98520/so...

It's that simple.

The thing that I wanted to include but couldn't figure out how to make it work, was redirecting the Help button to Mushpedia. It's a great resourse and I'm using it quite often, so that would be handy. If anyone has any ideas how to do it, let me know.
I've also tried to change chat background colour, but that didn't seem to work too.

Anyway, Enjoy!
