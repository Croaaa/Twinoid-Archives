A note on post-it notes:
Type the same message on each post-it note so they end up stacking. This will make it easier to access other items in the shelf.

So if you write (say) "same message" in the first post-it note, and also write "same message in the second post-it note, both post-its would stack.

After posting the fifth consecutive post-it note, you won't have to re-access the post-it pad... making it a lot faster to create more post-it notes.

(At least these were true in the spanish version. Haven't tried these in the english version yet but these should work).
