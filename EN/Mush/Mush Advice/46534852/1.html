You need Flash Player in order to play Mush, else you'll just have a black square in the middle of the page. The thing is, on non-desktop devices, it gets a little tricky… It does not depend on the device, but on its operating system.

On Android up to 4.3:
1. Allow installation of external apps: Settings → Security → check Unknown sources;
2. Install Flash Player for Android 4 or Android 2 & 3;
3. Use a Flash-compatible browser, such as Firefox or Dolphin Browser.

On Apple devices:
You cannot install Flash Player on these.

You may also not need to install Flash Player, with some mobile browsers who will make the game work in the cloud so that you only need to use it. I haven't tested this, but apparently Puffin Browser works for playing, but not posting in the chat. Be aware though that it will consume a lot of data since there's constant synchronisation between your phone and their Flash-computing servers.
