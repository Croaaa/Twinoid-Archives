" DON'T EAT THE CRAB DIP ".

Hi ! I've heard a lot of questions about : How can you get dirty ? What can't you do being dirty? Etc etc...
So here's a list which, I hope, will help the newcomers to know more about that statut.



Being dirty is shown by a public statut that everyone can see :

: « You are filthy and you stink... A quick spin under the shower wouldn't hurt. Take this chance to put one some deodorant. »



How to get dirty?

  • Working on the PILGRED.
  • Cooking, using the Coffee machine, and the Express Cooking ( microwave ).
  • Eating. ( Any kind of consumable ; Fruits, coffee, rations, cooked rations, drugs... )
  • Gardening.
  • Working on the researchs.
  • Repairs, even if you fail.
  • Checking the combustion chamber.
  • Extracting or inserting fuel.
  • Torturing. ( Lvl 4 Chao's skill )
  • Explorations.
  • Doing it.
  • Using the spore extracter.
  • Building a BluePrint.
  • Updraging drones.
  • Removing and installing a camera.
  • Dismantling objects, even if you fail.
  • Kicking someone, even if the kick fails.
  • Doing a surgery on yourself.
  • Healing yourself with bandages.
  • Distributor project "SNC Kitchen".
  • Sabotaging something. ( Mush's action )

You can't get dirty with these actions if you're carrying the apron.

---

The following actions/diseases will make you dirty no matter what.

  • Extracting a spore ( Mush's action )
  • Slim trap ( Mush's action unlocked level 7 )
  • Some diseases. ( Like the flue, food poisoning, gastro etc... )
  • Mass mushification ( Mush's action unlocked lvl 18 )

What CAN'T you do being dirty?

  • Working on the researchs.
  • Working on the projects.
  • Scanning planets. ( you can scan planets, just can't analyze them. )
  • Working on communication.
  • + ( from Gardening 101 ) " When trying to graft a fruit onto a plant, the plant must not be thirsty, dried out or sick, and the person doing it must not be dirty. If any of these things is the case, the plant will die and you will lose the fruit. "

If you see anything that must be corrected, PM me or post it under the comments.
