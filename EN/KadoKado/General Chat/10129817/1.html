After fiddling with CSS a little, I actually found a way to fix this problem... but it requires M-T to actually edit their CSS

Anyway, I'm not sure why there's a problem in Opera in the first place, so I just randomly tried things until it worked

Here's the CSS that should be added (the bolded parts are unintentionally bold, since the commenting style for CSS uses asterisks):
.smileyBox { display: table; } / Floating all the a tags causes the surrounding box to render incorrectly /
.smileyBox a { display: block; float: left; } / The display: block; part might be redundant, but it doesn't hurt anything /
I'm sure there's a better way to fix it, but it works over here, and it only requires a small amount of CSS! I tested my CSS in Opera 10.6 (obviously), Firefox 3.6, Chrome 7, and IE9... all look fine
