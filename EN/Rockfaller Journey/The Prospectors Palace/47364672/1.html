That is because Flash is stuck to version 11 in Linux for Firefox, and all new Flash MT games will be requiring Flash 12+. Adobe has decided three years ago that all Linuxers could only get the new versions through the Chrome plugin (which also works with Chromium and Opera, btw). The Linux Firefox plugin will only receive a few security updates for the next two years, after what support will be completely dropped.

You can use the freshplayerplugin wrapper so that Firefox will be able to use the Chrome built-in up-to-date version, if you wish so. I can't seem to find a more complete tutorial than the one on the French Ubuntu doc: copy/paste (one at a time) the four + one commands in a terminal, restart Firefox, and you're good to go.

sudo add-apt-repository ppa:nilarimogard/webupd8
sudo apt-get update
sudo apt-get install freshplayerplugin
sudo add-apt-repository --remove ppa:nilarimogard/webupd8

sudo apt-get remove adobe-flashplugin

You may also need to check in the Firefox Plugins manager (Ctrl+Shift+A) that Flash 18 is enabled and, if it is still here, that Flash 11 is disabled.
