Ok, surely they will ignore my suggestion but I need to try!

I think there should be a testing server of DinoRPG. What's a testing server?:

It's a standalone server (like the EN, FR... servers).
On this server is easier to progress. Among other options:
    Infinite movements
    Infinite gold
    Infinite
    Easily change the level of your Dinoz.
You cannot earn trophies (epic rewards, titles, status, etc).
A testing server is not a server where you can play like on the EN, FR, ES or DE servers. The purpose of this server is to test new options: the Mana War, a new quest or something else.

Please note:

The testing server would not be always open. It would be open when the devs of DinoRPG need to test something new.
Any player may enter, or it can be limited by selecting beta testers, etc.
The server could be only in French, or we can have a testing server for each language.
The purpose of the server is, and this is very important, check and report the bugs (the forum of this server will be only for these reports).
This means, for example:
The devs have created the new Mana War. They open the testing server and let any player/the selected players enter on the server. During a few days, the players can test ALL the War options and if they think something is wrong, they open a new thread on the forum, the admins read it and fix the bug.

And finally:

It cannot be used for free entertainment. For example if they're testing the Mana War, only the Dinotown and Dinoplaza areas will be available, and Papy Joe, Madam X, Yakuzi... etc, won't be there.
The Admins will be very attentive, so no spam in the forum. If you spam, you won't be able to enter to the testing server anymore. If needed, maybe our mods could deal with it.
I think we REALLY need it. For example, the Mana War is full of bugs. On the Spanish server I'm on the (current) top 1 clan. Due to a bug we went down to 0 of honor with the first war. Now it's impossible to play the war: if we declare a war, we lose honor; if we don't declare a war, we lose honor daily. We reported it to the Admins and they fixed it (now you can't lose 100 of honor in a simple war), but they won't restore our honor. Yeah, we, the top 1 clan, are helpless on the first Mana War and our only solution is to leave our clan and fight on a new one (losing 8M of treasure). Can you believe it?
I think that if we have a testing server, these bugs won't appear. It is not good to test in a real server, where the people pays real money and they have to coexist with a lot of bugs. I do not think this should be happening.

What do you think? I think it's a good idea and we need it. If you support the idea I can make it know on the French forum. I can even send a PM to a French admin.

Thank you very much.
