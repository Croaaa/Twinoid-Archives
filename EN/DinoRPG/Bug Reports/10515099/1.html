I already suggested this, but here it goes again:

Instead of raising the prices for everybody, this game could be programmed to raise the prices according to the items the player buys.

I explain: everybody, and for all eternity, has the current prices when they buy their first item. Then prices change for the second item, and so on. This would allow EVERYBODY to at least eventually buy an item. And then prices would raise for the second item, and so on.

This way, players that start the game later can come closer, it's unfair a player starting today will only buy items for more napos, and older players would have always a harder time to get new items. It would be a little bit like the turn system: in the beginning it takes little time, and then more and more. If MT really raises prices for everybody, it will be bad for new players, or those that get less napos from the events...
