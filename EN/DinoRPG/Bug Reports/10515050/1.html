As has already been posted in these forums:
To rewatch a normal fight
(from the end when the xp/gold rewards are displayed)
Firefox
View --> Page Style --> No Style
View --> Page Style --> Basic Page Style
Fight will replay

IE
Page --> Style --> No Style
Page --> Style --> Default Style
Fight will replay


To watch fights in the CotC
Go to the Battle tab and view the grid.
In the bottom left, select the crossed swords to view the fights or deselect them if you only want to see movements on the grid
You can view any or all fights in the current battle as long as the battle tab is visible
