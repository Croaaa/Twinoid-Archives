As most of us know, the Spanish and French servers get much more attention from MT than this server or the German one.

What if all servers were merged into one to eliminate this? All accounts and clans could be transferred onto one server (logically the French one as it has the most players), so no one would lose any progress. Epic items won in PVP events prior to the merge could have a small flag added to them to show what server they were originally won in.

The language barrier would be easy to overcome, as each user could select what language they want to view the site in, just like Facebook. There could also be seperate forums for each of the 4 languages currently represented.

Sure it would be a lot of work for MT, but it would be better for them in the long run as there would be more competition and more need to spend real money to keep up.

I understand that smaller/non paying members and clans may be disadvantaged, but this could be remedied by increasing the prizes for each event (since MT were going to be handing out 4 sets of prizes for each event anyway) and maybe increasing the number of qualifiers for the Dojo.
