IMO,

Tabbed gaming is almost breaking the function of the game.

The game wasn't designed to run separately in separate tabs, but to run as a whole in 1 window or Tab.

It's a website, and only has the data you request upon loading.

In order to play with multiple tabs the site would have to auto-refresh every second for the hundreds of players on this site alone.

A website is similar to a phone book, you can only see the numbers for the page you are on. If you want to see more, you'll need to change pages aka Refresh, after performing an action on another tab.

Sorry, but I don't see it as broken, and I don't see why the Devs would see it as broken either.
