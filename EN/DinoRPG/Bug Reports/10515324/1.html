I made a proposal in this thread and was asked to start new one, so the discussions don't intefere with each other.

My idea was to have marked threads, in which you can post your opinion only once, so that it would be easier for epoq or the programmers to see how the people on this server see things.

The reason to allow only one post per person is, that if a minority posts more often their opinion than the majority, the true result could be blured.

Yet, we all want to have productive discussions.
So the idea was born to have a thread, in which can be discussed, with as many post you think are needed to provide your opinion, and have another thread in which your final statment is posted.

To test this idea, you can post, what you think of this idea, provide ideas how the thread with the final statment should be marked, if the thread prior a "final-statment-thread" should be marked, if both threads should be started parallel or if the "final-statment-thread" should be opened after a certain ammount of time, or anthing else that comes in to your mind to this topic.

I'll see how the discussion about this matter goes in here. As soon as the discussion ends (or maybe after the War), I'll start a final statment thread.
