If you searched Gaussian Elimination on google your first hit should be Wikipedia which gives a pretty basic, yet helpful, example. Yes the example they use has easier coefficients so the elimination involves simpler numbers but since youre doing matrix math I would assume you are in highschool and should be able to figure out how to do the elimination. Fortunately, your equations come from a nice data set which allows for a solution to be found via Gaussian Elimination. If the data wasnt so nice you would have to do real matrix manipulation to find a solution (if one even existed).
Anyway, you basically need to get your equations from:

548a + 281b + 149c = 499
472a + 521b + 169c = 625
404a + 448b + 312c = 605

To the form:

Da + Eb + Fc = G
0a + Hb + Ic = J
0a + 0b + Kc = L

From there you can solve for a, b, and c.
