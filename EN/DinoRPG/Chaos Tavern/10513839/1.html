Two more things :
http://data.en.dinorpg.com/swf/dino.swf? v=0&data=89ZwZ9bZyXBoow5P&chk=34 112&flip=1&alpha=50&damages= 2

I removed the "amp" before the alpha.
Now with &alpha= you can set an alpha-transparency for the image.
(Alpha = 0= invisible, 100= full visible, 50=half-transparent, like a ghost)

I added &damages=
Now you can set the amount of damages of the image of the Dinoz.
(Damages = 0=no damages, 1=damaged, 2=seriousely damaged)


Also, flip=1 or flip=0 turn the dinoz, but it seems that it work only on the sdino image and not on the dino.
