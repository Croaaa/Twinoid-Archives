If we don't think there are duplicated skills used by the other invocations, we may be able to isolate down into the Kabuki with a few educated guesses.

We know that Vacuum Disk, Vivacious Wind, and Elasticity are all out. I would almost certainly add vaporous form and toxic cloud to the tier 3 skills that are out. Both are widely used and already powerful: I suspect the Kabuki didn't get an already widely-used skill. It leaves us with 7 skills that are possible, with:

1) Awakening (really bad if this is it)
2) Ejector Palms (sorta bad if this is it)
3) Plunging Attack (fairly usable and testable)
4) Stealth (would be ok)
5) Specialist (widely used and mostly ruled out)
6) Achilles Heel (widely used and mostly ruled out)
7) Sharp Eye (not horrible, but no precedent for gathering skills being the end requirement)

Since there are a lot of kabuki around, I would think that if it were any of the more common skills (3,5,6) it would be found at this point.

To me, it seems like they probably went perception line on water for either fatal hits or marsh to keep bubble kabuki from getting way stronger. It could be an off-skill like absolute zero or without mercy, but I suspect they went down perception.

Hard to want to test those out, however, as that is a pretty gimpy Kabuki in the meta-game and hard to obtain (cheaply) for easy testing.
