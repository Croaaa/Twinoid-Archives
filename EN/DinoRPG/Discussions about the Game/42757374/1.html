On the EN server there is still some minor content missing, that even the German server has - the server that always lagged behind and still has not got the Caushemesh update (and is not ever going to get it now).
The missions from Klutz and the dungeon in the magnetic steppes are probably not going to have any impact on the game, but it would be still nice to have them.

The PROs of it happening - if we bug Buzzard enough - in my eyes would be:

1. There seem to be no new game mechanics that need to be implanted, only the translation work. On the ES and FR server these things already exist and only need to be fitted to the EN server by language. The needed workload is thereby most likely limited.

2. In the last months/years there seems to be the trend that MT tries to get all servers to a similar level. Be it events running at the same time and probably soon by the same rules or the IDT. It looks like everything is intended to follow the same lines.

The CONs are:

1. By not having any real impact on the game power- or meta-gamewise the update would not lead to people paying money to rush through the new contents. So there is no real financial incentive for MT to do it.

2. The trend to unify things on the different servers may be a try to cut down costs/work time on the game, as devs could supervise events that run at the same time under the same rules more easily, and can get bugs out of the system on all servers at the same time, instead of always needing to adapt to invidual server issues. In that case the striving for getting the servers on the same level does not include the missing content and investing extra work for little reward.

The main purpose of this post is to ask what other players think about it. Do you think there might be a chance for us to get the missing content if enough people ask buzzard and thereby MT/devs for it? Do you even care to get it, or is it too irrelevant for the game in your eyes? Are there PROs and CONs beside the ones I mentioned? Or do you deem them invalid for some reason?

Any comments on topic are welcome.
