Items in the next level up are only unlocked if you have the beginning of that path already selected.

So if you have already levelled up on the Intelligence path to access Lightning Conductor and First Aid and THEN you get Focus, you have to level up again to open up Concentration and Regeneration.

I couldnt be bothered to read all the answers to see if anyone correctly answered the question, so at the risk of repeating the correct answer.

When you select "unlock" you unlock all skills that are next along all the various branches. But only in the element you leveld up in.

So if your dinoz had (lvl1)Focus/(lvl2)Lightning Conductor/(lvl3)Career Plan, then when you choose "unlock" it will unlock all branches from the skills you already have, these being,

Reincarnation(lvl4),Diamont Fangs(lvl3),Elemental Fission(lvl3),Concentration(lvl2),Regene ration(lvl2)

All of them will be unlocked, so it adviseable when unlocking skills to already have a plan based on your guess at how many elemental points your dinoz will get. This way if you know you want LeafyDawn/BlazingTwilight/Lightning Dance you can make sure that before you unlock you take each prequisite so that you unlock as many branches as possible with the fewest unlocks.
