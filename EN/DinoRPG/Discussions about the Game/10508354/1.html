OK i know about tamers market, but i was thinking if it is or would be possible to trade a dino for a dino between players.

If it is available to do so disregard my post.

If not i suggest that the developers should put this into the game to make trading a little more fun in the game.

I think to keep the balance in the game there should be a lvl cap for trading like this:

Dino Level:
1-10 1 level gap
11-20 2 lvl gap
21-30 3 lvl gap
31-40 4 lvl gap
41-50 5 lvl gap

then possibly as the game requires other cash sinks for the company to make buying gold coins more attractive to players you could make players with the lower level dino pay more if they want a dino above the lvl cap limit.

So it would be like:

Dino Level:
1-10 1 level gap = Free, each additional level 1k
11-20 2 lvl gap = Free, each additional level 2k
21-30 3 lvl gap = Free, each additional level 3k
31-40 4 lvl gap = Free, each additional level 4k
41-50 5 lvl gap = Free, each additional level 5k

So a guy trying to trade a lvl 1 dino for a lvl 50 will need to pay an additional 135,000.

I think this system would keep balance within the gamer and allow another depth to enter the game as people would start trading for more desireable looking dinos or for dinos that have a more desireable element base from fortunate leveling.
