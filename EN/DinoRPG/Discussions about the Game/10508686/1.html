If I have only 2 skills of lvl 1 and I unlocked lvl 2, do I have to unlock 3rd branch skills again after acquiring 3rd skill of lvl 1?

If you get level 2 skills, but don't unlock level 3 skills, you can go back to get the other skills caught up, so that you can unlock all level 3 skills at one time.

If you unlock only some level 3 skills, you would need to unlock the other skills, yes.
