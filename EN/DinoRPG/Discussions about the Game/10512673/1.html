Piemaster
This is very tricky, because I read option A and thought "yes I want that one" then I read B and thought "oh wait I want that too" and all the way through down to F, so this is a neat illustration of why it is very difficult to prioritise development resources in games like this. After some soul searching I think my order or preference would go:

D (I think this is the thing most likely to make the game significantly better in the long run. Would need to be done well though, with the right new skills)

A (Even with the balance patch a lot of skills, abilities and items are way out of balance)

C (Unlikely to make a huge difference, but can be done with comparatively few resources and everyone is always excited by new dino races. I think some new demons are long overdue, given they would be very easy to code)

E (Forums badly need updating, and the game engine could use a complete overhaul I think. Flash doesn't look like the future)

F (I think we have enough clan events and they are reasonably diverse, but it would be nice to see some effort put into improving the rules of the events we have to create the best experience possible)

B (New areas are nice, but we've had quite a lot lately and personally I haven't come anywhere close to finishing the content we have yet. Obviously making the world bigger is always nice though and higher-levelled players will really appreciate it)

So there,s my answer D, A, C, E, F, B in that order

I only wish that the official surveys were this thought provoking...
