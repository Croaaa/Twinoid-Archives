Are you using the same dino or different ones for the quest?

As far as I can say you are at the correct step, but I'm suspecting that you might need to use the dino that spoke to the Captain.
While the old Quests and Missions were clearly defined (Missions could only be done by dinos... well on that Mission, but Quests could be contributed to by any dino in correct location) the last Quests had a tendency to break with that custom. Sometimes the dino that did part of the quest had to do the next step too.
An example is the Sage Quest where the same dino that spoke to the Sommelier has to go to Guard Post for the next step.

So I would try to either use a dino that has at least the shard too, or the one that spoke to the Captain. (In case you did not try that yet).
