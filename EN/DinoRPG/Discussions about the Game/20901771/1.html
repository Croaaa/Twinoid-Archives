To start with, there are only three basic 'jobs' you need to worry about.

1. Fighter - These dinos will be used to progress in the PvE aspect of the game and will ultimately be your go-to dinos in PvP as well. Take the best combat skills available. Ignore anything that doesn't directly help you on the battlefield (or is a prerequisite of a skill that does). All irmas should ultimately be used on these dinos (although you shouldn't be buying irmas to start with)

2. Collector - These dinos will collect ingredients for you. Get the gathering skills in each element (such as Fisherman, Hunter, Search etc). Get as many of these skills as you can. Once you've got them all in one element you can grab them some fighting skills to give them more survivability.

3. U-Skiller - These dinos sole purpose in life is to get one or more 'U skills' (non-cumultive skills that affect all your dinos) and then freeze them so they don't count towards your limit. The most important of these is 'Leader' so start working towards this skill ASAP.
