Sept. 2015:

  • 1. Can we have somehow a way to comment fights implemented on DinoRPG?
    Devs: No, the fight motor is too old to do such a thing.
  • 2. Can we get an event calendar or be noticed in advance?
    Devs: We're trying to follow a schedule, but we prefer to announce such only at the end of each events.
  • 3. Can we put more offers on the Market?
    Devs: This is an intended feature to prevent abuses (so no).
  • 4. Could you change how the dinoz characteristics are set up, so you could update skills more efficiently in the future?
    Devs: If we change how it is set up, your dinoz may become unavailable afterwards.
  • 5. In Mana War defenses, we can view magic items each dinoz equipped with, could you do the same with basic items?
    Devs: No.
  • 6. Could you remove Flash dependencies from the game?
    Devs: We cannot, or else, we have to reprogram the game entirely.
  • 7. Could you add a permanent grid to add value when we're paying?
    Devs: Not.
  • 8. Can we know the CotC brackets beforehand?
    Devs: It will still be three leagues due to clan activity (they're talking about a change in last FR CotC). Brackets levels will be decided by community (probably: we (EN mods or me) will invite you to vote whenever those polls come around).
