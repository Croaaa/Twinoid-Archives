I finally got some info about the status for the research of the Tofufu's Invocation.

The research team, basically Tyalos is doing the builds on several servers, has 9 plans to cover all combination of 2 in the 3 main elements ().

Each plan covers one tree of each element.

The builds are named after the first Skill's name of the each Element tree.

For Example:
Ca-fo-ag = Carpace, Focus, Agility

This build will try to cover all Skills in the branch of Carpace, Focus and Agility.

With this method, only 9 Tofufus are needed to cover any combination of two elements out of three main elements.

To see the progress of each build and their element, the numbers of level-ups are given in brackets.
In addition the level is provided to see, if a branch is most likly to be fully reached or not.

For Example:
Ca-fo-ag level 17 (3-0-5) = There have been 3 Level ups in , 0 in , and 5 in .


Now let's come the current status of the builds:

En-in-st level 10 (2-2-1)
Sa-ce-mi level 14 (2-4-0)

En-ce-ag level 15 (3-1-3)
Ca-ce-st level 16 (1-0-4)
