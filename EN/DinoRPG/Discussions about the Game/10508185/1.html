Husaurus
"A" type skills are attacks. A Dinoz can have any number of these skills. "A" type skills are used randomly in place of a normal attack . The Element of the "A" skill does not have to match the Element of the attack it is replacing.

So, for instance a Dinoz might choose to use "Water Canon" instead of its normal Wood attack... which would be a good thing for a Sirain but a tragedy for a Gorilloz. Therefore, "A" skills are often best taken in a strong Element of that Dinoz... or you might find your Dinoz randomly nerfed in battle. There is an status object that allows turning off/on individual "A" skills and "E" skills... but the quest for this status object is not in the first area.

Husaurus
No.
A Skills replace Attacks.
P Skills are Permanent skills... they always apply.
E Skills are Enhancement skills... also "active" but are used before attacks instead of replacing them.
C Skills are Collecting skills.
S Skills are Special skills... having a particular effect.
U Skills are Universal skills... they apply to all your Dinoz.

On march 1, 2009
Husaurus
Each skill has skills above it in the skill tree (except for skills at the top of the tree). When you unlock skills, you unlock all skills immediately above the skills you have. After a skill is unlocked, they will become options the next time your Dinoz levels in that element.

So, the more skills you take before you unlock, the more value you get from unlocking. However, your Dinoz will not be able to unlock and take all skills in an element, so there is some strategy involved... however, without the full skill tree to plan ahead with, that strategy can be hard.

For a softpig, I would probably take all first level fire skills before unlocking fire . All three first level skills have good skills above them.
