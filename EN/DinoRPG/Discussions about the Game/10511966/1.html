You can convert notes to gold just by buying something on the market and then reselling it at the same price.

I think there will never be a 'buy now' action in the market, or a 'sell now' action either, because that would open up the same abuses as direct sales between players, and make unlawful multiaccounting more damaging to the game.

I don't know if there has been a request for a standard anti-sniping feature, that would add one minute to any auction when there is a bid in the last minute. This would seem to be the only request that might be considered... but I think it would be a pain to implement, that might outweigh the benefit.
