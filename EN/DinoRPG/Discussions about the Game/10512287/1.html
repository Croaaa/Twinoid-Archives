This is how it is in a chess tournament. If you don't have an opponent, you get a win, and are guarenteed an opponent next game.

Actually, I think the best way to format this tournament is in the same way chess tournaments are formated, using the Swiss System. It's already been suggested many times before, but I think it's the fairest way to do CoTC.

The first round is either drawn at random or seeded according to some prior order, such as rating or last year's performance. Players who win receive a point, those who draw receive half a point and players that lose receive no points. Win, lose, or draw, all players proceed to the next round where winners are pitted against winners, losers are pitted against losers, and so on. In subsequent rounds, players face opponents with the same (or almost the same) score. No player is paired up against the same opponent twice however.
~Wikipedia

Winning clans will fight winning clans with similiar power levels, same goes for losing vs losing. This system has been used since the 19th century, so it obviously works. I don't see much of a reason not to use this. First match pairings could be disputed, but that's about it.
