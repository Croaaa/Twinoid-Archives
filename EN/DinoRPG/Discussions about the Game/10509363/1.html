On july 5, 2009
insanedar
The reason behind this is that if can be carried over through level ups, then it is possible to abuse the system. Let's say you are level 1 and you get 25 from smashrooms. If at the time you level up, you don't choose a skill, then you remain as level one with full experience points. If the carries over from the fight, then the will continue to add up. You will keep on getting 25 from smashrooms until you have enough to be level 50.

Well it is possible to code around it by making it so that only the first fight through level up carry over. I don't know if they're looking into fixing it.
