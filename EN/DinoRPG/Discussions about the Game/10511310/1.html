Sorry if this has been addressed before, but I searched for it and couldn't find anything. If this has been covered already, please just tell me politely so the thread can be locked and we can all move on.

I'm sure most everyone knows about the lag glitch that sometimes happens when one of your dinos summons reinforcements in a fight. There is a similar glitch on the game Formula Wan were the races load really slowly, and I've found that, in both games, all you have to do to get past it is minimize your browser for a second or two, and then maximize it again. By the time you maximize it again the glitch will be fixed and things will be running normally again. I don't know if it is removing and then returning focus to the flash program that fixes it, but for whatever reason, it works.

I've tried it out here a few times now, and it's worked each time for me. Hopefully it will work for everyone else too.
