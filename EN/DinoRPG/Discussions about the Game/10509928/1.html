I think you misunderstood it. In order to learn skills from a higher level in the skill tree, you have to learn the skill that comes before them in the tree, and then "Unlock" them. Unlocking takes the level-up (instead of learning a skill), so you only do it when you have all the skills learned that you will want from the current level.

The next time you get the same element on a level-up, you'll have the option to learn one of those skills.
