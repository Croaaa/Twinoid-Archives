Merchant only decreases the prices on items in the dinoz shop. Well aiming for for all U-skills in one element is kind of hard, so it'll probably take a while. When you get a tier 1 skill, you get the choice to unlock tier 2 skills for the the tier 1 skill. So if you were to get intelligence, you have the choice to unlock the tier 2 skill path of that skill or get another tier 1 skill. For the main element of a dinoz, I suggest you get all tier 1 skills then unlock the tier 2 skills thus saving more levels for less unlocks.
