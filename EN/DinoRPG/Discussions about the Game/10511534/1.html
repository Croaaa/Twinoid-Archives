AndySQdancer
This trick is very useful in the Steppes, especially if same missions are being done by a group of 3 Dinoz. When the 3 move to a spot where there are fights to be done in the mission by pressing the mission button, feed the group some Irmas, so all 3 Dinoz in the group have a turn available. Then let the leader do the mission task requiring a fight and all 3 fight together. Then regroup so that the original 2nd Dinoz is the leader and do this Dinoz part of the mission task and again all 3 fight together. Finally, then regroup again so that the original 3rd Dinoz is the leader and do this Dinoz part of the mission task and again all 3 fight together. Saves lots of by fighting as a group and actually gives more to each Dinoz for that portion of the mission fight.
