On october 4, 2017
Kayin

Currently Chrome 61 removed its experimental flag "prefer-html5-over-Flash", and it was directly integrated as being per default in this current version (understand as HTML5 got priority over Flash). But they didn't add any button to re-toggle Flash back.

So solutions for Chrome are:
- Uninstall version 61 for version 60 and don't do any updates.
- Wait for version 62 at the end of October.

Google support already tell how you will be able to enable Flash in version 62+ (only on the English page it seems).
