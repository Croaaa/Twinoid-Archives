If it were true, yes. So there's no way it is. That would just make the turn delay pointless.

You can get large amounts for a single fight, but usually you won't.

But even if you only get 200 for each fight, with 21 Dinoz each fighting separately, you'd get 4200 for a full turn, which would translate into 4 extra turns, awarding another 800 for a 5th Irma's Potion, with 700 to spare after that. 26 turns in a day is a lot.

In practice, you'll spend some of that on Meat Pies or Cloud Burgers, and some Dinoz will be resting instead of fighting.
