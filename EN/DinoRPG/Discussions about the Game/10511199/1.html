[895738]
There are 3 main guides/paths:

Attack
Defence
Gather

Now if you are not an active participant of the clan wars, you can choose whatever path you prefer, that is you follow a random mixture that you think works well.

For the pteroz, an attack path would involve getting all the first level skills and then getting as many A or E skills as possible. Speed initiative are very important.

A defence path would involve getting as much upgrades as possible, as well as acquiring immunity from status debuffs.

A gathering path would involve checking the element skill lists carefully and going straight along a path/build of C skills or if you are planning to excavate, you would aim for the additional square skills.
