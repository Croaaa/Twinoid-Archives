Browser History consists of several discrete items depending on the browser:
. Temporary Internet Files (Cache)
. Cookies
. History of Sites viewed
. History of Downloads
. Form details
. Passwords
. Site Preferences

By default the "Clear History" option in both Firefox and Internet Explorer 8 will delete entries from multiple sections (e.g. Firefox 3.5 "Clear Recent History" is set to delete all except Site Preferences)

However, this is largely irrelevant to the issue at hand since it seems that the only requirement is drop the connection to the server so that, when you reconnect, the server displays the first "random" element again.

Clicking the "Disconnect" button rather than choosing a skill from the Career Path element would seem to achieve this.
