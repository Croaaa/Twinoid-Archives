On december 15, 2015
Kayin

You go through an API to get data for your external website, and the API restrict what you can get, while it's secured by being under Twinoid. To help players and clans manage things in events.

Nothing about automatized actions, interface change etc... you would get from Greasemonkey scripts, which are strictly forbidden on DinoRPG.

That's the difference between API and scripts.
