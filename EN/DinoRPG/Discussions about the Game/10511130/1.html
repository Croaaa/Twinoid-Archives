If the dinoz level's are different the XP they each gain will also be different. It's easier if you level dinoz with the same level in groups for a few different reasons.

Also the number of dino's in the group effects the XP each dinoz earns individually.

Solo = 100% XP
2-3 group = 75% XP
4 group = 60or65% ish XP
5 group = 50% XP

The total's are then tallied for the shown XP.
