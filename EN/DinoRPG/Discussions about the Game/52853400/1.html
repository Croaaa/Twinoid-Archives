On february 16, 2016
Kayin

Dev has decided to restrict ingredients gathering to avoid abuse (it won't strangle us as much as FR has known this summer):

  • It won't be possible to empty out more than 3 grids per day.
  • Those 3 grids are understood has one per ingredient types.
  • That restriction won't bind event grids.

This restriction is still in discussion, but already applied on FR server (I don't know for other server).
