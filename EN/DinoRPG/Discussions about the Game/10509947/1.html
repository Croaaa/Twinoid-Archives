The game uses nothing like AI at all. It uses a random number generator and every action is based on a dice roll. There's not "decision process" made in fights, just random dice rolls and dumb luck.
On august 29, 2009
[1165856]
Let me clarify - there is an order to the dice rolls - for example, it rolls for a successful E skill (highest level) first, then if that doesn't pass, it rolls for another E skill (next level down). Once all the E skills are rolled, it rolls for A skills - if none hit, it does a normal attack. If you have the skills that make your attacks "smarter" all this does is make target choice non-random, and eliminates attacks that would be ineffective against those targets from being rolled - but this is still very simple - not AI at all. It is random dice rolls, so it isn't worth complaining about - get better skills.
