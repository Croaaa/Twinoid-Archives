Kayin
Twinoid is mandatory, thus if you don't have your own account, create one asap. If you don't want it, you can stop playing.

1. Only one Twinoid account, it has different language settings, choose one as you see fit (for Muxxu users, some games have flags, it means you won't be able to interact with someone with a different language on those games).

2. D2N and DinoRPG have multiple servers, for each language, they counted as separate games, thus you will be able to connect your D2N/DinoRPG in FR/ESP/EN/DE accounts into your only one Twino account (If some people had connected their DinoRPG account with old Muxxu system, they will be automatically linked under Twino).

3. Twinoid will only appear as a dark grey bar, in place of the dark green one. You can see design changes if you play on DE.

4. I don't know if they change the way you link an account (and I don't know if it's because of trying to link it to a different language), but it's a bit tricky. You need to be unlogged from Twinoid, then go to that game homepage and login on it. The Twino login window will prompt like normal, then after that a second login will appear. Put your old login for that game in. And then you're linked.

5. Some MT games may appear as multilingual, but has only one server, like Kadokado. If you have different language accounts, you are a multi-account user, so for Twinoid, you link only once when it will come around for that game.

6. You can still use your browser links to go on each of your games.

7. The new message system will have a newer version update, removing the IM style system, into a real mailbox (real group discussion & mailing).

8. You will be able to use (in probably the same update as the message system v.2) smileys from all the game you play. Ain't it nice for you Andy?

9. A ban from one MT game will result as a Twinoid account ban. In somewhere in far future, a forum ban could count as an overall Twinoid forum ban. Please be aware of your own behaviour.
