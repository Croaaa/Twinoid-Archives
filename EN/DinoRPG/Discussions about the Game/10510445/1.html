POTAATO
Apparently the search bar here works the same as in Hammerfest, I guess it works the same for all MT games?

Advanced search: use + and - before each word.

For example:
quest
search every message relating to quests

fridge quest
search every message about quest OR fridge

+quest +fridge
search every message about quests AND fridge

+quest -fridge
search every message about quest BUT NOT fridge

+quest +"I've found all of them"
search every message about quest and the text "I've found all of them"

It doesn't help with long threads, but at least this can refine the results somewhat. MT still needs to update its search abilities to return better results.
