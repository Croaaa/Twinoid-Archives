You've already made this thread before. I'll cp my answer in.

I would suggest following these trees:

Wrath - 25% increase in next attack
Warm Blood - +4 initiative
Burning Heart - +12 Fire and +20 HP
Your gorillaz would attack harder, then get hits more often, then get a second decent attack element.


Mutation - +30
Ventral Pouch - extra equipment slot
Sumo - +100 HP
Basic supportive skills, still useful if you only get 1-3 water.


Celerity - +15% speed
Double Hit - +20% chance of multiattack
Career Plan - Reroll element
More hits, more hits, chance of a better ele


Strategy - Adopts best strategy according to enemy defenses
Analyse - Choose best target for each attack
Specialist - Removes weakest element
Last one is fantastic.

Picked these cause they don't rely on you getting lots in each element to be good still.
