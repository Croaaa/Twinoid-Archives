On every skill activation the Skill Forger has a chance to activate (like Acid Blood, Life Source, etc. on hits). When activated the dinoz with Skill Forger can use the copied skill. It seems the copy uses the element number of the copier (not the power of the original skill). For example a clanmate's dinoz used Kaar-Sher Beam, the dark wanwan copied it and later used it, but it done only 1 damage with it.

Actually the best use I can see for it is the copy of some Invocations (Fujin and Werewolf), not element dependent skills like Smelly Aura (it is working, I've seen it), Hypnosis, Black Hole, Sylphs, Aqueous Clone (confirmed too), etc.

I have a google sheet for mapping which skills can activate it and which skills were activated so if anyone have information feel free to share it with us (if there is something wierd about the activation I would be glad if you could share it too): Skill Forger mapping
