Each dinoz is a random combination of a certain number of features (in the case of wanwans, main colour, colour of the paws, colour of the eyes, shape of the ears, shape of the tail, etc).
So a wanwan with the exact same combination as the one you saw is probably not particularly frequent, simply due to the sheer number of possible combinations (it can take a while before the same combination comes back), but none of the features you listed are particularly uncommon.
