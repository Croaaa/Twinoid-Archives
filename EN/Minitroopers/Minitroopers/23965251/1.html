After setting up a localhost I was able to get the replays working, just as Pause said. It still requires internet connection but at least it's possible to save raid and epic mission replays. It's funny that the same HTML page won't work if you try to run it offline (without the localhost).

Anyway, once you have setup the HTML page on the localhost, all you have to do is copy and replace the lines containing the actual Flash data in order to watch a different replay. It's a couple of lines enclosed in a <script> tag containing the swfp variable which contains the fight data. For what I've seen, those two lines can even be from any server (.com, .es, .fr, .net) and the replay will work.

Manipulating the values is, unfortunately, still not possible .
