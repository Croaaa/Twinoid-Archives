Following on from Cripto's win of the Late-September Championship, here's the competition for October.

For those of you new to the game here's how it works. This is a "continuous championship" style competition where players may move up the ranking list be beating the army immediately above them. It will run for ONE MONTH only, then the winner will be crowned and a new championship will begin for November. The winner may choose a "theme" or condition of play for the following month, in order to have a bit of variety in which armies can take part. Or, of course, have no theme.

This month's theme is MAXIMUM OF TWO HELMET CHOICES OF ANY TYPE.

As an example, this is allowed: an army with soldier, soldier, medic, medic, comms, comms, sab, sab, scout, scout, pilot, pilot.
This would also be allowed: an army with soldier, medic, comms, comms, level-5-trooper.
However this is not allowed: an army with soldier, soldier, soldier.


_ _ _

1) Only armies without more than two of any helmet choice may take part this month. Any army that gets a third trooper of any one class will be removed from the listing

2) To sign up add a link to your HQ page (tinyurl is allowed to help with forum formatting issues) and army name then playername afterwards. This is to make it as easy as possible for everyone else to see which army name to use to fight.
e.g. http://tinyurl.com/knife-fighters Knife Fighters / pickupsticks
LINK then ARMY NAME then / PLAYER NAME

3) To move up the ranking, beat the person above you then post a link to the replay here. Copy and paste the list, adjusting your position, and re-post it.

4) MAKE SURE you use the correct list from the last post in the thread. MAKE SURE nobody else has posted at the same time as you. If this happens, then edit your post to correct it. Anyone that messes up the list regularly may be excluded from the following month so pay attention.

5) Only fights where you sent the challenge (are on the left) are allowed. Only direct challenge fights are allowed. Fights that end in BAT1/BAT2/BAT3 or MISS1/MISS2/MISS3 are not eligible.

6) Players may NOT go "inactive" however you may choose to remove yourself from the competition at any time.

7) Only one army per player is allowed.

The last correct post to this thread with the forum timestamp of October will be the ranking used to decide the winner.
