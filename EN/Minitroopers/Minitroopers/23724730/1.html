Only players with a dynamic IP can do this.
First try to find out if you actually have a dynamic ip.
To find out if you have a dynamic ip go to this web page and write down your ip number... What is my IP address.
It will look something like this IP Information: 202.99.999.99.
If you reload this web page the number will stay the same every time, we want this to change.
Later you will revisit this same web page and see if the number has changed.
You may only create new minitroopers recruits when the number changes.

Different players have different ways of changing the ip number. I don't know which way will work for you, you may need to experiment.

I have an external modem and here is how I change my ip number...
Save all work (remember passwords), delete all browser history (is this your computer?), close all programs.
Good? Right then...

with the computer still on (and my modem still on) I go to the wall and I unplug the modem (it is still running).
I then go back to the computer and watch a some youtube video (do not go to minitroopers).
I keep working my modem with youtube videos until it runs out of power and dies in the middle of that hot new single from that cool group.
When this happens the modem's power capacitors are empty (no power stored and no power going in).
With no power the modem forgets everything (including the IP number).

Then...
plug the modem back into the wall and give it time to reconnect.
When it's ready go back into your browser and revisit WhatIsMyIPAddress.com.
Look at the old number that you wrote down before, do you have a different number now?
If you do then you have a dynamic ip.
Now go back to minitroopers and try too create recruits for yourself.
Good?

There are better, faster ways to do this but you will need to talk with other players and experiment to find what works for you.
