Would anyone be interested in a Minitroopers Olympics?
Yes? No? Maybe? "Tell me more"?

I am considering running a competition.
Well, OK, maybe LOTS of competitions. If I'd thought of this before I'd have tried to launch it to coincide with the current Olympic Games, but I guess hindsight is a wonderful thing.

Aaaaanyway.

Here's the proposed format.
1) There are many events.
2) The events have strict entry conditions
3) Each event has a one-week sign up period
4) All teams that are signed up then play off in a single-elimination tournament
5) Matches will be "one challenge each" with a tie resulting in a playoff until the deadlock is broken
6) Each event will award a Gold, Silver and Bronze medal to the top three finishers
7) Players must declare the country they wish to represent when signing up
8 ) When everything is over and done with, we'll rank the countries by medals

Sounds good? Yeah, that's what I thought.

All we need is an event list. Here's how this is going to work.
People will propose events here. After being proposed, at least two other people must "second" the proposal for it to be confirmed.
An event may have up to four conditions attached to it.
Each event must have a name that is an Olympic event! Choices may be from any of the current Summer Olympic events.
You may want to visit http://en.wikipedia.org/wiki/Olympic_... for some ideas.

I'll get the ball rolling.

SHOOTING: CANISTRIAN CATEGORY

1. New armies must join this tournament.
2. The ones with the most points accumulated within four matches (i.e. the days that the rats appear) would win.
3. POINT SYSTEM:
+300 if done within 1st try
+150 if done within 2nd try
+0 if done within 3rd try
-50 if unable
NOTE: Sequential point system for each rat killed. That means 1 point for 1 rat, 3 points for 2 rats, 6 points for 3 rats, and so on.
3. In case of ties for medals,a sudden-death match (NORMAL ATTACK) will be implemented. Bo3 match.
4. NEW SYSTEM:
Gold Medal
Silver Medal
Bronze Medal
2nd Bronze Medal (Can be determined by a repechage round if required). This medal goes to the fourth place (or the winner of a two-way tie matchup), but if a three-way or more tie occurs,then a special 5th round (i.e. extermination) match is played.
5. For extermination matches, screenshots are required. For infiltration sudden-death matches, BAT/MISS replays are not allowed.
6. Extra points for melee attacks (+1 each).

SHOOTING:ARCADIAN CATEGORY

0. Infiltration refers to the mission that appears at the right side of the screen. As a related rule, the armies can only attack each other in the competition (to trigger the right-side mission on).
1. New armies must join this tournament.
2. The ones with the most points accumulated within four matches (i.e. the days that the infiltrate missions appear) would win.
3. POINT SYSTEM:
+300 if done within 1st try
+150 if done within 2nd try
+0 if done within 3rd try
-50 if unable
NOTE: Sequential point system for each troop killed. That means 1 point for 1 troop, 3 points for 2 troops, 6 points for 3 troops, and so on.
3. In case of ties for medals,a sudden-death match (NORMAL ATTACK) will be implemented. Bo3 match.
4. NEW SYSTEM:
Gold Medal
Silver Medal
Bronze Medal

5. For infiltration matches, screenshots are required. For infiltration sudden-death matches, BAT/MISS replays are not allowed.
6. Extra points for melee attacks (+1 each).
