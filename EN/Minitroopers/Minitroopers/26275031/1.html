A very worthy experiment!

I've always wondered what 'biggest threat', 'weakest', and 'easiest' meant in game terms.

I've been assuming that 'easiest' meant the target with the least aim penalties associated with it. And that 'weakest' meant the lowest health.

But who really knows?

Biggest threat is a complete mystery, as it could mean so many different things. Weapon with the largest damage? That would be a launcher of some kind. Or perhaps a machine gun if they added all the potential damage together. Some calculation based upon both potential damage and aim? And does it take into effect any skills? I'm much more concerned about that machine gunner firing at my trooper with Dodge than a launcher firing against him, because the launcher will be dodged! Only the devs could say.

The only one which seems to have a clear definition is 'closest', as there can really be only one definition for that term. But I've seen troopers set to closest fire across the board before in preference to a target which is much closer.

So again, it's all a mystery.

The only one which seems to have a clear definition is 'closest', as there can really be only one definition for that term. But I've seen troopers set to closest fire across the board before in preference to a target which is much closer.

It could be that a trooper's "Priority Target" take precedence over the trooper's "Preferred Victim". That might explain why that trooper hit the enemy across the board instead of the closest enemy. It makes sense too. If a trooper's preferred victim is the closest enemy for example, but his priority target is a Doctor, that trooper will aim for the Doctor if he is in range, even if that Doctor isn't the closest enemy. The reason for that is because taking down doctors comes in first before taking down troopers that are closest to that said trooper. In other words, it's the trooper's "priority".

Just my own untested theory formed from my limited knowledge. Because of this, I'll set all my troopers' "Priority Target" to default to see what happens now.
