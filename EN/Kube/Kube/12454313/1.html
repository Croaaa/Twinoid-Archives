Connection Problems


Certain games on Muxxu require players to be connected to the server in real-time. Due to technical reasons it could be that it's not possible to play in real-time on servers such as public networks, universities or businesses.

If you are connected to Muxxu on a server with a weak connection, it may be that you can't connect to the game server.

If you can't connect to the game server from your domestic connection, please ensure the following:

Your internet connection is working as normal. If you suffer from frequent connection problems, it may be difficult for you to connect to the server in real-time.
It may be that the real-time game server is not working at the moment, please try again in a couple of days.

Advanced Users: If you know how to configure your firewall, you should authorise outgoing TCP connections from play.[game].muxxu.com (for example play.kube.muxxu.com for the game Kube) and for port 6667.

Due to the large number of different operating systems and computers, no help can be provided concerning the technical configuration of your computer to access the game server.

Don't hesitate to ask other players for help in the Forum .
