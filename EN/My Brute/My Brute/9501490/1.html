Forking is choosint the other choice as one of your levelups - to see if you dont have anything better hiding in there. As long as you choose exact the same things each levelup - you would go the same path. But if you choose something else - all the choces that come afterwards are new.

Hatchet + 3 speed
Toughtened Skin + 3 agility

Lance + 3 speed

Dog A + 3 speed

Tragic Potion + 3 speed

It is still possible to get some of the same stuff (like dog A, shield) - but it is a gamble, so usually fork after your best stuff...
