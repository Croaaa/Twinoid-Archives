I'm trying to provide an exact formula for Health points, but I only have 2 relative new Brutes, so my insight of the game is still small. So far I only have a brute without skill/pet modification on Health and one with Vitality. For pets and other skills like Imortality I will either need to get them myself or rely on information from other players! (like you! yes you, sitting there in front of that screen!)


One thing I instantly noticed when recruiting a brute was that some have a sum of 7 and others of 8 for Strength + Agility + Speed. Those with 8 had a Health of 62 and those with 7 a health of 68. Reading the forum I saw that those 6 points in Health obviously came from 1 point more in Endurance, an important stat that for some strange reason is not shown on a brutes page. From the description 1 point of Endurence provides 6 points of Health.


Neither 62 nor 68 can be diveded evenly by 6, so there had to be a basic Health value to which Endurance adds further Health points. As a fan of round numbers I postulate that this basic Health value is 50 and the sum of Strength + Agility + Speed + Endurance for a starting brute is 10. So for example if Strength + Agility + Speed is 7 you will have a (hidden) Endurance of 3 and therefore 50 + 3*6 = 68 Health.


The basic formula for Health I came up with is:


Health = A + B + C

with:

A = basic Health value (50)
B = Health added for Endurance (basicly Endurance*6)
C = Health added for level


With every level up, each brute will either get 1 or 2 Health points if nothing else changes (like getting Endurance increase, pets or skills that increase Endurance/Health). The pattern is +1/+2/+1/+2/+1/+2 starting with +1 when going from level 1 to level 2.


So as part of the formula:

C = ((brute level - 1)*1,5)rounded down


As far as skills/pets go I have only Vitality right now and it modifies B as follows:


B = 6*((Endurance*VM)rounded down), with

Vitality modifier (VM)= 1,5 if you have Vitality and 1 if you don't have it
(don't forget that Vitality also adds +3 to Endurance before multipling)


For my own brutes this formula was true for each and every level!

I will need:

- exact history (to reverse calculate starting Endurance)
- Health at each level, if available

Nope, it works. The thing that confuses most players it seems is the fact that only 50% of the Health points that are provided by Endurence are added. So Vitality does only add 50% of B, not of the A and C parts.
For your brute that would mean he started with an Endurance of 2 (10 - sum of 8 for the other attributes). This got raised to 5 by Endurace. To this 5 is added 50% and then rounded down (7,5->7).

So you have:
A + B +C = 50 + 6*((5*1,5)rounded down) + ((4 - 1)*1,5)rounded down
= 50 + 6*7 + 4 = 96
