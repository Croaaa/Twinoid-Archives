I just wanted to start a thread about Android, so....
During the last weeks I tested a bunch of browsers on android, and all suffered from the same problem: the login did not work. But today I found one, and I'm in fact writing this post from my HTC Desire: The Miren Browser, to be downloaded on the usual market, does work with the muxxu login. It doesn't autofill the login dialog of mybrute.muxxu.com, but does with the main muxxu.com startpage once it knows your credentials. Also, the forum works better when accessed via the main page instead via the mybrute page. On mybrute, training and fighting works, too, but be warned, it uses lots juice, especially if you zoom.
I also found that it saves a bit of CPU if you disable that realtime message preview in your muxxu account preferences.
Hope this helped some of you!
