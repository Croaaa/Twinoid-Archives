Well, a destiny path is just a particular set of level up choices that you made. You can "fork" by changing a decision, which leads you down a new path. Generally, people find a good path, and start forking at certain levels in order to find a better path

Now, why does this matter? It's because you are reset to level 1 when you rank up. That's how people can improve their destiny path (your brute's level can't go down, it can only be reset)

If you want to find a good path via forking, make sure that you write down what you picked. It's not always the obvious choices that lead to a great brute

This is just me, but don't pick "new destiny" when you win a tournament. If you think your brute's terrible, try forking at the first level up. Most brutes have destinies that aren't totally terrible, so see what you can do with it before you ask for a new one
